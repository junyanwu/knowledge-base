# 实盘模拟检查清单（精简版）

## 📊 每日必做

### 开盘前（9:00-9:25）
- [ ] 检查数据源连接
- [ ] 查看隔夜新闻和公告
- [ ] 确认策略信号
- [ ] 制定当日计划

### 交易时段（9:30-15:00）
- [ ] 记录每笔交易（时间、代码、价格、数量、理由）
- [ ] 监控持仓盈亏
- [ ] 检查风控指标（止损线、仓位限制）
- [ ] 记录心理状态变化

### 收盘后（15:00-16:00）
- [ ] 更新持仓明细
- [ ] 计算当日盈亏
- [ ] 复盘交易决策
- [ ] 更新交易日志

| 维度 | 权重 | 指标 | 目标 | 实际 | 得分 |
|------|------|------|------|------|------|

- ≥85分：优秀，可转实盘

- 60-69分：一般，继续优化
- <60分：较差，暂停转实盘

## ✅ 转实盘条件检查表

### 必要条件（全部满足）

- [ ] 模拟时长 ≥ 3个月（60+交易日）
- [ ] 最大回撤 < 15%
- [ ] 夏普比率 > 1.0
- [ ] 交易次数 ≥ 30次
- [ ] 系统稳定性测试通过（30天无故障）

### 充分条件（至少满足5条）

- [ ] 累计收益率 > 5%（季度）
- [ ] 胜率 ≥ 55%
- [ ] 盈亏比 ≥ 1.5
- [ ] 信号执行率 ≥ 95%
- [ ] 相对基准超额收益 > 3%
- [ ] 经历极端行情测试（单日涨跌>3%）
- [ ] 心理压力测试通过

### 转实盘流程

```
第一阶段（1个月）：10-20%资金
    ↓
第二阶段（2-3个月）：50-70%资金
    ↓
第三阶段：100%资金
```

## ⚠️ 风险警示

**模拟与实盘差异：**
1. 心理压力：真实资金风险会导致执行力下降
2. 流动性：大单可能无法按预期价格成交
3. 滑点：实际交易成本可能更高
4. 系统稳定性：真实环境可能出现技术故障

**禁止行为：**
- ❌ 跳过小额试运行阶段
- ❌ 因模拟成功就过度自信
- ❌ 频繁修改策略参数
- ❌ 忽视止损纪律
- ❌ 不记录交易日志

## 🎯 快速自检问题

每周问自己：
1. 我能解释每一笔亏损的原因吗？
2. 我是否严格执行了所有信号？
3. 我的情绪是否影响了交易决策？
4. 策略是否有逻辑性变化需要调整？
5. 我准备好用真实资金交易了吗？

**如果所有答案都是肯定的，说明你准备好了！** 🚀
