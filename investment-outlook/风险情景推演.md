### 1.2 风险传导路径

```
地缘政治风险 → 能源价格波动 → 通胀压力 → 货币政策收紧 → 资产估值下行
      ↓              ↓
供应链中断 ← 贸易壁垒升级 ← 政策风险
      ↓
企业盈利下降 → 信用风险上升 → 流动性危机
```

## 二、一年期风险情景（2026）

### 2.1 主要风险识别

#### 2.1.1 地缘政治风险（高优先级）

**情景A：俄乌冲突升级**

| 维度 | 分析 |
|------|------|
| **现状** | 冲突进入第五年，"基础设施战争"阶段 |
| **触发条件** | 北约向乌克兰提供核武器零部件、俄使用"榛树"导弹 |
| **概率** | 30%（较2025年下降，因特朗普政府推动谈判） |
| **影响** | 能源价格飙升、粮食安全危机、欧洲经济衰退 |
| **预警指标** | 欧洲军工产出（2025年首次超越美国）、高超音速武器使用 |

**情景B：中东冲突扩大**

| 维度 | 分析 |
|------|------|
| **现状** | 巴以冲突持续、红海航运受阻 |
| **触发条件** | 伊朗直接介入、霍尔木兹海峡封锁 |
| **概率** | 25% |
| **影响** | 油价突破150美元/桶、全球通胀反弹、航运成本暴涨 |
| **预警指标** | 美军中东部署、伊朗核谈判进展、以色列军事行动 |

**情景C：台海局势紧张**

| 维度 | 分析 |
|------|------|
| **现状** | 美台互动增加、军售持续 |
| **触发条件** | 美国高层访台、台湾独立公投、解放军大规模演习 |
| **概率** | 15%（2026年相对平稳） |
| **影响** | 半导体供应链中断、中美全面对抗、区域战争风险 |
| **预警指标** | 美对台军售规模、解放军演习频率、台海军事部署 |

#### 2.1.2 经济与市场风险

**情景D：人民币汇率波动**

| 维度 | 分析 |
|------|------|
| **现状** | 人民币快速升值，离岸突破6.84关口 |
| **政策应对** | 外汇风险准备金率清零（2026年3月生效） |
| **风险方向** | 双向波动加大，短期升值压力 → 中期贬值压力 |
| **概率** | 50%波动加剧 |
| **影响** | 出口企业利润压缩、资本流动波动、A股外资进出频繁 |

**情景E：房地产市场二次探底**

| 维度 | 分析 |
|------|------|
| **现状** | 四年调整后底部特征显现，但分化加剧 |
| **触发条件** | 政策刺激效果衰减、购房者信心不足、房企债务违约 |
| **概率** | 35% |
| **影响** | 银行不良贷款上升、地方财政收入下降、消费信心走弱 |
| **预警指标** | 百城房价指数、商品房销售面积、房企信用利差 |

**情景F：地方政府债务风险**

| 维度 | 分析 |
|------|------|
| **现状** | "6+4+2"万亿元化债方案推进中，2026年为收官年 |
| **风险点** | 隐性债务化解进度、地方融资平台转型 |
| **概率** | 25%出现区域性风险暴露 |
| **影响** | 区域性金融风险、基建投资放缓、信用利差扩大 |

## 三、五年期风险情景（2026-2031）

### 3.1 结构性风险

#### 3.1.1 人口老龄化冲击

| 时间节点 | 老龄化程度 | 经济影响 |
|---------|-----------|---------|
| 2026年 | 65岁以上人口约14% | 养老金压力初显 |
| 2028年 | 约15% | 劳动力供给收缩 |
| 2030年 | 约16% | 医疗支出占比显著上升 |
| 2031年 | 约17% | 养老金缺口扩大 |

**投资影响**：
- 银发经济崛起（医疗、养老、康养）
- 劳动力密集型产业成本上升
- 储蓄率下降影响资本积累
- 房地产需求结构性萎缩

#### 3.1.2 美国债务危机

| 指标 | 2026年 | 2031年预估 |
|------|--------|-----------|
| 美国联邦债务/GDP | 约120% | 约140%+ |
| 利息支出/财政收入 | 约15% | 约25%+ |
| 美元信用风险 | 低 | 中高 |

**风险传导**：
- 美债收益率飙升 → 全球资产重定价
- 美元信用动摇 → 储备货币地位受挑战
- 全球通胀压力 → 央行政策两难

#### 3.1.3 气候变化加剧

| 情景 | 概率 | 潜在影响 |
|------|-----|---------|
| 极端天气频发 | 高 | 农业、保险业损失扩大 |
| 碳关税全面实施 | 高 | 出口企业成本上升 |
| 能源转型加速 | 中高 | 传统能源资产贬值 |

### 3.2 周期性风险

#### 3.2.1 全球经济周期

| 阶段 | 预计时间 | 特征 |
|------|---------|------|
| 衰退期 | 2026-2027 | 增长放缓、政策宽松 |
| 复苏期 | 2028-2029 | 增长回升、通胀可控 |
| 扩张期 | 2030-2031 | 潜在过热风险 |

#### 3.2.2 科技周期

| 领域 | 当前阶段 | 2026-2031展望 |
|------|---------|--------------|
| AI应用 | 快速扩张 | 泡沫形成→调整→成熟 |
| 新能源汽车 | 高速增长 | 渗透率见顶→洗牌 |
| 半导体 | 周期底部 | 库存出清→新周期启动 |

### 3.3 突发性风险（黑天鹅）

#### 3.3.1 大国冲突升级

| 情景 | 触发条件 | 概率 | 影响 |
|------|---------|-----|------|
| 中美军事冲突 | 台海危机失控 | <5% | 极端：全球供应链断裂 |
| 美俄直接对抗 | 北约直接参战 | <10% | 极端：核战争风险 |
| 区域战争蔓延 | 中东全面战争 | 10-15% | 高：能源危机 |

#### 3.3.2 技术崩溃事件

| 情景 | 概率 | 影响 |
|------|-----|------|
| AI系统失控 | 低 | 生产中断、信任危机 |
| 网络攻击瘫痪 | 中 | 金融系统、基础设施受损 |
| 卫星系统故障 | 低 | 通信、导航、金融交易中断 |

#### 3.3.3 全球金融危机

| 情景 | 触发因素 | 概率 | 传导路径 |
|------|---------|-----|---------|
| 美债违约 | 债务上限危机 | 15-20% | 美债收益率飙升→全球资产重定价 |
| 系统性银行危机 | 商业地产坏账 | 20-25% | 信用紧缩→经济衰退 |
| 新兴市场债务危机 | 美元走强+利率高位 | 25-30% | 资本外流→货币危机 |

| 等级 | 颜色 | 含义 | 应对策略 |
