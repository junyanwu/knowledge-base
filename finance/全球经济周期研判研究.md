# 全球经济周期研判研究：2024-2026年展望

> 研究时间：2026年3月
> 更新时间：2026-03-09
> 数据来源：IMF、世界银行、各国央行、国家统计局、第一财经、财联社等

---

## 研究摘要

### 核心结论

**全球经济正处于周期转换的关键节点。** 2024-2026年全球经济呈现以下特征：

1. **美国经济**：处于扩张后期，软着陆概率上升，但衰退风险尚未完全消除。美联储货币政策从紧缩转向中性，2025年进入降息周期。

2. **欧洲经济**：复苏步伐缓慢，能源转型和地缘政治冲击带来结构性挑战。欧洲央行维持相对宽松立场，但通胀压力仍存。

3. **中国经济**：从高速增长转向高质量发展阶段，2026年GDP目标设定为4.5%-5%。货币政策适度宽松，财政政策更加积极，结构性改革深化。

4. **新兴市场**：分化明显，东南亚和印度保持较高增速，拉美和中东欧面临挑战。

### 周期定位总结

| 经济体 | 当前阶段 | 2026年展望 | 关键风险 |
|--------|---------|-----------|---------|
| 美国 | 扩张后期 | 软着陆/温和衰退 | 通胀粘性、债务上限 |
| 欧元区 | 低速复苏 | 温和增长 | 能源价格、地缘政治 |
| 中国 | 结构转型 | 稳中求进 | 房地产、外需走弱 |
| 新兴市场 | 分化加剧 | 结构性机会 | 资本流出、汇率波动 |

### 投资策略核心观点

- **股票**：结构性机会为主，关注科技创新、高端制造、消费升级
- **债券**：中美利率下行周期，利率债配置价值提升
- **大宗商品**：有色金属受益于新能源需求，能源价格波动加大
- **黄金**：避险属性和抗通胀属性并重，配置价值凸显

## 第一章 全球经济周期阶段判断

### 1.1 美国经济周期位置

#### 经济增长态势

美国经济在2024-2025年展现出较强韧性，主要特征：

**GDP增长**：
- 2024年实际GDP增长约2.5%，超出市场预期
- 2025年增速预计放缓至1.8-2.0%
- 消费支出保持稳健，就业市场持续紧俏

**就业市场**：
- 失业率维持在4.0%左右历史低位
- 非农就业月均新增15-20万人
- 工资增长稳健，但增速放缓

**通胀走势**：
- CPI从2023年高点逐步回落
- 核心通胀仍高于2%目标
- 服务通胀粘性较强

#### 美联储货币政策

根据美联储官网最新信息（2026年1月）：

**政策框架审查**：
- 2025年进行货币政策策略、工具和沟通框架的周期性审查
- 2020年引入的平均通胀目标制（AIT）面临调整讨论

**利率路径**：
- 联邦基金利率已从5.25-5.50%高位逐步下调
- 降息节奏取决于通胀和就业数据
- 预计2026年政策利率维持在中性水平附近

**资产负债表**：
- 缩表进程持续
- 准备金水平保持充裕
- 关注流动性状况

#### 周期判断

根据NBER经济周期测定标准，美国经济：
- 最近一次经济高峰：2020年2月
- 最近一次经济低谷：2020年4月
- 当前处于扩张周期第70个月左右

**周期阶段定位**：**扩张后期 / 软着陆阶段**

判断依据：
1. 经济增速仍高于潜在增速
2. 就业市场保持紧俏
3. 通胀逐步回归目标
4. 收益率曲线从倒挂回归正常

**风险因素**：
- 高利率滞后效应
- 商业地产风险
- 信用卡债务上升
- 地缘政治冲击

### 1.2 欧洲经济复苏态势

#### 经济增长表现

欧洲经济在2024-2025年呈现"弱复苏"特征：

**GDP增长**：
- 2024年欧元区GDP增长约0.8%
- 2025年增速预计提升至1.2%
- 德国经济疲软拖累整体表现

**结构性挑战**：
- 能源转型成本高企
- 制造业竞争力下降
- 人口老龄化加剧
- 劳动力市场僵化

#### 欧洲央行政策取向

根据欧洲央行官网最新信息（2026年3月）：

**货币政策立场**：
- 维持价格稳定为核心目标
- 关注供给冲击对通胀的影响
- 警惕地缘政治和宏观经济上行风险

**政策工具**：
- 利率：存款便利利率、主要再融资利率、边际贷款便利利率
- 资产购买计划（APP、PEPP）
- 定向长期再融资操作（TLTRO）

**数字欧元进展**：
- 2026年起开展12个月试点
- 测试真实支付场景
- 评估关键功能可行性

**研究重点**（来自欧洲央行博客）：
- 人工智能对就业的影响
- 经济预测误差模式分析
- 关税对欧洲企业的影响
- 气候变化对财政政策和主权风险的影响

#### 周期判断

**周期阶段定位**：**低速复苏期**

判断依据：
1. 增速低于潜在增速
2. 通胀压力缓解
3. 货币政策维持宽松
4. 财政政策空间有限

**风险因素**：
- 俄乌冲突持续
- 能源价格波动
- 移民问题
- 英国脱欧后续影响

### 1.3 中国经济周期特征

#### 经济增长态势

根据中国宏观经济追踪数据（2026年3月最新）：

**2025年成绩单**：
- GDP增速：5.0%（圆满实现预期目标）
- GDP总量：140.19万亿元人民币
- 就业：城镇新增就业1267万人，失业率平均5.2%
- 城镇化率：67.9%

**2026年目标设定**：
- GDP增长目标：4.5%-5%（区间目标）
- CPI涨幅：2%左右
- 就业目标：新增就业1200万人以上

**2026年2月PMI数据**：
- 制造业PMI：49.0%（-0.3个百分点，低于临界点）
- 非制造业商务活动指数：49.5%（+0.1个百分点）
- 综合PMI产出指数：49.5%（-0.3个百分点）

**企业规模分化明显**：
- 大型企业PMI：51.5%（高于临界点）
- 中型企业PMI：47.5%（低于临界点）
- 小型企业PMI：44.8%（大幅低于临界点）

#### 通胀格局

**CPI数据（2026年1月）**：
- CPI同比：+0.2%
- 核心CPI：+0.8%
- 食品价格：-0.7%（猪肉-13.7%）
- 服务价格：+0.1%

**PPI数据（2026年1月）**：
- PPI同比：-1.4%
- PPI环比：+0.4%（转正）
- 有色金属价格大幅上涨（+16.1%）

#### 货币政策动态

根据央行最新表态（2026年3月）：

**政策基调**：
- 继续实施**适度宽松的货币政策**
- 保持社会融资条件相对宽松
- 当前融资成本处于历史低位

**流动性投放**：
- 2026年以来公开市场净投放中长期资金约2万亿元
- 债券融资占社融增量46%，融资结构深刻变化

**结构性工具**：
- 从"发挥作用"到"加量提质"
- 科技创新再贷款可能扩大额度
- 再贷款利率预计下调0.2-0.4个百分点

**汇率政策**：
- 保持人民币汇率基本稳定
- 企业汇率避险+人民币结算比例已达60%

#### 周期判断

**周期阶段定位**：**结构转型期 / 复苏初期**

判断依据：
1. 增速目标下调，从高速转向高质量
2. PMI回落至收缩区间，复苏基础待巩固
3. 通胀温和，政策空间充足
4. 货币政策明确宽松取向

**中国特色因素**：
- 政策逆周期调节力度大
- 房地产周期影响显著
- 人口结构变化影响深远
- 产业升级加速推进

## 第二章 周期驱动因素分析

### 2.1 货币政策影响

#### 全球货币政策分化

**美国**：
- 从紧缩转向中性/宽松
- 降息节奏取决于通胀
- 关注就业市场变化

**欧洲**：
- 维持相对宽松
- 关注通胀粘性
- 结构性工具使用

**中国**：
- 适度宽松取向明确
- 结构性工具加量

**日本**：
- 货币政策正常化进程中

```
货币政策 → 利率水平 → 资产价格 → 投资消费 → 经济增长
     ↓           ↓           ↓          ↓
   流动性     信用扩张     财富效应    就业改善
```

**中国传导特点**：
- 宽货币→宽信用传导存在时滞

- 结构性工具定向支持

- 人工智能对就业市场影响深远

**产业影响**：
- 科技行业：直接受益，估值提升

- 政府工作报告点名"建设高质量数据集"
- "十五五"末AI相关产业规模将达10万亿元以上

### 2.4 地缘政治冲击

#### 主要风险因素

**中东冲突**（2026年3月最新）：
- 霍尔木兹海峡航运受阻
- 化工品价格普涨
- 中国汽车出口中东受影响

**中美关系**：
- 科技竞争加剧
- 供应链重构
- 贸易摩擦持续

**俄乌冲突**：
- 能源价格波动
- 粮食安全风险
- 全球供应链中断

#### 影响机制

```
地缘政治冲击
     ↓
├── 能源价格波动 → 通胀上行 → 货币政策收紧
├── 供应链中断 → 生产成本上升 → 盈利下降
├── 贸易摩擦 → 出口下降 → 经济增长放缓
└── 风险偏好下降 → 资本流出 → 资产价格下跌
```

### 3.2 收益率曲线分析


### 4.1 周期定位与资产配置

#### 美林时钟与中国实践

```
        通胀
          ↑
    滞胀期 | 过热期

          |
    衰退期 | 复苏期
          |
          └───────→ 经济增长
```

**中国市场的特殊性**：
- 政策干预打破周期规律
- 周期阶段可能跳跃或缩短
- 需要结合政策因素综合判断

#### 当前周期定位

| 经济体 | 经济增长 | 通胀 | 周期阶段 | 推荐资产 |
|--------|---------|------|---------|---------|
| 美国 | 中低速 | 中等 | 扩张后期/软着陆 | 股票+短债 |
| 欧洲 | 低速 | 中等 | 复苏期 | 股票+利率债 |
| 中国 | 中速 | 低通胀 | 复苏初期 | 利率债+股票 |

### 4.2 行业轮动策略

## 第五章 研究结论与展望

### 5.1 主要结论

1. **全球经济周期分化**
   - 美国处于扩张后期，软着陆概率上升
   - 欧洲复苏步伐缓慢，结构性挑战突出
   - 中国进入结构转型期，政策托底效应显现
   - 新兴市场分化加剧，结构性机会并存

2. **货币政策转向宽松**
   - 全球央行从紧缩转向中性/宽松
   - 中国货币政策明确适度宽松
   - 结构性工具成为政策重点
   - 流动性环境改善利好风险资产

3. **周期驱动因素多元**
   - 货币政策影响减弱，财政政策发力
   - 技术创新（AI）重塑产业格局
   - 地缘政治带来持续不确定性
   - 新能源转型创造投资机会

4. **领先指标信号混合**
   - PMI整体走弱，但服务业相对稳健
   - 收益率曲线从倒挂回归正常
   - 信用利差有所扩大，但仍可控
   - 消费者信心待修复

### 5.2 风险提示

**上行风险**：
- AI技术突破超预期
- 政策刺激效果超预期
- 地缘政治缓和

**下行风险**：
- 通胀超预期反弹
- 地缘政治冲突升级
- 房地产市场风险
- 中美关系恶化
