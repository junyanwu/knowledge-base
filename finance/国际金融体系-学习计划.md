## 二、学习重点与目标

### 核心模块

#### 模块1: 国际收支理论（权重20%）
**学习重点**：
- 国际收支平衡表（经常账户、资本账户、金融账户、储备账户）
- 国际收支平衡与失衡
- 国际收支调节理论（弹性分析法、吸收分析法、货币分析法）
- 国际收支与宏观经济的关系
- 中国国际收支特点与演变

**预期成果**：
- 能够阅读和分析国际收支平衡表
- 理解国际收支失衡的原因和影响
- 掌握国际收支调节的政策工具

**实践应用**：
- 分析中国国际收支双顺差格局
- 评估经常账户顺差收窄的影响
- 解读外汇储备变动

#### 模块2: 汇率决定理论（权重25%）
**学习重点**：
- 购买力平价理论（绝对PPP、相对PPP）
- 利率平价理论（抛补利率平价、非抛补利率平价）
- 国际费雪效应
- 资产市场理论（货币分析法、资产组合分析法）
- 汇率超调模型（Dornbusch模型）
- 行为金融学视角下的汇率决定
- 汇率预测方法

**预期成果**：
- 理解汇率决定的各种理论框架
- 能够分析汇率变动的影响因素
- 掌握汇率预测的基本方法

**实践应用**：
- 分析人民币对美元汇率的决定因素
- 预测汇率短期和中长期走势
- 理解汇率政策工具（中间价、逆周期因子）

#### 模块3: 国际货币体系（权重20%）
**学习重点**：
- 国际货币体系演变
  - 金本位制（1870-1914）
  - 布雷顿森林体系（1944-1971）
  - 牙买加体系（1976至今）
- 国际储备货币体系
  - 美元的国际储备货币地位
  - 欧元、日元、人民币的国际化
  - 特别提款权（SDR）
- 国际货币体系改革
- 国际金融治理机制
  - IMF（国际货币基金组织）
  - World Bank（世界银行）
  - BIS（国际清算银行）
  - G20、G7等国际协调机制

**预期成果**：
- 理解国际货币体系的历史演变和未来趋势
- 掌握主要国际金融机构的职能
- 能够分析国际货币体系改革的方向

**实践应用**：
- 分析美元霸权的形成和挑战
- 评估人民币国际化的进展和障碍
- 理解SDR篮子货币调整的意义

#### 模块4: 国际资本流动（权重20%）
**学习重点**：
- 国际资本流动的类型
  - 直接投资（FDI）
  - 证券投资
  - 其他投资
  - 储备资产
- 国际资本流动理论
  - 新古典理论
  - 资产组合理论
  - 资产定价模型
- 国际资本流动的影响
  - 对经济增长的影响
  - 对金融稳定的影响
- 资本流动管理
  - 资本管制
  - 宏观审慎政策
  - 托宾税
- 中国资本账户开放进程

**预期成果**：
- 理解国际资本流动的动因和影响
- 掌握资本流动管理的政策工具
- 能够分析资本流动风险

**实践应用**：
- 分析外资流入A股的影响
- 评估资本外流风险
- 理解QFII、RQFII、沪深港通机制

#### 模块5: 外汇市场运作（权重15%）
**学习重点**：
- 外汇市场结构
  - 即期外汇市场
  - 远期外汇市场
  - 外汇掉期市场
  - 外汇期权市场
- 外汇交易机制
  - 外汇交易流程
  - 外汇市场参与者
  - 外汇定价机制
- 外汇风险管理
  - 外汇风险类型（交易风险、折算风险、经济风险）
  - 外汇风险对冲工具（远期、掉期、期权）
- 人民币外汇市场
  - 即期市场
  - 远期市场
  - 掉期市场
  - 期权市场
  - 离岸人民币市场（CNH）

**预期成果**：
- 理解外汇市场的运作机制
- 掌握外汇风险管理工具
- 能够进行外汇交易和风险对冲

**实践应用**：
- 分析外汇市场流动性
- 设计外汇风险对冲方案
- 理解人民币离岸市场运作

## 三、推荐学习资料

### 经典教材
1. **《国际金融学》（克鲁格曼）** - 国际公认的权威教材
   - 重点章节：国际收支、汇率决定、国际货币体系
   
2. **《国际金融》（姜波克）** - 国内经典教材
   - 重点章节：汇率理论、国际收支、国际货币体系
   - 特别关注：中国国际金融实践

3. **《国际经济学》（多恩布什）**
   - 重点章节：国际金融部分

### 专业书籍
1. **《汇率决定理论》（崔孟修）**
   - 重点：汇率理论的系统梳理

2. **《人民币国际化研究》**
   - 重点：人民币国际化进程与前景

3. **《国际金融危机史》（金德尔伯格）**
   - 重点：历次金融危机的演变与传导

### 在线课程
1. **Coursera - International Finance**
   - 授课机构：宾夕法尼亚大学沃顿商学院
   - 重点：汇率、国际资本流动

2. **edX - Global Financial Markets**
   - 重点：国际金融市场运作

### 研究报告与数据
1. **国际货币基金组织（IMF）**
   - World Economic Outlook（世界经济展望）
   - Global Financial Stability Report（全球金融稳定报告）
   - COFER数据（外汇储备货币构成）

2. **国际清算银行（BIS）**
   - Quarterly Review（季度回顾）
   - Triennial Central Bank Survey（三年期央行调查）
   - 有效汇率指数

3. **中国人民银行**
   - 国际收支平衡表
   - 人民币国际化报告
   - 跨境人民币业务数据

4. **国家外汇管理局**
   - 外汇储备数据
   - 跨境资金流动监测
   - 外汇市场数据

### 实务参考
1. **《外汇交易实务》**
   - 重点：外汇交易流程、风险管理

2. **《跨境人民币业务指南》**
   - 重点：人民币跨境结算、离岸市场

## 六、与量化投资的关系

### 直接应用场景
1. **汇率因子构建**：汇率变动、汇率波动率、利差因子
2. **跨境投资策略**：基于汇率预期的资产配置
3. **外汇量化交易**：汇率预测模型、套利策略
4. **宏观对冲策略**：利用汇率衍生品进行宏观对冲
5. **跨境套利**：A-H股套利、中美利率套利

### 知识融合点
- **宏观因子模型**：汇率、利差、资本流动纳入宏观因子
- **跨境资产配置**：汇率风险对冲、国家风险考量
- **外汇策略开发**：汇率预测、套利机会识别
- **风险模型优化**：外汇风险因子、全球风险因子
