## 一、投资组合理论

### 1.1 单一资产的收益与风险

#### 1.1.1 收益率计算

**持有期收益率：**
```
R = (P₁ - P₀ + D) / P₀
```
- **R**：持有期收益率
- **P₁**：期末价格
- **P₀**：期初价格
- **D**：期间分红

**案例：**
```
期初股价：100元
期末股价：110元
期间分红：5元

R = (110 - 100 + 5) / 100 = 15%
```

**预期收益率：**
```
E(R) = Σ pᵢ × Rᵢ
```
- **pᵢ**：第i种情况的概率
- **Rᵢ**：第i种情况的收益率

**案例：**
```
经济繁荣（概率30%）：收益率20%
经济平稳（概率50%）：收益率10%
经济衰退（概率20%）：收益率-5%

E(R) = 0.3×20% + 0.5×10% + 0.2×(-5%) = 10%
```

#### 1.1.2 风险度量

**方差与标准差：**
```
σ² = Σ pᵢ × [Rᵢ - E(R)]²

σ = √σ²
```

**案例：**
```
E(R) = 10%

σ² = 0.3×(20%-10%)² + 0.5×(10%-10%)² + 0.2×(-5%-10%)²
σ² = 0.3×0.01 + 0 + 0.2×0.0225
σ² = 0.003 + 0.0045 = 0.0075

σ = √0.0075 = 8.66%
```

**偏度与峰度：**
```
偏度（Skewness）：衡量分布的不对称性
峰度（Kurtosis）：衡量分布的尖峰程度

正态分布：偏度=0，峰度=3
```

### 1.2 资产组合的收益与风险

#### 1.2.1 组合收益率

**公式：**
```
Rₚ = Σ wᵢ × Rᵢ
```
- **Rₚ**：组合收益率
- **wᵢ**：第i种资产权重
- **Rᵢ**：第i种资产收益率

**案例：**
```
股票A权重：60%，收益率：15%
股票B权重：40%，收益率：8%

Rₚ = 0.6×15% + 0.4×8% = 9% + 3.2% = 12.2%
```

**组合预期收益率：**
```
E(Rₚ) = Σ wᵢ × E(Rᵢ)
```

#### 1.2.2 组合风险

**组合方差：**
```
σₚ² = ΣΣ wᵢwⱼσᵢⱼ
```

**展开形式：**
```
σₚ² = w₁²σ₁² + w₂²σ₂² + 2w₁w₂σ₁₂
```

**协方差：**
```
σ₁₂ = Cov(R₁, R₂) = E[(R₁ - E(R₁))(R₂ - E(R₂))]
```

**相关系数：**
```
ρ₁₂ = σ₁₂ / (σ₁×σ₂)

ρ ∈ [-1, 1]
```

#### 1.2.3 分散化效应

**案例：两只股票的组合**

```
股票A：E(Rₐ)=15%, σₐ=20%
股票B：E(Rᵦ)=10%, σᵦ=15%
相关性：ρₐᵦ=0.3
权重：wₐ=60%, wᵦ=40%

组合收益率：
E(Rₚ) = 0.6×15% + 0.4×10% = 13%

组合方差：
σₚ² = (0.6)²×(0.2)² + (0.4)²×(0.15)² + 2×0.6×0.4×0.3×0.2×0.15
σₚ² = 0.0144 + 0.0036 + 0.00432 = 0.02232

σₚ = √0.02232 = 14.94%

分散化效应：
组合标准差 < 加权平均标准差
14.94% < 0.6×20% + 0.4×15% = 18%
```

**分散化的极限：**
```
当N→∞时：
σₚ² = (1/N)σ̄² + (N-1)/N × σ̄̄² → σ̄̄²

σ̄̄²：资产间平均协方差
```

**案例：等权重组合**
```
100只股票，每只股票：
平均方差：σ̄² = 400（标准差20%）
平均协方差：σ̄̄² = 100

组合方差：
σₚ² = (1/100)×400 + (99/100)×100 = 4 + 99 = 103
σₚ = 10.15%

分散化降低风险约50%
```

### 1.3 马科维茨均值-方差模型

#### 1.3.1 有效前沿

**定义：**在给定风险水平下，收益最大的组合集合。

**求解有效前沿：**
```
min σₚ² = w'Σw
s.t. E(Rₚ) = w'μ ≥ R_target
     Σwᵢ = 1
```

**案例：两只股票**

```
股票A：E(Rₐ)=15%, σₐ=20%
股票B：E(Rᵦ)=10%, σᵦ=15%
ρₐᵦ = 0.3

有效前沿计算：
wₐ = 0% → E(R)=10%, σ=15%
wₐ = 20% → E(R)=11%, σ=13.2%
wₐ = 40% → E(R)=12%, σ=12.5%
wₐ = 60% → E(R)=13%, σ=14.9%
wₐ = 80% → E(R)=14%, σ=17.8%
wₐ = 100% → E(R)=15%, σ=20%

最小方差组合：
wₐ* = [σᵦ² - Cov(Rₐ,Rᵦ)] / [σₐ² + σᵦ² - 2Cov(Rₐ,Rᵦ)]
```

#### 1.3.2 最优风险组合

**引入无风险资产：**

```
无风险利率：Rᶠ = 3%
风险组合：E(Rₘ), σₘ

资本配置线（CAL）：
E(Rₚ) = Rᶠ + (E(Rₘ) - Rᶠ)/σₘ × σₚ
```

**切点组合（最优风险组合）：**
```
夏普比率最大化：
Sharpe Ratio = (E(Rₘ) - Rᶠ) / σₘ

切点组合是CAL与有效前沿的切点
```

### 1.4 两基金分离定理

**定理：**任何有效组合都可以由两个有效组合的线性组合生成。

**含义：**
```
投资者只需持有：
1. 无风险资产
2. 切点组合（最优风险组合）

不同投资者的差异仅在于两者的配置比例
```

**案例：**
```
切点组合：E(Rₘ)=12%, σₘ=15%
无风险利率：Rᶠ=3%

保守投资者：
wᶠ = 60%（无风险资产）
wₘ = 40%（风险组合）
E(Rₚ) = 0.6×3% + 0.4×12% = 6.6%
σₚ = 0.4×15% = 6%

激进投资者：
wᶠ = -20%（借入）
wₘ = 120%（风险组合）
E(Rₚ) = -0.2×3% + 1.2×12% = 13.8%
σₚ = 1.2×15% = 18%
```

## 二、资本资产定价模型（CAPM）

### 2.1 CAPM的假设

1. 投资者是风险厌恶的，追求效用最大化
2. 投资者具有相同的投资期限
3. 投资者对资产收益率的预期一致
4. 资产收益率服从正态分布
5. 市场是无摩擦的（无交易成本、税收）
6. 所有资产可无限分割
7. 投资者可以无风险利率借贷
8. 信息对所有投资者免费可得

### 2.2 CAPM的推导

#### 2.2.1 市场组合

**定义：**包含所有风险资产，权重按市值比例分配的组合。

**性质：**
```
市场组合是切点组合
所有投资者持有市场组合
```

#### 2.2.2 证券市场线（SML）

**单个资产的预期收益率：**
```
E(Rᵢ) = Rᶠ + βᵢ × [E(Rₘ) - Rᶠ]
```
- **E(Rᵢ)**：资产i的预期收益率
- **Rᶠ**：无风险利率
- **βᵢ**：资产i的Beta系数
- **E(Rₘ)**：市场组合的预期收益率
- **E(Rₘ) - Rᶠ**：市场风险溢价

**Beta系数：**
```
βᵢ = Cov(Rᵢ, Rₘ) / σₘ²
βᵢ = ρᵢₘ × σᵢ / σₘ
```

**案例：**
```
无风险利率：Rᶠ = 3%
市场收益率：E(Rₘ) = 10%
市场风险溢价：E(Rₘ) - Rᶠ = 7%

股票A：
βₐ = 1.2
E(Rₐ) = 3% + 1.2×7% = 11.4%

股票B：
βᵦ = 0.8
E(Rᵦ) = 3% + 0.8×7% = 8.6%
```

#### 2.2.3 资本市场线（CML）与证券市场线（SML）

**资本市场线（CML）：**
```
E(Rₚ) = Rᶠ + (E(Rₘ) - Rᶠ)/σₘ × σₚ

适用：有效组合
横轴：标准差（σ）
斜率：夏普比率
```

**证券市场线（SML）：**
```
E(Rᵢ) = Rᶠ + βᵢ × [E(Rₘ) - Rᶠ]

适用：单个资产或组合
横轴：Beta（β）
斜率：市场风险溢价
```

### 2.3 Beta系数

#### 2.3.1 Beta的含义

**β = 1**：资产风险等于市场风险
**β > 1**：资产风险高于市场风险（进攻型）
**β < 1**：资产风险低于市场风险（防御型）
**β < 0**：资产与市场负相关（如黄金）

#### 2.3.2 Beta的计算

**回归法：**
```
Rᵢₜ - Rᶠₜ = αᵢ + βᵢ(Rₘₜ - Rᶠₜ) + εᵢₜ
```

**案例：A股某股票**
```
回归数据：
过去60个月收益率数据
R² = 0.45（拟合优度）

回归结果：
β = 1.35
α = 0.5%
```

**组合Beta：**
```
βₚ = Σ wᵢ × βᵢ
```

**案例：**
```
股票A：βₐ=1.2, 权重wₐ=60%
股票B：βᵦ=0.8, 权重wᵦ=40%

βₚ = 0.6×1.2 + 0.4×0.8 = 1.04
```

### 2.4 CAPM的应用

#### 2.4.1 资本成本估算

**股权资本成本：**
```
Kₑ = Rᶠ + β × (E(Rₘ) - Rᶠ)
```

**案例：**
```
某公司：
无风险利率：Rᶠ = 3%
公司Beta：β = 1.2
市场风险溢价：E(Rₘ) - Rᶠ = 7%

股权资本成本：
Kₑ = 3% + 1.2×7% = 11.4%
```

**加权平均资本成本（WACC）：**
```
WACC = wₑ × Kₑ + wᵈ × Kᵈ × (1 - T)
```

#### 2.4.2 投资业绩评估

**詹森Alpha（Jensen's Alpha）：**
```
α = Rᵢ - [Rᶠ + βᵢ × (Rₘ - Rᶠ)]
```

**案例：**
```
某基金：
实际收益率：Rᵢ = 15%
Beta：βᵢ = 1.1
市场收益率：Rₘ = 12%
无风险利率：Rᶠ = 3%

预期收益率：
E(Rᵢ) = 3% + 1.1×(12% - 3%) = 12.9%

詹森Alpha：
α = 15% - 12.9% = 2.1%

α > 0：基金跑赢市场
```

#### 2.4.3 证券定价

**判断证券被高估或低估：**

```
如果 E(Rᵢ) > CAPM预测值 → 证券被低估 → 买入
如果 E(Rᵢ) < CAPM预测值 → 证券被高估 → 卖出
```

### 2.5 CAPM的检验与局限

#### 2.5.1 实证检验方法

**时间序列检验：**
```
检验α是否显著异于0

模型：
Rᵢₜ - Rᶠₜ = αᵢ + βᵢ(Rₘₜ - Rᶠₜ) + εᵢₜ

原假设：αᵢ = 0
```

**截面检验：**
```
检验β是否是收益率的唯一解释因素

模型：
R̄ᵢ = γ₀ + γ₁βᵢ + εᵢ

预期：γ₀ = Rᶠ, γ₁ = E(Rₘ) - Rᶠ
```

#### 2.5.2 Roll批评

**核心观点：**
1. 市场组合不可观测
2. 使用代理组合（如指数）检验CAPM存在问题
3. 即使代理组合有效，也不能证明CAPM正确

#### 2.5.3 市场异象

**规模效应（Size Effect）：**
```
小市值股票收益率 > 大市值股票收益率
CAPM无法解释
```

**价值效应（Value Effect）：**
```
低P/B股票收益率 > 高P/B股票收益率
CAPM无法解释
```

**动量效应（Momentum Effect）：**
```
过去表现好的股票，未来继续表现好
CAPM无法解释
```

### 2.6 CAPM的扩展

#### 2.6.1 零Beta CAPM

**假设：**存在无借贷限制。

**公式：**
```
E(Rᵢ) = E(R₀ₘ) + βᵢ × [E(Rₘ) - E(R₀ₘ)]
```
- **R₀ₘ**：与市场组合零相关的组合

#### 2.6.2 多期CAPM

**假设：**投资者关心多期消费。

**消费CAPM（CCAPM）：**
```
E(Rᵢ) = Rᶠ + βᵢᶜ × [E(Rₘ) - Rᶠ]

βᵢᶜ = Cov(Rᵢ, ΔC) / Var(ΔC)
```

## 三、A股市场实证案例

### 3.1 A股市场Beta计算

**数据：**
```
标的：沪深300成分股
期间：2019-2023年（5年）
频率：月度数据
市场指数：沪深300指数
无风险利率：3%
```

**Beta分布：**
```
β < 0.5：防御型，占比10%
0.5 ≤ β < 1.0：稳健型，占比35%
1.0 ≤ β < 1.5：均衡型，占比40%
β ≥ 1.5：进攻型，占比15%

平均Beta：1.05
```

### 3.2 CAPM在A股的有效性

**检验结果：**
```
R²平均值：0.35
α显著异于0的比例：25%

结论：CAPM在A股的解释力有限
```

**改进方向：**
1. 引入Fama-French三因子模型
2. 加入动量因子
3. 考虑流动性因子

### 3.3 投资组合优化案例

**案例：两只股票组合**

```
股票：贵州茅台（600519）
E(R) = 18%, σ = 25%, β = 1.2

股票：中国平安（601318）
E(R) = 12%, σ = 20%, β = 0.9

相关性：ρ = 0.6

最优组合（夏普比率最大）：
w茅台 = 55%, w平安 = 45%

组合收益：E(Rₚ) = 15.3%
组合风险：σₚ = 20.5%
Beta：βₚ = 1.07
夏普比率：SR = (15.3% - 3%) / 20.5% = 0.60
```

## 四、小结与思考题

### 4.1 核心要点

1. **投资组合理论**
   - 分散化降低非系统性风险
   - 有效前沿是给定风险下收益最大的组合
   - 两基金分离定理

2. **CAPM模型**
   - 资产预期收益率由Beta决定
   - E(Rᵢ) = Rᶠ + βᵢ × [E(Rₘ) - Rᶠ]
   - 证券市场线（SML）

3. **Beta系数**
   - β = Cov(Rᵢ, Rₘ) / σₘ²
   - 组合Beta是各资产Beta的加权平均

4. **CAPM应用**
   - 资本成本估算
   - 投资业绩评估
   - 证券定价

### 4.2 思考题

1. **计算题**：
   两只股票A和B，E(Rₐ)=15%, σₐ=20%, E(Rᵦ)=10%, σᵦ=15%, ρₐᵦ=0.5
   求最小方差组合的权重和组合标准差。

2. **分析题**：
   为什么分散化可以降低风险但不能消除所有风险？

3. **推导题**：
   推导证券市场线（SML）公式。

4. **案例题**：
   某股票β=1.3，过去5年平均收益率12%，市场收益率10%，无风险利率3%
   计算詹森Alpha，判断该股票是否被低估。
