# 三农问题研究

> 更新时间：2026-03-06

## 乡村振兴

### 战略进展

**2026年中央一号文件核心部署**

2026年2月3日，中共中央、国务院发布《关于锚定农业农村现代化 扎实推进乡村全面振兴的意见》，这是党的十八大以来第14个指导"三农"工作的中央一号文件。文件明确2026年是"十五五"开局之年，农业农村现代化关系中国式现代化全局和成色。

**主要成就（截至2025年）**

| 指标 | 数据 | 说明 |
|------|------|------|
| 粮食产量 | 1.43万亿斤 | 连续两年稳定在1.4万亿斤以上，总产和单产均创历史新高 |
| 大豆产量 | 2000万吨以上 | 连续4年稳定在此水平 |
| 肉类产量 | 首次超过1亿吨 | 历史性突破 |
| 农民人均可支配收入 | 24,456元 | 比上年实际增长6% |
| 城乡居民收入倍差 | 2.31:1 | 较2020年的2.56:1显著缩小 |
| 脱贫人口务工就业 | 3000万人以上 | 连续5年稳定 |
| 乡村常住人口 | 4.51亿人 | 较2024年减少1369万人 |

**常态化帮扶新阶段**

2025年是巩固拓展脱贫攻坚成果同乡村振兴有效衔接5年过渡期的最后一年，2026年进入常态化帮扶新阶段。重点举措包括：

- **动态管理**：防止返贫致贫监测帮扶覆盖全体农村人口，精准识别、快速响应
- **分类帮扶**：确定160个国家乡村振兴重点帮扶县和一批省级重点帮扶县
- **政策稳定**：保持中央财政常态化帮扶资金规模稳定，延续五级书记抓、东西部协作、中央单位定点帮扶等机制

### 典型经验

**"千万工程"引领**

学习运用"千万工程"经验走深走实，以推进乡村全面振兴为总抓手，坚持：
- 农业农村优先发展
- 城乡融合发展
- 因地制宜、片区化推进

## 农业现代化

### 技术进步

**科技创新重点方向**

1. **种业振兴行动**
   - 加快选育和推广突破性品种
   - 推进生物育种产业化

2. **智能农机装备**
   - 加快高端智能、丘陵山区适用农机装备研发应用
   - 中国已成为农用无人机保有量第一大国（全球约50万台，中国超过30万台）

3. **农业新质生产力**
   - 促进人工智能与农业发展相结合
   - 拓展无人机、物联网、机器人等应用场景
   - 从靠经验种地到靠数据经营、靠算法增效

4. **数字农业**
   - 建设"天空地"一体化农业监测网络
   - 推进农业生产智能化、全产业链数字化

**农业科技创新效能提升**

| 领域 | 重点任务 |
|------|----------|
| 创新机制 | 统筹科技创新平台基地建设，深化农业科研院所改革 |
| 创新主体 | 培育壮大农业领域科技领军企业，促进产学研融合 |
| 创新领域 | 聚焦种子、农机等关键领域攻关 |

### 规模经营

**现代农业经营体系建设**

1. **土地延包**
   - 2026年在除贵州、西藏外的29个省（区、市）全面开展二轮延包整省试点
   - 原则：大稳定、小调整

2. **规模经营**
   - 规范承包地经营权流转
   - 发展农业适度规模经营
   - 促进小农户和现代农业发展有机衔接

3. **经营主体**
   - 家庭农场：提升生产经营能力，向规模适度、生产集约、绿色高效方向发展
   - 农民合作社：提高发展质量，清理规范"空壳社"
   - 农业社会化服务：完善便捷高效的服务体系

**把农业建成现代化大产业**

| 特征 | 内涵 |
|------|------|
| 农林牧渔并举 | 践行大农业观、大食物观，构建多元化食物供给体系 |
| 产加销贯通 | 做"粮头食尾""畜头肉尾""农头工尾"文章，多环节增值 |
| 农文旅融合 | 发展乡村旅游、"小而美"文旅业态 |

## 城乡融合

### 要素流动

**推动城乡要素双向流动**

1. **人口市民化**
   - 科学有序推进农业转移人口市民化
   - 落实转移支付、新增建设用地指标、基础设施建设投资与市民化挂钩政策
   - 推行常住地登记户口提供基本公共服务制度

2. **土地权益**
   - 依法维护进城落户农民土地承包权、宅基地使用权、集体收益分配权
   - 探索建立自愿有偿退出的办法
   - 有序推进农村集体经营性建设用地入市

3. **人才下乡**
   - 发展壮大乡村人才队伍
   - 激励各类人才下乡服务和创业就业
   - 实施新一轮高校毕业生"三支一扶"计划

**投融资机制创新**

- 优先保障农业农村领域一般公共预算投入
- 用足用好专项债、超长期特别国债
- 健全财政金融协同支农投入机制
- 引导和规范农业农村领域社会投资

### 公共服务

**基础设施补短板**

| 领域 | 重点任务 |
|------|----------|
| 供水 | 分类提升农村供水保障水平，有条件地方推行县域统管 |
| 电力 | 提升农村电网供电保障能力，扩大充电设施覆盖范围 |
| 交通 | 农村老旧公路改造、过窄公路拓宽、次差路段提升 |
| 物流 | 推动农村客货邮融合发展，深化快递进村 |
| 网络 | 提升农村及偏远地区网络覆盖水平 |

**公共服务均等化**

- **教育**：稳慎优化农村学校布局，实施县域普通高中振兴行动计划
- **医疗**：实施医疗卫生强基工程，稳步提高医保基金在县乡村医疗卫生机构使用比例
- **养老**：以居家养老为基础，发展助餐服务、日间照料、康复护理等
- **托育**：发展县乡普惠托育服务

## 粮食安全

### 产量目标

**2026年粮食安全目标**

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 粮食产量 | 1.4万亿斤左右 | 保持稳定 |
| 人均粮食占有量 | 508.9公斤（2025年实际） | 远超国际公认400公斤安全线 |

**粮食安全形势三句话**

1. **总体有保障**：供给丰富、市场平稳、储备充足
2. **产需紧平衡**：供需总体紧平衡格局未改变
3. **责任不放松**：粮食安全这根弦始终要绷紧

### 耕地保护

**耕地红线**

- 严守耕地红线，严格耕地占补平衡管理
- 坚决打击各类破坏耕地违法行为
- 扎实推进农村乱占耕地建房整治
- 分区分类高质量推进高标准农田建设

**耕地质量提升**

| 措施 | 内容 |
|------|------|
| 高标准农田 | 完善立项、建设、验收和管护机制，强化资金监管 |
| 黑土地保护 | 实施新一轮黑土地保护工程 |
| 盐碱地治理 | 扎实推进盐碱地综合利用 |
| 酸化耕地治理 | 推进酸化耕地治理 |

### 种业振兴

**种业振兴行动重点**

- 加快选育和推广突破性品种
- 推进生物育种产业化
- 加强农业关键核心技术攻关

## 农民增收

### 收入来源

**2026年农民增收四项措施**

1. **稳定务农种粮收益**
   - 价格政策：合理确定稻谷、小麦最低收购价，完善棉花目标价格政策
   - 补贴政策：稳定耕地地力保护补贴、玉米大豆生产者补贴、稻谷补贴
   - 保险保障：强化稻谷、小麦、玉米、大豆保险保障，支持发展地方特色农产品保险

2. **发展县域富民产业**
   - 依托本地特色资源发展县域经济
   - 推动兴业、强县、富民一体发展
   - 拓展农民参与产业发展渠道，完善公平分享收益机制

3. **促进农民工稳岗就业**
   - 落实稳岗就业支持政策
   - 实施大规模职业技能培训
   - 扩大以工代赈项目覆盖范围
   - 保障农民工工资支付

4. **盘活农村闲置资源资产**
   - 发展壮大新型农村集体经济
   - 有序推进农村集体经营性建设用地入市
   - 增加农民财产性收入

### 消费升级

**扩大乡村消费举措**

| 方向 | 措施 |
|------|------|
| 消费业态 | 培育丰收市集、非遗工坊、休闲露营等新业态新模式 |
| 流通设施 | 支持县乡村流通基础设施建设更新，促进商贸流通降本增效 |
| 下乡产品 | 支持新能源汽车、智能家电、绿色建材下乡 |
| 回收体系 | 健全农村废旧家电家具等再生资源回收体系 |

**农民收入增长趋势**

- 2025年农民人均可支配收入达24,456元，实际增长6%
- 城乡居民收入倍差由2.56:1缩小到2.31:1
- 脱贫县农民人均可支配收入增速持续高于全国平均水平

## 投资启示

### 农业投资机会

**重点关注领域**

| 领域 | 投资方向 | 政策支持 |
|------|----------|----------|
| 智慧农业 | 农业物联网、智能农机、无人机植保 | 中央一号文件明确支持 |
| 种业 | 生物育种、突破性品种研发 | 种业振兴行动持续推进 |
| 高标准农田 | 农田建设、水利设施 | 政府投资重点领域 |
| 农产品加工 | 精深加工、冷链物流 | 延伸产业链、提升附加值 |
| 农村电商 | 产地预冷、仓储保鲜、分拣加工 | 农村电商高质量发展工程 |
| 乡村旅游 | "小而美"文旅业态 | 农文旅融合政策支持 |

**政策红利**

1. **财政投入**：农业农村领域一般公共预算优先保障
2. **金融支持**：支农支小再贷款、科技创新和技术改造再贷款
3. **用地保障**：农村集体经营性建设用地入市，盘活闲置土地房屋
4. **人才引进**：乡村产业带头人培育、农创客扶持

**风险提示**

- 农产品价格波动风险：部分农产品价格较为低迷
- 自然灾害风险：近两年每年因灾农作物受灾面积超1000万公顷
- 政策执行风险：避免"一刀切"、层层加码
- 市场风险：防止产业同质化、内卷式竞争
