## 一、国际收支平衡表

### 1.1 国际收支的定义

**国际收支（Balance of Payments, BOP）**：一国在一定时期内（通常为一年）居民与非居民之间全部经济交易的系统记录。

**关键概念：**
- **居民（Resident）**：在一国经济领土内具有经济利益中心的单位
- **非居民（Non-resident）**：不在该国经济领土内的单位
- **经济交易**：商品、服务、收入的交换，资产的转移

### 1.2 国际收支平衡表结构

#### 标准组成部分（IMF第六版）

```
国际收支平衡表
├── 经常账户（Current Account）
│   ├── 货物贸易（Goods）
│   ├── 服务贸易（Services）
│   ├── 初次收入（Primary Income）
│   └── 二次收入（Secondary Income）
├── 资本账户（Capital Account）
│   ├── 资本转移
│   └── 非生产性非金融资产
├── 金融账户（Financial Account）

│   ├── 金融衍生品
```

### 1.4 国际收支调节理论

#### 1.4.1 弹性分析法

**核心观点**：汇率变动通过影响进出口价格，进而影响贸易差额。

**马歇尔-勒纳条件：**
```
|ηx| + |ηm| > 1
```
- **ηx**：出口需求弹性
- **ηm**：进口需求弹性

**政策含义：**
- 只有当进出口需求弹性之和大于1时，本币贬值才能改善贸易差额
- 如果弹性不足，贬值可能恶化贸易差额（J曲线效应）

**J曲线效应：**
```
贸易差额
  ↑
  |         J曲线
  |        /
  |       /
  |      /
  |_____/
  |    ↓↑
  +----------→ 时间
     贬值
```

**时滞分析：**
1. **货币合同期**：贬值初期，进口成本立即上升，贸易差额恶化
2. **数量调整期**：进出口数量逐渐调整，贸易差额改善
3. **新均衡期**：贸易差额达到新的均衡

#### 1.4.2 吸收分析法

**核心观点**：贸易差额 = 国民收入 - 国内吸收。

**基本公式：**
```
B = Y - A
```
- **B**：贸易差额
- **Y**：国民收入
- **A**：国内吸收（C + I + G）

**贬值的影响：**
```
ΔB = ΔY - ΔA
```

**两种效应：**
1. **收入效应**：贬值 → 出口增加 → 收入增加 → B改善
2. **吸收效应**：贬值 → 进口成本上升 → 吸收减少 → B改善

**政策含义：**
- 贬值改善贸易差额的条件：收入增加 > 吸收增加
- 需要配合紧缩性财政货币政策

#### 1.4.3 货币分析法

**核心观点**：国际收支差额本质上是货币现象。

**基本公式：**
```
B = ΔR = ΔM^d - ΔM^s
```
- **B**：国际收支差额
- **ΔR**：外汇储备变动
- **ΔM^d**：货币需求变动
- **ΔM^s**：货币供给变动

**政策含义：**
- 国际收支顺差 → 货币需求 > 货币供给 → 资金流入
- 国际收支逆差 → 货币需求 < 货币供给 → 资金流出
- 改善国际收支需要控制货币供给

## 二、汇率决定理论

### 2.1 购买力平价理论

#### 2.1.1 绝对购买力平价

**核心观点**：汇率由两国物价水平之比决定。

**公式：**
```
E = P / P*
```
- **E**：汇率（本币/外币）
- **P**：本国物价水平
- **P***：外国物价水平

**理论基础：**
- 一价定律：相同商品在不同国家的价格应该相等
- 套利机制：价格差异 → 套利 → 价格趋同

**案例：**
```
假设：
美国iPhone价格：$1000
中国iPhone价格：¥7200
绝对PPP汇率：E = 7200/1000 = 7.2 ¥/$
```

**局限性：**
1. 忽略贸易成本（运输、关税）
2. 非贸易品存在（房产、服务）
3. 产品差异化
4. 物价指数选择困难

#### 2.1.2 相对购买力平价

**核心观点**：汇率变动由两国通胀率差异决定。

**公式：**
```
Δe/e ≈ π - π*
```
- **Δe/e**：汇率变动率
- **π**：本国通胀率
- **π***：外国通胀率

**案例：**
```
假设：
中国通胀率：2%
美国通胀率：3%
人民币对美元汇率变动：Δe/e ≈ 2% - 3% = -1%

即：人民币相对美元升值1%
```

**实证检验：**
- 短期：PPP预测效果差
- 长期：PPP有一定解释力
- 汇率偏离PPP可能持续多年

### 2.2 利率平价理论

#### 2.2.1 抛补利率平价（CIP）

**核心观点**：远期汇率由两国利率差异决定。

**公式：**
```
F/S = (1 + i) / (1 + i*)
```
- **F**：远期汇率
- **S**：即期汇率
- **i**：本国利率
- **i***：外国利率

**近似形式：**
```
(F - S) / S ≈ i - i*
```

**案例：**
```
假设：
即期汇率：7.0 ¥/$
中国利率：3%
美国利率：5%

远期汇率：
F = 7.0 × (1 + 3%) / (1 + 5%) = 6.87 ¥/$

远期贴水：(6.87 - 7.0) / 7.0 ≈ -1.86%
```

**套利机制：**
```
如果 F ≠ S × (1+i)/(1+i*)，则存在套利机会：

套利步骤：
1. 借入本币
2. 按即期汇率兑换外币
3. 投资外币资产
4. 签订远期合约锁定汇率
5. 到期后按远期汇率换回本币
6. 偿还本币借款，获利
```

#### 2.2.2 非抛补利率平价（UIP）

**核心观点**：预期汇率变动由两国利率差异决定。

**公式：**
```
E(S₊₁)/S = (1 + i) / (1 + i*)
```
- **E(S₊₁)**：预期未来即期汇率

**近似形式：**
```
E(Δe)/e ≈ i - i*
```

**案例：**
```
假设：
即期汇率：7.0 ¥/$
中国利率：3%
美国利率：5%

预期汇率变动：
E(Δe)/e = 3% - 5% = -2%

预期未来汇率：
E(S₊₁) = 7.0 × (1 - 2%) = 6.86 ¥/$
```

**实证检验：**
- UIP预测效果较差
- 风险溢价、交易成本、资本管制影响

### 2.3 资产市场理论

#### 2.3.1 货币分析法

**核心观点**：汇率由两国货币供求决定。

**弹性价格模型：**
```
e = (m - m*) - α(y - y*) + β(i - i*)
```
- **e**：汇率（对数形式）
- **m**：货币供给（对数）
- **y**：实际收入（对数）
- **i**：利率
- **α, β**：系数

**政策含义：**
- 货币供给增加 → 汇率贬值
- 收入增加 → 汇率升值
- 利率上升 → 汇率贬值（价格效应）

#### 2.3.2 资产组合分析法

**核心观点**：汇率由资产组合平衡决定。

**基本假设：**
1. 投资者持有本币资产、外币资产、本币
2. 资产不完全替代
3. 资产组合平衡决定汇率

**资产需求函数：**
```
M/P = L(i, i* + E(ė), W)
B/P = B(i, i* + E(ė), W)
F/P = F(i, i* + E(ė), W)
```
- **M**：货币需求
- **B**：本币债券需求
- **F**：外币资产需求
- **W**：财富

**政策含义：**
- 资产供给变化影响汇率
- 经常账户影响外币资产供给
- 货币政策影响资产组合

### 2.4 汇率超调模型

**Dornbusch模型（1976）**

**核心观点**：资产市场调整快于商品市场，导致汇率超调。

**基本假设：**
1. 短期价格粘性
2. 长期价格灵活
3. 资本完全流动
4. 非抛补利率平价成立

**超调机制：**
```
货币供给增加
  ↓
短期：价格不变，利率下降
  ↓
资本流出，汇率大幅贬值
  ↓
长期：价格上涨，汇率升值回归
```

**图形分析：**
```
汇率 (e)
  ↑
  |     超调
  |    /
  |   /
  |  /      长期均衡
  | /      __________
  |/______/
  +----------→ 时间
  短期超调  长期均衡
```

**案例：**
```
假设：
长期均衡汇率：7.0 ¥/$
货币供给增加10%

短期超调：
汇率可能贬值到7.5 ¥/$（超调7%）

长期回归：
汇率逐渐升值回到7.0 ¥/$
```

## 三、人民币汇率形成机制

### 3.1 汇率制度演变

#### 3.1.1 固定汇率时期（1949-1994年）

**特点：**
- 官方汇率固定
- 双重汇率制度
- 外汇管制严格

**问题：**
- 汇率扭曲
- 外汇短缺
- 出口缺乏激励

#### 3.1.2 汇率并轨（1994年）

**改革内容：**
- 官方汇率与调剂市场汇率并轨
- 建立银行间外汇市场
- 实行有管理的浮动汇率制度

**汇率并轨：**
```
官方汇率：5.8 ¥/$
调剂汇率：8.7 ¥/$
并轨汇率：8.7 ¥/$
```

#### 3.1.3 参考一篮子货币（2005年至今）

**2005年7月21日改革：**
- 实行以市场供求为基础、参考一篮子货币进行调节、有管理的浮动汇率制度
- 汇率一次性升值2.1%

**汇率波动区间：**
```
2007年：±0.5%
2012年：±1.0%
2014年：±2.0%
```

### 3.2 人民币汇率中间价形成机制

#### 3.2.1 中间价报价模型

**2015年8月11日改革前：**
```
中间价 = 前一日收盘价
```

**2015年"8·11汇改"后：**
```
中间价 = 前一日收盘价 + 一篮子货币汇率变化
```

**2016年后：**
```
中间价 = 收盘价 + 一篮子货币汇率变化 + 逆周期调节因子
```

**2020年后：**
```
中间价 = 收盘价 + 一篮子货币汇率变化
（淡出逆周期调节因子）
```
