## 执行摘要

本白皮书整合8周深度研究成果，为2026-2030年A股ETF投资提供完整的策略框架。核心结论：

- **情景概率**：乐观25%，基准50%，悲观25%
- **预期年化收益**：乐观25-40%，基准10-20%，悲观-15%至+5%
- **最大回撤控制目标**：-15%以内
- **核心策略**：情景化配置 + 核心卫星架构 + 动态风险管理

**V2.0更新要点**：

```
组合配置 = Σ(情景概率 × 情景配置)

示例计算：
基准情景(50%) × 基准配置 + 乐观情景(25%) × 乐观配置 + 悲观情景(25%) × 悲观配置

+ 25% × [科技40% + 消费25% + 金融15% + 现金10%]

= 最终组合：宽基15% + 高股息29% + 科技25% + 消费13% + 防御8% + 黄金5% + 现金10%
```

**再平衡触发条件**：
- 单一ETF权重偏离目标权重>5%
- 组合总权重偏离100%>3%
- 有ETF触发出局条件

├── 股票仓位：维持

### 6.3 ETF出局条件

**触发出局**（满足任一）：
- 年化收益连续2年<10%
- 最大回撤>20%
- 夏普比率连续2年<0.5
- 流动性不足（日均成交<5000万）

**替换原则**：
- 从备选池中选择最优替代
- 优先选择同行业/同风格ETF
- 保持组合整体风险水平

## 第七部分：绩效评估与归因

### 7.1 收益指标

| 指标 | 计算公式 | 目标值 |
|------|---------|-------|
| 年化收益率 | (期末净值/期初净值)^(252/n) - 1 | 12-18% |
| 累计收益率 | (期末净值 - 期初净值) / 期初净值 | 年度12-18% |
| 超额收益率 | 组合收益率 - 基准收益率 | 年度3-5% |

### 7.2 风险指标

| 指标 | 计算公式 | 目标值 |
|------|---------|-------|
| 最大回撤 | max(Peak - Trough) / Peak | <-15% |
| 年化波动率 | std(日收益率) × √252 | 15-18% |
| VaR（95%，日度） | -μ×Δt + z×σ×√Δt | <-2.0% |

### 7.3 风险调整收益指标

| 指标 | 计算公式 | 目标值 |
|------|---------|-------|
| 夏普比率 | (收益率 - 无风险利率) / 波动率 | >1.0 |
| 卡玛比率 | 年化收益率 / 最大回撤 | >1.0 |
| 索提诺比率 | (收益率 - 无风险利率) / 下行风险 | >1.2 |
