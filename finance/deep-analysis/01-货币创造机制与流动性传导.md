# 货币创造机制与流动性传导

## 摘要

本文深入分析中国货币创造机制与流动性传导体系，结合2024-2026年最新数据，探讨从央行基础货币投放到实体经济的传导路径，并评估其对A股市场流动性的影响。

---

## 一、商业银行货币创造机制

### 1.1 存款乘数理论

存款乘数是理解货币创造的核心概念。当央行向银行体系注入基础货币时，商业银行通过贷款-存款循环实现货币的派生扩张。

**基本公式：**
```
存款乘数 = 1 / 法定存款准备金率
```

**实际货币乘数：**
```
货币乘数 = M2 / 基础货币
```

根据中国人民银行2025年数据：
- **M2余额**：340.29万亿元（2025年末）
- **货币乘数约7-8倍**（估算值，受现金漏损率、超额准备金率影响）

### 1.2 中国货币乘数的演变

| 年份 | M2余额(万亿元) | 同比增速 | 备注 |
|------|---------------|---------|------|
| 2024末 | ~312 | 7.3% | 经济复苏期 |
| 2025末 | 340.29 | 8.5% | 稳增长政策发力 |
| 2026年1月 | 347.19 | 9.0% | 开年信贷投放强劲 |

货币乘数的影响因素：
1. **法定存款准备金率**：央行通过降准提高货币乘数
2. **超额准备金率**：银行持有超额准备金的意愿
3. **现金漏损率**：公众持有现金的比例
4. **定期存款比率**：影响流动性创造效率

### 1.3 2024-2026年降准周期分析

**关键降准操作：**

| 时间 | 降准幅度 | 释放资金 | 政策目标 |
|------|---------|---------|---------|
| 2024年2月 | 0.5% | 约1万亿元 | 支持实体经济 |
| 2024年9月 | 0.5% | 约1万亿元 | 稳增长、降成本 |
| 2025年 | 多次定向降准 | 累计约1.5万亿元 | 支持小微企业、绿色金融 |

**当前存款准备金率水平：**
- 大型金融机构：约10%
- 中小金融机构：约7%
- 农村金融机构：约6%

## 二、央行基础货币投放渠道

### 2.1 传统投放渠道

#### （1）外汇占款
- **历史地位**：2014年前是主要投放渠道
- **当前状况**：外汇占款规模稳定，不再是主要投放方式
- **2025年末外汇储备**：3.36万亿美元

#### （2）公开市场操作（OMO）
- **主要工具**：7天逆回购、14天逆回购
- **操作频率**：每日开展
- **2025年特点**：逆回购利率多次下调，引导市场利率下行

#### （3）中期借贷便利（MLF）
**2025年1月MLF余额：4.29万亿元**

MLF操作数据：
| 时期 | 余额变化 | 操作利率 | 政策信号 |
|------|---------|---------|---------|
| 2024Q4 | 小幅收缩 | 2.0% | 中性偏紧 |
| 2025Q1 | 适度扩张 | 1.9% | 降息周期 |
| 2025年末 | 4.3万亿 | 1.8% | 稳增长导向 |

### 2.2 创新投放渠道

#### （1）结构性货币政策工具

**2025年一季度末数据：**
- 存续工具：**10项**
- 余额合计：**5.9万亿元**

**2024年第四季度新设工具：**
1. **证券、基金、保险公司互换便利（SFISF）**
   - 额度：5000亿元
   - 目的：支持资本市场流动性

2. **股票回购增持再贷款**
   - 额度：3000亿元
   - 目的：稳定股票市场

#### （2）抵押补充贷款（PSL）
- 主要用于棚改、地下管廊等基础设施建设
- 规模收缩：从高峰期的3.5万亿元降至约2万亿元

#### （3）定向中期借贷便利（TMLF）
- 定向支持小微、民营企业
- 利率低于MLF约15BP

### 2.3 基础货币投放数据分析

**2025年基础货币变化：**
```
基础货币 ≈ M2 / 货币乘数
2025年末 ≈ 340万亿 / 7.5 ≈ 45万亿元
```

**投放渠道占比（估算）：**
| 渠道 | 占比 | 趋势 |
|------|------|------|
| MLF/OMO | 40% | 上升 |
| 外汇占款 | 25% | 稳定 |
| 降准释放 | 20% | 周期性 |
| 结构性工具 | 15% | 快速上升 |

## 三、流动性传导机制

### 3.1 传导链条

```
央行操作 → 银行间市场流动性 → 商业银行信贷 → 实体经济
    ↓              ↓                ↓            ↓
  降准/降息    资金利率下降     贷款利率下降   投资消费增加
```

### 3.2 传导效率分析

**银行间市场利率（2025年12月）：**
- 同业拆借加权平均利率：**1.36%**
- 质押式回购加权平均利率：**1.40%**
- 较上年同期下降约20-25BP

**信贷市场利率：**
- LPR（1年期）：约3.1%
- LPR（5年期）：约3.6%
- 企业贷款加权平均利率：约3.8%

**传导效率评估：**
| 环节 | 传导效果 | 评价 |
|------|---------|------|
| 央行→银行间市场 | 高效 | 利率敏感 |
| 银行间→信贷市场 | 中等 | 存在时滞 |
| 信贷→实体经济 | 相对滞后 | 需求约束 |

### 3.3 流动性传导的阻滞因素

1. **信贷需求不足**：实体经济融资需求偏弱
2. **银行风险偏好**：惜贷情绪影响传导效率
3. **监管约束**：资本充足率、流动性覆盖率等指标约束
4. **资金空转风险**：资金在金融体系内循环

## 四、2024-2026年货币投放数据分析

### 4.1 社会融资规模

**2025年全年社会融资规模增量：35.6万亿元**
- 比上年多增3.34万亿元
- 主要增量来源：政府债券净融资13.84万亿元

**2026年1月末社会融资规模存量：449.11万亿元**
- 同比增长8.2%
- 结构变化：政府债券占比上升至21.4%

### 4.2 信贷投放结构

**2025年人民币贷款增加：16.27万亿元**

分部门看：
| 部门 | 增量 | 占比 | 特点 |
|------|------|------|------|
| 住户贷款 | 4417亿元 | 2.7% | 短期减少、中长期增加 |
| 企(事)业单位贷款 | 15.47万亿元 | 95.1% | 中长期贷款为主 |
| 非银金融机构贷款 | -1103亿元 | -0.7% | 收缩 |

**2026年1月单月贷款增加：4.71万亿元**
- 企业贷款占绝对主导（4.45万亿元）
- 中长期贷款增加3.18万亿元，显示基建投资发力

### 4.3 货币供应量分析

**M1统计口径修订（2025年起）：**
新口径包括：
- 流通中货币（M0）
- 单位活期存款
- **个人活期存款（新增）**
- **非银行支付机构客户备付金（新增）**

**M1增速变化（新口径）：**
| 时期 | 余额(万亿元) | 同比增速 |
|------|-------------|---------|
| 2024年12月 | 111.31 | 1.2% |
| 2025年12月 | 115.51 | 3.8% |
| 2026年1月 | 117.97 | 4.9% |

**M2与M1剪刀差：**
- 2026年1月：M2增速9.0% - M1增速4.9% = **4.1个百分点**
- 剪刀差收窄，反映经济活跃度提升

## 五、对A股流动性的影响

### 5.1 流动性与股市的关系

**理论机制：**
```
货币宽松 → 市场利率下降 → 股票估值上升 → 股价上涨
         ↓
      资金流入股市 → 增量资金推动 → 行情上涨
```

### 5.2 2024-2026年流动性环境评估

**利好因素：**
1. **货币供应量稳健增长**：M2增速8.5%-9%
2. **政策利率持续下调**：降低市场无风险利率
3. **结构性工具直达资本市场**：SFISF、回购增持再贷款
4. **社会融资规模扩大**：充裕的融资支持

**利空因素：**
1. **资金空转风险**：部分资金未进入实体经济
2. **信贷结构失衡**：居民部门加杠杆意愿弱
3. **外部环境制约**：中美利差倒挂限制宽松空间

### 5.3 A股流动性指标跟踪

**核心指标：**
| 指标 | 当前值 | 信号 |
|------|-------|------|
| M2增速 | 9.0% | 中性偏多 |
| M1-M2剪刀差 | -4.1% | 收窄中 |
| 社融增速 | 8.2% | 稳健 |
| 银行间利率 | 1.36% | 宽松 |
| 北向资金 | 波动 | 中性 |

### 5.4 历史经验与投资启示

**历史规律：**
- M1增速触底回升往往预示股市底部
- 货币政策转向宽松后3-6个月，股市通常有正向反应
- 结构性货币政策工具对特定板块有直接支持作用

**2024-2026年投资启示：**

1. **关注政策信号**
   - 降准、降息是重要做多信号
   - MLF利率调整指引LPR方向

2. **跟踪流动性指标**
   - M1增速拐点是关键信号
   - 银行间利率过低需警惕资金空转

3. **把握结构性机会**
   - SFISF直接利好券商、保险
   - 回购增持再贷款支持高股息板块
   - PSL收缩意味着基建相关需谨慎

4. **风险提示**
   - 货币政策边际效果递减
   - 流动性宽松不等于股市上涨
   - 需结合经济基本面综合判断

## 六、结论

2024-2026年中国货币政策呈现以下特征：

1. **货币创造效率提升**：降准周期提高货币乘数，M2增速稳健在8.5%-9%

2. **投放渠道多元化**：从外汇占款主导向MLF/OMO和结构性工具转变

3. **传导效率改善**：银行间利率降至1.36%低位，但实体融资需求仍需提振

4. **资本市场支持加强**：创新工具直接支持股票市场流动性

5. **A股影响中性偏多**：流动性环境友好，但需关注传导效率和结构性问题

**投资者应重点关注**：M1增速拐点、政策利率变化、结构性工具额度使用情况，以及中美货币政策分化带来的外部制约。
