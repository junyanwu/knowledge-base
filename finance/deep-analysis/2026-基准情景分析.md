# 2026年基准情景分析：现状延续与温和增长

## 概率评估：50%

### 一、情景概述与核心假设

基准情景假设2026年全球经济延续当前态势，呈现温和增长格局，既无显著好转也无重大危机。核心特征包括：中美关系维持"竞争性共存"状态、美联储货币政策谨慎转向、中国经济在政策托底下实现弱复苏、中东局势紧张但可控。在此情景下，全球经济增速预计在3.0%-3.5%区间，接近IMF基准预测。

#### 1.1 中美关系现状延续

基准情景下，中美关系维持"竞争性共存"的基本框架。双方在贸易、技术、地缘政治等领域持续博弈，但避免全面对抗升级。

**贸易领域**：已加征关税维持现状，新增关税有限。双方在具体产品层面可能有有限度的关税排除，但整体关税水平保持高位。中美贸易额小幅下降或持平，中国出口企业继续面临关税压力，但已逐步适应并寻找替代市场。

**技术领域**：美方对华技术管制持续收紧，但节奏放缓。核心高端芯片、先进制造设备等领域管制维持或小幅加码，但消费电子、成熟制程等非核心领域管制边际放松。中国企业继续推进国产替代，但进度受制于技术瓶颈，难以快速突破。

**地缘政治**：台海局势维持"不统不独不武"现状，双方在南海、台海保持军事存在但避免直接冲突。美国继续推进"印太战略"，强化与日韩澳菲等盟友合作，但对中国核心利益的挑衅有所克制。

这一情景对中国经济的影响是：出口增长承压但可控，外资流入放缓但不大幅流出，技术进步放缓但仍在推进，经济增长更多依赖内需驱动。

#### 1.2 美联储政策谨慎转向

基准情景假设美联储在2025年底结束加息周期，2026年开始降息，但节奏和幅度低于乐观情景。

**通胀路径**：美国核心PCE通胀缓慢回落至2.5%-3.0%区间，但难以快速降至2%目标。服务通胀粘性较强，工资增长保持高位，房租回落缓慢。总体通胀维持震荡下行趋势，但存在反复风险。

**就业市场**：美国就业市场缓慢降温，失业率从当前3.5%-4.0%上升至4.5%-5.0%，实现软着陆而非硬着陆。职位空缺率下降，但裁员规模有限，就业市场对通胀的支撑作用减弱。

**降息路径**：2026年降息2-3次，每次25个基点，联邦基金利率降至4.0%-4.75%区间。降息节奏谨慎，每次决策高度依赖当期数据，市场预期波动较大。长期利率维持在较高水平，10年期美债收益率在3.5%-4.0%区间。

**对全球的影响**：美元指数温和走弱但维持强势，新兴市场资本流入有限，全球流动性改善幅度不大。对中国而言，人民币贬值压力减轻但未消除，央行货币政策空间有限，中美利差依然倒挂。

#### 1.3 中国经济弱复苏

基准情景下，中国经济增速在2026年达到4.5%-5.0%，实现政府增长目标，但复苏力度不强。

**消费复苏温和**：居民消费信心缓慢恢复，社会消费品零售总额增速5%-6%。服务业消费好于商品消费，旅游、餐饮等接触性消费恢复较好，但耐用品消费（家电、汽车、家具）增长疲软。消费者收入预期谨慎，储蓄倾向仍然较高。

**投资结构分化**：基础设施投资保持6%-7%增速，仍是稳增长的重要抓手。制造业投资增速5%-6%，结构向高端制造倾斜，但整体投资意愿不强。房地产开发投资继续负增长，降幅在-5%至-3%区间，销售端难以快速回暖，企业资金紧张制约投资。

**出口增长承压**：全球经济温和增长背景下，中国出口增速在0%-2%区间，难以大幅扩张。对欧美出口小幅下降或持平，对"一带一路"国家出口保持增长但增速放缓。出口结构持续优化，新能源汽车、光伏等产品成为亮点，但难以完全抵消传统出口下滑。

**政策支持持续**：财政政策保持积极，专项债规模维持较高水平，但增量有限。货币政策保持稳健，社融增速在8%-9%，M2增速在7%-8%。结构性政策工具持续发力，但对经济拉动的边际效应递减。

### 二、中东局势紧张但可控

基准情景假设中东地缘政治局势持续紧张，但各方保持克制，避免全面冲突升级。

**巴以冲突**：以色列与哈马斯冲突持续但强度下降，呈现"低强度长期化"特征。国际社会持续斡旋，但难以实现持久和平。加沙地区人道主义危机持续，但范围不再扩大。

**伊朗因素**：伊朗与以色列通过代理力量间接对抗，但避免直接军事冲突。伊朗核问题谈判停滞，但伊朗暂不进行核试验。伊朗支持的胡塞武装、真主党等继续对以色列及红海航运构成威胁，但攻击频率和强度可控。

**地区大国**：沙特、阿联酋等海湾国家维持平衡外交，既与美国保持安全合作，也与中国深化经贸关系。OPEC+产量政策保持灵活，油价维持在75-90美元/桶区间。

**对油价的影响**：布伦特原油价格在80-95美元/桶区间震荡，存在地缘政治溢价但不失控。对中国而言，油价维持中高水平增加生产成本和通胀压力，但在可控范围内。

### 三、A股市场中性预期

基准情景下，A股市场呈现结构性行情，整体涨幅有限但存在结构性机会。

#### 3.1 指数表现预测

- **上证指数**：目标区间3100-3400点，涨幅0%-10%
- **沪深300指数**：目标区间4000-4400点，涨幅5%-15%
- **创业板指**：目标区间2000-2300点，涨幅0%-15%
- **科创50指数**：目标区间900-1100点，涨幅-5%-15%

市场特征呈现"上有顶、下有底"的区间震荡格局。估值难以大幅扩张，但盈利增长支撑市场；结构性机会丰富，但板块轮动加快。

#### 3.2 市场驱动因素分析

**估值因素中性**：当前A股估值处于历史较低水平，向下空间有限；但在缺乏重大利好催化下，估值也难以大幅修复。沪深300市盈率维持在11-13倍区间，创业板指维持在25-30倍区间。

**盈利增长温和**：经济弱复苏背景下，A股上市公司盈利增长温和。预计全部A股2026年净利润增速在5%-8%，其中沪深300成分股增速6%-10%，创业板增速10%-15%。盈利增长是市场的主要支撑，但难以推动大幅上涨。

**流动性平衡**：美联储降息幅度有限，外资流入规模有限，预计2026年北向资金净流入500-1500亿元。国内货币政策保持稳健，两融余额温和回升，个人投资者风险偏好中性。流动性既不充裕也不紧张，市场呈现存量博弈特征。

**情绪谨慎乐观**：中美关系维持现状，经济数据符合预期，政策支持持续，市场情绪谨慎乐观。风险溢价维持在中性水平，市场波动率适中。

#### 3.3 结构性行情特征

基准情景下，市场呈现显著的结构性行情，板块分化明显：

**相对强势板块**：
- **高股息板块**（电力、高速公路、运营商）：涨幅5%-15%，提供稳定收益和防御属性
- **央企改革主题**：涨幅10%-20%，受益于国企改革深化
- **科技创新板块**（人工智能、半导体）：涨幅10%-25%，国产替代逻辑持续
- **新能源板块**：涨幅0%-15%，估值消化后有所修复

**相对弱势板块**：
- **房地产板块**：涨幅-10%-5%，行业景气度仍然低迷
- **消费板块**：涨幅0%-10%，复苏力度不强
- **银行板块**：涨幅0%-8%，息差收窄制约盈利

### 五、风险点识别

基准情景虽为概率最高情景（50%），但仍需关注以下风险：

#### 5.1 上行风险（转为乐观情景的触发因素）

1. **中美关系超预期改善**：领导人会晤达成重大共识，关税大幅削减
2. **美联储降息超预期**：通胀快速回落，降息节奏快于预期
3. **中国经济强复苏**：政策刺激效果超预期，消费、投资快速回暖
4. **地缘政治缓和**：中东冲突降级，乌克兰危机缓解

#### 5.2 下行风险（转为悲观情景的触发因素）

1. **中美关系恶化**：台海危机升级、技术制裁加码、贸易战升级
2. **美联储维持高利率**：通胀反弹导致降息推迟或取消
3. **中国经济失速**：房地产风险扩散、消费持续低迷、出口大幅下滑
4. **地缘冲突升级**：中东全面战争、台海危机、俄乌冲突扩大化
5. **金融危机**：美国商业地产危机、欧洲债务危机、新兴市场债务危机

#### 5.3 特别关注风险

**房地产市场风险**：房地产是中国经济的核心风险点。若房价持续下跌、房企违约蔓延、银行不良资产上升，可能引发系统性风险。关注房企债务到期高峰、银行房地产贷款占比、房价走势等指标。

**地方债务风险**：地方政府债务压力巨大，城投平台债务风险持续。若经济下行、土地出让收入下降，可能触发债务违约。关注地方债务化解进展、城投债收益率、财政转移支付等。

**金融风险**：中小银行资产质量恶化、影子银行风险、理财产品违约等潜在风险。关注银行不良贷款率、影子银行规模、理财产品收益率等。

### 七、投资启示

基准情景下，投资者应采取以下策略：

1. **保持耐心**：震荡市中不要追涨杀跌，耐心等待机会
2. **均衡配置**：兼顾防御和进攻，避免过度集中
3. **精选结构**：关注高股息、科技、新能源等结构性机会
4. **控制仓位**：保持中庸仓位（60%-70%），留有余地
5. **动态调整**：根据关键信号指标及时调整策略
6. **风险管理**：设定止损位，控制单次亏损幅度
7. **定期再平衡**：每季度进行一次组合再平衡

基准情景是最大概率情景，投资者应以稳健为主，在控制风险的前提下把握结构性机会。同时，要密切关注情景转换信号，及时调整策略应对变化。
