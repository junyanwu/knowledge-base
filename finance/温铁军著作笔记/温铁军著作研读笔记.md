# 温铁军著作研读笔记

> 更新时间：2026-03-06
> 作者：温铁军（1951-）
> 身份：三农问题专家、中国人民大学教授

## 核心理论

### 1. 成本转嫁论

**核心观点：**
- 工业化过程中的制度成本向"三农"转嫁
- 城乡二元体制下，危机成本主要由农村承担
- "三农"是危机"软着陆"的缓冲带

**理论依据：**
任何工业化都绕不开资本原始积累。中国原始积累不能对外，就只能对三农。三农之所以是问题，在于它向国家的工业化作了巨大的贡献。

**实践验证：**
- 1958-1960年苏联援华投资中断危机 → 第一次上山下乡
- 1968-1970年三线建设危机 → 第二次上山下乡
- 1974-1976年危机 → 第三次上山下乡
- 三次危机通过向农村转嫁成本实现"软着陆"

### 2. 三农问题理论

**核心观点：**
- 三农问题：农民、农村、农业问题
- 三农是中华民族文明的根基
- 农耕文明是"三生合一"：生态-生活-生产

**历史视角：**
- 中国几千年都是乡土文明，农业社会
- 农业社会涵养自然资源、维护环境生态的能力远大于工业文明
- 维护了中华文明绵延不绝的五千年历史

**现代化困境：**
- 工业化进程中，农业要素要按照工业定价
- 成本高于国际价格，没有政府保护就没人干农业
- 农业社会向工业社会转型中的结构性矛盾

### 3. 危机周期理论

**核心观点：**
- 经济危机是资本主义一般内生性矛盾
- 中国也存在周期性的经济危机
- 危机化解方式决定社会成本分配

**理论发展：**
- 1987年在《新华文摘》发表《危机论》，首次提出周期性危机问题
- 从生产过剩危机到资本过剩危机的演进
- 8次危机的规律性分析

### 4. 去依附理论

**核心观点：**
- 后发国家政治独立后需去除对原宗主国经济依附
- 中国通过土改实现第一次"去依附"
- 需要在思想、文化、制度各方面实现去依附

**关键要素：**
- 土地改革动员亿万农民
- 农村成为吸纳超发货币、稳定物价的"海绵"和"减压阀"
- 增量农产品稳定供给压低城市物价

## 八次危机

### 危机一：1958-1960年
**背景：** 苏联援华投资中断
**表现：** 工业化初期困境
**应对：** 第一次上山下乡
**结果：** 危机成本向农村转嫁，实现软着陆

### 危机二：1968-1970年
**背景：** 三线建设中的国家战略调整
**表现：** 经济结构失衡
**应对：** 第二次上山下乡
**结果：** 农村吸纳城市过剩劳动力

### 危机三：1974-1976年
**背景：** "四三方案"到"八二方案"的外债压力
**表现：** 最后一次上山下乡
**结果：** 通过农村再次化解危机

### 危机四：1979-1980年
**背景：** 改革开放初期
**表现：** 改革以来第一次经济危机
**应对：** 借助"三农"复苏
**特征：** 内源性危机

### 危机五：1989-1990年
**背景：** 改革深化期
**表现：** 改革以来第二次经济危机
**应对：** "三农"承接
**特征：** 内源性危机

### 危机六：1993-1994年
**背景：** 外向型经济转型
**表现：** 改革以来第三次经济危机
**应对：** 外向型转化
**特征：** 内源性危机向外向型转化

### 危机七：1997年
**背景：** 东南亚金融危机
**表现：** 输入型危机
**应对：** 扩大内需、西部大开发
**特征：** 第一次输入型危机
**启示：** 林毅夫提出罗斯福新政中国版

### 危机八：2008年
**背景：** 全球金融危机
**表现：** 输入型危机
**应对：** 4万亿投资、新农村建设
**特征：** 第二次输入型危机

### 八次危机的启示

**软着陆机制：**
- 三次上山下乡是危机软着陆的关键手段
- 城乡二元体制下，农村有效吸纳危机成本
- 改革开放后三农承接能力下降，多次硬着陆

**硬着陆代价：**
- 三农不再有过去的社会条件承接危机
- 导致社会问题甚至政治运动
- 政府财税金融乃至整个经济体制的重大变革

**规律总结：**
- 60年4次引进外资，各伴生2次危机
- 危机周期性爆发不可避免
- 化解手段决定社会成本分配

## 去依附

### 理论来源

**依附理论：**
- 埃及裔学者萨米尔·阿明提出
- 后发国家在政治独立后仍经济依附原宗主国
- 发展中国家陷入"发展主义陷阱"

**沃勒斯坦世界体系理论：**
- 核心国家、边缘国家、半边缘国家
- 不平等交换与依附关系

### 中国的去依附实践

**1949-1952年第一次通胀危机：**
1. **危机背景：**
   - 民国延续的长期高通胀
   - 经济发展内生新矛盾
   - 财政赤字压力

2. **应对措施：**
   - 土地改革提高农民生产积极性
   - 增量农产品稳定供给压低城市物价
   - 吸纳过量增发货币
   - 实现经济软着陆

3. **关键机制：**
   - 农村成为吸纳超发货币的"海绵"
   - 农村成为稳定物价的"减压阀"
   - 货币下乡并沉淀

**两次硬脱钩：**
1. **1950年：** 美国带领西方国家制裁封锁中国
2. **1960年：** 中苏硬脱钩
   - 偿还朝鲜战争军事援助
   - 支付改变对苏依附的制度成本
   - 被动实现去依附

### 当前去依附的挑战

**思想解殖：**
- 形成去西方中心主义的话语体系
- 解构殖民化话语体系
- 建立独立自主的思想理论体系

**文化去依附：**
- 在教育、医疗、文化、意识形态各方面
- 支撑独立话语体系的构建
- 应对西方中心主义的挑战

**经济去依附：**
- 减少对外部资源和市场的依赖
- 构建国内大循环为主体
- 发展自主可控的产业链

## 最新观点（2024-2025）

### 一、思想解殖与三个世界理论

**核心观点：**
- 做好中国研究，需先完成"思想解殖"
- 毛泽东"三个世界"理论是应对新冷战的重要思想武器
- 21世纪"毛泽东热"反映年轻人的求新知自觉性

**思想建设：**
- 继承第三世界思想家遗志
- 发展去西方中心主义话语体系
- 形成独立自主的思想理论

**历史借鉴：**
- 1950年和1960年两次硬脱钩的经验教训
- 中国被迫却彻底实现"去依附"
- 独立自主、自力更生的建设道路

### 二、全球过剩危机与资本问题

**危机诊断：**
- 全球危机根源是"资本过剩"
- 中国也进入金融资本过剩阶段
- 从生产过剩到资本过剩的演进

**中国现状：**
- 金融总量已达世界最高额度
- 银行出现"资产荒"：存款增加，贷款找不到投向
- 55%以上居民借贷投向房地产

**应对方向：**
- 工业供给侧改革（2015）
- 农业供给侧改革（2017）
- 金融供给侧改革（2019）
- 向乡村领域投资应对过剩

### 三、新质生产力与新型生产关系

**三个原则：**
1. 创新要素配置方式，全面深化生产关系改革
2. 粗放数量型增长改为质量效益增长
3. 不忽视传统产业，不搞单一模式

**发展困境：**
- 产业资本阶段资源禀赋不足
- 钢铁工业铁矿石90%以上靠进口
- 石油天然气能源70%靠进口
- 大豆进口90%以上

**转型要求：**
- 从"工业化+城市化"模式转型
- 应对"面源污染"、资源枯竭、环境破坏
- 解决社会两极分化问题

### 四、乡村振兴作为第三资产池

**三大资产池：**
1. **第一资产池：** 1999年后"铁公基"基础设施建设
   - 西部大开发、东北振兴、中部崛起
   - 高速公路网、机场码头建设

2. **第二资产池：** 2000年后房地产
   - 搭基本建设便车的机会收益
   - 现已出现过剩和金融泡沫

3. **第三资产池：** 乡村振兴
   - 2005年新农村建设开始投入
   - 已投入超过10万亿
   - 乡村振兴投资规模至少7万亿

**第三资产池的优势：**
- 农村村村"五通"（路、水、电、气、通信）
- 大量生态空间资源待开发
- 山、水、田、林、湖、草、沙等资源

**投资方向：**
- 生态产业化、产业生态化
- 养生、康养、文化、旅游等新业态
- 乡土文化内涵开发

### 五、乡村振兴与就业问题

**就业压力：**
- 贸易战导致企业转移
- 几百万家个体私营经济注销
- 几千万打工者"二次返乡"
- 需要乡村提供就业机会

**战略意义：**
- 乡村振兴是应对全球化挑战的"压舱石"
- 数千万返乡人员要有事做、有饭吃、有地方住
- 国家稳定的基石

**新业态激活：**
- 不是只能种地
- 需要把金融供给侧改革与农业供给侧改革结合
- 把资源开发权交给农民
- 鼓励市民下乡联合创业

### 六、全球化挑战与应对

**核心矛盾：**
- 全球资本过剩与中国资本过剩
- 过剩资本硬碰硬的竞争
- 恶性竞争加剧

**战略转变：**
- 从大进大出的外向型经济转型
- 国内大循环为主体
- 构建战略回旋空间

**应对策略：**
- 加强县域经济和乡村振兴
- 把产业留在县域
- 让农民获得更多产业增值收益
- 减少外出务工成本

## 投资启示

### 一、理解经济周期

**周期规律：**
- 经济危机是资本主义内生性矛盾
- 中国也存在8次经济危机周期
- 理解周期才能把握投资时机

**危机应对：**
- 关注国家逆周期调控政策
- 把握资产池转换机遇
- 避免在资产泡沫期进入

### 二、关注资产池转换

**历史经验：**
- 第一资产池：基础设施建设 → 获得第一桶金
- 第二资产池：房地产 → 机会收益
- 第三资产池：乡村振兴 → 当前机遇

**投资要点：**
- 跟上政策转型的节奏
- 第三资产池不是投农业，是投乡村振兴
- 关注生态产业化、产业生态化

### 三、理解成本转嫁机制

**三农的承接作用：**
- 危机成本往往向农村转嫁
- 乡村振兴是承接资本过剩的新渠道
- 农村资源价值重估

**投资启示：**
- 关注农村土地、生态资源价值
- 把握乡村物业和资源型资产开发
- 新业态：养生、康养、文化、旅游

### 四、去依附的投资逻辑

**外部依赖风险：**
- 钢铁：铁矿石90%以上进口
- 能源：石油天然气70%进口
- 农业：大豆90%以上进口

**投资方向：**
- 自主可控产业链
- 国产替代机会
- 资源安全和粮食安全

### 五、新质生产力与生产关系

**转型要求：**
- 从粗放数量型到质量效益型
- 新型生产关系改革
- 创新要素配置方式

**投资机会：**
- 新技术、新业态、新模式
- 传统产业升级改造
- 不搞单一模式，因地制宜

### 六、风险管理

**资本过剩风险：**
- 金融资产荒
- 资金滞存
- 投资回报率下降

**应对策略：**
- 关注供给侧结构性改革
- 把握国家战略投资方向
- 避免资本野蛮生长领域

**长期视角：**
- 中华农耕文明的根基
- 人与自然和谐共生的现代化
- 一万年农耕文明vs百年工业化

## 核心概念速查

### 关键术语

**三农问题：** 农民、农村、农业问题

**成本转嫁：** 工业化制度成本向农村转嫁

**软着陆：** 危机成本被农村有效吸纳

**硬着陆：** 危机在城市爆发，导致体制重大变革

**去依附：** 去除对原宗主国或主导国的经济依附

**三个世界：** 毛泽东提出的地缘政治理论

**第三资产池：** 乡村振兴作为资本投资的新领域

**资本过剩：** 金融资本超过实体经济投资需求

**生产过剩：** 工业产能超过市场需求

**新质生产力：** 质量效益型的新型生产力

**新型生产关系：** 与新质生产力相适应的生产关系

**生态产业化：** 将生态资源转化为产业

**产业生态化：** 产业发展符合生态要求

### 重要理论

**依附理论：** 后发国家政治独立后经济仍依附

**成本转嫁理论：** 危机成本向弱势群体转嫁

**危机周期理论：** 经济危机周期性爆发

**世界体系理论：** 核心-边缘-半边缘结构
