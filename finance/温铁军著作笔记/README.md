## 🎯 核心理论框架

### 成本转嫁论

温铁军的核心理论贡献：

> "在历次危机来临时，凡是能向农村直接转嫁危机代价的，城市产业资本就可以实现软着陆，原有体制就得以维持；凡是不能向农村直接转嫁危机代价的，就只能在城市硬着陆，从而导致政府财税、金融乃至整个经济体制的重大变革。"

**中国模式特点**：
- 城乡二元结构下的内向型制度成本转嫁
- 农村"蓄水池"功能吸纳城市危机
- 土地制度弹性提供缓冲空间

### 八次危机概览

| 序号 | 时期 | 类型 | 转嫁方式 |
|-----|------|------|---------|
| 1 | 1958-1960 | 外资依赖型 | 上山下乡 |
| 2 | 1968-1970 | 内源性 | 农村吸纳 |
| 3 | 1974-1976 | 外资依赖型 | 农村吸纳 |
| 4 | 1979-1980 | 内源性 | 农村改革 |
| 5 | 1988-1990 | 内源性 | 体制调整 |
| 6 | 1993-1994 | 内源性 | 分税制改革 |
| 7 | 1997-1998 | 输入型 | 国债投资 |
| 8 | 2008-2009 | 输入型 | 四万亿计划 |

### 主要矛盾诊断

**国内矛盾**：生产过剩 + 消费透支

**国际矛盾**：全球化解体 + 双重资本竞争

## 🔑 核心概念

| 概念 | 定义 |
|-----|------|
| **成本转嫁** | 危机成本从城市向农村的制度性转移 |
| **软着陆** | 通过转嫁实现经济平稳调整 |
| **硬着陆** | 无法转嫁导致的激进体制变革 |
| **蓄水池功能** | 农村吸纳过剩要素的制度功能 |
| **三农问题** | 农业、农村、农民问题的统称 |
| **新质生产力** | 绿色生产力，人与自然和谐共生 |
| **城乡二元结构** | 中国特有的城乡分割制度安排 |
