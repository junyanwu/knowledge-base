# 金融市场基础知识

## 一、金融市场概述

### 1.1 什么是金融市场
金融市场是资金供求双方进行资金融通的场所或机制，包括货币市场、资本市场、外汇市场、黄金市场等。

### 1.2 金融市场分类

#### 按交易期限分类
- **货币市场**: 短期资金市场，交易期限在一年以内
  - 同业拆借市场
  - 票据市场
  - 大额可转让定期存单市场
  - 短期债券市场

- **资本市场**: 长期资金市场，交易期限在一年以上
  - 股票市场
  - 债券市场
  - 基金市场

#### 按交易性质分类
- **发行市场（一级市场）**: 证券首次发行
- **流通市场（二级市场）**: 证券交易流通

#### 按交割方式分类
- **现货市场**: 即期交割
- **期货市场**: 远期交割
- **期权市场**: 权利交易

### 1.3 金融市场参与者
- **个人投资者**: 散户
- **机构投资者**: 基金、保险、券商、银行等
- **企业**: 融资方和投资方
- **政府**: 发行国债、监管市场
- **中央银行**: 货币政策调控

## 二、股票市场

### 2.1 股票基本概念
- **股票**: 公司所有权的凭证
- **股东**: 股票持有者
- **股息**: 公司分红
- **资本利得**: 股价上涨收益

### 2.2 股票分类
- **A股**: 人民币普通股（上海、深圳）
- **B股**: 人民币特种股（外币交易）
- **H股**: 香港上市内地企业
- **N股**: 纽约上市
- **S股**: 新加坡上市

### 2.3 股票指数
- **上证指数**: 上海证券交易所综合指数
- **深证成指**: 深圳证券交易所成份指数
- **创业板指**: 创业板综合指数
- **沪深300**: 沪深两市300只龙头股
- **中证500**: 中小盘代表指数

### 2.4 股票交易规则（A股）
- **交易时间**: 9:30-11:30, 13:00-15:00
- **涨跌幅限制**: 主板10%，创业板/科创板20%
- **交易单位**: 100股为一手
- **T+1制度**: 当日买入次日才能卖出

## 三、债券市场

### 3.1 债券基本概念
- **债券**: 债务凭证，承诺还本付息
- **票面利率**: 年化利息率
- **到期收益率**: 持有到期的年化收益

### 3.2 债券分类
- **国债**: 政府发行，信用最高
- **地方债**: 地方政府发行
- **企业债**: 企业发行
- **可转债**: 可转换为股票的债券
- **金融债**: 金融机构发行

### 3.3 债券收益来源
- **利息收入**: 定期获得的利息
- **资本利得**: 价格波动收益
- **再投资收益**: 利息再投资收益

## 四、基金市场

### 4.1 基金基本概念
- **基金**: 集合理财，专业投资
- **基金份额**: 持有比例
- **净值**: 基金单位价值

### 4.2 基金分类
- **股票型基金**: 80%以上投资股票
- **债券型基金**: 80%以上投资债券
- **混合型基金**: 股债灵活配置
- **货币型基金**: 投资短期货币工具
- **指数型基金**: 跟踪特定指数
- **ETF**: 交易所交易基金

### 4.3 基金费用
- **申购费**: 买入费用（通常1-1.5%）
- **赎回费**: 卖出费用（持有时间越长越低）
- **管理费**: 基金公司收取（年化0.5-2%）
- **托管费**: 银行托管费用

## 五、风险与收益

### 5.1 风险类型
- **市场风险**: 价格波动风险
- **信用风险**: 违约风险
- **流动性风险**: 变现困难
- **操作风险**: 人为失误
- **政策风险**: 政策变化影响

### 5.2 风险管理
- **分散投资**: 不要把鸡蛋放在一个篮子里
- **资产配置**: 股债商品等多元化
- **止损策略**: 设定止损点
- **对冲策略**: 期货期权对冲

### 5.3 收益与风险关系
```
高风险 → 高预期收益
低风险 → 低预期收益
无风险利率 → 国债收益率
```

## 六、投资策略

### 6.1 价值投资
- 寻找被低估的优质公司
- 长期持有
- 关注基本面

### 6.2 成长投资
- 投资高增长公司
- 关注行业前景
- 接受较高估值

### 6.3 技术分析
- 研究价格走势
- 使用技术指标
- 短期交易为主

### 6.4 被动投资
- 购买指数基金
- 低成本
- 长期持有
