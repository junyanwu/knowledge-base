# 技术分析详解

## 一、K线基础

### 1.1 K线构成
- **开盘价**: 交易时段第一笔成交价
- **收盘价**: 交易时段最后一笔成交价
- **最高价**: 交易时段内最高成交价
- **最低价**: 交易时段内最低成交价
- **实体**: 开盘价与收盘价之间的矩形
- **影线**: 最高/最低价与实体之间的线条

### 1.2 K线类型

| 类型 | 特征 | 含义 |
|------|------|------|
| 大阳线 | 实体长，收盘>>开盘 | 强势上涨 |
| 大阴线 | 实体长，收盘<<开盘 | 强势下跌 |
| 十字星 | 实体很小，开盘≈收盘 | 多空平衡 |
| 锤子线 | 下影线长，实体在上 | 底部反转信号 |
| 倒锤子 | 上影线长，实体在下 | 底部反转信号 |
| 流星线 | 上影线长，实体在上 | 顶部反转信号 |
| 上吊线 | 下影线长，实体在上 | 顶部反转信号 |

### 1.3 K线组合形态

**反转形态**
- 早晨之星：底部反转
- 黄昏之星：顶部反转
- 穿头破脚：强势反转
- 乌云盖顶：顶部反转
- 刺穿形态：底部反转

**持续形态**
- 红三兵：上涨持续
- 三只乌鸦：下跌持续
- 上升三法：上涨中继
- 下降三法：下跌中继

## 二、技术指标

### 2.1 趋势类指标

#### 移动平均线 (MA)
```
MA(n) = (P1 + P2 + ... + Pn) / n
```
- **短期**: MA5, MA10
- **中期**: MA20, MA60
- **长期**: MA120, MA250

**均线交叉策略**:
- 金叉：短期均线上穿长期均线 → 买入
- 死叉：短期均线下穿长期均线 → 卖出

#### MACD (指数平滑异同移动平均线)
```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD = (DIF - DEA) × 2
```

**应用规则**:
- DIF上穿DEA → 金叉买入
- DIF下穿DEA → 死叉卖出
- 柱状线由负转正 → 底部信号
- 柱状线由正转负 → 顶部信号

### 2.2 震荡类指标

#### RSI (相对强弱指标)
```
RSI = 100 - 100 / (1 + RS)
RS = 平均涨幅 / 平均跌幅
```

**应用规则**:
- RSI > 70: 超买区，可能回调
- RSI < 30: 超卖区，可能反弹
- RSI背离：价格创新高但RSI不创新高 → 反转信号

#### KDJ (随机指标)
```
RSV = (C - Ln) / (Hn - Ln) × 100
K = SMA(RSV, 3)
D = SMA(K, 3)
J = 3K - 2D
```

**应用规则**:
- K、D值 > 80: 超买
- K、D值 < 20: 超卖
- K线上穿D线: 金叉买入
- K线下穿D线: 死叉卖出

#### 布林带 (Bollinger Bands)
```
中轨 = MA(n)
上轨 = 中轨 + k × σ
下轨 = 中轨 - k × σ
```

**应用规则**:
- 价格触及上轨: 可能回调
- 价格触及下轨: 可能反弹
- 布林带收口: 波动率降低，可能突破
- 布林带开口: 波动率增加，趋势加强

### 2.3 成交量指标

#### OBV (能量潮)
```
今日OBV = 昨日OBV ± 今日成交量
上涨取正，下跌取负
```

**应用规则**:
- OBV上升，价格上升: 健康上涨
- OBV下降，价格上升: 量价背离，警惕
- OBV上升，价格下降: 底部积累

#### 成交量均线
- 5日均量线：短期成交量趋势
- 10日均量线：中期成交量趋势
- 量比：当日成交量 / 5日平均成交量

## 三、图表形态

### 3.1 反转形态

#### 头肩顶/底
```
       头
      /  \
左肩 /    \ 右肩
    /      \
   /        \
  颈线--------
```
- 头肩顶：顶部反转
- 头肩底：底部反转
- 突破颈线确认形态

#### 双顶/双底 (M顶/W底)
```
   /\     /\
  /  \   /  \
 /    \ /    \
-------M-------
```
- 双顶：两个高点基本相等
- 双底：两个低点基本相等
- 突破颈线确认

#### 圆弧顶/底
- 圆弧顶：缓慢形成的顶部
- 圆弧底：缓慢形成的底部
- 成交量逐渐变化

### 3.2 持续形态

#### 三角形
- 上升三角形：高点水平，低点抬高
- 下降三角形：低点水平，高点降低
- 对称三角形：波动收窄，方向待定

#### 旗形
- 上升旗形：下跌中继
- 下降旗形：上涨中继

#### 矩形
- 水平整理区间
- 突破方向决定走势

## 四、技术分析理论

### 4.1 道氏理论
- **趋势定义**: 一系列更高的高点和更高低点
- **趋势分类**: 主要趋势、次要趋势、小型趋势
- **确认原则**: 两个指数必须相互确认

### 4.2 波浪理论
```
上升5浪: 1-2-3-4-5
下降3浪: A-B-C

完整的8浪周期
```

**波浪特征**:
- 第1浪：起步浪，往往不被注意
- 第2浪：调整浪，回撤第1浪的部分
- 第3浪：主升浪，最强、最长
- 第4浪：调整浪，复杂多变
- 第5浪：冲高浪，可能衰竭
- A浪：下跌开始
- B浪：反弹浪
- C浪：主跌浪

### 4.3 江恩理论
- 时间周期分析
- 价格角度线
- 江恩角度：1×1, 1×2, 2×1 等

### 4.4 缠论
- 分型：顶分型、底分型
- 笔：相邻分型之间的连线
- 线段：笔的组合
- 中枢：三段重合区域
- 背驰：力度衰竭的判断

## 五、技术分析应用

### 5.1 入场信号
1. 趋势确认（均线多头排列）
2. 回调结束（支撑位企稳）
3. 指标金叉（MACD/KDJ）
4. 成交量放大
5. 形态突破

### 5.2 出场信号
1. 趋势反转（均线死叉）
2. 阻力位遇阻
3. 指标死叉/背离
4. 成交量萎缩
5. 形态破位

### 5.3 止损设置
- 固定比例止损：-5% 或 -10%
- 支撑位止损：跌破支撑位
- 均线止损：跌破重要均线
- ATR止损：根据波动率设置

### 5.4 风险管理
- 单笔风险：不超过总资金的2%
- 总仓位：不超过70%
- 分散投资：不超过3-5只股票
- 盈亏比：至少1:2
