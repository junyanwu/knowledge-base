# 中国宏观经济追踪

**更新时间：2026年3月2日**

---

## 一、2026年中国GDP增速预测

### 官方目标设定

根据中国宏观经济论坛（CMF）2026年3月研讨会专家共识：

| 指标 | 2025年实际 | 2026年预测目标 |
|------|-----------|---------------|
| 全国GDP增速目标 | 5%左右 | **4.5%—5%区间** |
| 31省份加权平均增速 | 5.3% | 约5%（下调0.3个百分点） |
| GDP总量 | 140万亿元（首次突破） | 预计147万亿元左右 |

### 核心判断

1. **区间目标成共识**：2026年GDP增长目标从2025年的"5%左右"调整为"4.5%—5%区间"，为应对内外部冲击预留政策弹性
2. **经济大省挑大梁**：广东、江苏、山东、浙江等经济大省大多明确了5%以上的增长目标
3. **名义与实际GDP双回升**：综合整治"反内卷"见效、周期性力量筑底，2026年将实现名义与实际GDP增速双回升

### 经济走势判断

- **观点一（前低后高）**："十五五"规划项目超前布局、微观主体资产负债表修复将推动经济逐步回升
- **观点二（前高后低）**：年初"开门红"诉求与后续结构转型需平衡
- **共识**：经济韧性强，长期向好的基本趋势没有改变

## 二、三驾马车表现分析

### 1. 消费：服务化转型加速

#### 2025年消费表现
- 第三产业增加值同比增长5.4%，高于整体GDP增速
- 服务消费成为消费增长新引擎

#### 2026年消费政策方向

**新消费增长点布局：**

| 地区 | 重点培育方向 |
|------|-------------|
| 广东 | 首店经济、低空消费、人工智能+消费融合 |
| 上海 | 首发经济、夜间经济、银发经济 |
| 北京 | 老字号活力激活、赛事经济 |
| 浙江 | 数智消费、悦己消费等新型业态 |

#### 消费修复预期

- **打破"地产弱则消费弱"惯性**：地产调整第五年起"挤出效应"弱化，将提振25-40岁群体消费倾向
- **服务消费潜力挖掘**：成为"两新"政策优化方向
- **消费修复潜力值得期待**

### 2. 投资：精准发力"止跌回稳"

#### 2026年投资目标
- 各地投资目标较2025年下调约2个百分点
- 但均保持正增长，着力扭转去年负增长态势

#### 投资三大发力方向

| 方向 | 具体措施 |
|------|---------|
| **传统基建更新与新型基建** | 北京"开一备三"项目储备机制、上海2550亿元重大工程投资计划 |
| **城市更新** | 京沪等地已公布项目清单，成为重要抓手 |
| **民生类投资** | 学前教育、健康养老等领域补短板力度加大 |

#### 政策性金融工具支持
- 5000亿元新型政策性金融工具已全额投放
- 可撬动约7万亿元固定资产投资
- 为一季度"开门红"奠定基础

### 3. 出口：外部环境复杂严峻

#### 外部环境挑战
- 世界经济复苏步履维艰
- 国际经贸格局深度调整
- 外部环境变乱交织

#### 应对策略
- 扩大制度型开放
- 国内国际双循环相互促进
- 持续成为世界经济增长的最大贡献者和最强稳定锚

## 三、货币政策和财政政策最新动向

### 货币政策：适度宽松

#### 政策定调
2026年2月27日中央政治局会议明确：**实施适度宽松的货币政策**

#### 具体措施预期

| 政策工具 | 预期操作 |
|---------|---------|
| 降准降息 | 适时推进，政策利率预计10bps幅度调降1-2次 |
| 5年期以上LPR | 重点下调，破解企业投资激励困境 |
| 银行负债成本 | 降低存款利率，优化负债结构 |

#### 银行业净息差企稳
- 2025年四季度商业银行净息差：**1.42%**（连续三个季度持平）
- 各类银行净息差表现：

| 银行类型 | 净息差 | 变化 |
|---------|-------|------|
| 大型商业银行 | 1.30% | -1bp |
| 股份制银行 | 1.56% | 持平 |
| 城商行 | 1.37% | 持平 |
| 农商行 | 1.60% | +2bps |

- 2026年净息差降幅预计收窄至约4个基点，为2022年以来首次年度息差降幅在低个位数水平

### 财政政策：更加积极

#### 政策定调
**继续实施更加积极的财政政策，强化改革举措与宏观政策协同**

#### 财政政策要点

| 方面 | 具体措施 |
|------|---------|
| 支出强度 | 保障财政支出强度 |
| 结构优化 | 优化财政支出结构 |
| 转移支付 | 加大对地方转移支付 |
| 赤字率 | 明确赤字率与新增债务规模 |

#### 地方化债方案
- "6+4+2"万亿元地方化债方案持续推进
- 2026年是6万亿隐性债务置换收官年
- 优化化债方式，给予地方政府更多自主权

## 四、房地产市场走势

### 市场现状：筑底企稳

#### 调整周期回顾
- 房地产市场历经**四年深度调整**（2022-2025）
- 房地产开发投资、商品房销售面积较2021年峰值分别下降超40%和50%
- 新开工面积回落至2004年水平

#### 2026年市场特征

| 特征 | 描述 |
|------|-----|
| **底部特征显现** | 量价指标已接近合理区间，调整空间基本释放 |
| **分化格局** | 总量下行态势趋缓，结构差异成为市场核心矛盾 |
| **政策底确立** | 从"救市"转向"稳市" |

### 政策转向：系统性稳市

#### 政策目标升级
- **原目标**：推动房地产市场止跌回稳
- **新目标**：着力稳定房地产市场
- **核心逻辑**：供需协同、精准施策

#### 四大政策支柱

| 支柱 | 内容 |
|------|-----|
| **需求端** | 延续宽松导向，公积金改革全面深化 |
| **供给端** | 优化土地供应，支持合理融资需求 |
| **金融支持** | 稳定房地产开发贷款，支持优质房企 |
| **长效机制** | 加快建立房地产发展新模式 |

### 市场展望

#### 标普预测
- 2026年全国商品房销售额预计约7.6万亿元
- 同比下降约10%，降幅显著收窄

#### 2026年回暖信号
- 核心城市成交量稳步回升
- 二手房价格止跌企稳
- 房企风险逐步出清
- 市场信心重塑

#### 关键判断
- 楼市单边下行周期已然终结
- 理性回暖新征程正式开启
- 这不是短期反弹，而是周期级别的趋势转向

## 五、经济转型进展

### 新质生产力培育

#### 科技创新投入
- 2025年全社会R&D经费投入同比增长8.1%
- R&D经费与GDP之比达**2.80%**
- **首次超过OECD国家平均水平**

#### 前沿领域突破
- 人工智能
- 量子科技
- 脑机接口
- 商业航天

#### 区域差异化布局

| 区域 | 发展方向 |
|------|---------|
| 东部沿海 | 依托科技与人才优势布局未来产业 |
| 中西部 | 结合资源禀赋深耕产业链上游 |
| 京津冀/长三角/粤港澳大湾区 | 国际科技创新中心建设 |

#### 风险警示
- 避免新兴产业"一哄而上"
- 聚焦创新生态构建
- 因地制宜挖掘要素优势

### 产业升级方向

#### 传统产业
- 加速转型
- 设备更新改造

#### 新兴产业
- 蓬勃发展
- 人工智能+消费融合
- 低空经济

#### 未来产业
- 超前布局
- 基础研究投入加大

### 经济结构优化

#### 2025年产业结构

| 产业 | 增加值 | 同比增长 |
|------|-------|---------|
| 第一产业 | - | 3.9% |
| 第二产业 | - | 4.5% |
| 第三产业 | - | 5.4% |

#### 结构特点
- 第三产业增速领先，服务业发展活力持续释放
- 科技创新推动产业创新
- 产业升级构筑新优势

## 六、核心结论与展望

### 稳：发展底盘牢固
- GDP总量突破140万亿元新台阶
- 5%增长对应超5万亿元经济增量
- 经济基础稳、优势多、韧性强、潜能大

### 进：转型动能增强
- 新质生产力不断发展壮大
- 服务消费成为新增长点
- 科技创新投入创历史新高

### 韧：抗压能力凸显
- 在复杂国际环境中应变破局
- 政策灵活性与适应性强
- 双循环格局持续完善

### 2026年关键词

| 关键词 | 内涵 |
|--------|------|
| **务实增长** | 区间目标设定，预留政策弹性 |
| **精准施策** | 供需协同，差异化发展 |
| **稳健转型** | 新质生产力培育与传统产业升级并重 |
| **风险化解** | 房地产止跌回稳，地方债务有序化解 |
