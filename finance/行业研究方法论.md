# 行业研究方法论

## 一、行业分析框架

### 1.1 行业生命周期

```
  规模
    ↑
    |        成熟期
    |       /¯¯¯¯¯¯\
    |      /        \
    |     / 成长期   \
    |    /            \
    |   /              \
    |__/________________\____→ 时间
       导入期        衰退期
```

| 阶段 | 特征 | 投资策略 |
|------|------|---------|
| 导入期 | 需求小、竞争少、高风险 | 风险投资 |
| 成长期 | 需求快速增长、竞争加剧 | 成长股投资 |
| 成熟期 | 需求稳定、竞争格局稳定 | 价值投资 |
| 衰退期 | 需求下降、竞争减少 | 回避或转型 |

### 1.2 行业竞争格局

#### 波特五力模型
```
        潜在进入者威胁
             ↓
    ┌─────────────────┐
    │                 │
替代品威胁 ←→ 行业竞争 ←→ 供应商议价能力
    │                 │
    └─────────────────┘
             ↓
       买方议价能力
```

**分析要点**:
1. **供应商议价能力**: 集中度、替代性、转换成本
2. **买方议价能力**: 集中度、价格敏感度、替代选择
3. **潜在进入者**: 进入壁垒、规模经济、品牌壁垒
4. **替代品威胁**: 替代品性价比、转换成本
5. **行业竞争**: 竞争者数量、产品差异化、退出壁垒

### 1.3 行业分类

#### 按周期性分类
- **周期性行业**: 金融、地产、汽车、钢铁、有色
- **防御性行业**: 食品饮料、医药、公用事业
- **成长性行业**: 科技、新能源、生物医药

#### 按产业链位置分类
- **上游**: 原材料、资源开采
- **中游**: 制造、加工
- **下游**: 消费、服务

## 二、行业分析方法

### 2.1 供需分析

#### 需求分析
- **市场规模**: 当前规模、增长潜力
- **需求驱动**: 人口、收入、政策、技术
- **需求弹性**: 价格弹性、收入弹性
- **替代品**: 替代品价格、性能

#### 供给分析
- **产能**: 当前产能、产能利用率
- **产能扩张**: 新增产能、扩张周期
- **成本结构**: 固定成本、变动成本
- **竞争格局**: 集中度、市场份额

#### 供需平衡
- 供不应求 → 价格上涨 → 企业盈利改善
- 供过于求 → 价格下跌 → 企业盈利恶化

### 2.2 成本分析

#### 成本结构
```
总成本 = 固定成本 + 变动成本
       = 原材料 + 人工 + 制造费用 + 销售+管理+财务费用
```

#### 成本曲线
- **规模经济**: 平均成本随产量增加而下降
- **学习曲线**: 成本随经验积累而下降
- **边际成本**: 多生产一单位的成本

#### 成本比较
- 与行业平均比较
- 与主要竞争对手比较
- 成本变化趋势

### 2.3 盈利分析

#### 盈利驱动因素
```
利润 = 收入 - 成本
     = 价格 × 销量 - 成本
```

#### 关键指标
- **毛利率**: (收入-成本)/收入
- **净利率**: 净利润/收入
- **ROE**: 净利润/净资产
- **ROA**: 净利润/总资产

#### 盈利周期
- 行业盈利有周期性
- 盈利高点往往是投资风险点
- 盈利低点往往是投资机会点

## 三、重点行业分析

### 3.1 银行业

#### 关键指标
- **净息差 (NIM)**: 利息收入-利息支出 / 生息资产
- **不良贷款率 (NPL)**: 不良贷款 / 总贷款
- **资本充足率**: 资本 / 风险加权资产
- **拨备覆盖率**: 拨备 / 不良贷款

#### 分析要点
1. 宏观经济对资产质量的影响
2. 利率政策对净息差的影响
3. 监管政策变化
4. 金融科技冲击

### 3.2 房地产业

#### 关键指标
- **销售面积**: 新房、二手房成交量
- **销售金额**: 成交金额
- **土地购置**: 土地成交面积、金额
- **新开工/竣工**: 开发投资先行指标
- **库存**: 待售面积

#### 分析要点
1. 政策调控（限购、限贷、限价）
2. 资金来源（开发贷、按揭、信托）
3. 土地供应和地价
4. 人口结构和城镇化

### 3.3 汽车行业

#### 关键指标
- **销量**: 月度、年度销量
- **库存**: 库存深度
- **产能利用率**: 产量/产能
- **价格**: 终端价格、折扣率

#### 分析要点
1. 政策影响（购置税、补贴）
2. 新能源转型
3. 智能化趋势
4. 出口市场

### 3.4 医药行业

#### 关键指标
- **研发投入**: R&D费用率
- **在研管线**: 产品数量、阶段
- **专利到期**: 仿制药威胁
- **带量采购**: 政策影响

#### 分析要点
1. 创新药研发能力
2. 仿制药竞争格局
3. 政策变化（集采、医保）
4. 人口老龄化需求

### 3.5 科技行业

#### 关键指标
- **收入增长**: 同比、环比
- **毛利率**: 产品竞争力
- **研发投入**: R&D费用率
- **用户数据**: MAU、ARPU

#### 分析要点
1. 技术迭代速度
2. 商业模式创新
3. 监管政策影响
4. 国际竞争格局

## 四、行业研究工具

### 4.1 数据来源

#### 宏观数据
- 国家统计局: GDP、CPI、PMI等
- 央行: 货币供应、信贷数据
- 海关: 进出口数据

#### 行业数据
- 行业协会: 行业统计
- 政府部门: 专项统计
- 第三方机构: 咨询报告

#### 公司数据
- 上市公司公告
- 财务报表
- 投资者交流

### 4.2 研究方法

#### 自上而下
1. 宏观经济分析
2. 行业周期判断
3. 公司选择

#### 自下而上
1. 公司分析
2. 行业比较
3. 宏观验证

#### 比较分析
- 横向比较: 同行业公司对比
- 纵向比较: 历史数据对比
- 国际比较: 与海外对比

### 4.3 调研方法

#### 实地调研
- 公司拜访
- 工厂参观
- 经销商访谈

#### 专家访谈
- 行业专家
- 产业链上下游
- 竞争对手

#### 数据验证
- 多数据源交叉验证
- 财务数据勾稽关系
- 逻辑推理

## 五、行业投资策略

### 5.1 行业轮动

#### 周期轮动
```
经济复苏 → 金融、可选消费
经济繁荣 → 工业、原材料
经济见顶 → 能源、必需消费
经济衰退 → 医药、公用事业
经济触底 → 科技、成长股
```

#### 政策驱动
- 政策支持行业：超配
- 政策限制行业：低配
- 政策中性行业：标配

### 5.2 行业配置

#### 核心配置
- 行业龙头
- 稳定增长行业
- 高股息行业

#### 卫星配置
- 成长性行业
- 周期性行业
- 主题投资

### 5.3 风险控制

#### 行业分散
- 单一行业配置不超过30%
- 相关性行业合计不超过50%

#### 动态调整
- 根据行业景气度调整
- 根据估值水平调整
- 根据风险偏好调整
