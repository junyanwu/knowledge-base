## 1. 核心概念

### 1.1 货币的本质与职能

#### 货币的本质
- **定义**：货币是固定充当一般等价物的特殊商品
- **本质特征**：
  - 一般等价物：能够表现一切商品价值
  - 特殊商品：本身具有价值（实物货币）或代表价值（信用货币）
  - 社会属性：体现商品生产者之间的社会经济关系

#### 货币的五大职能
1. **价值尺度**
   - 衡量商品价值的标准
   - 价格是价值的货币表现
   - 只需要观念上的货币

2. **流通手段**
   - 交易媒介功能
   - 需要现实的货币
   - 货币流通规律：M = PQ/V

3. **贮藏手段**
   - 财富保存功能
   - 需要现实的、足值的货币
   - 金属货币可自发调节货币流通量

4. **支付手段**
   - 延期支付功能
   - 债权债务清算
   - 信用货币时代的主要职能

5. **世界货币**
   - 国际贸易结算
   - 国际储备资产
   - 主要形式：美元、欧元、人民币（SDR篮子货币）

### 1.2 货币层次划分（中国央行标准）

| 层次 | 定义 | 特点 |
|------|------|------|
| **M0** | 流通中现金 | 流动性最强 |
| **M1** | M0 + 企业活期存款 | 狭义货币，反映现实购买力 |
| **M2** | M1 + 准货币（定期存款、储蓄存款等） | 广义货币，反映潜在购买力 |

**重要指标**：
- M1-M2剪刀差：反映经济活跃度和通胀预期
- M2/GDP：货币化程度指标
- 货币乘数：m = M/B（M为货币供应量，B为基础货币）

### 1.3 商业银行的资产负债业务

#### 负债业务（资金来源）
1. **存款业务**
   - 活期存款：无息或低息，流动性强
   - 定期存款：较高利率，期限固定
   - 储蓄存款：个人存款，受存款保险保护

2. **借款业务**
   - 同业拆借：银行间短期融资
   - 向央行借款：再贴现、SLF、MLF等
   - 发行金融债券：长期稳定资金来源

#### 资产业务（资金运用）
1. **贷款业务**
   - 企业贷款：流动资金贷款、固定资产贷款
   - 个人贷款：住房按揭、消费贷款、经营贷款
   - 信贷风险：信用风险、市场风险、操作风险

2. **证券投资**
   - 国债投资：低风险、流动性好
   - 金融债券投资：收益略高
   - 企业债券投资：收益较高，风险较大

3. **现金资产**
   - 库存现金
   - 存放央行款项（法定存款准备金、超额准备金）
   - 存放同业款项

#### 资产负债管理原则
- **安全性**：风险控制优先
- **流动性**：保证支付能力
- **盈利性**：追求合理收益
- 三性平衡：风险-收益权衡

## 2. 货币政策工具详解

### 2.1 传统三大政策工具

#### 1. 存款准备金制度

**定义**：商业银行按存款比例缴存央行的资金

**类型**：
- 法定存款准备金：强制缴存，央行规定比例
- 超额存款准备金：自愿存放，用于清算

**政策效应**：
```
准备金率上调 → 银行可贷资金减少 → 货币乘数下降 → 货币供应收缩
准备金率下调 → 银行可贷资金增加 → 货币乘数上升 → 货币供应扩张
```

**2024-2025年中国实践**：
- 多次降准释放长期流动性
- 定向降准支持小微企业、三农
- 存款准备金率动态调整机制

**量化分析**：
- 货币乘数 m = (1+c)/(r+e+c)
  - c: 现金漏损率
  - r: 法定准备金率
  - e: 超额准备金率

#### 2. 再贴现政策

**定义**：央行通过调整再贴现率影响商业银行融资成本

**政策机制**：
```
再贴现率 ↓ → 商业银行融资成本 ↓ → 放贷意愿 ↑ → 货币扩张
再贴现率 ↑ → 商业银行融资成本 ↑ → 放贷意愿 ↓ → 货币收缩
```

**特点**：
- 被动性：需要银行主动申请
- 告示效应：传递政策意图
- 最后贷款人：危机时期的重要工具

#### 3. 公开市场操作（OMO）

**定义**：央行在公开市场买卖有价证券

**操作类型**：
| 操作类型 | 方向 | 目的 |
|----------|------|------|
| 正回购 | 央行卖出证券、约定买回 | 回笼流动性 |
| 逆回购 | 央行买入证券、约定卖回 | 投放流动性 |
| 买断式买入 | 永久买入 | 投放长期流动性 |
| 卖断式卖出 | 永久卖出 | 回笼长期流动性 |

**中国央行OMO特点**：
- 以7天逆回购为主力操作工具
- 每日操作常态化
- 操作利率（7天逆回购利率）成为政策利率
- 2024年引入买断式逆回购工具

### 2.2 结构性货币政策工具

#### 常备借贷便利（SLF）
- **期限**：隔夜、7天、1个月
- **特点**：由金融机构主动发起，满足短期流动性需求
- **利率**：形成利率走廊上限

#### 中期借贷便利（MLF）
- **期限**：3个月、6个月、1年
- **特点**：央行主动操作，投放中期基础货币
- **利率**：MLF利率是重要的政策利率
- **操作方式**：合格银行以国债、央行票据等优质债券作为抵押

#### 定向中期借贷便利（TMLF）
- **期限**：1年（可续作两次，实际3年）
- **特点**：定向支持小微、民营企业
- **利率**：比MLF低15个基点

#### 抵押补充贷款（PSL）
- **特点**：支持棚改、地下管廊、重大水利工程
- **期限**：长期资金
- **利率**：较长期限的政策利率

#### 支农、支小再贷款
- **目的**：定向支持涉农领域和小微企业
- **利率**：优惠利率

### 2.3 利率走廊机制

```
                    SLF利率（利率走廊上限）
                          ↓
    市场利率（DR007） ←→ 政策利率（7天逆回购利率）
                          ↑
                    超额准备金利率（利率走廊下限）
```

**机制原理**：
- 市场利率在走廊内波动
- SLF利率为金融机构向央行借款的上限
- 超额准备金利率为闲置资金的保底收益
- 政策利率引导市场利率中枢

## 3. 利率传导机制

### 3.1 利率体系结构

#### 中国利率体系层次
```
央行政策利率
    ├── OMO利率（短期政策利率）
    ├── MLF利率（中期政策利率）
    └── 存款准备金利率
    
金融市场利率
    ├── 银行间拆借利率（SHIBOR、IBO）
    ├── 债券市场利率（国债、信用债）
    ├── 回购利率（DR、R系列）
    └── 同业存单利率
    
商业银行利率
    ├── 贷款市场报价利率（LPR）
    ├── 存款利率
    └── 贷款利率
```

### 3.2 利率传导链条

```
央行操作 → 政策利率 → 市场利率 → 银行负债成本 → 贷款利率 → 实体经济
    ↓
公开市场操作
调整MLF利率
    ↓
货币市场利率变动
（DR007、SHIBOR）
    ↓
债券市场利率变动
（国债收益率曲线）
    ↓
银行负债成本变化
（存款利率、同业负债）
    ↓
贷款定价变化
（LPR调整）
    ↓
企业融资成本变化
投资消费决策变化
```

### 3.3 利率期限结构

#### 收益率曲线形态
| 形态 | 描述 | 经济含义 |
|------|------|----------|
| 正常（上升） | 长期利率 > 短期利率 | 经济增长预期 |
| 平坦 | 长短期利率接近 | 经济增长放缓 |
| 倒挂 | 短期利率 > 长期利率 | 衰退预警信号 |
| 驼峰 | 中期利率最高 | 不确定性增加 |

#### 利率期限结构理论

1. **预期理论**
   - 长期利率 = 预期未来短期利率的几何平均
   - 公式：(1+rₙ)ⁿ = (1+r₁)(1+r₂)ⁿ⁻¹...(1+rₙ)
   - 含义：收益率曲线反映未来利率预期

2. **流动性溢价理论**
   - 长期债券需要流动性溢价补偿
   - 公式：rₙ = E(rₙ) + LPₙ
   - 含义：上升曲线为正常形态

3. **市场分割理论**
   - 不同期限市场相互独立
   - 供需决定各期限利率
   - 含义：央行可针对性影响特定期限

### 3.4 LPR改革与传导

#### LPR机制
- **构成**：MLF利率 + 银行加点
- **期限**：1年期、5年期以上
- **报价行**：18家商业银行
- **发布频率**：每月20日

#### LPR传导效应
```
MLF利率下调
    ↓
LPR报价下行
    ↓
贷款定价下限下移
    ↓
存量贷款重定价（一般贷款利率下降）
    ↓
实体经济融资成本降低
```

**5年期LPR特殊意义**：
- 与房贷利率挂钩
- 房地产调控的重要工具
- 存量房贷利率调整机制

```python
import akshare as ak
import pandas as pd
from datetime import datetime

class MonetaryDataTracker:
    def __init__(self):
        self.data_dir = './data/monetary/'
    
    def fetch_shibor(self, start_date, end_date):
        """获取SHIBOR利率"""
        df = ak.rate_interbank(market="shibor", 
                               symbol=" Shibor利率", 
                               start_date=start_date,
                               end_date=end_date)
        return df
    
    def fetch_money_supply(self):
        """获取货币供应量数据"""
        df = ak.macro_china_m2()
        return df
    
    def fetch_social_financing(self):
        """获取社会融资规模数据"""
        df = ak.macro_china_shrzgm()
        return df
    
    def fetch_lpr(self):
        """获取LPR报价数据"""
        df = ak.macro_china_lpr()
        return df
    
    def fetch_bond_yield(self):
        """获取国债收益率曲线"""
        df = ak.bond_china_yield(start_date=datetime.now().strftime('%Y%m%d'))
        return df
    
    def calculate_term_spread(self, yield_data):
        """计算期限利差"""
        spread_10y_2y = yield_data['10年期'] - yield_data['2年期']
        spread_10y_1y = yield_data['10年期'] - yield_data['1年期']
        return {'10Y-2Y': spread_10y_2y, '10Y-1Y': spread_10y_1y}
    
    def monitor_policy_rate(self):
        """监控政策利率变化"""
        omo_rate = self.get_omo_rate()
        mlf_rate = self.get_mlf_rate()
        lpr_1y = self.get_lpr('1Y')
        lpr_5y = self.get_lpr('5Y')
        
        return {
            'OMO_7D': omo_rate,
            'MLF_1Y': mlf_rate,
            'LPR_1Y': lpr_1y,
            'LPR_5Y': lpr_5y,
            'spread_LPR5Y_LPR1Y': lpr_5y - lpr_1y
        }
```

## 附录：重要公式汇总

### 货币乘数
```
简单乘数：m = 1/r
完整乘数：m = (1+c)/(r+e+c)
其中：c=现金比率，r=法定准备金率，e=超额准备金率
```

### 货币需求函数（弗里德曼）
```
Md/P = f(Y, W, rm, rb, re, 1/P·dP/dt, u)
其中：Y=恒久收入，W=财富比例，rm=货币预期收益率，
      rb=债券预期收益率，re=股票预期收益率
```

### 费雪效应
```
名义利率 = 实际利率 + 预期通胀率
i = r + πe
```

### 泰勒规则
```
i = r* + π + 0.5(π - π*) + 0.5(y - y*)
其中：i=名义利率，r*=均衡实际利率，π=通胀率，
      π*=目标通胀率，y=实际产出，y*=潜在产出
```
