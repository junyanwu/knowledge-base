
## 本地模型使用策略

### 适合的任务 ✅
- 知识总结与归纳
- 代码生成与优化
- 策略框架设计
- 文档整理

### 局限性与应对 ⚠️
| 局限性 | 应对方法 |
|--------|----------|
| 速度较慢 (1-2 分钟/次) | 提前准备好问题，批量生成 |
| 知识截止 (2024 年前) | 结合网络搜索补充最新信息 |
| 复杂推理有限 | 分解为多个简单问题 |
| 代码可能出错 | 必须人工审查和测试 |

### 最佳实践
```
1. 问题具体化：不要问"怎么优化策略"，问"如何优化 Z-Score 阈值参数"
2. 分步进行：复杂任务分解为多个小问题
3. 交叉验证：重要信息多渠道确认
4. 实践检验：生成的代码必须回测验证
```
