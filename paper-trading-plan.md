# 实盘模拟交易方案

## 一、模拟交易方案设计

### 1.1 模拟环境配置

#### 数据源
- **行情数据**：使用实时数据（延迟<5秒）
- **历史数据**：至少3年历史数据用于回测验证
- **数据质量**：包含Tick级别数据，支持复权处理

#### 交易规则
- **初始资金**：建议使用与计划实盘等额的虚拟资金（如10万、50万）
- **交易时间**：与真实市场完全一致（A股: 9:30-11:30, 13:00-15:00）
- **手续费模拟**：
  - A股：万分之2.5佣金 + 印花税千分之1（卖出）
  - ETF：万分之2佣金，无印花税
  - 滑点模型：按成交额0.1%-0.3%模拟冲击成本

#### 订单执行
- **限价单**：按指定价格或更优价格成交
- **市价单**：按当前最优价格成交 + 滑点
- **成交量限制**：单笔不超过日成交量5%
- **成交延迟**：模拟订单排队时间（大单拆分执行）

### 1.2 模拟周期规划

| 阶段 | 时长 | 目标 | 重点验证 |
|------|------|------|----------|
| 第一阶段 | 1-2周 | 熟悉系统 | 交易流程、风控逻辑 |
| 第二阶段 | 1-2月 | 策略验证 | 收益稳定性、回撤控制 |
| 第三阶段 | 2-3月 | 压力测试 | 极端行情、异常情况处理 |

### 1.3 交易日志规范

每次交易需记录：

- 持仓明细

- 策略执行偏差
- 市场环境描述
- 心理状态评分（1-10）

## 二、评估指标体系

### 2.1 收益类指标

| 指标 | 计算公式 | 目标值 | 说明 |
|------|----------|--------|------|
| 累计收益率 | (期末-期初)/期初 | >5% (季度) | 基础收益指标 |
| 年化收益率 | 累计收益 × (365/天数) | >15% | 标准化比较 |
| 超额收益 | 策略收益 - 基准收益 | >0 | 相对表现 |
| 收益波动比 | 年化收益/年化波动率 | >0.5 | 风险调整收益 |

### 2.2 风险类指标

| 指标 | 计算公式 | 阈值 | 说明 |
|------|----------|------|------|
| 最大回撤 | max(Peak - Valley)/Peak | <15% | 极端亏损幅度 |
| 最大回撤时长 | 从回撤开始到恢复天数 | <60天 | 恢复能力 |
| 下行波动率 | σ(负收益) | <15% | 下行风险 |
| VaR (95%) | 历史分位数法 | <3% (日) | 在险价值 |

### 2.3 策略质量指标

| 指标 | 计算公式 | 目标值 | 说明 |
|------|----------|--------|------|
| 胜率 | 盈利次数/总次数 | >55% | 交易成功率 |
| 盈亏比 | 平均盈利/平均亏损 | >1.5 | 单笔收益质量 |
| 夏普比率 | (Rp-Rf)/σp | >1.0 | 风险调整收益 |
| Calmar比率 | 年化收益/最大回撤 | >1.5 | 回撤调整收益 |
| Sortino比率 | (Rp-Rf)/σd | >1.5 | 下行风险调整 |

### 2.4 执行效率指标

| 指标 | 计算方式 | 目标值 | 说明 |
|------|----------|--------|------|
| 信号执行率 | 实际执行/信号总数 | >95% | 系统可靠性 |
| 执行偏差 | 实际价格-信号价格 | <0.5% | 执行质量 |
| 成交率 | 成交量/委托量 | >98% | 流动性适应 |
| 平均成交时间 | 委托到成交时间 | <30秒 | 执行效率 |

### 2.5 综合评分模型

```
总分 = Σ(指标得分 × 权重)

评分维度：
- 收益表现 (30分)：累计收益率、年化收益率、超额收益
- 风险控制 (25分)：最大回撤、下行波动率、VaR
- 策略质量 (25分)：胜率、盈亏比、夏普比率
- 执行效率 (20分)：信号执行率、执行偏差、成交率

评级标准：
- 优秀：≥85分，可直接转实盘
- 良好：70-84分，小资金试运行
- 一般：60-69分，继续优化
- 较差：<60分，暂停转实盘
```

## 三、转实盘检查清单

### 3.1 时间条件（必须满足）

- [ ] **模拟时长 ≥ 3个月**（至少60个交易日）
- [ ] **经历完整市场周期**（包含上涨、下跌、震荡行情）
- [ ] **经历极端事件**（至少1次市场大幅波动，如单日涨跌>3%）

### 3.2 收益条件（至少满足3条）

- [ ] 累计收益率 > 5%（季度）
- [ ] 年化收益率 > 15%
- [ ] 相对基准超额收益 > 3%
- [ ] 夏普比率 > 1.0
- [ ] 连续盈利月数 ≥ 2个月

### 3.3 风险条件（必须全部满足）

- [ ] 最大回撤 < 15%
- [ ] 单日最大亏损 < 3%
- [ ] 连续亏损天数 < 10天
- [ ] 无系统性风险事件（如熔断、系统故障导致的异常交易）

### 3.4 策略条件（至少满足4条）

- [ ] 胜率 ≥ 55%
- [ ] 盈亏比 ≥ 1.5
- [ ] 交易次数 ≥ 30次（样本足够）
- [ ] 信号执行率 ≥ 95%
- [ ] 策略逻辑稳定（无频繁修改参数）
- [ ] 回测与实盘差异 < 20%

### 3.5 心理准备（自我评估）

- [ ] 能够接受连续亏损（测试：模拟账户连续亏损5天后仍能冷静执行）
- [ ] 理解策略风险点，并能解释每次亏损原因
- [ ] 制定止损计划并严格执行
- [ ] 准备好应急预案（如系统故障、数据异常）

### 3.6 技术准备（必须完成）

- [ ] 交易系统稳定性测试通过（连续运行30天无崩溃）
- [ ] 数据源备用方案就绪（至少2个数据源）
- [ ] 订单执行监控告警配置完成
- [ ] 风控系统测试通过（触及止损线自动平仓）
- [ ] 日志系统完善（所有交易可追溯）

### 3.7 资金管理准备

- [ ] 确定实盘资金规模（建议初始≤模拟资金的50%）
- [ ] 设置仓位限制（单只≤20%，总仓位≤80%）
- [ ] 设置止损线（总资金回撤≥10%暂停交易）
- [ ] 准备追加资金计划（可选）

### 3.8 合规与文档

- [ ] 策略说明书完成
- [ ] 风险揭示书完成
- [ ] 交易记录完整保存
- [ ] 合规审查通过（如适用）

## 四、分阶段转实盘路径

### 4.1 第一阶段：小额试运行（1个月）

**资金**：计划资金的10-20%
**目标**：
- 验证实盘环境与模拟的一致性
- 检验心理压力下的执行力
- 测试真实滑点和手续费

**通过标准**：
- 执行偏差 < 1%
- 心理状态稳定
- 无操作失误

### 4.2 第二阶段：逐步加仓（2-3个月）

**资金**：逐步提升至计划资金的50-70%
**目标**：
- 验证策略在更大资金下的表现
- 优化仓位管理
- 积累实盘经验

**通过标准**：
- 各项指标与模拟期一致
- 最大回撤在可控范围内

### 4.3 第三阶段：满仓运行

**资金**：计划资金的100%
**条件**：
- 前两个阶段全部达标
- 策略稳定运行≥6个月
- 持续监控和优化

## 五、风险提示

### 5.1 模拟与实盘的差异

| 维度 | 模拟环境 | 实盘环境 | 差异影响 |
|------|----------|----------|----------|
| 心理压力 | 无真实损失 | 真实资金风险 | 可能导致执行力下降 |
| 流动性 | 理想成交 | 可能无法成交 | 影响大单执行 |
| 滑点 | 估算值 | 真实市场 | 成本可能更高 |
| 系统稳定性 | 模拟系统 | 真实交易系统 | 可能有技术故障 |

### 5.2 注意事项

1. **不要因为模拟成功就过度自信** - 实盘会有不可预见的风险
2. **不要跳过小资金试运行阶段** - 这是验证心理承受力的关键
3. **不要频繁修改策略参数** - 应保持策略一致性
4. **不要忽视止损** - 模拟期养成的好习惯在实盘必须坚持
5. **持续记录和复盘** - 即使转实盘后也要保持模拟期的纪律

## 八、总结

实盘模拟是量化交易从理论走向实践的必经之路。通过至少3个月的严格模拟，建立完整的评估体系，确保策略在收益、风险、执行等多个维度都达到要求，才能平稳过渡到实盘交易。

**核心原则**：
1. 模拟即实盘 - 用真实资金的心态对待模拟
2. 数据说话 - 用量化指标评估，不凭感觉
3. 循序渐进 - 分阶段、小资金逐步过渡
4. 风险第一 - 宁可错过机会，不可冒险失控
