# ETF策略实盘模拟方案 V2.0

## 一、方案概述

### 1.1 设计目标

本方案旨在为ETF量化策略提供从回测到实盘的完整模拟验证流程，确保策略在真实市场环境下的稳定性和可靠性。

### 1.2 核心原则

| 原则 | 说明 |
|------|------|
| **模拟即实盘** | 用真实资金的心态对待模拟交易 |
| **数据说话** | 用量化指标评估，不凭主观感觉 |
| **循序渐进** | 分阶段、小资金逐步过渡 |
| **风险第一** | 宁可错过机会，不可冒险失控 |
| **持续迭代** | 根据模拟结果不断优化策略 |

### 1.3 适用策略

本方案适用于以下ETF策略：
- CTA趋势跟踪策略（MA金叉+成交量确认）
- 智能定投策略（估值分位调整）
- 行业轮动策略（动量排名）
- 风险平价策略（波动率均衡）

## 六、风险控制机制

### 6.1 三级预警系统

```yaml
# 黄色预警（警戒）
触发条件:
  - VIX > 25
  - 偏离度 > 2.0σ
  - 仓位偏离 > 10%
  - 最大回撤 > 12%
应对措施:
  - 密切监控市场
  - 准备调整仓位
  - 暂停新增开仓

# 橙色预警（严重）
触发条件:
  - VIX > 30
  - 偏离度 > 2.5σ
  - 仓位偏离 > 15%
  - 最大回撤 > 15%
应对措施:
  - 降低仓位至50%
  - 启动止损机制
  - 执行对冲操作

# 红色预警（危险）
触发条件:
  - VIX > 40
  - 偏离度 > 3.0σ
  - 仓位偏离 > 20%
  - 最大回撤 > 20%
应对措施:
  - 清仓至20%以下
  - 启动应急基金
  - 全面对冲风险
```

### 6.2 止损机制


## 八、心理建设

### 8.1 模拟与实盘的心理差异

| 维度 | 模拟交易 | 实盘交易 | 应对策略 |
|------|----------|----------|----------|
| 亏损感受 | 数字变化 | 真实损失 | 小额试运行建立耐受 |
| 决策压力 | 无压力 | 高压力 | 严格按信号执行 |
| 情绪波动 | 平静 | 焦虑/贪婪 | 记录情绪日记 |
| 执行纪律 | 容易坚持 | 容易动摇 | 自动化交易 |

### 8.2 心理压力测试清单

- [ ] 能否接受连续5天亏损？
- [ ] 能否在亏损时坚持信号？
- [ ] 能否在盈利时不贪婪？
- [ ] 能否在市场恐慌时保持冷静？
- [ ] 能否严格执行止损？

### 8.3 情绪记录

每日记录情绪评分（1-10分）：
- 1-3分：极度焦虑/恐惧
- 4-6分：正常波动
- 7-10分：过度自信/贪婪

**警戒线**：连续3天情绪评分<4或>8，应暂停交易并反思。

## 九、常见问题与解决方案

### 9.1 模拟期常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 执行偏差过大 | 滑点模型不准 | 调高滑点参数 |
| 回撤过大 | 风控不严格 | 降低仓位、收紧止损 |
| 胜率过低 | 策略失效 | 检查信号逻辑、优化参数 |
| 样本不足 | 时间太短 | 延长模拟期 |
| 情绪干扰 | 心理建设不足 | 记录情绪日记、严格执行 |

### 9.2 转实盘常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 实盘表现差于模拟 | 心理压力、滑点差异 | 小额试运行、调整预期 |
| 无法执行止损 | 贪婪/恐惧 | 自动化止损、分步执行 |
| 过度交易 | 追涨杀跌 | 严格遵守信号、记录反思 |
| 系统故障 | 技术、网络问题 | 备用方案、应急预案 |
