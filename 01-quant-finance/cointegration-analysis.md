# Cointegration Analysis

## Key Concepts:
- **Cointegration** measures if a linear combination of non-stationary time series is stationary
- **Engle-Granger method** for two-variable case
- **Johansen test** for multiple variables

## Mathematical Formulation:
For series $X_t$ and $Y_t$, we test if $\alpha X_t + \beta Y_t$ is stationary

## Applications:
- Pairs trading strategy development
- Mean-reversion arbitrage opportunities
