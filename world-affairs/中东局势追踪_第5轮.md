- **"真实承诺4"军事行动**：使用先进导弹和无人机袭击美军舰艇及基地

- **主要袭击目标**：
  - 美国驻伊拉克埃尔比勒总领馆（已摧毁）
  - 美国在巴林的军事基地
  - 阿联酋杰贝阿里港的美军舰艇

## 二、石油市场影响

### 2.1 供应格局

| 指标 | 数据 |
|------|------|
| 伊朗原油日产量 | 320-330万桶 |
| 全球占比 | 约3% |
| 探明储量 | 超2000亿桶（全球第3-4位） |
| 中国进口依赖 | 13%-14% |

### 2.2 关键基础设施受损

- **哈尔克岛（Kharg Island）**：处理伊朗90%以上原油出口的码头发生爆炸
- **杜库姆（Duqm）港口**：遭到袭击
- **霍尔木兹海峡**：油轮运输几近停摆

### 2.3 油价预测

| 机构 | 预测 |
|------|------|
| **高盛** | 油价上涨约18美元/桶（相当于霍尔木兹海峡封锁情景） |
| **汇丰银行** | 若未波及能源基础设施，短线跳涨5美元/桶 |
| **极端情景** | 伊朗放话"不许一滴石油流出"，油价或上每桶200美元 |

### 2.4 市场现状

- 全球原油市场供应目前相对充足
- 伊朗面临严重且持续中断的可能性较低
- 爆发全面石油危机的概率暂时不大（牛津经济研究院）

- **临时领导委员会**：已开始工作
- **全国哀悼期**：40天

- **中国促成的和解**：是否被利用成为焦点
- **多国卷入**：越来越多国家被卷入中东冲突，局势迅速多点开花

### 3.3 美方表态

- **特朗普声明**：下令美军参与打击是基于"伊朗即将率先发动攻击"的判断
- **美以目标**：以色列两重目的已达到，伊朗将回击

## 五、投资策略建议

### 5.1 资产配置建议

**防御性配置**：
- 增加黄金、国债等避险资产比重
- 关注能源板块：石油公司、油服企业

- 新能源替代主题（长期）
- 国防军工板块
