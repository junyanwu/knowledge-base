## 核心发现与投资启示总结

### 经济影响
- 俄罗斯经济展现出超出预期的韧性，但面临结构性挑战
- 乌克兰遭受毁灭性打击，重建需超4860亿美元
- 欧洲经历能源危机和通胀冲击，经济增速放缓
- 中国从俄罗斯获得能源进口机遇，但面临地缘政治压力

### 能源格局
- 全球能源贸易流向发生根本性重构
- 欧洲加速能源转型，可再生能源投资大幅增长
- 能源价格经历剧烈波动后趋于稳定
- 中国能源安全战略稳步推进，新能源产业全球领先

### 粮食安全
- 全球粮食价格经历剧烈波动后回落
- 发展中国家粮食安全压力持续
- 中国粮食安全总体可控，大豆依存度是结构性问题
- 农业科技和种业投资机会明确
