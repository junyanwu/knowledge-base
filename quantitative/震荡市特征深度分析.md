## 1. 震荡市定义与识别

### 1.1 震荡市的理论定义

震荡市（Sideways Market / Range-bound Market）是指价格在一定区间内反复波动，缺乏明确趋势方向的市场状态。

**核心特征**:
- 价格上下波动但无突破
- 均值回归特征明显
- 波动率相对较低
- 成交量萎缩或无序
- 趋势策略失效

### 1.2 多维度震荡识别指标

#### 1.2.1 ADX趋势强度指标

**Average Directional Index (ADX)** 用于衡量趋势强度，而非方向。

**计算公式**:
```python
ADX = 100 * MA(|+DI - -DI| / (+DI + -DI), n)
```

**震荡市判定标准**:
- ADX < 20：强震荡市
- 20 ≤ ADX < 25：弱震荡市
- ADX ≥ 25：趋势市

#### 1.2.2 ATR波动率指标

**Average True Range (ATR)** 衡量价格波动幅度。

**计算公式**:
```python
TR = max(High - Low, |High - Prev_Close|, |Low - Prev_Close|)
ATR = MA(TR, n)
```

**震荡市判定标准**:
- ATR < ATR_MA * 0.7：低波动震荡
- ATR相对历史分位数 < 30%：历史低位震荡

#### 1.2.3 布林带宽度指标

**Band Width** 衡量价格波动区间的收敛程度。

**计算公式**:
```python
Upper_Band = MA(Close, n) + k * Std(Close, n)
Lower_Band = MA(Close, n) - k * Std(Close, n)
Band_Width = (Upper_Band - Lower_Band) / MA(Close, n)
```

**震荡市判定标准**:
- Band Width < Band_Width_MA * 0.8：收缩震荡
- Band Width历史分位数 < 20%：极窄震荡

```python
import numpy as np
import pandas as pd
from typing import Tuple, Dict

class SidewaysMarketDetector:
    """震荡市综合识别系统"""
    
    def __init__(self,
                 adx_threshold: float = 25.0,
                 atr_lookback: int = 20,
                 bb_lookback: int = 20,
                 bb_std: float = 2.0):
        """
        初始化参数
        
        Args:
            adx_threshold: ADX阈值，低于此值为震荡
            atr_lookback: ATR计算周期
            bb_lookback: 布林带计算周期
            bb_std: 布林带标准差倍数
        """
        self.adx_threshold = adx_threshold
        self.atr_lookback = atr_lookback
        self.bb_lookback = bb_lookback
        self.bb_std = bb_std
    
    def calculate_adx(self, df: pd.DataFrame, period: int = 14) -> pd.Series:
        """计算ADX指标"""
        high = df['high']
        low = df['low']
        close = df['close']
        
        # 计算+DM和-DM
        plus_dm = high.diff()
        minus_dm = -low.diff()
        
        plus_dm = plus_dm.where((plus_dm > minus_dm) & (plus_dm > 0), 0)
        minus_dm = minus_dm.where((minus_dm > plus_dm) & (minus_dm > 0), 0)
        
        # 计算TR
        tr1 = high - low
        tr2 = abs(high - close.shift())
        tr3 = abs(low - close.shift())
        tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
        
        # 计算ATR
        atr = tr.rolling(period).mean()
        
        # 计算+DI和-DI
        plus_di = 100 * (plus_dm.rolling(period).mean() / atr)
        minus_di = 100 * (minus_dm.rolling(period).mean() / atr)
        
        # 计算DX和ADX
        dx = 100 * abs(plus_di - minus_di) / (plus_di + minus_di)
        adx = dx.rolling(period).mean()
        
        return adx
    
    def calculate_atr(self, df: pd.DataFrame, period: int = 14) -> pd.Series:
        """计算ATR指标"""
        high = df['high']
        low = df['low']
        close = df['close']
        
        tr1 = high - low
        tr2 = abs(high - close.shift())
        tr3 = abs(low - close.shift())
        tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
        
        atr = tr.rolling(period).mean()
        return atr
    
    def calculate_bandwidth(self, df: pd.DataFrame, 
                           period: int = 20, 
                           std_dev: float = 2.0) -> pd.Series:
        """计算布林带宽度"""
        close = df['close']
        
        # 计算移动平均和标准差
        ma = close.rolling(period).mean()
        std = close.rolling(period).std()
        
        # 计算上下轨
        upper_band = ma + std_dev * std
        lower_band = ma - std_dev * std
        
        # 计算带宽
        bandwidth = (upper_band - lower_band) / ma
        
        return bandwidth
    
    def detect_sideways(self, df: pd.DataFrame) -> pd.DataFrame:
        """
        综合识别震荡市
        
        Returns:
            DataFrame包含:
            - adx: ADX指标值
            - adx_signal: ADX信号(1=震荡, 0=趋势)
            - atr_percentile: ATR历史分位数
            - atr_signal: ATR信号
            - bandwidth_percentile: 带宽历史分位数
            - bandwidth_signal: 带宽信号
            - sideways_score: 综合震荡评分(0-1)
            - sideways_state: 震荡状态(强震荡/弱震荡/趋势市)
        """
        result = pd.DataFrame(index=df.index)
        
        # 1. ADX指标
        result['adx'] = self.calculate_adx(df)
        result['adx_signal'] = (result['adx'] < self.adx_threshold).astype(int)
        
        # 2. ATR指标
        atr = self.calculate_atr(df, self.atr_lookback)
        atr_ma = atr.rolling(252).mean()
        result['atr'] = atr
        result['atr_ratio'] = atr / atr_ma
        result['atr_percentile'] = atr.rolling(252).rank(pct=True)
        result['atr_signal'] = (result['atr_percentile'] < 0.3).astype(int)
        
        # 3. 布林带宽度
        bandwidth = self.calculate_bandwidth(df, self.bb_lookback, self.bb_std)
        bandwidth_ma = bandwidth.rolling(252).mean()
        result['bandwidth'] = bandwidth
        result['bandwidth_ratio'] = bandwidth / bandwidth_ma
        result['bandwidth_percentile'] = bandwidth.rolling(252).rank(pct=True)
        result['bandwidth_signal'] = (result['bandwidth_percentile'] < 0.2).astype(int)
        
        # 4. 综合震荡评分
        # 权重分配: ADX 40%, ATR 30%, BandWidth 30%
        result['sideways_score'] = (
            result['adx_signal'] * 0.4 +
            result['atr_signal'] * 0.3 +
            result['bandwidth_signal'] * 0.3
        )
        
        # 5. 震荡状态分类
        def classify_state(score):
            if score >= 0.7:
                return '强震荡'
            elif score >= 0.4:
                return '弱震荡'
            else:
                return '趋势市'
        
        result['sideways_state'] = result['sideways_score'].apply(classify_state)
        
        return result
    
    def analyze_sideways_statistics(self, df: pd.DataFrame) -> Dict:
        """
        分析震荡市统计特征
        
        Returns:
            包含震荡市频率、持续时间、强度分布等统计信息
        """
        sideways = self.detect_sideways(df)
        
        # 识别震荡市区间
        is_sideways = sideways['sideways_score'] >= 0.4
        sideways_changes = is_sideways.astype(int).diff()
        
        # 统计震荡市次数
        start_indices = df.index[sideways_changes == 1].tolist()
        end_indices = df.index[sideways_changes == -1].tolist()
        
        # 处理边界情况
        if len(start_indices) == 0 and is_sideways.iloc[0]:
            start_indices = [df.index[0]]
        if len(end_indices) == 0 and is_sideways.iloc[-1]:
            end_indices = [df.index[-1]]
        if len(start_indices) > len(end_indices):
            end_indices.append(df.index[-1])
        if len(end_indices) > len(start_indices):
            start_indices.insert(0, df.index[0])
        
        # 计算持续时间
        durations = []
        intensities = []
        for start, end in zip(start_indices, end_indices):
            duration = len(df.loc[start:end])
            durations.append(duration)
            
            # 计算平均震荡强度
            period_scores = sideways.loc[start:end, 'sideways_score']
            intensities.append(period_scores.mean())
        
        # 统计信息
        stats = {
            'total_periods': len(df),
            'sideways_periods': is_sideways.sum(),
            'sideways_ratio': is_sideways.mean(),
            'sideways_count': len(start_indices),
            'avg_duration': np.mean(durations) if durations else 0,
            'max_duration': np.max(durations) if durations else 0,
            'min_duration': np.min(durations) if durations else 0,
            'avg_intensity': np.mean(intensities) if intensities else 0,
            'high_intensity_ratio': sum(1 for i in intensities if i >= 0.7) / len(intensities) if intensities else 0,
            'state_distribution': sideways['sideways_state'].value_counts().to_dict()
        }
        
        return stats
```

```python
def calculate_price_range(df: pd.DataFrame, 
                          lookback: int = 20) -> pd.DataFrame:
    """
    计算价格震荡区间
    
    Args:
        df: 价格数据
        lookback: 回看周期
        
    Returns:
        包含区间特征的DataFrame
    """
    result = pd.DataFrame(index=df.index)
    
    # 滚动最高价和最低价
    result['rolling_high'] = df['high'].rolling(lookback).max()
    result['rolling_low'] = df['low'].rolling(lookback).min()
    
    # 区间宽度
    result['range_width'] = (result['rolling_high'] - result['rolling_low']) / df['close']
    
    # 价格位置（在区间中的位置）
    result['price_position'] = (
        (df['close'] - result['rolling_low']) / 
        (result['rolling_high'] - result['rolling_low'])
    )
    
    # 区间中值
    result['range_mid'] = (result['rolling_high'] + result['rolling_low']) / 2
    
    # 价格偏离中值的程度
    result['deviation_from_mid'] = (
        (df['close'] - result['range_mid']) / result['range_mid']
    )
    
    # 区间突破信号
    result['upper_breakout'] = (
        df['close'] > result['rolling_high'].shift(1)
    ).astype(int)
    result['lower_breakout'] = (
        df['close'] < result['rolling_low'].shift(1)
    ).astype(int)
    
    return result
```

#### 2.1.2 均值回归特征

**均值回归强度计算**:

```python
def calculate_mean_reversion_strength(df: pd.DataFrame, 
                                      lookback: int = 20) -> pd.DataFrame:
    """
    计算均值回归强度
    
    Args:
        df: 价格数据
        lookback: 回看周期
        
    Returns:
        包含均值回归特征的DataFrame
    """
    result = pd.DataFrame(index=df.index)
    
    # 移动平均
    result['ma'] = df['close'].rolling(lookback).mean()
    
    # 价格偏离MA
    result['deviation'] = (df['close'] - result['ma']) / result['ma']
    
    # 标准差
    result['std'] = df['close'].rolling(lookback).std()
    
    # Z-score
    result['z_score'] = (df['close'] - result['ma']) / result['std']
    
    # 均值回归强度（基于ADF检验）
    from statsmodels.tsa.stattools import adfuller
    
    def calculate_adf_score(series):
        try:
            adf_result = adfuller(series.dropna(), autolag='AIC')
            # ADF统计量越小，均值回归越强
            # 转换为0-1评分
            adf_stat = adf_result[0]
            critical_values = adf_result[4]
            
            # 如果ADF统计量小于1%临界值，评分=1
            # 如果大于10%临界值，评分=0
            # 线性插值
            if adf_stat < critical_values['1%']:
                return 1.0
            elif adf_stat > critical_values['10%']:
                return 0.0
            else:
                # 线性插值
                range_val = critical_values['10%'] - critical_values['1%']
                position = critical_values['10%'] - adf_stat
                return position / range_val
        except:
            return 0.5
    
    # 滚动计算ADF评分
    result['mean_reversion_score'] = df['close'].rolling(
        window=lookback * 3
    ).apply(calculate_adf_score, raw=False)
    
    # Hurst指数（衡量长期记忆性）
    def calculate_hurst(series):
        """计算Hurst指数"""
        try:
            lags = range(2, min(100, len(series) // 2))
            tau = [np.sqrt(np.std(np.subtract(series[lag:], series[:-lag]))) 
                   for lag in lags]
            
            # 对数回归
            poly = np.polyfit(np.log(lags), np.log(tau), 1)
            return poly[0] * 2.0
        except:
            return 0.5
    
    result['hurst'] = df['close'].rolling(
        window=lookback * 5
    ).apply(calculate_hurst, raw=False)
    
    # Hurst < 0.5 表示均值回归
    # Hurst = 0.5 表示随机游走
    # Hurst > 0.5 表示趋势
    result['is_mean_reverting'] = (result['hurst'] < 0.5).astype(int)
    
    return result
```

```python
def analyze_volatility_clustering(df: pd.DataFrame) -> Dict:
    """
    分析波动率聚类特征
    
    Returns:
        波动率聚类统计
    """
    # 计算收益率
    returns = df['close'].pct_change().dropna()
    
    # 计算绝对收益（波动率代理）
    abs_returns = returns.abs()
    
    # GARCH模型拟合
    from arch import arch_model
    
    try:
        # 拟合GARCH(1,1)模型
        model = arch_model(returns * 100, vol='Garch', p=1, q=1)
        results = model.fit(disp='off')
        
        # 提取参数
        alpha = results.params['alpha[1]']  # ARCH系数
        beta = results.params['beta[1]']    # GARCH系数
        persistence = alpha + beta           # 波动率持续性
        
        return {
            'arch_coefficient': alpha,
            'garch_coefficient': beta,
            'volatility_persistence': persistence,
            'is_clustered': persistence > 0.8,  # 高持续性
            'long_memory': persistence > 0.95   # 长记忆
        }
    except:
        return {
            'arch_coefficient': None,
            'garch_coefficient': None,
            'volatility_persistence': None,
            'is_clustered': False,
            'long_memory': False
        }


def calculate_volatility_regime(df: pd.DataFrame, 
                                lookback: int = 20) -> pd.DataFrame:
    """
    计算波动率状态
    
    Returns:
        包含波动率状态特征的DataFrame
    """
    result = pd.DataFrame(index=df.index)
    
    # 计算收益率
    returns = df['close'].pct_change()
    
    # 历史波动率
    result['volatility'] = returns.rolling(lookback).std() * np.sqrt(252)
    
    # 波动率MA
    result['volatility_ma'] = result['volatility'].rolling(252).mean()
    
    # 波动率分位数
    result['volatility_percentile'] = result['volatility'].rolling(252).rank(pct=True)
    
    # 波动率状态
    def classify_volatility(percentile):
        if percentile < 0.25:
            return '低波动'
        elif percentile < 0.75:
            return '中等波动'
        else:
            return '高波动'
    
    result['volatility_state'] = result['volatility_percentile'].apply(
        classify_volatility
    )
    
    # 波动率变化率
    result['volatility_change'] = result['volatility'].pct_change()
    
    # 波动率加速度
    result['volatility_acceleration'] = result['volatility_change'].diff()
```

```python
def analyze_volume_characteristics(df: pd.DataFrame, 
                                   lookback: int = 20) -> pd.DataFrame:
    """
    分析成交量特征
    
    Returns:
        包含成交量特征的DataFrame
    """
    result = pd.DataFrame(index=df.index)
    
    # 成交量MA
    result['volume_ma'] = df['volume'].rolling(lookback).mean()
    
    # 成交量比率
    result['volume_ratio'] = df['volume'] / result['volume_ma']
    
    # 成交量分位数
    result['volume_percentile'] = df['volume'].rolling(252).rank(pct=True)
    
    # 成交量状态
    def classify_volume(percentile):
        if percentile < 0.2:
            return '极度萎缩'
        elif percentile < 0.4:
            return '萎缩'
        elif percentile < 0.6:
            return '正常'
        elif percentile < 0.8:
            return '放量'
        else:
            return '极度放量'
    
    result['volume_state'] = result['volume_percentile'].apply(classify_volume)
    
    # 成交量波动
    result['volume_std'] = df['volume'].rolling(lookback).std()
    result['volume_cv'] = result['volume_std'] / result['volume_ma']  # 变异系数
    
    # 价量相关性
    result['price_volume_corr'] = df['close'].pct_change().rolling(
        lookback
    ).corr(df['volume'].pct_change())
    
    # 成交量趋势
    result['volume_trend'] = df['volume'].rolling(lookback).apply(
        lambda x: np.polyfit(range(len(x)), x, 1)[0] if len(x) > 1 else 0,
        raw=False
    )
    
    return result
```

```python
class SidewaysIntensityClassifier:
    """震荡市强度分级器"""
    
    def __init__(self):
        self.detector = SidewaysMarketDetector()
    
    def classify_intensity(self, df: pd.DataFrame) -> pd.DataFrame:
        """
        对震荡市进行强度分级
        
        Returns:
            包含强度分级和策略建议的DataFrame
        """
        result = self.detector.detect_sideways(df)
        
        # 强度分级
        def get_intensity_level(score):
            if score >= 0.8:
                return '极强震荡'
            elif score >= 0.6:
                return '强震荡'
            elif score >= 0.4:
                return '弱震荡'
            elif score >= 0.2:
                return '趋势萌芽'
            else:
                return '强趋势'
        
        result['intensity_level'] = result['sideways_score'].apply(get_intensity_level)
        
        # 策略建议
        def get_strategy_suggestion(row):
            intensity = row['intensity_level']
            
            if intensity == '极强震荡':
                return '网格交易、高频套利'
            elif intensity == '强震荡':
                return '均值回归、网格交易'
            elif intensity == '弱震荡':
                return '波段交易、区间交易'
            elif intensity == '趋势萌芽':
                return '趋势跟踪、动量策略'
            else:
                return '趋势跟踪、突破策略'
        
        result['strategy_suggestion'] = result.apply(get_strategy_suggestion, axis=1)
        
        # 风险提示
        def get_risk_warning(row):
            intensity = row['intensity_level']
            
            if intensity in ['极强震荡', '强震荡']:
                return '趋势策略风险高，避免追涨杀跌'
            elif intensity == '弱震荡':
                return '方向不明，建议轻仓观望'
            elif intensity == '趋势萌芽':
                return '趋势可能形成，可尝试建仓'
            else:
                return '趋势明确，可顺势操作'
        
        result['risk_warning'] = result.apply(get_risk_warning, axis=1)
        
        return result
    
    def generate_intensity_report(self, df: pd.DataFrame) -> Dict:
        """生成震荡强度报告"""
        classified = self.classify_intensity(df)
        
        report = {
            '总体统计': {
                '总交易日': len(df),
                '极强震荡占比': (classified['intensity_level'] == '极强震荡').mean(),
                '强震荡占比': (classified['intensity_level'] == '强震荡').mean(),
                '弱震荡占比': (classified['intensity_level'] == '弱震荡').mean(),
                '趋势萌芽占比': (classified['intensity_level'] == '趋势萌芽').mean(),
                '强趋势占比': (classified['intensity_level'] == '强趋势').mean()
            },
            '平均特征': {
                '平均ADX': classified['adx'].mean(),
                '平均ATR分位数': classified['atr_percentile'].mean(),
                '平均带宽分位数': classified['bandwidth_percentile'].mean(),
                '平均震荡评分': classified['sideways_score'].mean()
            },
            '持续时间分析': self._analyze_duration_by_intensity(classified)
        }
        
        return report
    
    def _analyze_duration_by_intensity(self, classified: pd.DataFrame) -> Dict:
        """分析不同强度等级的持续时间"""
        durations = {}
        
        for level in ['极强震荡', '强震荡', '弱震荡', '趋势萌芽', '强趋势']:
            mask = classified['intensity_level'] == level
            if mask.sum() > 0:
                # 计算持续段
                changes = mask.astype(int).diff()
                starts = classified.index[changes == 1].tolist()
                ends = classified.index[changes == -1].tolist()
                
                if len(starts) > 0:
                    avg_duration = np.mean([len(classified.loc[s:e]) for s, e in zip(starts, ends)])
                    durations[level] = {
                        '出现次数': len(starts),
                        '平均持续天数': avg_duration
                    }
        
        return durations
```

```python
def validate_sideways_detection(etf_data: Dict[str, pd.DataFrame]) -> pd.DataFrame:
    """
    验证震荡识别准确率
    
    使用未来收益率验证震荡识别的有效性:
    - 在震荡市中，未来收益率应该接近0且波动率低
    - 在趋势市中，未来收益率应该有方向性且波动率高
    """
    detector = SidewaysMarketDetector()
    results = []
    
    for code, df in etf_data.items():
        # 识别震荡市
        sideways = detector.detect_sideways(df)
        
        # 计算未来收益率
        df['future_return_5d'] = df['close'].pct_change(5).shift(-5)
        df['future_return_10d'] = df['close'].pct_change(10).shift(-10)
        df['future_return_20d'] = df['close'].pct_change(20).shift(-20)
        
        # 计算未来波动率
        df['future_vol_5d'] = df['close'].pct_change().rolling(5).std().shift(-5) * np.sqrt(252)
        df['future_vol_10d'] = df['close'].pct_change().rolling(10).std().shift(-10) * np.sqrt(252)
        df['future_vol_20d'] = df['close'].pct_change().rolling(20).std().shift(-20) * np.sqrt(252)
        
        # 震荡市验证
        sideways_mask = sideways['sideways_score'] >= 0.4
        trend_mask = sideways['sideways_score'] < 0.4
        
        # 震荡市未来收益（应该接近0）
        sideways_return_5d = df.loc[sideways_mask, 'future_return_5d'].mean()
        sideways_return_10d = df.loc[sideways_mask, 'future_return_10d'].mean()
        sideways_return_20d = df.loc[sideways_mask, 'future_return_20d'].mean()
        
        # 震荡市未来波动（应该低）
        sideways_vol_5d = df.loc[sideways_mask, 'future_vol_5d'].mean()
        sideways_vol_10d = df.loc[sideways_mask, 'future_vol_10d'].mean()
        sideways_vol_20d = df.loc[sideways_mask, 'future_vol_20d'].mean()
        
        # 趋势市未来收益（应该有方向性）
        trend_return_5d = df.loc[trend_mask, 'future_return_5d'].mean()
        trend_return_10d = df.loc[trend_mask, 'future_return_10d'].mean()
        trend_return_20d = df.loc[trend_mask, 'future_return_20d'].mean()
        
        # 趋势市未来波动（应该高）
        trend_vol_5d = df.loc[trend_mask, 'future_vol_5d'].mean()
        trend_vol_10d = df.loc[trend_mask, 'future_vol_10d'].mean()
        trend_vol_20d = df.loc[trend_mask, 'future_vol_20d'].mean()
        
        # 计算识别准确率
        # 震荡市：未来收益绝对值小且波动率低
        sideways_accuracy = (
            (abs(sideways_return_10d) < 0.02) and  # 收益率绝对值<2%
            (sideways_vol_10d < df['future_vol_10d'].mean())  # 波动率低于平均
        )
        
        # 趋势市：未来收益有方向性且波动率高
        trend_accuracy = (
            (abs(trend_return_10d) > abs(sideways_return_10d)) and  # 收益率绝对值更大
            (trend_vol_10d > sideways_vol_10d)  # 波动率更高
        )
        
        result = {
            'ETF代码': code,
            '震荡市占比': sideways_mask.mean(),
            
            # 震荡市特征
            '震荡市10日收益': sideways_return_10d,
            '震荡市10日波动': sideways_vol_10d,
            '震荡市识别准确': sideways_accuracy,
            
            # 趋势市特征
            '趋势市10日收益': trend_return_10d,
            '趋势市10日波动': trend_vol_10d,
            '趋势市识别准确': trend_accuracy,
            
            # 综合评估
            '总体准确率': (sideways_accuracy + trend_accuracy) / 2
        }
        
        results.append(result)
    
    return pd.DataFrame(results)


def validate_with_known_periods(etf_data: Dict[str, pd.DataFrame],
                               known_sideways_periods: List[Tuple[str, str, str]]) -> pd.DataFrame:
    """
    使用已知的震荡市时期验证识别准确率
    
    Args:
        etf_data: ETF数据
        known_sideways_periods: 已知震荡市时期列表 [(ETF代码, 开始日期, 结束日期), ...]
        
    Returns:
        验证结果
    """
    detector = SidewaysMarketDetector()
    results = []
    
    for code, start_date, end_date in known_sideways_periods:
        if code not in etf_data:
            continue
        
        df = etf_data[code]
        
        # 识别震荡市
        sideways = detector.detect_sideways(df)
        
        # 已知震荡市期间
        start = pd.to_datetime(start_date)
        end = pd.to_datetime(end_date)
        known_mask = (df.index >= start) & (df.index <= end)
        
        # 识别为震荡市的期间
        detected_mask = sideways['sideways_score'] >= 0.4
        
        # 计算准确率、召回率、F1
        true_positive = (known_mask & detected_mask).sum()
        false_positive = (~known_mask & detected_mask).sum()
        false_negative = (known_mask & ~detected_mask).sum()
        true_negative = (~known_mask & ~detected_mask).sum()
        
        precision = true_positive / (true_positive + false_positive) if (true_positive + false_positive) > 0 else 0
        recall = true_positive / (true_positive + false_negative) if (true_positive + false_negative) > 0 else 0
        f1 = 2 * precision * recall / (precision + recall) if (precision + recall) > 0 else 0
        
        result = {
            'ETF代码': code,
            '已知震荡期': f'{start_date} ~ {end_date}',
            '已知震荡天数': known_mask.sum(),
            '识别震荡天数': detected_mask.sum(),
            '真正例': true_positive,
            '假正例': false_positive,
            '假负例': false_negative,
            '真负例': true_negative,
            '精确率': precision,
            '召回率': recall,
            'F1分数': f1
        }
        
        results.append(result)
    
    return pd.DataFrame(results)
```

3. **低突破率**: 价格突破上下轨后快速回落

#### 6.1.3 波动率特征
1. **低波动率**: 历史波动率处于低位

#### 6.1.4 成交量特征
