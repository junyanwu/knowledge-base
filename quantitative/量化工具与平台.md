### 1.2 主流回测框架详解

import backtrader as bt

class MyStrategy(bt.Strategy):
    def __init__(self):
        self.sma = bt.indicators.SMA(self.data.close, period=20)

        if self.data.close > self.sma:
            self.buy()
        elif self.data.close < self.sma:

### 1.3 回测核心考量

#### 数据质量
- **幸存者偏差**：避免只用当前存续股票
- **前视偏差**：确保只使用历史可得信息
- **数据清洗**：处理停牌、分红、拆股

#### 交易成本建模
```python
class TransactionCostModel:
    def __init__(self, commission=0.0003, slippage=0.0005):
        self.commission = commission  # 手续费率
        self.slippage = slippage      # 滑点率
    
    def calculate_cost(self, price, quantity):
        # 手续费
        commission_cost = price * quantity * self.commission
        # 滑点（假设价格不利变动）
        slippage_cost = price * quantity * self.slippage
        return commission_cost + slippage_cost
```

#### 绩效评估指标
- **收益指标**：年化收益、累计收益、月度收益
- **风险指标**：最大回撤、波动率、下行风险
- **风险调整收益**：夏普比率、索提诺比率、卡玛比率
- **策略稳定性**：胜率、盈亏比、连续亏损期

### 1.4 回测框架最佳实践

#### 事件驱动架构
```python
class BacktestEngine:
    """事件驱动回测引擎"""
    
    def __init__(self, data_handler, strategy, portfolio, execution):
        self.data_handler = data_handler
        self.strategy = strategy
        self.portfolio = portfolio
        self.execution = execution
        self.events_queue = Queue()
    
    def run_backtest(self):
        """执行回测"""
        while True:
            # 更新市场数据
            if self.data_handler.continue_backtest:
                self.data_handler.update_bars()
            else:
                break
            
            # 处理事件队列
            while not self.events_queue.empty():
                event = self.events_queue.get()
                
                if event.type == 'MARKET':
                    self.strategy.calculate_signals(event)
                elif event.type == 'SIGNAL':
                    self.portfolio.update_signal(event)
                elif event.type == 'ORDER':
                    self.execution.execute_order(event)
                elif event.type == 'FILL':
                    self.portfolio.update_fill(event)
```

### 2.3 数据处理最佳实践

#### 数据清洗
```python
def clean_data(df):
    """数据清洗"""
    # 处理缺失值
    df = df.fillna(method='ffill')
    
    # 处理异常值
    df['close'] = df['close'].clip(lower=df['close'].quantile(0.01),
                                    upper=df['close'].quantile(0.99))
    
    # 处理停牌
    df = df[df['volume'] > 0]
    
    # 复权处理
    df['adjusted_close'] = df['close'] * df['adj_factor']
    
    return df
```

**LSTM示例**：
```python
import torch
import torch.nn as nn

class LSTMModel(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(LSTMModel, self).__init__()
        self.lstm = nn.LSTM(input_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, output_dim)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        out = self.fc(lstm_out[:, -1, :])
        return out

# 创建模型
model = LSTMModel(input_dim=10, hidden_dim=64, output_dim=1)
```
