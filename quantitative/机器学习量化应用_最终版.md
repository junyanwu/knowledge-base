### 1.2 选股（Stock Selection）

**核心价值**：
- 发现传统因子无法捕捉的**非线性关系**
- **自动化因子挖掘**，降低人力成本70%+
- **动态适应**市场变化，持续优化

**技术演进路径**：

#### 第一代：传统多因子模型（2010-2015）
```python
# 线性加权模型
预期收益 = Σ(因子暴露 × 因子收益) + ε
```

**优点**：可解释性强、计算简单  
**缺点**：无法捕捉非线性、因子交互效应弱

#### 第二代：机器学习增强（2015-2020）
```python
# XGBoost/LightGBM非线性模型
预期收益 = f(因子组合)  # f为非线性函数
```

**实战案例**：
- **国信证券多因子选股**：XGBoost+因子正交化，年化25%，夏普1.8
- **国盛金工机器学习系列**：随机森林因子重要性，自动筛选有效因子
- **LightGBM快速训练**：日频调仓，训练时间<5分钟，支持GPU加速

#### 第三代：深度学习端到端（2020-2023）
```python
class StockSelectionModel(nn.Module):
    """端到端深度学习选股模型"""
    def __init__(self, input_dim, hidden_dim=64):
        super().__init__()
        self.lstm = nn.LSTM(input_dim, hidden_dim, num_layers=2, batch_first=True)
        self.attention = nn.MultiheadAttention(hidden_dim, num_heads=4)
        self.fc = nn.Linear(hidden_dim, 1)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        attn_out, _ = self.attention(lstm_out, lstm_out, lstm_out)
        return self.fc(attn_out[:, -1, :])
```

**优势**：自动特征学习、捕捉长期依赖、端到端优化

#### 第四代：图神经网络+LLM（2023-2026）
```python
class NodeTransformerModel(nn.Module):
    """股票关系网络+Transformer"""
    def __init__(self):
        self.gnn = GNN(node_features=64, edge_features=16)
        self.transformer = TransformerEncoder(d_model=64, nhead=4)
    
    def forward(self, stock_features, graph_edges):
        # 图神经网络捕捉股票关系
        node_embeddings = self.gnn(stock_features, graph_edges)
        # Transformer捕捉全局依赖
        global_features = self.transformer(node_embeddings)
        return global_features
```

**2026年突破**：
- **Node Transformer架构**：捕捉股票间关系网络（供应链、资金流、行业关联）
- **LLM因子生成**：R&D-Agent自动化因子挖掘，研发效率提升10x
- **多模态融合**：文本+价量+财务+另类数据
