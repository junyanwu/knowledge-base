## 1. 风险平价理论推导

### 1.1 核心思想

**传统权重配置的问题**：
- 60/40组合：股票60%，债券40%
- 但风险贡献：股票约90%，债券约10%
- 风险高度集中，未真正分散化

**风险平价的解决方案**：
- 让每个资产对组合的总风险贡献相等
- 资金权重不等，但风险贡献相等
- 实现真正的风险分散化

### 1.2 数学推导

**组合波动率**：
```
σp = √(w'Σw)
```

其中：
- w：权重向量
- Σ：协方差矩阵
- σp：组合波动率

**边际风险贡献（MRC）**：
```
MRCi = ∂σp / ∂wi = (Σw)i / σp
```

推导：
```
σp = (w'Σw)^0.5

∂σp / ∂wi = 0.5 × (w'Σw)^(-0.5) × ∂(w'Σw) / ∂wi
          = 0.5 × (1/σp) × 2 × (Σw)i
          = (Σw)i / σp
```

**风险贡献（RC）**：
```
RCi = wi × MRCi = wi × (Σw)i / σp
```

**风险贡献比例**：
```
RC%i = RCi / σp = wi × (Σw)i / σp²
```

**风险平价条件**：
```
RCi = σp / n  ∀i

即：wi × (Σw)i = σp² / n
```

### 1.3 优化问题

**目标函数**：
```
min Σ (RCi - σp/n)²
```

**约束条件**：
```
Σ wi = 1
wi ≥ 0
```

**完整优化问题**：
```
min Σ (wi × (Σw)i / σp - σp/n)²
s.t. Σ wi = 1
     wi ≥ 0
```

**Python实现**：

```python
import numpy as np
from scipy.optimize import minimize

def risk_parity_objective(weights, cov_matrix):
    """
    风险平价目标函数
    
    参数:
    weights: 权重
    cov_matrix: 协方差矩阵
    
    返回:
    目标函数值（风险贡献差异的平方和）
    """
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 边际风险贡献
    marginal_risk_contrib = cov_matrix @ weights / portfolio_vol
    
    # 风险贡献
    risk_contrib = weights * marginal_risk_contrib
    
    # 目标风险贡献
    target_risk_contrib = portfolio_vol / len(weights)
    
    # 目标函数
    objective = np.sum((risk_contrib - target_risk_contrib) ** 2)
    
    return objective

def risk_parity_weights(cov_matrix, method='scipy'):
    """
    风险平价权重计算
    
    参数:
    cov_matrix: 协方差矩阵
    method: 优化方法 ('scipy' or 'closed_form')
    
    返回:
    风险平价权重
    """
    n_assets = cov_matrix.shape[0]
    
    if method == 'scipy':
        # 数值优化
        constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        bounds = [(0.001, 1) for _ in range(n_assets)]  # 避免零权重
        
        result = minimize(
            risk_parity_objective,
            x0=[1/n_assets] * n_assets,
            args=(cov_matrix,),
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        weights = result.x
        
    elif method == 'closed_form':
        # 闭式解（假设资产不相关）
        # 权重与波动率成反比
        vols = np.sqrt(np.diag(cov_matrix))
        weights = (1 / vols) / np.sum(1 / vols)
    
    return weights / np.sum(weights)  # 归一化

# 示例：4资产风险平价
vols = np.array([0.20, 0.15, 0.10, 0.08])  # 年化波动率
correlations = np.array([
    [1.00, 0.50, 0.30, 0.20],
    [0.50, 1.00, 0.40, 0.25],
    [0.30, 0.40, 1.00, 0.35],
    [0.20, 0.25, 0.35, 1.00]
])

cov_matrix = np.outer(vols, vols) * correlations

# 风险平价权重
rp_weights = risk_parity_weights(cov_matrix)
print("风险平价权重:")
for i, w in enumerate(rp_weights):
    print(f"资产{i+1}: {w:.2%}")

# 验证风险贡献
portfolio_vol = np.sqrt(rp_weights @ cov_matrix @ rp_weights)
marginal_risk = cov_matrix @ rp_weights / portfolio_vol
risk_contrib = rp_weights * marginal_risk
risk_contrib_pct = risk_contrib / portfolio_vol

print("\n风险贡献:")
for i, rc in enumerate(risk_contrib_pct):
    print(f"资产{i+1}: {rc:.2%}")

print(f"\n组合波动率: {portfolio_vol:.4f}")
```

## 2. 等风险贡献模型

### 2.1 模型扩展

**通用风险预算模型**：
```
min Σ (RCi / σp - bi)²
s.t. Σ wi = 1
     wi ≥ 0
```

其中：
- bi：资产i的目标风险预算
- Σ bi = 1

**特例**：
- 风险平价：bi = 1/n（等风险贡献）
- 逆波动率加权：bi ∝ 1/σi
- 自定义风险预算：根据投资观点设定

### 2.2 Python实现

```python
def risk_budget_weights(cov_matrix, risk_budget=None):
    """
    风险预算权重计算
    
    参数:
    cov_matrix: 协方差矩阵
    risk_budget: 目标风险预算 (默认等风险贡献)
    
    返回:
    风险预算权重
    """
    n_assets = cov_matrix.shape[0]
    
    # 默认等风险贡献
    if risk_budget is None:
        risk_budget = np.ones(n_assets) / n_assets
    
    # 确保风险预算和为1
    risk_budget = risk_budget / np.sum(risk_budget)
    
    def risk_budget_objective(weights):
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 边际风险贡献
        marginal_risk_contrib = cov_matrix @ weights / portfolio_vol
        
        # 风险贡献比例
        risk_contrib_pct = weights * marginal_risk_contrib / portfolio_vol
        
        # 目标函数
        objective = np.sum((risk_contrib_pct - risk_budget) ** 2)
        
        return objective
    
    # 优化
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0.001, 1) for _ in range(n_assets)]
    
    result = minimize(
        risk_budget_objective,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints,
        options={'ftol': 1e-12, 'maxiter': 1000}
    )
    
    weights = result.x
    
    return weights / np.sum(weights)

# 示例：自定义风险预算

    # 计算风险贡献

    print(f"\n场景{i+1}:")
    print("权重:", ' '.join([f"{w:.2%}" for w in weights]))
    print("风险贡献:", ' '.join([f"{rc:.2%}" for rc in risk_contrib_pct]))
    print(f"组合波动率: {portfolio_vol:.4f}")
```

```python
def asset_class_risk_budget(asset_classes, risk_budget_dict):
    """
    基于资产类别的风险预算
    
    参数:
    asset_classes: 资产类别映射 {'股票': ['ETF1', 'ETF2'], ...}
    risk_budget_dict: 风险预算 {'股票': 0.4, '债券': 0.4, ...}
    
    返回:
    完整风险预算向量
    """
    # 构建风险预算向量
    n_assets = sum([len(assets) for assets in asset_classes.values()])
    risk_budget = np.zeros(n_assets)
    
    idx = 0
    for asset_class, assets in asset_classes.items():
        # 类别内等风险贡献
        class_budget = risk_budget_dict[asset_class] / len(assets)
        for _ in assets:
            risk_budget[idx] = class_budget
            idx += 1
    
    return risk_budget / np.sum(risk_budget)  # 归一化
```

```python
def factor_risk_budget(factor_exposures, factor_cov_matrix, factor_budget):
    """
    基于因子的风险预算
    
    参数:
    factor_exposures: 因子暴露矩阵 (n_assets, n_factors)
    factor_cov_matrix: 因子协方差矩阵 (n_factors, n_factors)
    factor_budget: 因子风险预算
    
    返回:
    资产权重
    """
    n_assets = factor_exposures.shape[0]
    n_factors = factor_exposures.shape[1]
    
    def factor_budget_objective(weights):
        # 因子暴露
        portfolio_factor_exposure = weights @ factor_exposures
        
        # 因子风险贡献
        portfolio_factor_var = portfolio_factor_exposure @ factor_cov_matrix @ portfolio_factor_exposure
        marginal_factor_risk = factor_cov_matrix @ portfolio_factor_exposure / np.sqrt(portfolio_factor_var)
        factor_risk_contrib = portfolio_factor_exposure * marginal_factor_risk / np.sqrt(portfolio_factor_var)
        
        # 目标函数
        objective = np.sum((factor_risk_contrib - factor_budget) ** 2)
        
        return objective
    
    # 优化
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0.001, 1) for _ in range(n_assets)]
    
    result = minimize(
        factor_budget_objective,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

```python
def dynamic_risk_budget(returns, lookback_window=60, target_vol=0.10):
    """
    动态风险预算
    
    参数:
    returns: 收益率序列
    lookback_window: 滚动窗口
    target_vol: 目标波动率
    
    返回:
    动态权重序列
    """
    n_periods = len(returns)
    n_assets = returns.shape[1]
    
    weights_series = []
    dates = []
    
    for i in range(lookback_window, n_periods):
        # 滚动协方差
        window_returns = returns.iloc[i-lookback_window:i]
        cov_matrix = window_returns.cov().values * 252  # 年化
        
        # 风险平价权重
        weights = risk_budget_weights(cov_matrix)
        
        # 目标波动率调整
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        leverage = target_vol / portfolio_vol
        weights = weights * leverage
        
        weights_series.append(weights)
        dates.append(returns.index[i])
    
    return pd.DataFrame(weights_series, index=dates, columns=returns.columns)
```

## 4. 杠杆调整

### 4.1 杠杆原理

**问题**：
- 风险平价组合通常波动率较低（5-8%）
- 无法满足投资者的目标波动率要求（通常10-15%）
- 需要通过杠杆提高风险水平

**杠杆公式**：
```
w_leveraged = w × (目标波动率 / 组合波动率)
```

**约束**：
- 杠杆上限（通常2-3倍）
- 融资成本
- 流动性风险

### 4.2 杠杆优化

```python
def leverage_adjustment(weights, cov_matrix, target_vol, max_leverage=2.0, 
                       funding_cost=0.02):
    """
    杠杆调整
    
    参数:
    weights: 原始权重
    cov_matrix: 协方差矩阵
    target_vol: 目标波动率
    max_leverage: 最大杠杆
    funding_cost: 融资成本
    
    返回:
    杠杆后权重、杠杆倍数、调整后收益
    """
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 理论杠杆
    leverage = target_vol / portfolio_vol
    
    # 限制杠杆
    leverage = min(leverage, max_leverage)
    
    # 杠杆后权重
    leveraged_weights = weights * leverage
    
    # 计算融资成本影响
    # 假设无风险利率为融资成本
    funding_impact = -funding_cost * (leverage - 1)
    
    return {
        'original_weights': weights,
        'leveraged_weights': leveraged_weights,
        'leverage': leverage,
        'portfolio_vol': portfolio_vol,
        'target_vol': target_vol,
        'funding_impact': funding_impact,
        'adjusted_weights_sum': np.sum(leveraged_weights)
    }

# 示例
weights = risk_budget_weights(cov_matrix)

print("原始权重:", result['original_weights'])

print("杠杆倍数:", result['leverage'])
print("组合波动率:", result['portfolio_vol'])
print("融资成本影响:", result['funding_impact'])
```

### 4.3 杠杆风险管理

```python
def leverage_risk_management(weights, cov_matrix, target_vol, max_leverage=2.0,
                             drawdown_limit=0.20, vol_limit=0.15):
    """
    杠杆风险管理
    
    参数:
    weights: 权重
    cov_matrix: 协方差矩阵
    target_vol: 目标波动率
    max_leverage: 最大杠杆
    drawdown_limit: 最大回撤限制
    vol_limit: 波动率上限
    
    返回:
    风险管理后的杠杆
    """
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 初始杠杆
    leverage = target_vol / portfolio_vol
    
    # 波动率约束
    if leverage * portfolio_vol > vol_limit:
        leverage = vol_limit / portfolio_vol
    
    # 杠杆约束
    leverage = min(leverage, max_leverage)
    
    # 回撤约束（假设波动率与回撤成正比）
    # 经验法则：最大回撤约等于2倍波动率
    expected_drawdown = 2 * leverage * portfolio_vol
    if expected_drawdown > drawdown_limit:
        leverage = drawdown_limit / (2 * portfolio_vol)
    
    return leverage

# 示例
leverage = leverage_risk_management(weights, cov_matrix, target_vol=0.12)
print(f"风险管理后的杠杆: {leverage:.2f}x")
```

## 5. ETF风险平价组合实战

### 5.1 ETF风险度量

```python
class ETFRiskParityPortfolio:
    """ETF风险平价组合"""
    
    def __init__(self, etf_returns, etf_names=None, target_vol=0.10):
        """
        初始化
        
        参数:
        etf_returns: ETF收益率DataFrame
        etf_names: ETF名称
        target_vol: 目标波动率
        """
        self.returns = etf_returns
        self.etf_names = etf_names if etf_names else etf_returns.columns.tolist()
        self.target_vol = target_vol
        
        # 计算协方差矩阵
        self.cov_matrix = etf_returns.cov().values * 252
        self.volatilities = etf_returns.std().values * np.sqrt(252)
    
    def calculate_risk_metrics(self, weights):
        """
        计算风险指标
        
        参数:
        weights: 权重
        
        返回:
        风险指标
        """
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ self.cov_matrix @ weights)
        
        # 边际风险贡献
        marginal_risk = self.cov_matrix @ weights / portfolio_vol
        
        # 风险贡献
        risk_contrib = weights * marginal_risk
        risk_contrib_pct = risk_contrib / portfolio_vol
        
        # 风险预算差异
        target_risk_pct = np.ones(len(weights)) / len(weights)
        risk_budget_error = np.sqrt(np.mean((risk_contrib_pct - target_risk_pct) ** 2))
        
        return {
            'portfolio_volatility': portfolio_vol,
            'marginal_risk': marginal_risk,
            'risk_contribution': risk_contrib,
            'risk_contribution_pct': risk_contrib_pct,
            'risk_budget_error': risk_budget_error
        }
    
    def build_risk_parity_portfolio(self, risk_budget=None, leverage=True):
        """
        构建风险平价组合
        
        参数:
        risk_budget: 风险预算（None为等风险贡献）
        leverage: 是否使用杠杆
        
        返回:
        组合配置
        """
        # 风险平价权重
        weights = risk_budget_weights(self.cov_matrix, risk_budget)
        
        # 风险指标
        risk_metrics = self.calculate_risk_metrics(weights)
        
        # 杠杆调整
        if leverage:
            leverage_mult = self.target_vol / risk_metrics['portfolio_volatility']
            leverage_mult = min(leverage_mult, 2.0)  # 最大杠杆2倍
            final_weights = weights * leverage_mult
        else:
            leverage_mult = 1.0
            final_weights = weights
        
        return {
            'weights': dict(zip(self.etf_names, weights)),
            'final_weights': dict(zip(self.etf_names, final_weights)),
            'leverage': leverage_mult,
            'risk_metrics': risk_metrics
        }
    
    def backtest(self, weights, rebalance_freq='M'):
        """
        回测
        
        参数:
        weights: 目标权重
        rebalance_freq: 再平衡频率
        
        返回:
        回测结果
        """
        # 初始化
        portfolio_value = 1.0
        portfolio_values = [portfolio_value]
        current_weights = np.array(list(weights.values()))
        
        # 再平衡日期
        rebalance_dates = self.returns.resample(rebalance_freq).last().index
        
        for date, daily_returns in self.returns.iterrows():
            # 计算当日收益
            daily_return = (current_weights * daily_returns.values).sum()
            portfolio_value *= (1 + daily_return)
            portfolio_values.append(portfolio_value)
            
            # 权重漂移
            current_weights = current_weights * (1 + daily_returns.values)
            current_weights /= current_weights.sum()
            
            # 再平衡
            if date in rebalance_dates:
                current_weights = np.array(list(weights.values()))
        
        # 构建结果
        portfolio_series = pd.Series(
            portfolio_values,
            index=[self.returns.index[0] - pd.Timedelta(days=1)] + self.returns.index.tolist()
        )
        
        returns = portfolio_series.pct_change().dropna()
        
        # 绩效指标
        annual_return = returns.mean() * 252
        annual_vol = returns.std() * np.sqrt(252)
        sharpe = annual_return / annual_vol
        
        # 最大回撤
        cumulative = portfolio_series / portfolio_series.cummax()
        drawdown = (cumulative - 1)
        max_drawdown = drawdown.min()
        
        # 卡尔玛比率
        calmar = annual_return / abs(max_drawdown)
        
        return {
            'portfolio_values': portfolio_series,
            'returns': returns,
            'annual_return': annual_return,
            'annual_volatility': annual_vol,
            'sharpe_ratio': sharpe,
            'max_drawdown': max_drawdown,
            'calmar_ratio': calmar
        }
    
    def compare_with_benchmark(self, rp_weights, benchmark_weights):
        """
        与基准对比
        
        参数:
        rp_weights: 风险平价权重
        benchmark_weights: 基准权重
        
        返回:
        对比结果
        """
        # 风险平价回测
        rp_result = self.backtest(
            dict(zip(self.etf_names, rp_weights))
        )
        
        # 基准回测
        benchmark_result = self.backtest(
            dict(zip(self.etf_names, benchmark_weights))
        )
        
        comparison = pd.DataFrame({
            'Risk Parity': [
                rp_result['annual_return'],
                rp_result['annual_volatility'],
                rp_result['sharpe_ratio'],
                rp_result['max_drawdown'],
                rp_result['calmar_ratio']
            ],
            'Benchmark': [
                benchmark_result['annual_return'],
                benchmark_result['annual_volatility'],
                benchmark_result['sharpe_ratio'],
                benchmark_result['max_drawdown'],
                benchmark_result['calmar_ratio']
            ]
        }, index=['Annual Return', 'Annual Volatility', 'Sharpe Ratio', 
                  'Max Drawdown', 'Calmar Ratio'])
        
        return comparison

# 示例：构建ETF风险平价组合
```

```python
def compare_risk_parity_vs_traditional(etf_returns, etf_names):
    """
    对比风险平价与传统配置
    
    参数:
    etf_returns: ETF收益率
    etf_names: ETF名称
    
    返回:
    对比分析
    """
    # 协方差矩阵
    cov_matrix = etf_returns.cov().values * 252
    
    # 1. 等权重组合
    equal_weights = np.ones(len(etf_names)) / len(etf_names)
    
    # 2. 60/40组合
    traditional_weights = np.zeros(len(etf_names))
    traditional_weights[0] = 0.6  # 股票60%
    traditional_weights[2] = 0.4  # 债券40%
    
    # 3. 风险平价组合
    rp_weights = risk_budget_weights(cov_matrix)
    
    # 4. 逆波动率组合
    vols = np.sqrt(np.diag(cov_matrix))
    inverse_vol_weights = (1 / vols) / np.sum(1 / vols)
    
    # 计算各组合指标
    portfolios = {
        '等权重': equal_weights,
        '60/40': traditional_weights,
        '风险平价': rp_weights,
        '逆波动率': inverse_vol_weights
    }
    
    results = []
    
    for name, weights in portfolios.items():
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 风险贡献
        risk_contrib_pct = weights * (cov_matrix @ weights) / portfolio_vol**2
        
        # 风险预算差异
        target_risk = np.ones(len(weights)) / len(weights)
        risk_budget_error = np.std(risk_contrib_pct - target_risk)
        
        # 风险集中度（最大风险贡献）
        max_risk_contrib = np.max(risk_contrib_pct)
        
        results.append({
            'Portfolio': name,
            'Volatility': portfolio_vol,
            'Risk Budget Error': risk_budget_error,
            'Max Risk Contrib': max_risk_contrib,
            'Risk Contrib': risk_contrib_pct
        })
    
    return pd.DataFrame(results)
```

### 6.1 条件风险平价

```python
def conditional_risk_parity(returns, regimes, target_vol=0.10):
    """
    条件风险平价（考虑市场状态）
    
    参数:
    returns: 收益率序列
    regimes: 市场状态标签
    target_vol: 目标波动率
    
    返回:
    不同市场状态下的权重
    """
    regime_weights = {}
    
    for regime in np.unique(regimes):
        # 该状态下的收益率
        regime_returns = returns[regimes == regime]
        
        # 协方差矩阵
        cov_matrix = regime_returns.cov().values * 252
        
        # 风险平价权重
        weights = risk_budget_weights(cov_matrix)
        
        # 杠杆调整
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        leverage = target_vol / portfolio_vol
        
        regime_weights[regime] = {
            'weights': weights,
            'leverage': leverage,
            'volatility': portfolio_vol
        }
    
    return regime_weights
```

### 6.2 鲁棒风险平价

```python
def robust_risk_parity(returns, n_bootstrap=100, target_vol=0.10):
    """
    鲁棒风险平价（Bootstrap方法）
    
    参数:
    returns: 收益率序列
    n_bootstrap: Bootstrap次数
    target_vol: 目标波动率
    
    返回:
    鲁棒权重
    """
    n_samples = len(returns)
    weights_list = []
    
    for _ in range(n_bootstrap):
        # Bootstrap样本
        bootstrap_idx = np.random.choice(n_samples, n_samples, replace=True)
        bootstrap_returns = returns.iloc[bootstrap_idx]
        
        # 协方差矩阵
        cov_matrix = bootstrap_returns.cov().values * 252
        
        # 风险平价权重
        weights = risk_budget_weights(cov_matrix)
        weights_list.append(weights)
    
    # 权重均值
    weights_array = np.array(weights_list)
    mean_weights = np.mean(weights_array, axis=0)
    
    # 权重标准差（衡量稳定性）
    std_weights = np.std(weights_array, axis=0)
    
    # 协方差矩阵（全样本）
    full_cov = returns.cov().values * 252
    portfolio_vol = np.sqrt(mean_weights @ full_cov @ mean_weights)
    leverage = target_vol / portfolio_vol
    
    return {
        'weights': mean_weights,
        'weight_std': std_weights,
        'leverage': leverage,
        'volatility': portfolio_vol
    }
```

### 6.3 风险平价+动量

```python
def risk_parity_with_momentum(returns, lookback=60, momentum_window=12):
    """
    风险平价+动量策略
    
    参数:
    returns: 收益率序列
    lookback: 协方差窗口
    momentum_window: 动量窗口
    
    返回:
    动态权重序列
    """
    n_periods = len(returns)
    n_assets = returns.shape[1]
    
    weights_series = []
    dates = []
    
    for i in range(max(lookback, momentum_window), n_periods):
        # 协方差矩阵
        window_returns = returns.iloc[i-lookback:i]
        cov_matrix = window_returns.cov().values * 252
        
        # 风险平价权重
        rp_weights = risk_budget_weights(cov_matrix)
        
        # 动量信号
        momentum_returns = returns.iloc[i-momentum_window:i].sum()
        momentum_signal = (momentum_returns > 0).astype(int)  # 正收益为1，否则0
        
        # 调整权重
        adjusted_weights = rp_weights * momentum_signal
        
        # 归一化
        if adjusted_weights.sum() > 0:
            adjusted_weights = adjusted_weights / adjusted_weights.sum()
        else:
            adjusted_weights = rp_weights  # 无动量信号时用风险平价
        
        weights_series.append(adjusted_weights)
        dates.append(returns.index[i])
    
    return pd.DataFrame(weights_series, index=dates, columns=returns.columns)
```

## 7. 总结与实践要点

### 7.1 核心要点

1. **风险平价原理**
   - 等风险贡献，而非等资金权重
   - 真正的风险分散化
   - 需要杠杆达到目标波动率

2. **数学推导**
   - 边际风险贡献：MRCi = (Σw)i / σp
   - 风险贡献：RCi = wi × MRCi
   - 风险平价条件：RCi = σp / n

3. **风险预算**
   - 灵活设定风险预算
   - 基于资产类别或因子
   - 动态风险预算适应市场变化

4. **杠杆管理**
   - 目标波动率调整
   - 杠杆上限约束
   - 融资成本影响

### 7.2 ETF应用要点

1. **资产选择**
   - 低相关性资产
   - 流动性充足
   - 成本低廉

2. **风险度量**
   - 使用历史协方差矩阵
   - 考虑波动率聚类
   - 定期再平衡

3. **杠杆实施**
   - 使用期货或杠杆ETF
   - 控制融资成本
   - 管理流动性风险

4. **绩效评估**
   - 夏普比率
   - 最大回撤
   - 风险分散度

### 7.3 模型局限

1. **协方差矩阵估计误差**
2. **杠杆风险**
3. **流动性约束**
4. **交易成本**
