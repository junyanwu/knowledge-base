# 行业轮动模型深度分析

## 一、行业轮动理论框架

### 1.1 行业轮动的经济学基础

行业轮动（Sector Rotation）是指不同行业在不同经济周期阶段表现存在系统性差异的现象。其理论基础包括：

**经济周期理论：**
- 美林时钟将经济周期划分为四个阶段
- 不同阶段对应不同的优势行业
- 周期性行业 vs 防御性行业的切换

**产业生命周期：**
- 导入期：新兴行业，高风险高成长
- 成长期：快速增长，资本密集
- 成熟期：增长放缓，现金充裕
- 衰退期：需求下降，行业整合

**产业链传导：**
- 上游 → 中游 → 下游的景气传导
- 价格信号沿产业链传递
- 库存周期驱动短期轮动

### 1.2 行业分类体系

**申万行业分类（A股标准）：**
- 一级行业：31个（如银行、非银金融、食品饮料等）
- 二级行业：134个
- 三级行业：246个

**中信行业分类：**
- 一级行业：30个
- 与申万类似，部分调整

**GICS全球行业分类：**
- 11个一级行业板块
- 24个二级行业
- 69个三级行业

**轮动常用分组：**
- 周期类：钢铁、有色、煤炭、化工、建材
- 成长类：电子、计算机、通信、传媒
- 消费类：食品饮料、家电、纺织服装、医药
- 金融类：银行、非银金融、房地产
- 稳定类：公用事业、交通运输、建筑

### 1.3 行业轮动的驱动因素

行业轮动受三大因素驱动：

| 驱动因素 | 具体指标 | 影响行业 |
|----------|----------|----------|
| 宏观经济 | GDP、PMI、CPI、PPI | 周期vs防御 |
| 流动性 | 利率、M2、社融、信用利差 | 成长vs价值 |
| 市场情绪 | 换手率、涨跌比、北向资金 | 高βvs低β |

## 二、景气度指标构建

### 2.1 景气度概念

景气度反映行业的繁荣程度，是行业轮动的核心信号。景气度高意味着行业盈利增长预期强，股价表现通常较好。

### 2.2 景气度指标体系

#### 量价指标

**营业收入增速：**
$$Revenue\_Growth = \frac{Revenue_t - Revenue_{t-4}}{Revenue_{t-4}}$$

**利润增速：**
$$Profit\_Growth = \frac{Profit_t - Profit_{t-4}}{Profit_{t-4}}$$

**PPI分项：**
行业产品价格上涨反映需求旺盛，是景气度的重要信号。

#### 盈利指标

**ROE趋势：**
$$ROE\_Trend = \frac{ROE_t}{MA(ROE, 4)} - 1$$

**毛利率变化：**
$$Margin\_Change = Margin_t - Margin_{t-4}$$

**盈利预测上调比例：**
$$Earnings\_Revision = \frac{N_{upgrades} - N_{downgrades}}{N_{total}}$$

#### 行业特色指标

**金融行业：**
- 利率曲线形态（陡峭度）
- 信贷增速
- 不良贷款率
- 保险保费增速

**地产行业：**
- 商品房销售面积
- 土地成交溢价率
- 房价指数
- 开发投资增速

**消费行业：**
- 社会消费品零售总额
- 消费者信心指数
- 居民可支配收入
- 网购渗透率

**科技行业：**
- 半导体销售额
- 手机出货量
- 云计算市场规模
- 研发投入增速

### 2.3 综合景气度构建

**景气度打分公式：**

$$Score_i = \sum_{j=1}^{M} w_j \cdot rank(X_{i,j})$$

其中：
- $X_{i,j}$：行业$i$的第$j$个指标
- $rank$：横截面排名标准化
- $w_j$：指标权重

**权重确定方法：**
1. 等权法：$w_j = 1/M$
2. IC加权：$w_j = IC_j / \sum IC$
3. 主成分分析：提取第一主成分载荷

### 2.4 Python实现

```python
import pandas as pd
import numpy as np

class SectorMomentum:
    """
    行业景气度评分系统
    """
    def __init__(self, price_data, fundamental_data):
        self.prices = price_data
        self.fundamentals = fundamental_data
        
    def compute_momentum(self, window=20):
        """
        计算动量得分
        """
        returns = self.prices.pct_change(window)
        return returns.iloc[-1]
    
    def compute_fundamental_score(self):
        """
        计算基本面景气度
        """
        # 收入增速
        revenue_growth = self.fundamentals['revenue'].pct_change(4)
        # 利润增速
        profit_growth = self.fundamentals['profit'].pct_change(4)
        # ROE变化
        roe_change = self.fundamentals['roe'] - self.fundamentals['roe'].shift(4)
        
        # 综合得分
        score = (
            0.4 * revenue_growth.rank() +
            0.4 * profit_growth.rank() +
            0.2 * roe_change.rank()
        )
        return score
    
    def compute_sentiment_score(self, volume_data, northbound_data):
        """
        计算情绪指标
        """
        # 换手率
        turnover = volume_data / volume_data.rolling(20).mean()
        # 北向资金流入
        north_flow = northbound_data.diff(5)
        
        score = 0.5 * turnover.rank() + 0.5 * north_flow.rank()
        return score
    
    def get_composite_score(self):
        """
        综合景气度得分
        """
        momentum = self.compute_momentum()
        fundamental = self.compute_fundamental_score()
        sentiment = self.compute_sentiment_score()
        
        composite = (
            0.3 * momentum.rank() +
            0.5 * fundamental.rank() +
            0.2 * sentiment.rank()
        )
        return composite
```

## 三、动量轮动策略

### 3.1 动量效应原理

动量效应（Momentum Effect）是指过去表现好的资产在未来一段时间内继续表现好的现象。Jegadeesh和Titman（1993）首次系统性地证明了动量效应的存在。

**动量形成原因：**
- 信息传播滞后
- 投资者行为偏差（反应不足）
- 机构投资者的羊群效应

### 3.2 行业动量计算

**时间序列动量：**
$$Momentum_{TS} = \prod_{t=T-k+1}^{T}(1 + r_{i,t}) - 1$$

**横截面动量：**
$$Momentum_{CS} = \frac{r_{i,T-k:T} - \bar{r}_{T-k:T}}{\sigma_{T-k:T}}$$

**动量得分（排除近期反转）：**
$$Momentum\_Score = r_{T-12:T-1} - r_{T-1:T}$$

### 3.3 行业动量轮动策略

**策略逻辑：**
1. 计算各行业过去N月动量
2. 选择动量最强的前K个行业
3. 等权或按动量强度加权持有
4. 定期（月度/季度）调仓

**参数选择：**
- 形成期：3-12月（常用6月）
- 持有期：1-3月（常用1月）
- 行业数量：3-5个

**策略公式：**
$$w_i = \begin{cases} \frac{M_i}{\sum_{j \in TopK} M_j} & \text{if } rank(M_i) \leq K \\ 0 & \text{otherwise} \end{cases}$$

### 3.4 动量轮动改进

#### 动量+波动率调整

$$Adjusted\_Momentum = \frac{Momentum}{\sigma}$$

选择风险调整后动量最高的行业。

#### 动量+均值回归

$$Signal = Momentum_{12M} + \alpha \cdot (Mean - Price)$$

在动量基础上加入均值回归因子。

#### 相对动量

$$Relative\_Momentum = r_{sector} - r_{market}$$

选择相对市场表现最强的行业。

### 3.5 Python实现

```python
def sector_momentum_rotation(returns, formation_period=6, holding_period=1, n_sectors=5):
    """
    行业动量轮动策略
    
    参数:
        returns: 行业收益率DataFrame
        formation_period: 形成期（月）
        holding_period: 持有期（月）
        n_sectors: 选中行业数量
    
    返回:
        策略净值曲线
    """
    n_periods = len(returns)
    portfolio_returns = []
    
    for t in range(formation_period, n_periods, holding_period):
        # 计算形成期累计收益（排除最近1月）
        momentum = (1 + returns.iloc[t-formation_period:t-1]).prod() - 1
        
        # 选择动量最强的行业
        top_sectors = momentum.nlargest(n_sectors).index
        
        # 等权持有
        weights = pd.Series(0, index=returns.columns)
        weights[top_sectors] = 1 / n_sectors
        
        # 计算持有期收益
        period_return = (returns.iloc[t:t+holding_period] * weights).sum().sum()
        portfolio_returns.append(period_return)
    
    return pd.Series(portfolio_returns).cumsum()

def adaptive_momentum(returns, lookback_range=(3, 12)):
    """
    自适应动量：根据不同形成期的动量综合打分
    """
    scores = pd.DataFrame(index=returns.columns)
    
    for period in range(lookback_range[0], lookback_range[1] + 1):
        momentum = (1 + returns.iloc[-period:]).prod() - 1
        scores[f'period_{period}'] = momentum
    
    # 综合得分
    scores['composite'] = scores.mean(axis=1)
    return scores['composite']
```

## 四、价值轮动策略

### 4.1 行业估值体系

价值轮动基于行业估值高低进行配置，低估值行业预期获得估值修复收益。

**常用估值指标：**

| 指标 | 公式 | 适用行业 |
|------|------|----------|
| PE | 股价/每股收益 | 盈利稳定行业 |
| PB | 股价/每股净资产 | 资产密集型 |
| PS | 股价/每股销售额 | 亏损/高增长行业 |
| EV/EBITDA | 企业价值/息税折旧摊销前利润 | 高负债行业 |
| PEG | PE/净利润增长率 | 成长股 |

### 4.2 行业相对估值

**估值分位数：**
$$Valuation\_Percentile = \frac{PE_{current} - PE_{min}}{PE_{max} - PE_{min}}$$

**估值Z-score：**
$$Valuation\_Z = \frac{PE_{current} - PE_{mean}}{PE_{std}}$$

**相对估值：**
$$Relative\_PE = \frac{PE_{sector}}{PE_{market}}$$

### 4.3 价值轮动策略

**策略逻辑：**
1. 计算各行业估值指标
2. 选择估值最低的行业（低估）
3. 等待估值修复
4. 定期调仓

**估值修复周期：**
- 短期：1-3月
- 中期：6-12月
- 长期：2-3年

### 4.4 价值+成长轮动

结合估值与成长性进行轮动：

**PEG轮动：**
$$PEG = \frac{PE}{Growth\_Rate}$$

选择PEG最低的行业。

**价值成长得分：**
$$Score = w_1 \cdot (-PE\_rank) + w_2 \cdot Growth\_rank$$

### 4.5 Python实现

```python
def sector_value_rotation(pe_data, pb_data, n_sectors=5, holding_period=1):
    """
    行业价值轮动策略
    """
    portfolio_returns = []
    
    for t in range(12, len(pe_data), holding_period):
        # 计算PE分位数
        pe_percentile = (pe_data.iloc[t] - pe_data.iloc[t-12:t].min()) / \
                        (pe_data.iloc[t-12:t].max() - pe_data.iloc[t-12:t].min())
        
        # 计算PB分位数
        pb_percentile = (pb_data.iloc[t] - pb_data.iloc[t-12:t].min()) / \
                        (pb_data.iloc[t-12:t].max() - pb_data.iloc[t-12:t].min())
        
        # 综合估值得分（越低越好）
        value_score = 0.6 * pe_percentile + 0.4 * pb_percentile
        
        # 选择估值最低的行业
        cheap_sectors = value_score.nsmallest(n_sectors).index
        
        # 返回选中行业
        portfolio_returns.append({
            'date': pe_data.index[t],
            'sectors': list(cheap_sectors),
            'score': value_score[cheap_sectors].mean()
        })
    
    return portfolio_returns

def value_momentum_combo(returns, pe_data, value_weight=0.5):
    """
    价值+动量组合轮动
    """
    # 动量得分
    momentum = (1 + returns.iloc[-6:]).prod() - 1
    
    # 价值得分（低估值得分高）
    value = -pe_data.iloc[-1].rank()
    
    # 综合得分
    composite = value_weight * value.rank() + (1 - value_weight) * momentum.rank()
    
    return composite
```

## 五、机器学习行业轮动

### 5.1 特征工程

机器学习轮动需要构建丰富的特征：

**价格特征：**
- 动量：多周期动量（1月、3月、6月、12月）
- 波动率：历史波动率、下行波动率
- 换手率：成交量、换手率
- 技术指标：RSI、MACD、布林带

**基本面特征：**
- 估值：PE、PB、PS、PEG
- 盈利：ROE、毛利率、净利率
- 成长：收入增速、利润增速
- 质量：资产负债率、现金流质量

**宏观特征：**
- 利率：国债收益率、期限利差
- 流动性：M2增速、社融增速
- 通胀：CPI、PPI
- 经济：PMI、工业增加值

**情绪特征：**
- 北向资金：流入流出
- 融资余额：变化率
- 换手率：行业换手率
- 分析师预期：盈利预测调整

### 5.2 标签构造

**分类标签：**
$$Label = \begin{cases} 1 & \text{if } r_{i,t+1} > r_{market,t+1} + \theta \\ 0 & \text{if } |r_{i,t+1} - r_{market,t+1}| \leq \theta \\ -1 & \text{if } r_{i,t+1} < r_{market,t+1} - \theta \end{cases}$$

**回归标签：**
$$Label = r_{i,t+1}$$

**排名标签：**
$$Label = rank(r_{i,t+1})$$

### 5.3 模型选择

**树模型：**
- 随机森林：稳健、不易过拟合
- XGBoost/LightGBM：精度高、速度快
- CatBoost：处理类别特征能力强

**神经网络：**
- LSTM：捕捉时序依赖
- Transformer：注意力机制
- MLP：简单高效

**集成学习：**
- 多模型投票
- Stacking
- 分层模型

```python
import pandas as pd
import numpy as np
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import TimeSeriesSplit
import lightgbm as lgb

class MLSectorRotation:
    """
    机器学习行业轮动
    """
    def __init__(self, features, labels, model_type='lgb'):
        self.features = features
        self.labels = labels
        self.model_type = model_type
        self.model = None
        
    def build_features(self, price_data, fundamental_data, macro_data):
        """
        构建特征
        """
        features = pd.DataFrame()
        
        # 动量特征
        for period in [1, 3, 6, 12]:
            features[f'momentum_{period}'] = price_data.pct_change(period * 20)
        
        # 波动率特征
        features['volatility'] = price_data.pct_change().rolling(20).std()
        
        # 估值特征
        features['pe'] = fundamental_data['pe']
        features['pb'] = fundamental_data['pb']
        
        # 宏观特征
        features['rate_10y'] = macro_data['rate_10y']
        features['m2_growth'] = macro_data['m2_growth']
        
        return features.fillna(method='ffill')
    
    def train(self, train_features, train_labels):
        """
        训练模型
        """
        if self.model_type == 'rf':
            self.model = RandomForestClassifier(
                n_estimators=200,
                max_depth=5,
                min_samples_leaf=10,
                random_state=42
            )
        elif self.model_type == 'lgb':
            self.model = lgb.LGBMClassifier(
                n_estimators=200,
                max_depth=5,
                learning_rate=0.05,
                random_state=42
            )
        
        self.model.fit(train_features, train_labels)
        
    def predict(self, features):
        """
        预测
        """
        return self.model.predict_proba(features)
    
    def backtest(self, features, labels, n_splits=5):
        """
        回测（使用时序交叉验证）
        """
        tscv = TimeSeriesSplit(n_splits=n_splits)
        results = []
        
        for train_idx, test_idx in tscv.split(features):
            X_train, X_test = features.iloc[train_idx], features.iloc[test_idx]
            y_train, y_test = labels.iloc[train_idx], labels.iloc[test_idx]
            
            self.train(X_train, y_train)
            pred_proba = self.predict(X_test)
            
            # 选择预测概率最高的行业
            top_sectors = pred_proba[:, 1].argsort()[-5:]
            
            results.append({
                'predictions': top_sectors,
                'actuals': y_test.iloc[top_sectors].values
            })
        
        return results
    
    def feature_importance(self):
        """
        特征重要性
        """
        if self.model is None:
            return None
        
        importance = pd.DataFrame({
            'feature': self.features.columns,
            'importance': self.model.feature_importances_
        }).sort_values('importance', ascending=False)
        
        return importance
```

```python
def composite_rotation_strategy(momentum_score, value_score, sentiment_score, ml_score):
    """
    复合轮动策略
    """
    composite = (
        0.30 * momentum_score.rank() +
        0.30 * (-value_score).rank() +  # 低估值得分高
        0.25 * sentiment_score.rank() +
        0.15 * ml_score.rank()
    )
    
    # 选择综合得分最高的5个行业
    top_sectors = composite.nlargest(5)
    
    # 按得分加权
    weights = top_sectors / top_sectors.sum()
    
    return weights
```

```python
def sector_rotation_backtest(etf_prices, rebalance_freq='M'):

    # 计算特征
    returns = etf_prices.pct_change()
    momentum_6m = (1 + returns.rolling(120)).prod() - 1
    
    # 回测循环
    portfolio_value = [1.0]
    holdings = []
    
    for i in range(120, len(etf_prices), 20):  # 月度调仓
        # 计算动量
        mom = momentum_6m.iloc[i-1]
        
        # 选择动量前5
        top_5 = mom.nlargest(5).index
        
        # 计算下期收益
        next_return = returns.iloc[i:i+20][top_5].mean().mean()
        portfolio_value.append(portfolio_value[-1] * (1 + next_return))
        
        holdings.append({
            'date': etf_prices.index[i],
            'sectors': list(top_5),
            'value': portfolio_value[-1]
        })
    
    return pd.Series(portfolio_value), pd.DataFrame(holdings)
```

## 七、风险控制

### 7.1 行业集中度风险

**风险度量：**
$$Concentration = \sum_{i=1}^{N} w_i^2$$

**控制方法：**
- 单一行业权重上限（如20%）
- 行业数量下限（如至少3个）
- 行业分散度约束

### 7.2 换手率控制

**换手率公式：**
$$Turnover = \frac{1}{2}\sum_{i=1}^{N}|w_{i,t} - w_{i,t-1}|$$

**控制方法：**
- 延长调仓周期
- 设置换手率上限
- 引入换手惩罚

### 7.3 回撤控制

**止损机制：**
```python
def apply_stop_loss(portfolio_value, max_drawdown=0.15):
    """
    应用止损
    """
    peak = portfolio_value.cummax()
    drawdown = (portfolio_value - peak) / peak
    
    if drawdown.iloc[-1] < -max_drawdown:
        return True  # 触发止损
    return False
```
