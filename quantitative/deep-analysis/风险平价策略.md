# 风险平价策略深度分析

## 一、风险平价理论基础

### 1.1 传统资产配置的局限性

传统的资产配置方法（如60/40股债组合）存在显著的"风险集中"问题。尽管债券配置比例较高，但由于股票波动率远高于债券，组合风险主要由股票贡献：

**传统60/40组合风险贡献分析：**
- 股票权重：60%
- 债券权重：40%
- 股票波动率：≈20%
- 债券波动率：≈5%
- 组合波动率：≈12%
- **股票风险贡献：约90%**
- **债券风险贡献：约10%**

这意味着所谓的"分散化"实际上是伪分散——组合表现几乎完全由股票决定。

### 1.2 风险平价核心理念

风险平价（Risk Parity）由Edward Qian于2005年提出，核心思想是：**让各类资产对组合总风险的贡献相等**。

**风险平价的目标：**
$$RC_i = RC_j, \quad \forall i, j$$

其中$RC_i$为资产$i$的风险贡献。

### 1.3 风险贡献的定义

资产$i$的风险贡献定义为：

$$RC_i = w_i \cdot \frac{\partial \sigma_p}{\partial w_i} = w_i \cdot \frac{(\Sigma w)_i}{\sigma_p}$$

其中：
- $w_i$：资产$i$的权重
- $\Sigma$：协方差矩阵
- $\sigma_p$：组合波动率

**风险贡献的经济含义：**
- 边际风险贡献：$\frac{\partial \sigma_p}{\partial w_i}$ 表示权重微增对组合风险的边际影响
- 总风险贡献：权重 × 边际风险贡献

**组合波动率公式：**
$$\sigma_p = \sqrt{w^T \Sigma w}$$

**风险贡献满足性质：**
$$\sum_{i=1}^{N} RC_i = \sigma_p$$

## 二、资产类别风险贡献计算

### 2.1 数学推导

给定$N$个资产，风险平价要求：

$$RC_i = RC_j, \quad \forall i,j$$

即：

$$w_i \cdot \frac{(\Sigma w)_i}{\sigma_p} = w_j \cdot \frac{(\Sigma w)_j}{\sigma_p}$$

化简得：

$$w_i \cdot (\Sigma w)_i = w_j \cdot (\Sigma w)_j$$

### 2.2 求解方法

#### 等权风险贡献（ERC）优化

**目标函数：**
$$\min_w \sum_{i=1}^{N} \sum_{j=1}^{N} (RC_i - RC_j)^2$$

**约束条件：**
$$\sum_{i=1}^{N} w_i = 1, \quad w_i \geq 0$$

#### 对数变换法

将风险贡献相等转化为优化问题：

$$\min_w \sum_{i=1}^{N} \left[ \ln(RC_i) - \ln\left(\frac{\sigma_p}{N}\right) \right]^2$$

#### 闭式解（简化情况）

当资产相关性为零时，风险平价权重有解析解：

$$w_i = \frac{1/\sigma_i}{\sum_{j=1}^{N} 1/\sigma_j}$$

即权重与波动率成反比。

### 2.3 Python实现

```python
import numpy as np
from scipy.optimize import minimize

def risk_parity_weights(cov_matrix):
    """
    计算风险平价权重
    
    参数:
        cov_matrix: 协方差矩阵 (N x N)
    
    返回:
        weights: 风险平价权重
    """
    n = cov_matrix.shape[0]
    
    def objective(w):
        # 组合波动率
        port_vol = np.sqrt(w @ cov_matrix @ w)
        # 各资产风险贡献
        marginal_contrib = cov_matrix @ w
        risk_contrib = w * marginal_contrib / port_vol
        # 目标：风险贡献方差最小
        target_rc = port_vol / n
        return np.sum((risk_contrib - target_rc) ** 2)
    
    # 约束条件
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = tuple((0, 1) for _ in range(n))
    
    # 初始猜测（逆波动率加权）
    vols = np.sqrt(np.diag(cov_matrix))
    w0 = (1 / vols) / np.sum(1 / vols)
    
    result = minimize(objective, w0, method='SLSQP',
                      bounds=bounds, constraints=constraints)
    
    return result.x

def risk_contribution(weights, cov_matrix):
    """
    计算各资产的风险贡献
    """
    port_vol = np.sqrt(weights @ cov_matrix @ weights)
    marginal_contrib = cov_matrix @ weights
    rc = weights * marginal_contrib / port_vol
    rc_pct = rc / port_vol  # 风险贡献百分比
    return rc, rc_pct
```

### 2.4 风险平价示例

**假设资产特征：**
| 资产 | 预期收益 | 波动率 | 相关性矩阵 |
|------|---------|--------|-----------|
| 股票 | 8% | 20% | 1.0 0.2 0.1 |
| 债券 | 4% | 5% | 0.2 1.0 0.0 |
| 商品 | 6% | 15% | 0.1 0.0 1.0 |

**风险平价权重计算：**
```python
# 协方差矩阵
vols = np.array([0.20, 0.05, 0.15])
corr = np.array([[1.0, 0.2, 0.1],
                 [0.2, 1.0, 0.0],
                 [0.1, 0.0, 1.0]])
cov = np.outer(vols, vols) * corr

weights = risk_parity_weights(cov)
# 结果约: 股票 15%, 债券 55%, 商品 30%
```

## 三、杠杆与风险平价的结合

### 3.1 杠杆的必要性

风险平价组合通常给低波动资产（如债券）更高权重，导致组合整体波动率较低（约5-8%）。为获得与股票可比的收益，需要引入杠杆。

**杠杆的作用：**
- 提高低波动资产的收益贡献
- 实现目标波动率
- 保持风险贡献相等

### 3.2 目标波动率法

设定目标波动率$\sigma_{target}$，计算杠杆倍数：

$$Leverage = \frac{\sigma_{target}}{\sigma_{RP}}$$

其中$\sigma_{RP}$为风险平价组合的自然波动率。

**杠杆后权重：**
$$w_i^{levered} = w_i \times Leverage$$

### 3.3 杠杆成本考虑

使用杠杆需要考虑融资成本：

$$R_{levered} = Leverage \times R_{portfolio} - (Leverage - 1) \times r_f$$

其中$r_f$为无风险利率。

**调整后的风险平价目标：**
$$\max_w Sharpe = \frac{w^T \mu - (L-1) r_f}{\sqrt{w^T \Sigma w}}$$

### 3.4 杠杆风险

**风险放大：**
- 波动率放大
- 回撤放大
- 极端行情下可能爆仓

**流动性风险：**
- 融资成本上升
- 追加保证金要求
- 强制平仓风险

**杠杆管理建议：**
- 杠杆倍数控制在2倍以内
- 设置止损线
- 动态调整杠杆水平

### 5.2 风险平价组合代码

```python
import pandas as pd
import numpy as np
from scipy.optimize import minimize

class RiskParityPortfolio:
    def __init__(self, returns_df):
        """
        初始化风险平价组合
        
        参数:
            returns_df: 收益率DataFrame，列为资产，行为时间
        """
        self.returns = returns_df
        self.cov_matrix = self._estimate_cov()
        
    def _estimate_cov(self, method='ledoit_wolf'):
        """
        协方差矩阵估计
        """
        if method == 'sample':
            return self.returns.cov().values
        elif method == 'ledoit_wolf':
            from sklearn.covariance import LedoitWolf
            lw = LedoitWolf().fit(self.returns.values)
            return lw.covariance_
        elif method == 'ewma':
            span = min(60, len(self.returns) // 4)
            return self.returns.ewm(span=span).cov().iloc[-1].values
    
    def optimize(self):
        """
        求解风险平价权重
        """
        n = self.cov_matrix.shape[0]
        
        def objective(w):
            port_vol = np.sqrt(w @ self.cov_matrix @ w)
            marginal = self.cov_matrix @ w
            rc = w * marginal / port_vol
            target = port_vol / n
            return np.sum((rc - target) ** 2)
        
        constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        bounds = tuple((0.02, 0.5) for _ in range(n))  # 限制权重范围
        
        # 初始权重（逆波动率）
        vols = np.sqrt(np.diag(self.cov_matrix))
        w0 = (1 / vols) / np.sum(1 / vols)
        
        result = minimize(objective, w0, method='SLSQP',
                          bounds=bounds, constraints=constraints)
        
        self.weights = result.x
        return self.weights
    
    def risk_contribution(self):
        """
        计算风险贡献
        """
        port_vol = np.sqrt(self.weights @ self.cov_matrix @ self.weights)
        marginal = self.cov_matrix @ self.weights
        rc = self.weights * marginal / port_vol
        return rc / port_vol  # 百分比
    
    def leverage(self, target_vol=0.10):
        """
        计算杠杆比例
        """
        current_vol = np.sqrt(self.weights @ self.cov_matrix @ self.weights)
        return target_vol / current_vol
    
    def rebalance(self, new_returns):
        """
        组合再平衡
        """
        self.returns = new_returns
        self.cov_matrix = self._estimate_cov()
        return self.optimize()
```

### 5.3 回测框架

```python
def backtest_risk_parity(etf_returns, rebalance_freq='M'):
    """
    风险平价回测
    
    参数:
        etf_returns: ETF收益率DataFrame
        rebalance_freq: 再平衡频率 ('M'=月度, 'Q'=季度)
    
    返回:
        组合净值曲线
    """
    dates = etf_returns.index
    lookback = 252  # 一年历史数据
    
    portfolio_values = [1.0]
    weights_history = []
    
    for i, date in enumerate(dates[lookback:]):
        # 使用历史数据估计协方差
        hist_returns = etf_returns.iloc[i:i+lookback]
        
        # 优化权重
        rp = RiskParityPortfolio(hist_returns)
        weights = rp.optimize()
        weights_history.append(weights)
        
        # 计算下期收益
        if i + lookback < len(dates):
            next_return = etf_returns.iloc[i + lookback]
            port_return = np.dot(weights, next_return)
            portfolio_values.append(portfolio_values[-1] * (1 + port_return))
    
    return pd.Series(portfolio_values, index=dates[lookback-1:])
```

```python
def portfolio_metrics(portfolio_values):
    """
    计算组合绩效指标
    """
    returns = portfolio_values.pct_change().dropna()
    
    metrics = {
        '年化收益': returns.mean() * 252,
        '年化波动': returns.std() * np.sqrt(252),
        '夏普比率': returns.mean() / returns.std() * np.sqrt(252),
        '最大回撤': (portfolio_values / portfolio_values.cummax() - 1).min(),
        '卡玛比率': returns.mean() * 252 / abs((portfolio_values / portfolio_values.cummax() - 1).min()),
        '月度胜率': (returns > 0).mean(),
    }
    
    return metrics
```

## 六、风险平价的风险与局限

### 6.1 模型风险

**协方差估计误差：**
- 历史数据不等于未来
- 极端行情下相关性可能急剧变化
- 收缩估计、随机波动模型可缓解

**因子模型依赖：**
- 风险平价假设协方差矩阵稳定
- 实际中市场状态转换频繁

### 6.2 杠杆风险

**放大损失：**
- 2008年金融危机期间，部分杠杆风险平价策略损失惨重
- 流动性枯竭时难以去杠杆

**融资成本：**
- 杠杆成本侵蚀收益
- 利率上升时期尤为明显

### 6.3 市场风险

**相关性失效：**
- 极端行情下资产相关性趋近于1
- "分散化消失"现象

**通胀风险：**
- 债券权重高，通胀上行期受损
- 需要纳入抗通胀资产（TIPS、商品）

### 6.4 应对措施

**动态协方差：**
- 使用GARCH模型估计时变波动率
- 加入尾部风险指标

**风险平价变体：**
- **风险预算（Risk Budgeting）：** 允许不等风险贡献
- **层级风险平价：** 先资产类别内平价，再类别间平价
- **条件风险平价：** 根据市场状态动态调整

**尾部风险对冲：**
- 配置VIX期货
- 购买看跌期权
- 尾部风险基金

## 七、总结与建议

### 7.1 核心要点

1. **风险平价本质：** 让各类资产对组合风险的贡献相等，实现真正的分散化
2. **杠杆必要性：** 低波动组合需要杠杆才能获得可观收益
3. **A股实践：** 股票波动率高导致债券权重很高，需要多资产类别
4. **动态调整：** 协方差矩阵需要定期更新，适应市场变化

### 7.2 实施建议

**资产选择：**
- 至少包含4-5个低相关性资产类别
- 权益、固收、商品、海外资产均衡配置
- 考虑通胀保值资产

**杠杆管理：**
- 目标波动率设定在8-12%为宜
- 杠杆倍数控制在2倍以内
- 设置止损和动态降杠杆机制

**再平衡频率：**
- 月度或季度再平衡
- 协方差矩阵更新频率与再平衡频率匹配
- 极端行情下可临时调整

**风险监控：**
- 实时监控各资产风险贡献
- 关注资产相关性变化
- 建立尾部风险预警机制

风险平价是一种科学、系统的资产配置方法，在A股市场同样具有应用价值。关键是根据A股特点进行本地化调整，并严格控制杠杆风险。
