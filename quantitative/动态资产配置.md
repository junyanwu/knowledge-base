# 动态资产配置

## 1. 概述

### 1.1 定义

**动态资产配置：** 根据市场环境变化，动态调整资产配置比例的策略。

### 1.2 分类

| 类型 | 特点 | 频率 |
|------|------|------|
| 战略资产配置 | 长期目标配置 | 年度调整 |
| 战术资产配置 | 短期机会配置 | 月度/季度调整 |
| 动态资产配置 | 规则化调整 | 日度/周度调整 |

## 2. 战略资产配置

### 2.1 长期目标配置

**原理：** 基于长期投资目标和风险承受能力，确定基准配置。

**方法：**
```python
def strategic_allocation(expected_returns, cov_matrix, risk_aversion=2.5):
    """
    战略资产配置（均值方差优化）
    
    参数:
    expected_returns: 期望收益
    cov_matrix: 协方差矩阵
    risk_aversion: 风险厌恶系数
    
    返回:
    最优权重
    """
    from scipy.optimize import minimize
    
    n_assets = len(expected_returns)
    
    def neg_utility(weights):
        portfolio_return = weights @ expected_returns
        portfolio_var = weights @ cov_matrix @ weights
        utility = portfolio_return - 0.5 * risk_aversion * portfolio_var
        return -utility
    
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    result = minimize(
        neg_utility,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x

# 示例
expected_returns = np.array([0.08, 0.06, 0.04, 0.03])
cov_matrix = np.array([
    [0.04, 0.01, 0.005, 0.002],
    [0.01, 0.0225, 0.0075, 0.003],
    [0.005, 0.0075, 0.01, 0.004],
    [0.002, 0.003, 0.004, 0.0064]
])

strategic_weights = strategic_allocation(expected_returns, cov_matrix)
print(f'战略配置: {strategic_weights}')
```

### 2.2 再平衡策略

**固定比例再平衡：**
```python
def rebalance_fixed(current_weights, target_weights, threshold=0.05):
    """
    固定比例再平衡
    
    参数:
    current_weights: 当前权重
    target_weights: 目标权重
    threshold: 偏离阈值
    
    返回:
    是否需要再平衡、调整权重
    """
    # 偏离度
    deviation = np.abs(current_weights - target_weights)
    
    # 是否需要再平衡
    need_rebalance = np.any(deviation > threshold)
    
    if need_rebalance:
        adjust_weights = target_weights - current_weights
    else:
        adjust_weights = np.zeros_like(current_weights)
    
    return need_rebalance, adjust_weights
```

**时间驱动再平衡：**
```python
def time_driven_rebalance(last_rebalance, current_date, frequency='quarterly'):
    """
    时间驱动再平衡
    
    参数:
    last_rebalance: 上次再平衡日期
    current_date: 当前日期
    frequency: 频率（monthly, quarterly, annually）
    
    返回:
    是否需要再平衡
    """
    days_diff = (current_date - last_rebalance).days
    
    if frequency == 'monthly':
        rebalance_days = 30
    elif frequency == 'quarterly':
        rebalance_days = 90
    elif frequency == 'annually':
        rebalance_days = 365
    else:
        rebalance_days = 90
    
    return days_diff >= rebalance_days
```

## 3. 战术资产配置

### 3.1 市场时机

**价值指标：**
```python
def valuation_timing(current_valuation, historical_mean, threshold=1.0):
    """
    估值时机
    
    参数:
    current_valuation: 当前估值（PE、PB等）
    historical_mean: 历史均值
    threshold: 触发阈值
    
    返回:
    配置建议
    """
    valuation_ratio = current_valuation / historical_mean
    
    if valuation_ratio < 1 - threshold:
        return 'overweight'  # 低估，超配
    elif valuation_ratio > 1 + threshold:
        return 'underweight'  # 高估，低配
    else:
        return 'neutral'  # 中性
```

**动量指标：**
```python
def momentum_timing(returns, lookback=12):
    """
    动量时机
    
    参数:
    returns: 收益率
    lookback: 回溯期（月）
    
    返回:
    配置建议
    """
    momentum = (1 + returns).rolling(lookback).apply(np.prod, raw=True) - 1
    
    if momentum.iloc[-1] > 0:
        return 'overweight'
    elif momentum.iloc[-1] < 0:
        return 'underweight'
    else:
        return 'neutral'
```

def dynamic_risk_parity(returns, window=60, target_vol=0.10):
    """
    动态风险平价
    
    参数:
    returns: 收益率序列
    window: 滚动窗口
    target_vol: 目标波动率
    
    返回:
    动态权重序列

    n_periods = len(returns)
    n_assets = returns.shape[1]
    
    weights_series = []

### 4.2 因子动态配置

```python
def factor_dynamic_allocation(factor_returns, factor_budget, window=60):
    """
    因子动态配置
    
    参数:
    factor_returns: 因子收益率
    factor_budget: 因子风险预算
    window: 滚动窗口
    
    返回:
    动态因子暴露
    """
    n_periods = len(factor_returns)
    n_factors = factor_returns.shape[1]
    
    factor_exposure_series = []
    
    for i in range(window, n_periods):
        # 滚动因子协方差
        window_returns = factor_returns.iloc[i-window:i]
        factor_cov = window_returns.cov().values
        
        # 因子风险预算
        exposure = factor_risk_budget_optimization(factor_cov, factor_budget)
        factor_exposure_series.append(exposure)
    
    return pd.DataFrame(factor_exposure_series, index=factor_returns.index[window:])

def factor_risk_budget_optimization(factor_cov, factor_budget):
    """
    因子风险预算优化
    """
    n_factors = len(factor_budget)
    
    def objective(exposure):
        factor_var = exposure @ factor_cov @ exposure
        marginal_factor_risk = factor_cov @ exposure
        factor_risk_contrib = exposure * marginal_factor_risk
        factor_risk_pct = factor_risk_contrib / factor_var
        
        return np.sum((factor_risk_pct - factor_budget) ** 2)
    
    from scipy.optimize import minimize
    
    constraints = {'type': 'eq', 'fun': lambda e: np.sum(e) - 1}
    
    result = minimize(
        objective,
        x0=[1/n_factors] * n_factors,
        method='SLSQP',
        constraints=constraints
    )
    
    return result.x
```

### 4.3 波动率目标配置

```python
def volatility_targeting(returns, target_vol=0.10, window=60):
    """
    波动率目标配置
    
    参数:
    returns: 收益率序列
    target_vol: 目标波动率
    window: 滚动窗口
    
    返回:
    动态杠杆
    """
    # 滚动波动率
    rolling_vol = returns.rolling(window).std() * np.sqrt(252)
    
    # 动态杠杆
    leverage = target_vol / rolling_vol
    
    # 限制杠杆
    leverage = leverage.clip(0.5, 2.0)
    
    return leverage
```

## 5. 实战案例

### 5.1 完整动态配置策略

```python
class DynamicAssetAllocation:
    """动态资产配置策略"""
    
    def __init__(self, assets, strategic_weights):
        self.assets = assets
        self.strategic_weights = strategic_weights
        self.current_weights = strategic_weights.copy()
    
    def tactical_adjustment(self, signals):
        """
        战术调整
        
        参数:
        signals: 信号字典 {'equity': 'overweight', ...}
        
        返回:
        调整后权重
        """
        weights = self.strategic_weights.copy()
        
        for asset, signal in signals.items():
            if signal == 'overweight':
                weights[asset] *= 1.2
            elif signal == 'underweight':
                weights[asset] *= 0.8
        
        # 归一化
        weights = weights / weights.sum()
        
        return weights
    
    def risk_adjustment(self, weights, cov_matrix, target_vol=0.10):
        """
        风险调整
        
        参数:
        weights: 当前权重
        cov_matrix: 协方差矩阵
        target_vol: 目标波动率
        
        返回:
        调整后权重
        """
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 杠杆
        leverage = target_vol / portfolio_vol
        leverage = np.clip(leverage, 0.5, 2.0)
        
        # 调整权重
        adjusted_weights = weights * leverage
        
        return adjusted_weights, leverage
    
    def rebalance(self, current_weights, target_weights, threshold=0.05):
        """
        再平衡
        
        参数:
        current_weights: 当前权重
        target_weights: 目标权重
        threshold: 再平衡阈值
        
        返回:
        是否需要再平衡
        """
        deviation = np.abs(current_weights - target_weights)
        return np.any(deviation > threshold)
```

### 5.2 多策略组合

```python
def multi_strategy_allocation(returns, strategies):
    """
    多策略组合
    
    参数:
    returns: 收益率序列
    strategies: 策略列表
    
    返回:
    组合权重
    """
    strategy_weights = []
    
    for strategy in strategies:
        weights = strategy(returns)
        strategy_weights.append(weights)
    
    # 等权组合
    combined_weights = np.mean(strategy_weights, axis=0)
    
    return combined_weights
```

## 6. 绩效评估

### 6.1 指标计算

```python
def evaluate_dynamic_allocation(portfolio_returns, benchmark_returns):
    """
    评估动态配置策略
    
    参数:
    portfolio_returns: 策略收益率
    benchmark_returns: 基准收益率
    
    返回:
    绩效指标
    """
    # 年化收益
    annual_return = portfolio_returns.mean() * 252
    benchmark_return = benchmark_returns.mean() * 252
    
    # 年化波动率
    annual_vol = portfolio_returns.std() * np.sqrt(252)
    benchmark_vol = benchmark_returns.std() * np.sqrt(252)
    
    # 夏普比率
    sharpe = annual_return / annual_vol
    benchmark_sharpe = benchmark_return / benchmark_vol
    
    # 最大回撤
    cumulative = (1 + portfolio_returns).cumprod()
    drawdown = (cumulative - cumulative.cummax()) / cumulative.cummax()
    max_drawdown = drawdown.min()
    
    # 信息比率
    active_return = portfolio_returns - benchmark_returns
    information_ratio = active_return.mean() / active_return.std() * np.sqrt(252)
    
    return {
        'annual_return': annual_return,
        'benchmark_return': benchmark_return,
        'annual_vol': annual_vol,
        'benchmark_vol': benchmark_vol,
        'sharpe': sharpe,
        'benchmark_sharpe': benchmark_sharpe,
        'max_drawdown': max_drawdown,
        'information_ratio': information_ratio
    }
```

## 7. 总结

动态资产配置要点：

1. **战略配置**：长期目标、基准配置
2. **战术配置**：市场时机、宏观因子
3. **动态调整**：风险平价、因子暴露、波动率目标
4. **再平衡**：阈值驱动、时间驱动
5. **绩效评估**：夏普比率、信息比率、最大回撤

**注意事项：**
- 交易成本
- 估计误差
- 模型风险
- 流动性约束
