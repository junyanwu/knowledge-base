## 一、时间序列基础概念

### 1.1 时间序列的定义

**时间序列**：按时间顺序排列的一组观测值。

**表示**：
```
{Xₜ, t = 1, 2, ..., T}
或
X₁, X₂, ..., Xₜ, ...
```

**特点**：
- 时间依赖性：当前值与过去值相关
- 趋势性：长期向上或向下的趋势
- 季节性：周期性波动
- 随机性：不可预测的波动

### 1.2 时间序列的分类

#### 按性质分类

**1) 平稳时间序列**
- 均值恒定
- 方差恒定
- 协方差只与时滞有关

**2) 非平稳时间序列**
- 均值、方差或协方差随时间变化
- 包含趋势、季节性等

#### 按频率分类

**1) 低频数据**
- 年度数据、季度数据、月度数据

**2) 高频数据**
- 日数据、小时数据、分钟数据
- 逐笔数据（tick data）

### 1.3 时间序列分析的目标

**1) 描述性分析**
- 识别趋势、季节性、周期性
- 计算描述性统计量

**2) 预测**
- 基于历史数据预测未来值
- 评估预测精度

**3) 解释性分析**
- 分析变量间的关系
- 识别因果关系

**4) 控制与优化**
- 监控过程稳定性
- 优化决策

## 二、平稳性

### 2.1 严平稳

**定义**：时间序列 {Xₜ} 的任意有限维分布不随时间平移而改变。

**数学表达**：
```
对任意 t₁, t₂, ..., tₙ 和任意 k，
(X_{t₁}, X_{t₂}, ..., X_{tₙ}) 的联合分布与
(X_{t₁+k}, X_{t₂+k}, ..., X_{tₙ+k}) 的联合分布相同
```

### 2.2 弱平稳（宽平稳）

**定义**：时间序列 {Xₜ} 满足以下三个条件：

**条件1（均值恒定）**：
```
E(Xₜ) = μ（常数）
```

**条件2（方差恒定）**：
```
Var(Xₜ) = σ²（常数）
```

**条件3（协方差只与时滞有关）**：
```
Cov(Xₜ, X_{t+k}) = γ(k)
```

**自协方差函数**：
```
γ(k) = Cov(Xₜ, X_{t+k}) = E[(Xₜ - μ)(X_{t+k} - μ)]
```

**自相关函数（ACF）**：
```
ρ(k) = γ(k) / γ(0) = Cov(Xₜ, X_{t+k}) / Var(Xₜ)
```

### 2.3 白噪声

**定义**：如果 {εₜ} 是独立同分布的随机变量，且

```
E(εₜ) = 0
Var(εₜ) = σ²
Cov(εₜ, εₛ) = 0, t ≠ s
```

则称 {εₜ} 为白噪声序列，记为 εₜ ~ WN(0, σ²)。

**性质**：
```
γ(k) = {
  σ², k = 0
  0,  k ≠ 0
}

ρ(k) = {
  1, k = 0
  0, k ≠ 0
}
```

### 2.4 平稳性检验

#### 1) 图形检验

**时间序列图**：
- 平稳序列：围绕均值上下波动
- 非平稳序列：有明显的趋势或波动幅度变化

**自相关图**：
- 平稳序列：ACF快速衰减
- 非平稳序列：ACF缓慢衰减

#### 2) 单位根检验

**ADF检验（Augmented Dickey-Fuller Test）**

**原假设**：存在单位根（非平稳）

**检验模型**：
```
ΔXₜ = α + βt + γX_{t-1} + Σ δᵢ ΔX_{t-i} + εₜ
```

**检验统计量**：
```
t = γ̂ / SE(γ̂)
```

**判断准则**：
```
如果 t < 临界值，则拒绝原假设，序列平稳
如果 t > 临界值，则不能拒绝原假设，序列非平稳
```

**案例**：
```python
from statsmodels.tsa.stattools import adfuller

# ADF检验
result = adfuller(data, maxlag=12, regression='c')
adf_statistic = result[0]
p_value = result[1]
critical_values = result[4]

print(f'ADF统计量: {adf_statistic:.4f}')
print(f'p值: {p_value:.4f}')
print('临界值:')
for key, value in critical_values.items():
    print(f'  {key}: {value:.4f}')

# 判断
if p_value < 0.05:
    print('序列平稳（拒绝原假设）')
else:
    print('序列非平稳（不能拒绝原假设）')
```

**PP检验（Phillips-Perron Test）**
- 不需要指定滞后阶数
- 对异方差更稳健

**KPSS检验**
- 原假设：序列平稳
- 与ADF检验互补

## 三、ARMA模型

### 3.1 AR模型（自回归模型）

**AR(p)模型**：
```
Xₜ = c + φ₁X_{t-1} + φ₂X_{t-2} + ... + φₚX_{t-p} + εₜ
```

或写为：
```
Xₜ = c + Σ φᵢX_{t-i} + εₜ
```

**特征方程**：
```
φ(z) = 1 - φ₁z - φ₂z² - ... - φₚzᵖ = 0
```

**平稳性条件**：特征方程的所有根在单位圆外。

**案例：AR(1)模型**
```
Xₜ = c + φX_{t-1} + εₜ

平稳条件：|φ| < 1

均值：E(Xₜ) = c / (1 - φ)
方差：Var(Xₜ) = σ² / (1 - φ²)
自相关：ρ(k) = φ^k
```

**AR(p)的自相关函数**：
```
AR(1): ρ(k) = φ₁^k（指数衰减）
AR(2): ρ(k) = A₁λ₁^k + A₂λ₂^k（指数衰减或震荡衰减）
```

### 3.2 MA模型（移动平均模型）

**MA(q)模型**：
```
Xₜ = μ + εₜ + θ₁ε_{t-1} + θ₂ε_{t-2} + ... + θ_qε_{t-q}
```

或写为：
```
Xₜ = μ + εₜ + Σ θᵢε_{t-i}
```

**性质**：
- 任何MA(q)模型都是平稳的
- 可逆性条件：特征方程的所有根在单位圆外

**案例：MA(1)模型**
```
Xₜ = μ + εₜ + θε_{t-1}

均值：E(Xₜ) = μ
方差：Var(Xₜ) = (1 + θ²)σ²
自相关：
  ρ(1) = θ / (1 + θ²)
  ρ(k) = 0, k > 1
```

**MA(q)的自相关函数**：
```
ρ(k) = {
  θ_k + Σ θᵢθ_{i+k} / (1 + θ₁² + ... + θ_q²), k ≤ q
  0, k > q
}
```

MA模型的ACF在滞后q后截尾。

### 3.3 ARMA模型（自回归移动平均模型）

**ARMA(p,q)模型**：
```
Xₜ = c + φ₁X_{t-1} + ... + φₚX_{t-p} + εₜ + θ₁ε_{t-1} + ... + θ_qε_{t-q}
```

或写为：
```
φ(B)Xₜ = c + θ(B)εₜ

其中：
φ(B) = 1 - φ₁B - φ₂B² - ... - φₚBᵖ
θ(B) = 1 + θ₁B + θ₂B² + ... + θ_qB_q
B：滞后算子，BXₜ = X_{t-1}
```

**性质**：
- 平稳性：AR部分的特征根在单位圆外
- 可逆性：MA部分的特征根在单位圆外

**ARMA(1,1)的ACF**：
```
ρ(k) 按指数衰减（或震荡衰减）
```

### 3.4 ARMA模型的定阶

#### 1) ACF和PACF

**AR(p)模型**：
```
ACF：拖尾（指数衰减）
PACF：p阶截尾
```

**MA(q)模型**：
```
ACF：q阶截尾
PACF：拖尾
```

**ARMA(p,q)模型**：
```
ACF：拖尾
PACF：拖尾
```

#### 2) 信息准则

**AIC（Akaike Information Criterion）**：
```
AIC = -2 ln(L) + 2k
```
- L：似然函数最大值
- k：参数个数

**BIC（Bayesian Information Criterion）**：
```
BIC = -2 ln(L) + k ln(T)
```
- T：样本量

**选择准则**：选择使AIC或BIC最小的模型。

**案例**：
```python
from statsmodels.tsa.arima.model import ARIMA
import warnings
warnings.filterwarnings('ignore')

# 模型选择
best_aic = float('inf')
best_order = None

for p in range(5):
    for q in range(5):
        try:
            model = ARIMA(data, order=(p, 0, q))
            results = model.fit()
            if results.aic < best_aic:
                best_aic = results.aic
                best_order = (p, 0, q)
        except:
            continue

print(f'最优模型: ARMA{best_order}')
print(f'AIC: {best_aic:.2f}')
```

## 四、ARIMA模型

### 4.1 差分运算

**一阶差分**：
```
ΔXₜ = Xₜ - X_{t-1}
```

**d阶差分**：
```
Δ^d Xₜ = Δ(Δ^{d-1} Xₜ)
```

**案例**：
```
原始序列：X₁, X₂, X₃, X₄, X₅
一阶差分：ΔX₂ = X₂ - X₁, ΔX₃ = X₃ - X₂, ...
二阶差分：Δ²X₃ = ΔX₃ - ΔX₂, Δ²X₄ = ΔX₄ - ΔX₃, ...
```

### 4.2 ARIMA模型

**ARIMA(p,d,q)模型**：
```
φ(B)(1-B)^d Xₜ = c + θ(B)εₜ
```

- **p**：自回归阶数
- **d**：差分阶数
- **q**：移动平均阶数

**等价形式**：
```
对 Δ^d Xₜ 建立ARMA(p,q)模型
```

**案例：ARIMA(1,1,1)**
```
ΔXₜ = φΔX_{t-1} + εₜ + θε_{t-1}
```

### 4.3 ARIMA建模流程

**步骤1：平稳性检验**
```
检验原序列是否平稳
如果非平稳，进行差分
确定差分阶数 d
```

**步骤2：模型识别**
```
对平稳序列绘制ACF和PACF
识别AR和MA的阶数 p 和 q
```

**步骤3：参数估计**
```
使用极大似然估计或最小二乘估计
估计φ₁, ..., φₚ, θ₁, ..., θ_q
```

**步骤4：模型诊断**
```
检验残差是否为白噪声
Ljung-Box检验
Q-Q图检验正态性
```

**步骤5：预测**
```
使用估计的模型进行预测
计算预测区间
评估预测精度
```

### 4.4 案例分析：股票收益率预测

**数据准备**：
```python
import pandas as pd
import numpy as np
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.tsa.stattools import adfuller
import matplotlib.pyplot as plt

# 读取股票数据
data = pd.read_csv('stock_price.csv', index_col='date', parse_dates=True)
returns = np.log(data['close']).diff().dropna()

# 平稳性检验
result = adfuller(returns)
print(f'ADF统计量: {result[0]:.4f}')
print(f'p值: {result[1]:.4f}')

# 绘制ACF和PACF
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
plot_acf(returns, ax=axes[0], lags=20)
plot_pacf(returns, ax=axes[1], lags=20)
plt.show()

# 模型拟合
model = ARIMA(returns, order=(1, 0, 1))
results = model.fit()
print(results.summary())

# 残差诊断
residuals = results.resid
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].plot(residuals)
axes[0].set_title('残差序列')
axes[1].hist(residuals, bins=30, density=True)
axes[1].set_title('残差分布')
plt.show()

# 预测
forecast = results.forecast(steps=10)
print('未来10期预测:')
print(forecast)
```

## 五、GARCH模型

### 5.1 波动率聚类

**现象**：金融时间序列的波动率呈现聚类现象，即大幅波动之后往往跟随大幅波动，小幅波动之后往往跟随小幅波动。

**特征**：
- 收益率序列本身平稳
- 收益率的平方或绝对值序列存在自相关
- 波动率随时间变化

### 5.2 ARCH模型

**ARCH(q)模型（Autoregressive Conditional Heteroskedasticity）**：

**均值方程**：
```
Xₜ = μ + εₜ
εₜ = σₜ zₜ, zₜ ~ N(0,1)
```

**方差方程**：
```
σₜ² = ω + α₁ε_{t-1}² + α₂ε_{t-2}² + ... + α_qε_{t-q}²
```

**约束条件**：
```
ω > 0, αᵢ ≥ 0
ω + Σ αᵢ < 1（平稳性条件）
```

**性质**：
```
E(εₜ) = 0
Var(εₜ) = ω / (1 - Σ αᵢ)
```

**ARCH效应检验（LM检验）**：
```
原假设：H₀: α₁ = α₂ = ... = α_q = 0
检验统计量：LM = T R² ~ χ²(q)
```

### 5.3 GARCH模型

**GARCH(p,q)模型**：

**均值方程**：
```
Xₜ = μ + εₜ
εₜ = σₜ zₜ, zₜ ~ N(0,1)
```

**方差方程**：
```
σₜ² = ω + Σ αᵢε_{t-i}² + Σ βⱼσ_{t-j}²
     = ω + α₁ε_{t-1}² + ... + α_qε_{t-q}² + β₁σ_{t-1}² + ... + β_pσ_{t-p}²
```

**约束条件**：
```
ω > 0, αᵢ ≥ 0, βⱼ ≥ 0
Σ αᵢ + Σ βⱼ < 1（平稳性条件）
```

**GARCH(1,1)模型**：
```
σₜ² = ω + αε_{t-1}² + βσ_{t-1}²

无条件方差：
Var(εₜ) = ω / (1 - α - β)
```

**案例**：
```python
from arch import arch_model

# 读取收益率数据
returns = pd.read_csv('returns.csv')['return']

# 拟合GARCH(1,1)模型
model = arch_model(returns, vol='Garch', p=1, q=1)
results = model.fit()
print(results.summary())

# 波动率预测
forecast = results.forecast(horizon=5)
print('未来5期波动率预测:')
print(np.sqrt(forecast.variance[-1:]))

# 绘制条件波动率
fig, ax = plt.subplots(figsize=(12, 6))
ax.plot(results.conditional_volatility)
ax.set_title('条件波动率')
ax.set_xlabel('时间')
ax.set_ylabel('波动率')
plt.show()
```

### 5.4 GARCH模型的扩展

#### 1) GJR-GARCH

**方差方程**：
```
σₜ² = ω + αε_{t-1}² + γε_{t-1}²I(ε_{t-1}<0) + βσ_{t-1}²
```

**特点**：捕捉杠杆效应（负收益对波动率的影响更大）

#### 2) EGARCH（Exponential GARCH）

**方差方程（对数形式）**：
```
ln(σₜ²) = ω + α|z_{t-1}| + γz_{t-1} + β ln(σ_{t-1}²)
```

**特点**：
- 无需非负约束
- 直接捕捉杠杆效应

#### 3) TGARCH（Threshold GARCH）

**方差方程**：
```
σₜ² = ω + αε_{t-1}² + γε_{t-1}²I(ε_{t-1}<0) + βσ_{t-1}²
```

### 5.5 GARCH模型在风险管理中的应用

**VaR（Value at Risk）计算**：

```
VaR = μ + σₜ × z_α

其中：
- μ：收益率均值
- σₜ：条件波动率
- z_α：标准正态分布的α分位数

95% VaR: z_{0.05} = -1.645
99% VaR: z_{0.01} = -2.326
```

**案例**：
```python
# 计算VaR
mu = returns.mean()
sigma = results.conditional_volatility.iloc[-1]
VaR_95 = mu + sigma * (-1.645)
VaR_99 = mu + sigma * (-2.326)

print(f'95% VaR: {VaR_95:.4f}')
print(f'99% VaR: {VaR_99:.4f}')
```

## 六、VAR模型

### 6.1 VAR模型定义

**VAR(p)模型（Vector Autoregression）**：
```
Yₜ = c + A₁Y_{t-1} + A₂Y_{t-2} + ... + A_pY_{t-p} + εₜ
```

其中：
- Yₜ：k维向量（k个变量）
- c：k维常数向量
- Aᵢ：k×k系数矩阵
- εₜ：k维白噪声向量

**案例：VAR(1)模型（两变量）**
```
Yₜ = [y₁ₜ, y₂ₜ]'

[y₁ₜ]   [c₁]   [a₁₁  a₁₂] [y₁,t-1]   [ε₁ₜ]
[y₂ₜ] = [c₂] + [a₂₁  a₂₂] [y₂,t-1] + [ε₂ₜ]
```

### 6.2 VAR模型的平稳性

**平稳性条件**：特征方程的根在单位圆内。

**特征方程**：
```
det(I - A₁z - A₂z² - ... - A_pzᵖ) = 0
```

**检验方法**：
```python
from statsmodels.tsa.api import VAR

# 拟合VAR模型
model = VAR(data)
results = model.fit(maxlags=2)

# 检查平稳性
print(results.is_stable())
```

### 6.3 VAR模型的估计

**似然函数**：
```
L(A₁, ..., A_p, Σ) ∝ |Σ|^{-T/2} exp(-1/2 Σ εₜ' Σ^{-1} εₜ)
```

**估计方法**：
1. OLS估计（方程逐个估计）
2. 极大似然估计

**案例**：
```python
import pandas as pd
from statsmodels.tsa.api import VAR

# 数据准备
data = pd.read_csv('macro_data.csv')
data = data[['GDP', 'CPI', 'Interest']]

# 拟合VAR模型
model = VAR(data)
results = model.fit(maxlags=4, ic='aic')

# 查看结果
print(results.summary())
```

### 6.4 脉冲响应分析

**脉冲响应函数（IRF）**：分析一个变量的冲击对其他变量的动态影响。

**定义**：
```
IRF(h) = ∂Y_{t+h} / ∂εₜ
```

**案例**：
```python
# 脉冲响应分析
irf = results.irf(10)

# 绘制脉冲响应图
irf.plot(orth=False)
plt.show()

# 正交化脉冲响应（需要识别假设）
irf.plot(orth=True)
plt.show()
```

### 6.5 方差分解

**方差分解**：分析不同冲击对预测误差方差的贡献。

**预测误差方差分解**：
```
Var(Y_{t+h}) = Σ Dⱼ ε_{t+h-j}

方差分解：
R_{ij}(h) = Σ (Dⱼ)_{ik}² / Var(Y_{i,t+h})
```

**案例**：
```python
# 方差分解
fevd = results.fevd(10)
fevd.plot()
plt.show()
```

### 6.6 Granger因果检验

**Granger因果关系**：如果X的过去值有助于预测Y的未来值，则称X Granger引起Y。

**检验方法**：
```
原假设 H₀：X不Granger引起Y

检验统计量：
F = [(RSSᵣ - RSSᵤ)/p] / [RSSᵤ/(T-2p-1)]

RSSᵣ：受约束模型（不含X的滞后项）的残差平方和
RSSᵤ：不受约束模型的残差平方和
```

**案例**：
```python
from statsmodels.tsa.stattools import grangercausalitytests

# Granger因果检验
data = pd.read_csv('macro_data.csv')
data = data[['GDP', 'CPI']]

# 检验CPI是否Granger引起GDP
gc_res = grangercausalitytests(data[['GDP', 'CPI']], maxlag=4)
print(gc_res)
```

## 八、小结与思考题

### 8.1 核心要点

1. **时间序列基础**
   - 平稳性是建模的前提
   - 白噪声是基本构建块

2. **ARMA模型**
   - AR(p)：自回归，ACF拖尾，PACF截尾
   - MA(q)：移动平均，ACF截尾，PACF拖尾
   - ARMA(p,q)：ACF和PACF都拖尾

3. **ARIMA模型**
   - 差分消除趋势
   - 建模流程：检验→识别→估计→诊断→预测

4. **GARCH模型**
   - 捕捉波动率聚类
   - 风险管理应用

5. **VAR模型**
   - 多变量时间序列建模
   - 脉冲响应与方差分解

### 8.2 思考题

1. **计算题**：
   给定AR(1)模型 Xₜ = 0.5X_{t-1} + εₜ，εₜ ~ WN(0,1)，求：
   - E(Xₜ)
   - Var(Xₜ)
   - ρ(1), ρ(2)

2. **分析题**：
   解释GARCH模型如何捕捉金融时间序列的波动率聚类现象。

3. **应用题**：
   使用ARIMA(1,1,1)模型对某股票价格建模，写出模型表达式并解释参数含义。

4. **案例题**：
   某宏观经济数据包含GDP、CPI、利率三个变量，如何使用VAR模型分析货币政策对实体经济的影响？
