# 行业景气度分析框架

## 一、行业景气度概念与定义

### 1.1 什么是行业景气度

**定义**：行业景气度是对某一行业当前经营状况和未来发展趋势的综合评价，反映行业的繁荣或萧条程度。

**核心维度**：
- **需求景气**：订单、销量、收入增长
- **供给景气**：产能利用率、库存水平
- **价格景气**：产品价格、毛利率
- **盈利景气**：净利润、ROE、现金流

### 1.2 景气度周期的四个阶段

```
        ┌─────────────────────────────────────┐
        │                                     │
   高   │    ★繁荣期          ★衰退期        │
   ↑    │    需求旺盛          需求下滑        │
   景   │    产能紧张          库存积压        │
   气   │    价格上涨          价格下跌        │
   度   │    盈利高增          盈利下滑        │
        │                                     │
        │    ★复苏期          ★萧条期        │
        │    需求回暖          需求低迷        │
        │    去库存完成        产能过剩        │
        │    价格企稳          价格低迷        │
   低   │    盈利改善          亏损增加        │
        └─────────────────────────────────────┘
              时间 →
```

### 1.3 A股行业景气度特征

| 特征 | A股表现 | 投资启示 |
|------|---------|----------|
| **周期性强** | 周期股景气波动剧烈 | 把握周期拐点 |
| **政策驱动** | 政策影响景气方向 | 关注产业政策 |
| **估值敏感** | 景气与估值背离常发生 | 结合估值判断 |
| **预期先行** | 股价领先景气见顶/底 | 预期管理重要 |

## 二、行业景气度指标体系

### 2.1 宏观领先指标

#### 2.1.1 PMI体系

**制造业PMI**：
$$PMI = 0.2 \times 新订单 + 0.25 \times 生产 + 0.1 \times 从业人员 + 0.2 \times 供应商配送时间 + 0.25 \times 原材料库存$$

**分项指标意义**：
| 分项 | 含义 | 领先/同步 |
|------|------|-----------|
| 新订单 | 需求端 | 领先 |
| 新出口订单 | 外需 | 领先 |
| 生产 | 供给端 | 同步 |
| 产成品库存 | 库存压力 | 滞后 |
| 原材料库存 | 补库意愿 | 领先 |
| 购进价格 | 成本压力 | 同步 |

**行业映射**：
```python
PMI新订单上涨 → 
    ├── 机械行业（资本开支增加）
    ├── 汽车行业（消费需求提升）
    └── 电子行业（产业链需求）

PMI产成品库存下降 → 
    ├── 去库存接近尾声
    └── 关注补库存机会
```

#### 2.1.2 PPI体系

**PPI同比**：
- 上游行业景气度的直接指标
- 传导机制：PPI上游 → PPI中游 → PPI下游 → CPI

**PPIRM（原材料购进价格指数）**：
- 企业成本端压力
- 与PPI剪刀差反映毛利率变化

**行业PPI分析框架**：
```
PPI分行业数据：
├── 采掘工业PPI → 煤炭、有色、石油开采
├── 原材料工业PPI → 钢铁、化工、建材
├── 加工工业PPI → 机械、汽车零部件
└── 生活资料PPI → 食品、纺织、服装
```

### 2.2 中观行业指标

#### 2.2.1 库存周期指标

**库存周期四阶段**：

| 阶段 | 库存状态 | 需求状态 | 景气判断 | 投资策略 |
|------|----------|----------|----------|----------|
| **主动补库存** | ↑ | ↑ | 高景气 | 持有 |
| **被动补库存** | ↑ | ↓ | 景气恶化 | 减仓 |
| **主动去库存** | ↓ | ↓ | 低景气 | 观望 |
| **被动去库存** | ↓ | ↑ | 景气复苏 | 布局 |

**库存指标构建**：
$$库存周转率 = \frac{营业成本}{平均存货}$$

$$库存销售比 = \frac{产成品存货}{销售收入}$$

**A股库存周期特征**：
- 平均库存周期：3-4年
- 主动补库存阶段：平均持续12-15个月
- 被动去库存阶段：最佳布局时点

#### 2.2.2 产能利用率

**定义**：
$$产能利用率 = \frac{实际产量}{设计产能} \times 100\%$$

**景气判断标准**：
- >85%：产能紧张，景气高，资本开支增加
- 75%-85%：正常水平
- <75%：产能过剩，景气低

**A股行业产能利用率监控**：
| 行业 | 数据来源 | 关注频率 |
|------|----------|----------|
| 钢铁 | 中国钢铁工业协会 | 月度 |
| 水泥 | 数字水泥网 | 周度/月度 |
| 化工 | 百川盈孚 | 周度 |
| 电子 | 行业协会 | 季度 |

#### 2.2.3 价格与盈利指标

**价格体系**：
```
上游：期货价格（沪铜、沪铝、螺纹钢、动力煤）
中游：现货价格（水泥、玻璃、化工品）
下游：终端价格（汽车、家电、消费品）
```

**盈利能力指标**：
- **毛利率** = (收入 - 成本) / 收入
- **净利率** = 净利润 / 收入
- **ROE** = 净利润 / 净资产
- **ROIC** = NOPAT / 投入资本

### 2.3 微观企业指标

#### 2.3.1 财务指标

**成长性指标**：
$$营收增速 = \frac{本期营收 - 上期营收}{上期营收}$$

$$扣非净利润增速 = \frac{本期扣非净利润 - 上期扣非净利润}{上期扣非净利润}$$

**质量指标**：
$$经营现金流/净利润 > 1 \Rightarrow 盈利质量好$$

$$应收账款周转率 = \frac{营业收入}{平均应收账款}$$

#### 2.3.2 调研与高频数据

**调研数据**：
- 券商行业调研报告
- 管理层业绩指引
- 订单跟踪（如光伏组件订单）

**高频数据**：
| 行业 | 高频指标 | 数据频率 |
|------|----------|----------|
| 房地产 | 30城商品房成交面积 | 日/周 |
| 汽车 | 乘联会销量数据 | 周/月 |
| 航运 | BDI指数、CCFI指数 | 日 |
| 水泥 | 磨机开工率 | 周 |
| 钢铁 | 高炉开工率 | 周 |
| 发电 | 六大电厂耗煤量 | 日 |

## 三、行业盈利周期分析

### 3.1 盈利周期的三个维度

**维度一：盈利能力周期**
$$ROE_t = 净利率_t \times 资产周转率_t \times 权益乘数_t$$

**A股ROE分解特征**：
- 周期股：ROE波动主要来自净利率
- 成长股：ROE提升主要来自周转率
- 金融股：ROE稳定但权益乘数高

**维度二：盈利增长周期**
$$盈利增速 = 营收增速 + 利润率扩张速度$$

**维度三：盈利质量周期**
- 经营现金流/净利润
- 非经常性损益占比
- 商誉减值风险

### 3.2 盈利周期与股价的关系

**戴维斯双击**：
```
盈利上升 + 估值上升 = 股价大涨
├── 阶段1：盈利触底回升（预期改善）
├── 阶段2：盈利确认增长（估值修复）
└── 阶段3：盈利超预期（估值扩张）
```

**戴维斯双杀**：
```
盈利下降 + 估值下降 = 股价大跌
├── 阶段1：盈利见顶回落（预期恶化）
├── 阶段2：盈利确认下滑（估值压缩）
└── 阶段3：盈利低于预期（估值崩塌）
```

**A股盈利周期数据**：
- 盈利周期长度：约3.5年
- 股价领先盈利：约3-6个月
- 盈利见顶信号：ROE连续2季度下滑

### 3.3 行业盈利周期比较

| 行业 | 盈利周期特征 | 周期长度 | 关键驱动 |
|------|-------------|---------|----------|
| **钢铁** | 强周期，剧烈波动 | 3-4年 | 地产、基建投资 |
| **化工** | 中等周期 | 2-3年 | 油价、下游需求 |
| **电子** | 弱周期，成长主导 | 2-3年 | 技术升级、换机周期 |
| **医药** | 弱周期，防御属性 | 不明显 | 政策、研发管线 |
| **银行** | 与经济周期同步 | 3-4年 | 信贷周期、息差 |
| **消费** | 稳定增长 | 不明显 | 居民收入、消费升级 |

## 四、景气度与股价的关系

### 4.1 理论框架：预期差模型

**核心公式**：
$$股票收益 = 盈利增长 + 估值变化 + 分红收益$$

其中：
$$估值变化 = f(预期差) = f(实际景气 - 预期景气)$$

**三种预期差**：
1. **正向预期差**：实际景气 > 预期景气 → 估值修复 → 股价上涨
2. **零预期差**：实际景气 = 预期景气 → 估值稳定 → 股价跟随盈利
3. **负向预期差**：实际景气 < 预期景气 → 估值压缩 → 股价下跌

### 4.2 景气度投资的三种策略

#### 策略一：景气度跟随策略

**逻辑**：买入景气上行的行业，卖出景气下行的行业

**指标构建**：
```python
景气度得分 = w1 * 营收增速_环比变化 
           + w2 * 净利润增速_环比变化
           + w3 * ROE_环比变化
           + w4 * 毛利率_环比变化
```

**操作规则**：
- 景气度得分 > 阈值 → 买入/超配
- 景气度得分 < -阈值 → 卖出/低配

#### 策略二：景气度反转策略

**逻辑**：在景气低点布局，等待景气复苏

**关键识别**：
```
景气反转信号：
├── 库存周期：被动去库存 → 主动补库存
├── 盈利周期：ROE触底回升
├── 价格周期：产品价格触底反弹
└── 政策周期：产业政策转向支持
```

**风险控制**：
- 信号确认后再介入（避免"接飞刀"）
- 分批建仓，控制仓位
- 设置止损线

#### 策略三：景气度+估值双因子策略

**核心思想**：景气度决定方向，估值决定赔率

**四象限策略**：
```
         高估值
           │
    ②高景气高估值    │    ①高景气低估值
    （持有）         │    （重仓买入）
 ───────────────────────────────────── 低景气
    ③低景气高估值    │    ④低景气低估值
    （回避）         │    （左侧布局）
           │
         低估值
```

**A股历史回测**：
| 象限 | 年化收益 | 夏普比率 | 最大回撤 |
|------|---------|---------|---------|
| ①高景气低估值 | 22.5% | 1.15 | -18% |
| ②高景气高估值 | 12.3% | 0.65 | -28% |
| ③低景气高估值 | -5.2% | -0.25 | -35% |
| ④低景气低估值 | 8.7% | 0.45 | -22% |

### 4.3 A股景气度投资的特殊考量

**政策因素**：
- 产业政策对景气的影响权重更大
- 需建立政策跟踪体系
- 示例：新能源补贴退坡、集采政策

**投资者结构**：
- 散户占比高，景气预期容易过度反应
- 景气高点往往伴随估值泡沫
- 需警惕"景气陷阱"

**数据质量**：
- 行业数据来源分散，口径不一
- 高频数据质量参差不齐
- 需要交叉验证多个数据源

### 5.1 量化模型构建

**模型一：景气度打分模型**

```python
def calculate_prosperity_score(industry):
    # 宏观因子（权重30%）
    macro_score = (
        0.3 * PMI_new_order_zscore +
        0.2 * PPI_yoy_zscore +
        0.2 * credit_growth_zscore +
        0.3 * investment_growth_zscore
    )
    
    # 中观因子（权重40%）
    mid_score = (
        0.25 * revenue_growth_zscore +
        0.25 * profit_growth_zscore +
        0.25 * inventory_turnover_zscore +
        0.25 * capacity_utilization_zscore
    )
    
    # 微观因子（权重30%）
    micro_score = (
        0.3 * roe_change_zscore +
        0.3 * gross_margin_change_zscore +
        0.2 * cashflow_quality_zscore +
        0.2 * forecast_revision_zscore
    )
    
    return 0.3 * macro_score + 0.4 * mid_score + 0.3 * micro_score
```

## 六、实战要点

### 6.1 景气度跟踪流程

**周度跟踪**：
1. 更新高频数据（价格、产量、库存）
2. 计算景气度得分变化
3. 识别景气度拐点信号

**月度跟踪**：
1. 更新PMI、PPI等宏观数据
2. 更新行业产量、销量数据
3. 计算景气度综合得分
4. 调整行业配置权重

**季度跟踪**：
1. 更新财务数据（营收、利润、ROE）
2. 分析盈利周期位置
3. 调整长期配置策略

### 6.2 景气度拐点识别

**顶部信号**：
- ROE连续2季度下滑
- 库存增速 > 需求增速
- 毛利率见顶回落
- 分析师开始下调盈利预测

**底部信号**：
- ROE连续2季度企稳
- 库存去化接近尾声
- 产品价格触底反弹
- 政策开始转向支持

### 6.3 风险控制

**模型风险**：
- 数据滞后：财报数据存在披露延迟
- 数据失真：统计口径变化、数据修正
- 过度拟合：历史规律可能失效

**市场风险**：
- 景气见顶但估值泡沫继续扩大
- 景气见底但估值继续压缩
- 政策黑天鹅事件

**应对措施**：
- 多数据源交叉验证
- 设置止损/止盈线
- 保持组合分散化
- 动态调整仓位

## 八、关键结论

### 为什么传统行业轮动在A股失效

1. **景气度指标失效**
   - 数据滞后：A股财报披露期长，数据时效性差
   - 指标失真：统计口径变化、数据修正频繁
   - 政策干扰：政策变化打断景气周期

2. **预期过度反应**
   - 散户占比高，景气高点追涨，景气低点杀跌
   - 景气见顶时估值泡沫大，见底时估值压缩过度

3. **行业快速轮动**
   - A股行业轮动速度远快于美股
   - 传统季度调仓跟不上市场变化

### 改进方案

```
核心改进方向：
├── 高频数据替代低频数据
│   └── 周度/日度数据替代月度/季度数据
├── 多维度景气度综合判断
│   └── 量、价、利、险四维模型
├── 预期差管理
│   └── 关注景气预期与实际的偏差
└── 动态调仓机制
    └── 月度/周度调仓替代季度调仓
```
