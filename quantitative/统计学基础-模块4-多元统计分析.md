## 1. 多元正态分布

### 1.1 多元正态分布定义

**概率密度函数：**
```
f(x) = (2π)^(-k/2) |Σ|^(-1/2) exp(-1/2 (x-μ)'Σ⁻¹(x-μ))
```

其中：
- x：k维随机向量
- μ：k维均值向量
- Σ：k×k协方差矩阵
- |Σ|：协方差矩阵行列式

**性质：**

1. **边缘分布**：多元正态的边缘分布仍是正态分布
   ```
   X₁ ~ N(μ₁, Σ₁₁)
   ```

2. **条件分布**：多元正态的条件分布仍是正态分布
   ```
   X₁|X₂ ~ N(μ₁|₂, Σ₁|₂)
   
   μ₁|₂ = μ₁ + Σ₁₂Σ₂₂⁻¹(X₂ - μ₂)
   Σ₁|₂ = Σ₁₁ - Σ₁₂Σ₂₂⁻¹Σ₂₁'
   ```

3. **线性组合**：多元正态的线性组合仍是正态分布
   ```
   Y = AX + b ~ N(Aμ + b, AΣA')
   ```

### 1.2 协方差矩阵

**定义：**
```
Σ = Cov(X) = E[(X-μ)(X-μ)']
```

**样本协方差矩阵：**
```python
import numpy as np

def sample_covariance(X):
    """
    计算样本协方差矩阵
    
    参数:
    X: 数据矩阵 (n, k)
    
    返回:
    协方差矩阵 (k, k)
    """
    n = X.shape[0]
    X_centered = X - X.mean(axis=0)
    cov = X_centered.T @ X_centered / (n - 1)
    return cov

# 示例
X = np.random.randn(100, 3)
cov = sample_covariance(X)
print('样本协方差矩阵:')
print(cov)
```

**相关矩阵：**
```python
def correlation_matrix(cov):
    """
    从协方差矩阵计算相关矩阵
    
    参数:
    cov: 协方差矩阵
    
    返回:
    相关矩阵
    """
    std = np.sqrt(np.diag(cov))
    corr = cov / np.outer(std, std)
    return corr
```

### 1.4 多元正态性检验

**Mardia检验：**
```python
from scipy.stats import chi2

def mardia_test(X):
    """
    Mardia多元正态性检验
    
    参数:
    X: 数据矩阵 (n, k)
    
    返回:
    skewness_stat: 偏度统计量
    kurtosis_stat: 峰度统计量
    p_values: p值
    """
    n, k = X.shape
    
    # 中心化
    X_centered = X - X.mean(axis=0)
    
    # 协方差矩阵
    S = X_centered.T @ X_centered / n
    S_inv = np.linalg.inv(S)
    
    # Mahalanobis距离
    d = np.sum(X_centered @ S_inv * X_centered, axis=1)
    
    # 多元偏度
    skewness = np.sum(d**3) / n**2
    skewness_stat = n * skewness / 6
    skewness_p = 1 - chi2.cdf(skewness_stat, k*(k+1)*(k+2)/6)
    
    # 多元峰度
    kurtosis = np.mean(d**2)
    kurtosis_stat = (kurtosis - k*(k+2)) / np.sqrt(8*k*(k+2)/n)
    kurtosis_p = 2 * (1 - chi2.cdf(abs(kurtosis_stat), 1))
    
    return {
        'skewness_stat': skewness_stat,
        'skewness_p': skewness_p,
        'kurtosis_stat': kurtosis_stat,
        'kurtosis_p': kurtosis_p
    }
```

## 2. 主成分分析

### 2.1 PCA理论基础

**目标：** 找到数据方差最大的方向

**数学推导：**

最大化方差：
```
max Var(a'X) = a'Σa
s.t. a'a = 1
```

拉格朗日函数：
```
L = a'Σa - λ(a'a - 1)
```

一阶条件：
```
Σa = λa
```

**主成分：**
- 第一主成分：协方差矩阵最大特征值对应的特征向量
- 第k主成分：第k大特征值对应的特征向量

**方差贡献：**
```
贡献率 = λₖ / Σλᵢ
累计贡献率 = Σⱼ₌₁ᵏ λⱼ / Σλᵢ
```

### 2.2 PCA计算步骤

**步骤：**

1. 标准化数据
2. 计算协方差矩阵
3. 特征值分解
4. 选择主成分
5. 计算主成分得分

**Python实现：**
```python
import numpy as np
from scipy.linalg import eigh

def pca_manual(X, n_components=None):
    """
    主成分分析
    
    参数:
    X: 数据矩阵 (n, k)
    n_components: 保留的主成分数量
    
    返回:
    components: 主成分 (k, n_components)
    explained_variance: 方差贡献
    scores: 主成分得分 (n, n_components)
    """
    # 标准化
    X_std = (X - X.mean(axis=0)) / X.std(axis=0)
    
    # 协方差矩阵
    cov = X_std.T @ X_std / (X.shape[0] - 1)
    
    # 特征值分解
    eigenvalues, eigenvectors = eigh(cov)
    
    # 降序排列
    idx = np.argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[idx]
    eigenvectors = eigenvectors[:, idx]
    
    # 选择主成分
    if n_components:
        eigenvalues = eigenvalues[:n_components]
        eigenvectors = eigenvectors[:, :n_components]
    
    # 方差贡献
    explained_variance = eigenvalues / eigenvalues.sum()
    
    # 主成分得分
    scores = X_std @ eigenvectors
    
    return {
        'components': eigenvectors,
        'explained_variance': explained_variance,
        'scores': scores
    }
```

## 3. 因子分析

### 3.1 因子分析模型

**模型：**
```
X = ΛF + ε
```

其中：
- X：可观测变量 (p维)
- Λ：因子载荷矩阵 (p×k)
- F：公共因子 (k维)
- ε：特殊因子 (p维)

**假设：**
```
E(F) = 0
Cov(F) = I
E(ε) = 0
Cov(ε) = Ψ (对角矩阵)
Cov(F, ε) = 0
```

**协方差结构：**
```
Cov(X) = ΛΛ' + Ψ
```

### 3.2 因子载荷估计

**主成分法：**
```python
from sklearn.decomposition import FactorAnalysis

def factor_analysis_pca(X, n_factors):
    """
    主成分法因子分析
    
    参数:
    X: 数据矩阵 (n, p)
    n_factors: 因子数量
    
    返回:
    因子载荷、特殊方差
    """
    # 标准化
    X_std = (X - X.mean(axis=0)) / X.std(axis=0)
    
    # 协方差矩阵
    R = np.corrcoef(X_std.T)
    
    # 特征值分解
    eigenvalues, eigenvectors = np.linalg.eigh(R)
    
    # 降序排列
    idx = np.argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[idx]
    eigenvectors = eigenvectors[:, idx]
    
    # 因子载荷
    loadings = eigenvectors[:, :n_factors] * np.sqrt(eigenvalues[:n_factors])
    
    # 特殊方差
    communalities = np.sum(loadings**2, axis=1)
    specific_variance = 1 - communalities
    
    return {
        'loadings': loadings,
        'specific_variance': specific_variance,
        'communalities': communalities
    }
```

### 3.3 因子旋转

**正交旋转（Varimax）：**
```python
from sklearn.decomposition import FactorAnalysis
from scipy.linalg import orth

def varimax_rotation(loadings, max_iter=100):
    """
    Varimax旋转
    
    参数:
    loadings: 因子载荷矩阵 (p, k)
    max_iter: 最大迭代次数
    
    返回:
    旋转后的载荷
    """
    p, k = loadings.shape
    
    # 初始化旋转矩阵
    R = np.eye(k)
    
    for _ in range(max_iter):
        # 计算旋转后载荷
        rotated = loadings @ R
        
        # 计算梯度
        gradient = loadings.T @ (rotated**3 - rotated @ np.diag(np.sum(rotated**2, axis=0)) / p)
        
        # SVD
        U, _, Vt = np.linalg.svd(gradient)
        
        # 更新旋转矩阵
        R_new = U @ Vt
        
        # 检查收敛
        if np.max(np.abs(R_new - R)) < 1e-6:
            break
        
        R = R_new
    
    return loadings @ R, R

# 示例
loadings = fa.components_.T
rotated_loadings, R = varimax_rotation(loadings)
print(f'旋转后载荷:\n{rotated_loadings}')
```

### 3.4 因子数量选择

**平行分析：**
```python
def parallel_analysis(X, n_iterations=100):
    """
    平行分析确定因子数量
    
    参数:
    X: 数据矩阵 (n, p)
    n_iterations: 迭代次数
    
    返回:
    推荐因子数量
    """
    n, p = X.shape
    
    # 实际数据的特征值
    R = np.corrcoef(X.T)
    actual_eigenvalues = np.linalg.eigvalsh(R)[::-1]
    
    # 随机数据的特征值
    random_eigenvalues = np.zeros((n_iterations, p))
    for i in range(n_iterations):
        X_random = np.random.randn(n, p)
        R_random = np.corrcoef(X_random.T)
        random_eigenvalues[i] = np.linalg.eigvalsh(R_random)[::-1]
    
    # 平均随机特征值
    mean_random = random_eigenvalues.mean(axis=0)
    
    # 比较实际与随机特征值
    n_factors = np.sum(actual_eigenvalues > mean_random)
    
    return n_factors
```

## 4. 判别分析

### 4.1 线性判别分析（LDA）

**目标：** 找到最优投影方向，使类间距离最大化，类内距离最小化

**Fisher准则：**
```
J(w) = w'S_Bw / w'S_Ww
```

其中：
- S_B：类间散布矩阵
- S_W：类内散布矩阵

**求解：**
```
S_Bw = λS_Ww
```

**Python实现：**
```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis

def lda_manual(X, y):
    """
    线性判别分析
    
    参数:
    X: 数据矩阵 (n, p)
    y: 类别标签 (n,)
    
    返回:
    判别函数、投影数据
    """
    # 类别
    classes = np.unique(y)
    n_classes = len(classes)
    
    # 总体均值
    overall_mean = X.mean(axis=0)
    
    # 类内散布矩阵
    S_W = np.zeros((X.shape[1], X.shape[1]))
    for c in classes:
        X_c = X[y == c]
        mean_c = X_c.mean(axis=0)
        S_W += (X_c - mean_c).T @ (X_c - mean_c)
    
    # 类间散布矩阵
    S_B = np.zeros((X.shape[1], X.shape[1]))
    for c in classes:
        X_c = X[y == c]
        n_c = X_c.shape[0]
        mean_c = X_c.mean(axis=0)
        S_B += n_c * (mean_c - overall_mean).reshape(-1, 1) @ (mean_c - overall_mean).reshape(1, -1)
    
    # 求解特征值问题
    eigenvalues, eigenvectors = np.linalg.eig(np.linalg.inv(S_W) @ S_B)
    
    # 降序排列
    idx = np.argsort(eigenvalues)[::-1]
    eigenvectors = eigenvectors[:, idx].real
    
    # 投影
    X_lda = X @ eigenvectors[:, :n_classes-1]
    
    return X_lda, eigenvectors

# 使用sklearn
lda = LinearDiscriminantAnalysis(n_components=2)
X_lda = lda.fit_transform(X, y)

print(f'判别函数:\n{lda.coef_}')
print(f'分类准确率: {lda.score(X, y)}')
```

## 5. 聚类分析

### 5.1 K-means聚类

**算法步骤：**
1. 随机选择k个初始中心
2. 将每个样本分配到最近的中心
3. 重新计算每个簇的中心
4. 重复步骤2-3直到收敛

**Python实现：**
```python
from sklearn.cluster import KMeans

def kmeans_manual(X, k, max_iter=100):
    """
    K-means聚类
    
    参数:
    X: 数据矩阵 (n, p)
    k: 簇数量
    max_iter: 最大迭代次数
    
    返回:
    标签、簇中心
    """
    n, p = X.shape
    
    # 随机初始化中心
    centers = X[np.random.choice(n, k, replace=False)]
    
    for _ in range(max_iter):
        # 分配样本到最近的簇
        distances = np.sqrt(((X[:, np.newaxis] - centers) ** 2).sum(axis=2))
        labels = np.argmin(distances, axis=1)
        
        # 更新簇中心
        new_centers = np.array([X[labels == i].mean(axis=0) for i in range(k)])
        
        # 检查收敛
        if np.all(centers == new_centers):
            break
        
        centers = new_centers
    
    return labels, centers

# 使用sklearn
```

### 5.3 聚类评估

**轮廓系数：**
```python
from sklearn.metrics import silhouette_score, silhouette_samples

def evaluate_clustering(X, labels):
    """
    评估聚类效果
    
    参数:
    X: 数据矩阵
    labels: 聚类标签
    
    返回:
    评估指标
    """
    # 轮廓系数
    silhouette_avg = silhouette_score(X, labels)
    silhouette_values = silhouette_samples(X, labels)
    
    # 簇内距离
    n_clusters = len(np.unique(labels))
    intra_distances = []
    for i in range(n_clusters):
        cluster_points = X[labels == i]
        center = cluster_points.mean(axis=0)
        intra_distance = np.sqrt(((cluster_points - center)**2).sum(axis=1)).mean()
        intra_distances.append(intra_distance)
    
    return {
        'silhouette_score': silhouette_avg,
        'silhouette_values': silhouette_values,
        'intra_distances': intra_distances
    }
```

```python
    plt.plot(range(1, max_k + 1), inertias, 'bo-')
    plt.xlabel('簇数量 k')
    plt.ylabel('簇内平方和')
    plt.title('肘部法则')
    plt.savefig('elbow_method.png')
    
    # 自动选择最优k（简化：使用二阶导数）
    if len(inertias) > 2:
        second_derivative = np.diff(np.diff(inertias))
        optimal_k = np.argmax(second_derivative) + 2
    else:
        optimal_k = 2
    
    return optimal_k
```

## 6. 金融应用

### 6.1 资产配置

**基于PCA的资产配置：**
```python
def pca_based_allocation(returns, n_factors=3):
    """
    基于PCA的资产配置
    
    参数:
    returns: 资产收益率 (n_periods, n_assets)
    n_factors: 主成分数量
    
    返回:
    最优权重
    """
    from sklearn.decomposition import PCA
    from scipy.optimize import minimize
    
    # PCA降维
    pca = PCA(n_components=n_factors)
    factors = pca.fit_transform(returns)
    factor_exposure = pca.components_.T
    
    # 因子协方差矩阵
    factor_cov = np.cov(factors.T)
    
    # 特质风险
    residuals = returns - factors @ factor_exposure.T
    idio_var = np.var(residuals, axis=0)
    
    # 完整协方差矩阵
    cov_matrix = factor_exposure @ factor_cov @ factor_exposure.T + np.diag(idio_var)
    
    # 均值方差优化
    expected_returns = returns.mean(axis=0)
    
    def neg_sharpe(weights):
        port_return = weights @ expected_returns
        port_vol = np.sqrt(weights @ cov_matrix @ weights)
        return -port_return / port_vol
    
    n_assets = returns.shape[1]
    constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    result = minimize(neg_sharpe, [1/n_assets]*n_assets, bounds=bounds, constraints=constraints)
    
    return result.x
```

### 6.2 风险管理

**基于PCA的风险分解：**
```python
def pca_risk_decomposition(returns, portfolio_weights, n_factors=5):
    """
    基于PCA的风险分解
    
    参数:
    returns: 资产收益率
    portfolio_weights: 组合权重
    n_factors: 主成分数量
    
    返回:
    风险贡献
    """
    from sklearn.decomposition import PCA
    
    # PCA
    pca = PCA(n_components=n_factors)
    factors = pca.fit_transform(returns)
    factor_exposure = pca.components_.T
    
    # 组合因子暴露
    portfolio_exposure = portfolio_weights @ factor_exposure
    
    # 因子协方差
    factor_cov = np.cov(factors.T)
    
    # 风险贡献
    factor_variance = portfolio_exposure @ factor_cov @ portfolio_exposure
    factor_vol = np.sqrt(factor_variance)
    
    # 各因子贡献
    factor_contrib = []
    for i in range(n_factors):
        contrib = portfolio_exposure[i]**2 * factor_cov[i, i] / factor_variance
        factor_contrib.append(contrib)
    
    return {
        'factor_exposure': portfolio_exposure,
        'factor_volatility': factor_vol,
        'factor_contribution': factor_contrib
    }
```

### 6.3 风格分析

**收益率风格分析：**
```python
def style_analysis(fund_returns, factor_returns):
    """
    收益率风格分析
    
    参数:
    fund_returns: 基金收益率
    factor_returns: 因子收益率 (n_periods, n_factors)
    
    返回:
    风格暴露
    """
    from scipy.optimize import minimize
    
    n_factors = factor_returns.shape[1]
    
    # 约束：权重非负且和为1
    constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_factors)]
    
    # 最小化跟踪误差
    def tracking_error(weights):
        portfolio_returns = factor_returns @ weights
        return np.std(fund_returns - portfolio_returns)
    
    result = minimize(tracking_error, [1/n_factors]*n_factors, bounds=bounds, constraints=constraints)
    
    return result.x
```
