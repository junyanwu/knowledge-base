# 时间序列分析 - 模块4：状态空间模型与卡尔曼滤波

## 1. 状态空间模型基础

### 1.1 模型形式

**状态方程：**
```
αₜ = Tₜαₜ₋₁ + Rₜηₜ
```

**观测方程：**
```
yₜ = Zₜ'αₜ + εₜ
```

其中：
- αₜ：状态向量
- yₜ：观测向量
- Tₜ：状态转移矩阵
- Zₜ：观测矩阵
- ηₜ ~ N(0, Qₜ)：状态噪声
- εₜ ~ N(0, Hₜ)：观测噪声

### 1.2 常见时间序列的状态空间表示

**ARMA(p,q)模型：**
```python
def arma_to_statespace(ar, ma):
    """
    ARMA模型转状态空间
    
    参数:
    ar: AR系数 [φ₁, φ₂, ..., φₚ]
    ma: MA系数 [θ₁, θ₂, ..., θᵧ]
    
    返回:
    状态空间矩阵
    """
    p = len(ar)
    q = len(ma)
    r = max(p, q + 1)
    
    # 状态转移矩阵
    T = np.zeros((r, r))
    T[0, :p] = ar
    T[1:, :-1] = np.eye(r-1)
    
    # 观测矩阵
    Z = np.zeros(r)
    Z[0] = 1
    
    # 状态噪声协方差
    R = np.zeros((r, 1))
    R[0] = 1
    
    Q = np.array([[1]])
    
    return T, Z, R, Q
```

## 2. 卡尔曼滤波

### 2.1 滤波算法

**预测步骤：**
```
αₜ|ₜ₋₁ = Tₜαₜ₋₁|ₜ₋₁
Pₜ|ₜ₋₁ = TₜPₜ₋₁|ₜ₋₁Tₜ' + RₜQₜRₜ'
```

**更新步骤：**
```
vₜ = yₜ - Zₜ'αₜ|ₜ₋₁
Fₜ = Zₜ'Pₜ|ₜ₋₁Zₜ + Hₜ
Kₜ = Pₜ|ₜ₋₁ZₜFₜ⁻¹
αₜ|ₜ = αₜ|ₜ₋₁ + Kₜvₜ
Pₜ|ₜ = Pₜ|ₜ₋₁ - KₜFₜKₜ'
```

**Python实现：**
```python
def kalman_filter(y, T, Z, R, Q, H, a0, P0):
    """
    卡尔曼滤波
    
    参数:
    y: 观测序列
    T, Z, R, Q, H: 状态空间矩阵
    a0: 初始状态
    P0: 初始状态协方差
    
    返回:
    滤波状态、对数似然
    """
    n = len(y)
    a = a0.copy()
    P = P0.copy()
    
    a_filtered = []
    P_filtered = []
    log_likelihood = 0
    
    for t in range(n):
        # 预测
        a_pred = T @ a
        P_pred = T @ P @ T.T + R @ Q @ R.T
        
        # 预测误差
        v = y[t] - Z.T @ a_pred
        F = Z.T @ P_pred @ Z + H
        
        # 卡尔曼增益
        K = P_pred @ Z / F
        
        # 更新
        a = a_pred + K * v
        P = P_pred - K * F @ K.T
        
        a_filtered.append(a.copy())
        P_filtered.append(P.copy())
        
        # 对数似然
        log_likelihood += -0.5 * (np.log(2 * np.pi * F) + v**2 / F)
    
    return a_filtered, P_filtered, log_likelihood
```

### 2.2 使用statsmodels

```python
from statsmodels.tsa.statespace.kalman_filter import KalmanFilter

# 定义状态空间模型
kf = KalmanFilter(
    k_states=2,  # 状态维度
    k_obs=1,     # 观测维度
    transition=T,
    observation=Z,
    selection=R,
    state_cov=Q,
    obs_cov=H
)

# 滤波
results = kf.filter(y)

print(f'滤波状态: {results.filtered_states}')
print(f'预测状态: {results.predicted_states}')
```

## 3. 状态空间模型应用

### 3.1 时变参数模型

```python
from statsmodels.tsa.statespace.structural import UnobservedComponents

# 局部水平模型
model = UnobservedComponents(y, level='local level')
results = model.fit()

print(results.summary())

# 提取状态
level = results.level
print(f'时变水平: {level}')
```

trend = results.level

## 4. 金融应用

### 4.1 动态因子模型

```python
def dynamic_factor_model(returns, n_factors=3):
    """
    动态因子模型
    
    参数:
    returns: 资产收益率 (n_periods, n_assets)
    n_factors: 因子数量
    
    返回:
    因子收益、因子暴露
    """
    from statsmodels.tsa.statespace.dynamic_factor import DynamicFactor
    
    model = DynamicFactor(
        returns,
        k_factors=n_factors,
        factor_order=1
    )
    results = model.fit()
    
    # 因子收益
    factors = results.factors.filtered
    
    # 因子暴露
    loadings = results.loadings
    
    return factors, loadings
```

### 4.2 时变波动率

```python
from statsmodels.tsa.statespace.structural import UnobservedComponents

def time_varying_volatility(returns):
    """
    时变波动率模型
    
    参数:
    returns: 收益率序列
    
    返回:
    时变波动率
    """
    # 对数波动率
    log_var = np.log(returns**2)
    
    # 状态空间模型
    model = UnobservedComponents(log_var, level='local level')
    results = model.fit()
    
    # 提取时变波动率
    volatility = np.exp(results.level / 2)
    
    return volatility
```

## 5. 总结

状态空间模型要点：

1. **模型形式**：状态方程 + 观测方程
2. **卡尔曼滤波**：预测 + 更新
3. **应用场景**：时变参数、结构分解、动态因子

**优势：**
- 处理缺失数据
- 时变参数估计
- 结构分解
- 实时更新
