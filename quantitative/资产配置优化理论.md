## 1. 现代投资组合理论（MPT）基础

### 1.1 理论起源与发展

**历史背景**：
- 1952年：Harry Markowitz发表《Portfolio Selection》
- 1990年：Markowitz获诺贝尔经济学奖
- 核心贡献：量化风险与收益的关系

**基本假设**：
1. 投资者是理性的，追求效用最大化
2. 投资者根据期望收益和风险（方差）做决策
3. 收益率服从正态分布
4. 市场是有效的
5. 无交易成本和税收

### 1.2 核心概念

**期望收益率**：
```
E(Rp) = Σ wi × E(Ri)
```

其中：
- E(Rp)：组合期望收益率
- wi：资产i的权重
- E(Ri)：资产i的期望收益率

**组合方差**：
```
σp² = Σ Σ wi × wj × σij
    = Σ wi² × σi² + Σ Σ wi × wj × ρij × σi × σj
```

其中：
- σp²：组合方差
- σij：资产i和j的协方差
- ρij：相关系数
- σi：资产i的标准差

**组合标准差**：
```
σp = √σp²
```

## 2. 马科维茨均值-方差模型

### 2.1 数学模型

**目标函数**：
```
max E(Rp) - (λ/2) × σp²
```

其中：
- λ：风险厌恶系数（0-10）
- E(Rp)：期望收益
- σp²：方差

**约束条件**：
```
Σ wi = 1    （权重和为1）
wi ≥ 0      （不允许卖空）
```

**拉格朗日函数**：
```
L = Σ wi × E(Ri) - (λ/2) × Σ Σ wi × wj × σij 
    - γ × (Σ wi - 1)
```

**Python实现**：

```python
import numpy as np
import pandas as pd
from scipy.optimize import minimize
import matplotlib.pyplot as plt

class MeanVarianceOptimizer:
    """均值-方差优化器"""
    
    def __init__(self, expected_returns, cov_matrix, risk_free_rate=0.02):
        """
        初始化
        
        参数:
        expected_returns: 期望收益率 (n,)
        cov_matrix: 协方差矩阵 (n, n)
        risk_free_rate: 无风险利率
        """
        self.expected_returns = expected_returns
        self.cov_matrix = cov_matrix
        self.risk_free_rate = risk_free_rate
        self.n_assets = len(expected_returns)
    
    def portfolio_return(self, weights):
        """组合期望收益"""
        return weights @ self.expected_returns
    
    def portfolio_volatility(self, weights):
        """组合波动率"""
        return np.sqrt(weights @ self.cov_matrix @ weights)
    
    def portfolio_sharpe(self, weights):
        """组合夏普比率"""
        ret = self.portfolio_return(weights)
        vol = self.portfolio_volatility(weights)
        return (ret - self.risk_free_rate) / vol
    
    def minimize_volatility(self, target_return):
        """
        最小化波动率（给定目标收益）
        
        参数:
        target_return: 目标收益率
        
        返回:
        最优权重
        """
        def neg_volatility(weights):
            return self.portfolio_volatility(weights)
        
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
            {'type': 'eq', 'fun': lambda w: self.portfolio_return(w) - target_return}
        ]
        
        bounds = [(0, 1) for _ in range(self.n_assets)]
        
        result = minimize(
            neg_volatility,
            x0=[1/self.n_assets] * self.n_assets,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return result.x
    
    def maximize_return(self, target_volatility):
        """
        最大化收益（给定目标波动率）
        
        参数:
        target_volatility: 目标波动率
        
        返回:
        最优权重
        """
        def neg_return(weights):
            return -self.portfolio_return(weights)
        
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
            {'type': 'eq', 'fun': lambda w: self.portfolio_volatility(w) - target_volatility}
        ]
        
        bounds = [(0, 1) for _ in range(self.n_assets)]
        
        result = minimize(
            neg_return,
            x0=[1/self.n_assets] * self.n_assets,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return result.x
    
    def maximize_sharpe(self):
        """
        最大化夏普比率
        
        返回:
        最优权重
        """
        def neg_sharpe(weights):
            return -self.portfolio_sharpe(weights)
        
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        ]
        
        bounds = [(0, 1) for _ in range(self.n_assets)]
        
        result = minimize(
            neg_sharpe,
            x0=[1/self.n_assets] * self.n_assets,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return result.x
    
    def minimize_variance(self):
        """
        最小化方差
        
        返回:
        最优权重
        """
        def portfolio_variance(weights):
            return weights @ self.cov_matrix @ weights
        
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        ]
        
        bounds = [(0, 1) for _ in range(self.n_assets)]
        
        result = minimize(
            portfolio_variance,
            x0=[1/self.n_assets] * self.n_assets,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return result.x
```

### 2.3 约束优化

```python
def constrained_optimization(expected_returns, cov_matrix, 
                            min_weights=None, max_weights=None,
                            sector_constraints=None):
    """
    带约束的均值方差优化
    
    参数:
    expected_returns: 期望收益率
    cov_matrix: 协方差矩阵
    min_weights: 最小权重限制
    max_weights: 最大权重限制
    sector_constraints: 行业约束 {'科技': (0.2, 0.4)}
    
    返回:
    最优权重
    """
    n_assets = len(expected_returns)
    
    # 默认边界
    if min_weights is None:
        min_weights = [0.0] * n_assets
    if max_weights is None:
        max_weights = [1.0] * n_assets
    
    bounds = [(min_weights[i], max_weights[i]) for i in range(n_assets)]
    
    # 目标函数
    def neg_sharpe(weights):
        ret = weights @ expected_returns
        vol = np.sqrt(weights @ cov_matrix @ weights)
        return -ret / vol
    
    # 约束条件
    constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
    
    # 行业约束
    if sector_constraints:
        for sector, (min_weight, max_weight) in sector_constraints.items():
            # 假设sector是一个资产索引列表
            constraints.append({
                'type': 'ineq',
                'fun': lambda w, s=sector: np.sum(w[s]) - min_weight
            })
            constraints.append({
                'type': 'ineq',
                'fun': lambda w, s=sector: max_weight - np.sum(w[s])
            })
    
    # 优化
    result = minimize(
        neg_sharpe,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x

# 示例：带约束的ETF组合
assets = ['股票ETF', '债券ETF', '商品ETF', 'REITs', '货币ETF']

min_weights = [0.10, 0.10, 0.05, 0.05, 0.05]  # 最小权重
max_weights = [0.60, 0.60, 0.30, 0.30, 0.20]  # 最大权重

optimal_weights = constrained_optimization(
    expected_returns, 
    cov_matrix,
    min_weights=min_weights,
    max_weights=max_weights
)

print("带约束的最优权重:")
for asset, weight in zip(assets, optimal_weights):
    print(f"{asset}: {weight:.2%}")
```

## 3. 有效前沿理论与计算

### 3.1 有效前沿定义

**有效前沿**：在给定风险水平下，能够实现最高期望收益的组合集合。

**特征**：
1. 上凸曲线
2. 最小方差组合在最左侧
3. 最大夏普组合在有效前沿上
4. 无法同时提高收益和降低风险

### 3.2 有效前沿计算

```python
class EfficientFrontier:
    """有效前沿计算器"""
    
    def __init__(self, expected_returns, cov_matrix, risk_free_rate=0.02):
        self.expected_returns = expected_returns
        self.cov_matrix = cov_matrix
        self.risk_free_rate = risk_free_rate
        self.n_assets = len(expected_returns)
        self.optimizer = MeanVarianceOptimizer(
            expected_returns, cov_matrix, risk_free_rate
        )
    
    def calculate_efficient_frontier(self, n_points=100):
        """
        计算有效前沿
        
        参数:
        n_points: 点数
        
        返回:
        收益率数组, 波动率数组
        """
        # 最小方差组合
        min_var_weights = self.optimizer.minimize_variance()
        min_var_return = self.optimizer.portfolio_return(min_var_weights)
        
        # 最大收益率（单资产最大收益）
        max_return = np.max(self.expected_returns)
        
        # 生成目标收益率
        target_returns = np.linspace(min_var_return, max_return, n_points)
        
        # 计算每个目标收益对应的最小波动率
        frontier_volatilities = []
        frontier_weights = []
        
        for target_ret in target_returns:
            try:
                weights = self.optimizer.minimize_volatility(target_ret)
                vol = self.optimizer.portfolio_volatility(weights)
                frontier_volatilities.append(vol)
                frontier_weights.append(weights)
            except:
                pass
        
        return target_returns[:len(frontier_volatilities)], \
               np.array(frontier_volatilities), \
               np.array(frontier_weights)
    
    def plot_efficient_frontier(self, n_points=100, figsize=(12, 8)):
        """
        绘制有效前沿
        
        参数:
        n_points: 点数
        figsize: 图形大小
        """
        returns, volatilities, weights = self.calculate_efficient_frontier(n_points)
        
        # 最小方差组合
        min_var_weights = self.optimizer.minimize_variance()
        min_var_return = self.optimizer.portfolio_return(min_var_weights)
        min_var_vol = self.optimizer.portfolio_volatility(min_var_weights)
        
        # 最大夏普组合
        max_sharpe_weights = self.optimizer.maximize_sharpe()
        max_sharpe_return = self.optimizer.portfolio_return(max_sharpe_weights)
        max_sharpe_vol = self.optimizer.portfolio_volatility(max_sharpe_weights)
        max_sharpe_ratio = self.optimizer.portfolio_sharpe(max_sharpe_weights)
        
        # 绘图
        plt.figure(figsize=figsize)
        
        # 有效前沿
        plt.plot(volatilities, returns, 'b-', linewidth=2, label='有效前沿')
        
        # 最小方差组合
        plt.scatter(min_var_vol, min_var_return, 
                   c='red', s=100, marker='*', label='最小方差组合', zorder=5)
        
        # 最大夏普组合
        plt.scatter(max_sharpe_vol, max_sharpe_return,
                   c='green', s=100, marker='*', label='最大夏普组合', zorder=5)
        
        # 资本市场线（CML）
        cml_returns = self.risk_free_rate + max_sharpe_ratio * np.array([0, max(volatilities)])
        cml_vols = np.array([0, max(volatilities)])
        plt.plot(cml_vols, cml_returns, 'g--', linewidth=1.5, label='资本市场线')
        
        # 无风险利率
        plt.scatter(0, self.risk_free_rate, c='orange', s=80, marker='D', 
                   label='无风险资产', zorder=5)
        
        # 单个资产
        asset_vols = np.sqrt(np.diag(self.cov_matrix))
        plt.scatter(asset_vols, self.expected_returns, c='gray', s=50, 
                   alpha=0.6, label='单个资产')
        
        plt.xlabel('波动率（标准差）', fontsize=12)
        plt.ylabel('期望收益率', fontsize=12)
        plt.title('有效前沿与资本市场线', fontsize=14)
        plt.legend(loc='best', fontsize=10)
        plt.grid(True, alpha=0.3)
        plt.tight_layout()
        plt.show()
        
        return {
            'min_variance': {
                'weights': min_var_weights,
                'return': min_var_return,
                'volatility': min_var_vol
            },
            'max_sharpe': {
                'weights': max_sharpe_weights,
                'return': max_sharpe_return,
                'volatility': max_sharpe_vol,
                'sharpe_ratio': max_sharpe_ratio
            }
        }

# 示例
```

### 3.3 资本市场线（CML）

**定义**：
```
E(Rp) = Rf + (E(Rm) - Rf) / σm × σp
```

其中：
- Rf：无风险利率
- E(Rm)：市场组合期望收益
- σm：市场组合波动率
- σp：组合波动率

**特征**：
1. 连接无风险资产和最大夏普组合
2. 线性关系
3. 所有有效组合都在CML上

```python
def capital_market_line(risk_free_rate, market_return, market_vol, 
                        target_vols):
    """
    资本市场线
    
    参数:
    risk_free_rate: 无风险利率
    market_return: 市场组合收益
    market_vol: 市场组合波动率
    target_vols: 目标波动率数组
    
    返回:
    CML收益率
    """
    sharpe_ratio = (market_return - risk_free_rate) / market_vol
    cml_returns = risk_free_rate + sharpe_ratio * target_vols
    return cml_returns
```

### 3.4 证券市场线（SML）

**定义**：
```
E(Ri) = Rf + βi × (E(Rm) - Rf)
```

其中：
- βi：资产i的贝塔系数

**Beta计算**：
```
βi = Cov(Ri, Rm) / Var(Rm) = ρim × σi / σm
```

```python
def security_market_line(risk_free_rate, market_return, market_vol, 
                        asset_returns, asset_vols, correlations):
    """
    证券市场线
    
    参数:
    risk_free_rate: 无风险利率
    market_return: 市场收益
    market_vol: 市场波动率
    asset_returns: 资产收益率
    asset_vols: 资产波动率
    correlations: 与市场的相关性
    
    返回:
    Beta值, SML收益率
    """
    # Beta值
    betas = correlations * asset_vols / market_vol
    
    # SML收益率
    market_premium = market_return - risk_free_rate
    sml_returns = risk_free_rate + betas * market_premium
    
    return betas, sml_returns
```

## 4. 分散化效应与相关性分析

### 4.1 分散化效应

**核心原理**：
组合风险低于各资产风险的加权平均，取决于资产间的相关性。

**风险降低公式**：
```
σp ≤ Σ wi × σi
```

**分散化收益**：
```
Diversification Benefit = Σ wi × σi - σp
```

```python
def calculate_diversification_benefit(weights, volatilities, correlations):
    """
    计算分散化收益
    
    参数:
    weights: 权重
    volatilities: 波动率
    correlations: 相关性矩阵
    
    返回:
    分散化收益
    """
    # 加权平均波动率
    weighted_avg_vol = np.sum(weights * volatilities)
    
    # 协方差矩阵
    cov_matrix = np.outer(volatilities, volatilities) * correlations
    
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 分散化收益
    diversification_benefit = weighted_avg_vol - portfolio_vol
    
    return {
        'weighted_avg_vol': weighted_avg_vol,
        'portfolio_vol': portfolio_vol,
        'diversification_benefit': diversification_benefit,
        'risk_reduction_pct': diversification_benefit / weighted_avg_vol
    }

# 示例：不同相关性下的分散化效应
weights = np.array([0.4, 0.3, 0.3])
vols = np.array([0.20, 0.15, 0.12])

# 相关性矩阵（低相关）
low_corr = np.array([
    [1.0, 0.2, 0.1],
    [0.2, 1.0, 0.3],
    [0.1, 0.3, 1.0]
])

# 相关性矩阵（高相关）
high_corr = np.array([
    [1.0, 0.8, 0.7],
    [0.8, 1.0, 0.9],
    [0.7, 0.9, 1.0]
])

result_low = calculate_diversification_benefit(weights, vols, low_corr)
result_high = calculate_diversification_benefit(weights, vols, high_corr)

print("低相关性组合:")
print(f"加权平均波动率: {result_low['weighted_avg_vol']:.4f}")
print(f"组合波动率: {result_low['portfolio_vol']:.4f}")
print(f"分散化收益: {result_low['diversification_benefit']:.4f}")

print("\n高相关性组合:")
print(f"加权平均波动率: {result_high['weighted_avg_vol']:.4f}")
```

### 4.2 相关性分析

```python
def analyze_correlation_impact(weights, volatilities, n_scenarios=20):
    """
    分析相关性对组合风险的影响
    
    参数:
    weights: 权重
    volatilities: 波动率
    n_scenarios: 场景数
    
    返回:
    相关性范围, 组合波动率范围
    """
    # 相关性范围
    corr_values = np.linspace(0, 1, n_scenarios)
    portfolio_vols = []
    
    for corr in corr_values:
        # 构建相关性矩阵
        n = len(weights)
        corr_matrix = np.eye(n) * (1 - corr) + np.ones((n, n)) * corr
        
        # 协方差矩阵
        cov_matrix = np.outer(volatilities, volatilities) * corr_matrix
        
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        portfolio_vols.append(portfolio_vol)
    
    # 绘图
    plt.figure(figsize=(10, 6))
    plt.plot(corr_values, portfolio_vols, 'b-', linewidth=2)
    plt.xlabel('平均相关系数', fontsize=12)
    plt.ylabel('组合波动率', fontsize=12)
    plt.title('相关性对组合风险的影响', fontsize=14)
    plt.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.show()
```

### 4.3 边际风险贡献

**边际风险贡献（MRC）**：
```
MRCi = ∂σp / ∂wi = Cov(Ri, Rp) / σp
```

**风险贡献（RC）**：
```
RCi = wi × MRCi
```

**风险贡献比例**：
```
RC%i = RCi / σp
```

```python
def calculate_risk_contribution(weights, cov_matrix):
    """
    计算风险贡献
    
    参数:
    weights: 权重
    cov_matrix: 协方差矩阵
    
    返回:
    边际风险贡献, 风险贡献, 风险贡献比例
    """
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 边际风险贡献
    marginal_risk_contrib = cov_matrix @ weights / portfolio_vol
    
    # 风险贡献
    risk_contrib = weights * marginal_risk_contrib
    
    # 风险贡献比例
    risk_contrib_pct = risk_contrib / portfolio_vol
    
    return {
        'marginal_risk_contrib': marginal_risk_contrib,
        'risk_contrib': risk_contrib,
        'risk_contrib_pct': risk_contrib_pct,
        'portfolio_vol': portfolio_vol
    }

# 示例
```

```python
def build_etf_correlation_matrix(etf_returns, method='pearson', window=None):
    """
    构建ETF相关性矩阵
    
    参数:
    etf_returns: ETF收益率
    method: 相关性计算方法 ('pearson', 'spearman', 'kendall')
    window: 滚动窗口（None表示全样本）
    
    返回:
    相关性矩阵
    """
    if window:
        # 滚动相关性
        rolling_corr = etf_returns.rolling(window).corr()
        return rolling_corr
    else:
        # 全样本相关性
        if method == 'pearson':
            corr_matrix = etf_returns.corr()
        elif method == 'spearman':
            corr_matrix = etf_returns.corr(method='spearman')
        elif method == 'kendall':
            corr_matrix = etf_returns.corr(method='kendall')
        
        return corr_matrix

def analyze_correlation_stability(etf_returns, window=60):

    分析相关性稳定性
    
    参数:
    etf_returns: ETF收益率
    window
    返回:
    相关性稳定性指标
    """
    # 滚动相关性
    n_assets = etf_returns.shape[1]
    rolling_corr = etf_returns.rolling(window).corr()
    
    # 计算相关性标准差（衡量稳定性）
    corr_stability = {}
    
    for i in range(n_assets):
        for j in range(i+1, n_assets):
            pair = f"{etf_returns.columns[i]}_{etf_returns.columns[j]}"
            pair_corr = rolling_corr.iloc[:, i].iloc[::n_assets+1].iloc[:, j]
            corr_stability[pair] = {
                'mean': pair_corr.mean(),
                'std': pair_corr.std(),
                'min': pair_corr.min(),
                'max': pair_corr.max()
            }
    
    return pd.DataFrame(corr_stability).T
```

```python
def multi_objective_optimization(expected_returns, cov_matrix, 
                                 risk_aversion_range=(1, 10),
                                 n_portfolios=50):
    """
    多目标优化（收益-风险权衡）
    
    参数:
    expected_returns: 期望收益率
    cov_matrix: 协方差矩阵
    risk_aversion_range: 风险厌恶系数范围
    n_portfolios: 组合数
    
    返回:
    多目标优化结果
    """
    risk_aversions = np.linspace(
        risk_aversion_range[0], 
        risk_aversion_range[1], 
        n_portfolios
    )
    
    results = []
    
    for risk_aversion in risk_aversions:
        # 均值方差优化
        n_assets = len(expected_returns)
        
        def neg_utility(weights):
            ret = weights @ expected_returns
            var = weights @ cov_matrix @ weights
            utility = ret - 0.5 * risk_aversion * var
            return -utility
        
        constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        bounds = [(0, 1) for _ in range(n_assets)]
        
        result = minimize(
            neg_utility,
            x0=[1/n_assets] * n_assets,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        weights = result.x
        ret = weights @ expected_returns
        vol = np.sqrt(weights @ cov_matrix @ weights)
        
        results.append({
            'risk_aversion': risk_aversion,
            'return': ret,
            'volatility': vol,
            'weights': weights
        })
    
    return pd.DataFrame(results)
```

## 7. 总结与实践要点

### 7.1 核心要点

1. **现代投资组合理论（MPT）**
   - 量化风险与收益的关系
   - 分散化降低风险
   - 有效前沿是最优组合集合

2. **马科维茨均值-方差模型**
   - 目标函数：最大化效用
   - 约束条件：权重和为1
   - 求解方法：二次规划

3. **有效前沿**
   - 上凸曲线
   - 最小方差组合在左侧
   - 最大夏普组合在有效前沿上

4. **分散化效应**
   - 相关性越低，分散化效果越好
   - 风险贡献衡量资产对组合风险的贡献
   - 边际风险贡献指导权重调整

### 7.2 ETF组合配置实践要点

1. **数据准备**
   - 使用历史收益率估计期望收益
   - 使用历史协方差矩阵估计风险
   - 注意数据频率和窗口选择

2. **参数估计**
   - 期望收益：历史平均、风险溢价模型、分析师预测
   - 协方差矩阵：历史协方差、指数加权、收缩估计
   - 无风险利率：国债收益率

3. **优化设置**
   - 权重约束：最小/最大权重
   - 行业约束：行业权重限制
   - 交易成本：考虑换手成本

4. **再平衡策略**
   - 定期再平衡：月度、季度
   - 阈值再平衡：偏离一定比例
   - 交易成本权衡

### 7.3 模型局限性与改进

**局限性**：
1. 假设收益率服从正态分布
2. 历史数据无法预测未来
3. 估计误差被放大
4. 忽略尾部风险

**改进方向**：
1. 使用Black-Litterman模型融入主观观点
2. 使用风险平价模型实现风险分散
3. 使用鲁棒优化降低估计误差影响
4. 使用条件风险价值（CVaR）衡量尾部风险
