# 资产配置方法

## 概述

资产配置是量化投资的核心环节，决定如何在多个资产间分配资金。不同的配置方法有不同的风险收益特征，选择合适的配置方法对策略表现至关重要。

---

## 1. 风险平价（Risk Parity）

### 核心思想
使每个资产对组合的总风险贡献相等，而非资金相等。

### 数学原理

#### 风险贡献
单个资产i的风险贡献：
```
RC_i = w_i * ∂σ_p / ∂w_i = w_i * (Σw)_i / σ_p
```

其中：
- w_i: 资产i的权重
- Σ: 协方差矩阵
- σ_p: 组合标准差

#### 风险平价条件
```
RC_1 = RC_2 = ... = RC_n = σ_p / n
```

### Python实现

#### 1.1 基础实现
```python
import numpy as np
from scipy.optimize import minimize

def risk_parity_weights(cov_matrix):
    """
    计算风险平价权重
    
    参数：
    - cov_matrix: 协方差矩阵（n×n）
    
    返回：
    - weights: 风险平价权重
    """
    n = cov_matrix.shape[0]
    
    def risk_parity_objective(w):
        # 组合波动率
        port_vol = np.sqrt(np.dot(w, np.dot(cov_matrix, w)))
        
        # 各资产风险贡献
        marginal_contrib = np.dot(cov_matrix, w)
        risk_contrib = w * marginal_contrib / port_vol
        
        # 目标：风险贡献相等
        target_contrib = port_vol / n
        
        # 差异的平方和
        return np.sum((risk_contrib - target_contrib) ** 2)
    
    # 约束条件
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},  # 权重和为1
    ]
    
    # 边界条件
    bounds = [(0.01, 1) for _ in range(n)]  # 最小1%权重
    
    # 初始猜测（等权重）
    w0 = np.ones(n) / n
    
    # 优化
    result = minimize(
        risk_parity_objective,
        w0,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints,
        options={'ftol': 1e-12, 'maxiter': 1000}
    )
    
    return result.x

# 示例
cov_matrix = np.array([
    [0.04, 0.01, 0.01],
    [0.01, 0.09, 0.02],
    [0.01, 0.02, 0.16]
])

```

```python
def rolling_risk_parity(returns, lookback=252, rebalance_freq=21):
```

### 优点
- 风险分散，不依赖收益预测
- 对低波动资产友好
- 回撤较小

### 缺点
- 需要杠杆放大低波动资产
- 协方差矩阵估计误差敏感
- 假设资产间关系稳定

### 改进方法
1. **风险平价 + 风险预算**：允许不同资产有不同的风险贡献
2. **鲁棒协方差估计**：使用收缩估计或因子模型
3. **动态风险平价**：根据市场环境调整风险预算

## 2. 最小方差组合（Minimum Variance）

### 核心思想
在所有可能的组合中，选择波动率最小的组合。

### 数学原理
```
min w'Σw
s.t. w'1 = 1
```

### Python实现

#### 2.1 基础实现
```python
def minimum_variance_weights(cov_matrix, allow_short=False):
    """
    计算最小方差组合权重
    
    参数：
    - cov_matrix: 协方差矩阵
    - allow_short: 是否允许做空
    """
    n = cov_matrix.shape[0]
    
    def variance(w):
        return np.dot(w, np.dot(cov_matrix, w))
    
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    ]
    
    if allow_short:
        bounds = [(-1, 1) for _ in range(n)]
    else:
        bounds = [(0, 1) for _ in range(n)]
    
    w0 = np.ones(n) / n
    
    result = minimize(
        variance,
        w0,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

#### 2.2 闭式解（允许做空）
```python
def minimum_variance_closed_form(cov_matrix):
    """
    最小方差组合的闭式解（允许做空）
    
    w = Σ^{-1} * 1 / (1' * Σ^{-1} * 1)
    """
    cov_inv = np.linalg.inv(cov_matrix)
    ones = np.ones(cov_matrix.shape[0])
    
    weights = np.dot(cov_inv, ones) / np.dot(ones, np.dot(cov_inv, ones))
    
    return weights
```

```python
    带权重约束的最小方差组合
    
    参数：
    - min_weight: 最小权重
    - max_weight: 最大权重

        variance,

        method='SLSQP',

        constraints=constraints
    )
    
    return result.x
```

### 优点
- 理论上波动率最低
- 不需要收益预测
- 实现简单

### 缺点
- 可能过度集中
- 对协方差估计敏感
- 不考虑收益

### 改进方法
1. **权重约束**：限制单资产权重
2. **收缩估计**：改善协方差矩阵
3. **因子模型**：用因子协方差代替资产协方差

## 3. 最大分散化（Maximum Diversification）

### 核心思想
最大化分散化比率，即加权平均波动率与组合波动率的比值。

### 数学原理

分散化比率：
```
DR = (w'σ) / sqrt(w'Σw)
```

目标：
```
max (w'σ) / sqrt(w'Σw)
s.t. w'1 = 1, w >= 0
```

### Python实现

#### 3.1 基础实现
```python
def maximum_diversification_weights(cov_matrix, vols):
    """
    计算最大分散化组合权重
    
    参数：
    - cov_matrix: 协方差矩阵
    - vols: 各资产波动率向量
    """
    n = cov_matrix.shape[0]
    vols = np.array(vols)
    
    def neg_diversification_ratio(w):
        port_vol = np.sqrt(np.dot(w, np.dot(cov_matrix, w)))
        weighted_vol = np.dot(w, vols)
        return -weighted_vol / port_vol
    
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    ]
    
    bounds = [(0.01, 1) for _ in range(n)]
    
    w0 = np.ones(n) / n
    
    result = minimize(
        neg_diversification_ratio,
        w0,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x

# 计算分散化比率
def calculate_diversification_ratio(weights, cov_matrix, vols):
    """
    计算分散化比率
    """
    port_vol = np.sqrt(np.dot(weights, np.dot(cov_matrix, weights)))
    weighted_vol = np.dot(weights, vols)
    
    return weighted_vol / port_vol
```

## 4. Black-Litterman模型

### 核心思想
结合市场均衡观点和投资者主观观点，生成后验预期收益。

### 数学原理

#### 市场均衡收益
```
Π = δΣw_mkt
```

其中：
- Π: 市场均衡收益
- δ: 风险厌恶系数
- Σ: 协方差矩阵
- w_mkt: 市场组合权重

#### 投资者观点
```
P * μ = Q + ε
ε ~ N(0, Ω)
```

其中：
- P: 观点矩阵
- Q: 观点值
- Ω: 观点不确定性

#### 后验收益
```
μ_BL = [(τΣ)^{-1} + P'Ω^{-1}P]^{-1} * [(τΣ)^{-1}Π + P'Ω^{-1}Q]
```

### Python实现

#### 4.1 Black-Litterman模型
```python
def black_litterman(cov_matrix, market_caps, views, view_confidences,
                    risk_aversion=2.5, tau=0.05):
    """
    Black-Litterman模型
    
    参数：
    - cov_matrix: 协方差矩阵
    - market_caps: 市值向量
    - views: 观点字典 {资产索引: 观点值}
    - view_confidences: 观点置信度向量
    - risk_aversion: 风险厌恶系数
    - tau: 缩放参数
    
    返回：
    - posterior_returns: 后验收益
    - posterior_weights: 后验权重
    """
    n = cov_matrix.shape[0]
    
    # 市场组合权重
    w_mkt = market_caps / market_caps.sum()
    
    # 市场均衡收益
    pi = risk_aversion * np.dot(cov_matrix, w_mkt)
    
    # 构建观点矩阵P和Q
    k = len(views)  # 观点数量
    P = np.zeros((k, n))
    Q = np.zeros(k)
    
    for i, (asset_idx, view_value) in enumerate(views.items()):
        P[i, asset_idx] = 1
        Q[i] = view_value
    
    # 观点不确定性矩阵Ω
    omega = np.diag(view_confidences ** 2)
    
    # 计算后验收益
    tau_cov_inv = np.linalg.inv(tau * cov_matrix)
    omega_inv = np.linalg.inv(omega)
    
    # 后验均值
    posterior_mean = np.linalg.inv(
        tau_cov_inv + np.dot(P.T, np.dot(omega_inv, P))
    ) @ (
        np.dot(tau_cov_inv, pi) + np.dot(P.T, np.dot(omega_inv, Q))
    )
    
    # 后验协方差
    posterior_cov = np.linalg.inv(
        tau_cov_inv + np.dot(P.T, np.dot(omega_inv, P))
    )
    
    # 计算最优权重
    posterior_weights = np.linalg.inv(risk_aversion * cov_matrix) @ posterior_mean
    
    # 归一化
    posterior_weights = posterior_weights / posterior_weights.sum()
    
    return {
        'posterior_returns': posterior_mean,
        'posterior_cov': posterior_cov,
        'posterior_weights': posterior_weights,
        'market_equilibrium_returns': pi,
        'market_weights': w_mkt
    }
```

#### 4.2 示例：相对观点
```python
def black_litterman_relative_view(cov_matrix, market_caps, relative_views,
                                   risk_aversion=2.5, tau=0.05):
    """
    Black-Litterman模型 - 相对观点
    
    参数：
    - relative_views: 相对观点列表
      例如：[('Asset1', 'Asset2', 0.02, 0.01)] 表示Asset1比Asset2高2%，置信度1%
    """
    n = cov_matrix.shape[0]
    assets = list(range(n))
    
    # 市场权重和均衡收益
    w_mkt = market_caps / market_caps.sum()
    pi = risk_aversion * np.dot(cov_matrix, w_mkt)
    
    # 构建观点矩阵
    k = len(relative_views)
    P = np.zeros((k, n))
    Q = np.zeros(k)
    omega = np.zeros(k)
    
    for i, (asset1, asset2, view_diff, confidence) in enumerate(relative_views):
        P[i, asset1] = 1
        P[i, asset2] = -1
        Q[i] = view_diff
        omega[i] = confidence ** 2
    
    omega = np.diag(omega)
    
    # 计算后验
    tau_cov_inv = np.linalg.inv(tau * cov_matrix)
    omega_inv = np.linalg.inv(omega)
    
    posterior_mean = np.linalg.inv(
        tau_cov_inv + np.dot(P.T, np.dot(omega_inv, P))
    ) @ (
        np.dot(tau_cov_inv, pi) + np.dot(P.T, np.dot(omega_inv, Q))
    )
    
    posterior_weights = np.linalg.inv(risk_aversion * cov_matrix) @ posterior_mean
    posterior_weights = posterior_weights / posterior_weights.sum()
    
    return posterior_mean, posterior_weights
```

### 优点
- 结合客观和主观信息
- 观点可以相对或绝对
- 结果稳健

### 缺点
- 观点置信度设定主观
- 依赖市场均衡假设
- 实现复杂

```python
def volatility_targeting_weights(returns, target_vol=0.15, lookback=63):
    """
    波动率目标配置
    
    调整杠杆使组合波动率接近目标
    """
    # 计算滚动波动率
    rolling_vol = returns.rolling(lookback).std() * np.sqrt(252)
    
    # 计算杠杆比例
    leverage = target_vol / rolling_vol
    
    # 限制杠杆范围
    leverage = leverage.clip(0.5, 2.0)
    
    # 基础权重（等权重）
    base_weights = pd.Series(1/len(returns.columns), index=returns.columns)
    
    # 调整后的权重
    adjusted_weights = base_weights * leverage
    
    return adjusted_weights
```

```python
def momentum_based_allocation(returns, lookback=252, top_n=3):
    """
    动量驱动的资产配置
    
    选择过去表现最好的资产
    """
    weights_history = []
    assets = returns.columns
    
    for i in range(lookback, len(returns)):
        window = returns.iloc[i-lookback:i]
        
        # 计算动量（收益率）
        momentum = window.sum()
        
        # 选择top_n资产
        top_assets = momentum.nlargest(top_n).index
        
        # 分配权重（风险平价）
        top_returns = window[top_assets]
        cov_matrix = top_returns.cov().values
        weights = risk_parity_weights(cov_matrix)
        
        # 构建完整权重向量
        full_weights = pd.Series(0, index=assets)
        full_weights[top_assets] = weights
        
        weights_history.append(full_weights)
    
    return pd.DataFrame(weights_history, index=returns.index[lookback:])
```

#### 5.4 风险预算动态配置
```python
def dynamic_risk_budgeting(returns, risk_budgets, lookback=126, rebalance_freq=21):
    """
    动态风险预算配置
    
    根据市场环境调整风险预算
    """
    n_assets = returns.shape[1]
    assets = returns.columns
    
    weights_history = []
    dates = []
    
    for i in range(lookback, len(returns), rebalance_freq):
        window = returns.iloc[i-lookback:i]
        cov_matrix = window.cov().values
        
        # 根据波动率调整风险预算
        vols = window.std() * np.sqrt(252)
        market_vol = vols.mean()
        
        # 高波动时降低股票风险预算
        if market_vol > 0.25:
            adjusted_budgets = risk_budgets.copy()
            adjusted_budgets['equity'] *= 0.7
            adjusted_budgets['bond'] *= 1.2
        else:
            adjusted_budgets = risk_budgets
        
        # 转换为向量
        budget_vector = np.array([adjusted_budgets.get(asset, 1/n_assets) 
                                   for asset in assets])
        
        # 风险预算优化
        weights = risk_budget_weights(cov_matrix, budget_vector)
        
        weights_history.append(weights)
        dates.append(returns.index[i])
    
    return pd.DataFrame(weights_history, index=dates, columns=assets)

def risk_budget_weights(cov_matrix, risk_budget):
    """
    风险预算权重计算
    """
    n = cov_matrix.shape[0]
    
    def objective(w):
        port_vol = np.sqrt(np.dot(w, np.dot(cov_matrix, w)))
        marginal_contrib = np.dot(cov_matrix, w)
        risk_contrib = w * marginal_contrib / port_vol
        
        # 目标：风险贡献与风险预算一致
        return np.sum((risk_contrib / port_vol - risk_budget) ** 2)
    
    constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
    bounds = [(0.01, 1) for _ in range(n)]
    w0 = np.ones(n) / n
    
    result = minimize(objective, w0, method='SLSQP', 
                     bounds=bounds, constraints=constraints)
    
    return result.x
```

## 应用到V4.2的建议

### 短期改进（优先级高）
1. **风险平价配置**
   - 使用风险平价分配各资产权重
   - 确保各资产风险贡献相等
   - 滚动更新协方差矩阵

2. **波动率目标**
   - 设定目标波动率（如15%）
   - 根据市场波动率调整杠杆
   - 高波动降仓，低波动加仓

### 中期改进（优先级中）
3. **动态配置**
   - 根据市场环境切换配置方案
   - 牛市增加股票权重
   - 熊市增加债券权重

4. **风险预算**
   - 设定各资产的风险预算
   - 动态调整风险预算
   - 监控风险贡献

### 长期改进（优先级低）
5. **Black-Litterman**
   - 结合市场均衡和主观观点
   - 根据策略信号形成观点
   - 更新后验收益和权重
