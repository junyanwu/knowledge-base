```python
def stat_arb_strategy(price_pairs, lookback=60, entry_z=2, exit_z=0.5):
    """
    统计套利策略
    
    参数:
    price_pairs: 价格对
    lookback: 回溯期
    entry_z: 入场阈值
    exit_z: 出场阈值
    
    返回:
    交易信号
    """
    # 协整检验
    spread = price_pairs['price1'] - price_pairs['price2']
    
    # 标准化
    spread_mean = spread.rolling(lookback).mean()
    spread_std = spread.rolling(lookback).std()
    z_score = (spread - spread_mean) / spread_std
    
    # 交易信号
    signal = pd.Series(0, index=spread.index)
    
    # 开仓
    signal[z_score > entry_z] = -1  # 做空价差
    signal[z_score < -entry_z] = 1   # 做多价差
    
    # 平仓
    signal[abs(z_score) < exit_z] = 0
    
    return signal
```

```python
def risk_parity_allocation(assets, risk_budget=None):
    """
    风险平价配置
    
    参数:
    assets: 资产数据
    risk_budget: 风险预算
    
    返回:
    配置权重
    """
    # 协方差矩阵
    cov_matrix = assets.cov()
    
    # 风险预算（默认等风险）
    if risk_budget is None:
        risk_budget = np.ones(len(assets.columns)) / len(assets.columns)
    
    # 优化
    from scipy.optimize import minimize
    
    def risk_budget_objective(weights):
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        marginal_risk = cov_matrix @ weights / portfolio_vol
        risk_contrib = weights * marginal_risk
        target_risk = portfolio_vol / len(weights)
        return np.sum((risk_contrib - target_risk) ** 2)
    
    n_assets = len(assets.columns)
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    result = minimize(
        risk_budget_objective,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

### 3.2 核心策略

**多因子模型：**
```python
def multi_factor_model(factors, returns):
    """
    多因子模型
    
    参数:
    factors: 因子数据
    returns: 收益率
    
    返回:
    因子收益、因子暴露
    """
    from sklearn.linear_model import LinearRegression
    
    # 因子收益
    model = LinearRegression()
    model.fit(factors, returns)
    factor_returns = model.coef_
    
    # 因子暴露
    factor_exposure = factors
    
    return factor_returns, factor_exposure

# 因子示例
```

**市场中性：**
```python
def market_neutral_portfolio(alpha, beta, market_return):
    """
    市场中性组合
    
    参数:
    alpha: 股票Alpha
    beta: 股票Beta
    market_return: 市场收益
    
    返回:
    对冲后收益
    """
    # 多头组合
    long_return = alpha + beta * market_return
    
    # 空头对冲
    hedge_return = -beta * market_return
    
    # 市场中性收益
    neutral_return = long_return + hedge_return
    
    return neutral_return
```

## 4. 策略失效案例

### 4.1 2015年A股股灾

**背景：**
- 2015年6月-8月
- A股大幅下跌
- 量化策略普遍失效

**失效原因：**
1. **流动性枯竭**
   - 千股跌停
   - 无法卖出
   - 止损失效

2. **模型失效**
   - 历史数据无法预测极端事件
   - 相关性剧变
   - 对冲失效

3. **杠杆风险**
   - 杠杆放大损失
   - 强制平仓
   - 踩踏

**教训：**
- 流动性风险是最大风险
- 极端情况下的模型失效
- 杠杆需谨慎

### 4.2 2020年美股熔断

**背景：**
- 2020年3月
- 疫情冲击
- 美股多次熔断

**失效原因：**
1. **相关性崩溃**
   - 所有资产同时下跌
   - 分散化失效

2. **波动率飙升**
   - VIX指数涨至80+
   - 期权定价失效

3. **流动性危机**
   - 国债市场流动性枯竭
   - ETF折价

**教训：**
- 极端情况下所有资产相关性趋近1
- 需要真正的分散化
- 现金是最好的对冲

**过度杠杆：**
- 杠杆放大收益，也放大损失
- 极端情况下强制平仓
- 保持适度杠杆

**模型依赖：**
- 历史不代表未来
- 极端情况模型失效
- 保持敬畏之心

**流动性风险：**
- 流动性是最重要的
- 流动性危机无法预测
- 保持流动性储备

**集中风险：**
- 分散化是免费的午餐
- 相关性在危机时失效
- 需要真正的分散
