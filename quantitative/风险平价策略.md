# 风险平价策略

## 1. 策略原理

### 1.1 核心思想

**风险平价：** 每个资产对组合的总风险贡献相等，而非资金权重相等。

**对比：**
```
传统60/40组合：
- 股票60%，债券40%
- 但股票风险贡献≈90%，债券风险贡献≈10%

风险平价组合：
- 股票约15%，债券约85%
- 股票风险贡献=债券风险贡献=50%
```

### 1.2 风险贡献公式

**边际风险贡献：**
```
MRCᵢ = ∂σₚ / ∂wᵢ = Cov(rᵢ, rₚ) / σₚ
```

**风险贡献：**
```
RCᵢ = wᵢ × MRCᵢ = wᵢ × Cov(rᵢ, rₚ) / σₚ
```

**风险贡献比例：**
```
MRC%ᵢ = RCᵢ / σₚ = wᵢ × βᵢ
```

## 2. 策略实现

### 2.1 基础风险平价

```python
import numpy as np
from scipy.optimize import minimize

def risk_parity_weights(cov_matrix):
    """
    风险平价权重
    
    参数:
    cov_matrix: 协方差矩阵
    
    返回:
    风险平价权重
    """
    n_assets = cov_matrix.shape[0]
    
    def risk_budget_objective(weights):
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 边际风险贡献
        marginal_risk = cov_matrix @ weights / portfolio_vol
        
        # 风险贡献
        risk_contrib = weights * marginal_risk
        
        # 目标：风险贡献相等
        target_risk = portfolio_vol / n_assets
        
        return np.sum((risk_contrib - target_risk) ** 2)
    
    # 约束和边界
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    # 优化
    result = minimize(
        risk_budget_objective,
        x0=[1/n_assets] * n_assets,
        method='SLSQP',
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

### 2.2 带目标波动率的风险平价

```python
def risk_parity_with_target_vol(cov_matrix, target_vol=0.10):
    """
    带目标波动率的风险平价
    
    参数:
    cov_matrix: 协方差矩阵
    target_vol: 目标波动率
    
    返回:
    权重、杠杆
    """
    # 风险平价权重
    weights = risk_parity_weights(cov_matrix)
    
    # 组合波动率
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    
    # 杠杆
    leverage = target_vol / portfolio_vol
    
    # 最终权重
    final_weights = weights * leverage
    
    return final_weights, leverage
```

### 3.2 资产选择

```python
def select_assets_for_risk_parity(returns, n_assets=5):
    """
    为风险平价选择资产
    
    参数:
    returns: 候选资产收益率
    n_assets: 目标资产数
    
    返回:
    选中资产
    """
    # 计算相关性
    corr_matrix = returns.corr()
    
    # 选择低相关资产
    selected = [returns.columns[0]]
    
    while len(selected) < n_assets:
        min_corr_sum = float('inf')
        best_asset = None
        
        for asset in returns.columns:
            if asset in selected:
                continue
            
            # 计算与已选资产的平均相关性
            corr_sum = sum([abs(corr_matrix.loc[asset, s]) for s in selected])
            
            if corr_sum < min_corr_sum:
                min_corr_sum = corr_sum
                best_asset = asset
        
        selected.append(best_asset)
    
    return selected
```

```python
class RiskParityBacktest:
    """风险平价回测"""
    
    def __init__(self, returns, rebalance_freq=20, target_vol=0.10):
        self.returns = returns
        self.rebalance_freq = rebalance_freq
        self.target_vol = target_vol
    
    def run(self):
        """运行回测"""
        n_periods = len(self.returns)
        weights_history = []
        portfolio_returns = []
        
        for i in range(self.rebalance_freq, n_periods, self.rebalance_freq):

                cov_matrix, self.target_vol

        annual_return = portfolio_returns.mean() * 252
        
        # 年化波动率
        annual_vol = portfolio_returns.std() * np.sqrt(252)
        
        # 夏普比率
        sharpe = annual_return / annual_vol
        
        # 最大回撤
        cumulative = (1 + portfolio_returns).cumprod()
        drawdown = (cumulative - cumulative.cummax()) / cumulative.cummax()
        max_drawdown = drawdown.min()
        
        return {
            'annual_return': annual_return,
            'annual_vol': annual_vol,
            'sharpe': sharpe,
            'max_drawdown': max_drawdown
        }
```

```python
    """
    风险平价 vs 60/40组合
    
    参数:
    returns: 收益率
    
    返回:
    对比结果
    """

    # 绩效对比

    comparison = pd.DataFrame({
        '60/40': perf_60_40,
        'Risk Parity': perf_rp
    })
```

```python
def all_weather_portfolio():
    """
    全天候组合（Ray Dalio）
    
    资产配置:
    - 30% 股票
    - 40% 长期国债
    - 15% 中期国债
    - 7.5% 黄金
    - 7.5% 商品
    """
    assets = ['US Stocks', 'Long-term Bonds', 'Mid-term Bonds', 'Gold', 'Commodities']
    weights = np.array([0.30, 0.40, 0.15, 0.075, 0.075])
    
    # 假设协方差矩阵
    volatilities = np.array([0.15, 0.08, 0.05, 0.15, 0.20])
    correlations = np.array([
        [1.00, -0.30, -0.20, 0.00, 0.30],
        [-0.30, 1.00, 0.80, 0.20, -0.10],
        [-0.20, 0.80, 1.00, 0.15, -0.05],
        [0.00, 0.20, 0.15, 1.00, 0.30],
        [0.30, -0.10, -0.05, 0.30, 1.00]
    ])
    
    cov_matrix = np.outer(volatilities, volatilities) * correlations
    
    # 风险贡献
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    marginal_risk = cov_matrix @ weights / portfolio_vol
    risk_contrib = weights * marginal_risk
    risk_contrib_pct = risk_contrib / portfolio_vol
    
    print('全天候组合:')
    print(f'权重: {weights}')
    print(f'风险贡献: {risk_contrib_pct}')
    
    return weights, risk_contrib_pct
```

### 5.2 风险平价基金

```python
def risk_parity_fund_example():
    """
    风险平价基金示例
    
    特点:
    - 低波动率目标（8-10%）
    - 高杠杆（1.5-2.5倍）
    - 多资产分散
    """
    # 资产
    assets = ['US Equity', 'Intl Equity', 'EM Equity', 
              'US Treasury', 'US TIPS', 'US IG Bond',
              'Commodities', 'Gold']
    
    # 历史波动率
    volatilities = np.array([0.15, 0.18, 0.22, 0.05, 0.06, 0.08, 0.20, 0.18])
    
    # 相关性（简化）
    correlations = np.eye(len(assets)) * 0.3 + (1 - np.eye(len(assets))) * 0.3
    
    # 协方差
    cov_matrix = np.outer(volatilities, volatilities) * correlations
    
    # 风险平价权重
    weights = risk_parity_weights(cov_matrix)
    
    # 目标波动率10%，计算杠杆
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    leverage = 0.10 / portfolio_vol
    
    # 最终配置
    final_weights = weights * leverage
    
    print('风险平价基金:')
    for asset, weight in zip(assets, final_weights):
        print(f'{asset}: {weight:.2%}')
    print(f'\n杠杆: {leverage:.2f}x')
    
    return final_weights
```

## 6. 优缺点分析

### 6.1 优点

1. **风险分散**：真正实现风险分散
2. **波动率稳定**：组合波动率更稳定
3. **适应性强**：各种市场环境表现稳健
4. **可解释性强**：风险贡献清晰

### 6.2 缺点

1. **杠杆风险**：需要杠杆达到目标波动率
2. **估计误差**：协方差矩阵估计困难
3. **流动性风险**：部分资产流动性差
4. **交易成本**：频繁再平衡增加成本

风险平价策略要点：

1. **核心原理**：风险贡献相等

4. **回测绩效**：夏普比率高、回撤小
