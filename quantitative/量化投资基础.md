# 量化投资基础知识

## 一、量化投资概述

### 1.1 什么是量化投资
量化投资是利用数学模型、统计分析、计算机程序来进行投资决策的方法。它通过数据挖掘和算法模型，寻找市场中的投资机会。

### 1.2 量化投资特点
- **客观性**: 基于数据和模型，减少人为情绪干扰
- **系统性**: 有完整的投资流程和风险控制体系
- **及时性**: 高频交易快速响应市场变化
- **分散性**: 通常持有大量标的，分散风险

### 1.3 量化投资与传统投资对比

| 特征 | 量化投资 | 传统投资 |
|------|---------|---------|
| 决策依据 | 数据和模型 | 基本面分析 |
| 投资范围 | 广泛分散 | 相对集中 |
| 情绪影响 | 低 | 高 |
| 交易频率 | 可高可低 | 相对较低 |
| 透明度 | 模型可验证 | 主观判断多 |

## 二、量化策略类型

### 2.1 趋势跟踪策略
- **原理**: 趋势一旦形成会持续一段时间
- **实现**: 移动平均线、突破策略
- **适用**: 明显的趋势市场

```python
# 简单移动平均线策略示例
def ma_strategy(prices, short_window=5, long_window=20):
    signals = []
    short_ma = prices.rolling(short_window).mean()
    long_ma = prices.rolling(long_window).mean()

    for i in range(len(prices)):
        if short_ma[i] > long_ma[i]:
            signals.append(1)  # 买入
        else:
            signals.append(-1)  # 卖出
    return signals
```

### 2.2 均值回归策略
- **原理**: 价格偏离均值后会回归
- **实现**: 布林带、RSI、统计套利
- **适用**: 震荡市场

### 2.3 因子投资策略
- **价值因子**: 低估值股票
- **动量因子**: 过去表现好的股票
- **质量因子**: 财务稳健的公司
- **低波动因子**: 波动率较低的股票
- **规模因子**: 小盘股效应

### 2.4 套利策略
- **统计套利**: 配对交易
- **期现套利**: 期货与现货价差
- **跨期套利**: 不同月份期货价差
- **跨市场套利**: A股与H股价差

### 2.5 高频交易策略
- **做市策略**: 提供流动性赚取价差
- **动量策略**: 捕捉短期价格变化
- **套利策略**: 快速捕捉定价错误

### 3.3 回测分析
```python
# 简单回测框架
def backtest(signals, prices, initial_capital=100000):
    capital = initial_capital
    position = 0
    portfolio_value = []

    for i, signal in enumerate(signals):
        if signal == 1 and position == 0:  # 买入
            position = capital / prices[i]
            capital = 0
        elif signal == -1 and position > 0:  # 卖出
            capital = position * prices[i]
            position = 0

        portfolio_value.append(capital + position * prices[i])

    return portfolio_value
```

## 四、常用量化指标

### 4.1 收益指标
- **累计收益率**: 总收益
- **年化收益率**: 换算成年度收益
- **超额收益**: 相对基准的收益

### 4.2 风险指标
- **波动率**: 收益率标准差
- **最大回撤**: 从高点到低点的最大跌幅
- **下行风险**: 负收益的波动率

### 4.3 风险调整收益指标
- **夏普比率**: (收益率 - 无风险利率) / 波动率
- **索提诺比率**: (收益率 - 无风险利率) / 下行风险
- **卡玛比率**: 年化收益 / 最大回撤

```python
# 计算夏普比率
def sharpe_ratio(returns, risk_free_rate=0.03):
    excess_returns = returns - risk_free_rate / 252
    return excess_returns.mean() / excess_returns.std() * np.sqrt(252)

# 计算最大回撤
def max_drawdown(portfolio_value):
    peak = portfolio_value.expanding(min_periods=1).max()
    drawdown = (portfolio_value - peak) / peak
    return drawdown.min()
```

### 4.4 交易成本
- **手续费**: 券商佣金
- **印花税**: 卖出时0.1%
- **滑点**: 实际成交价与理论价差
- **冲击成本**: 大单交易对价格的影响

## 八、2026年量化投资动态

### 8.1 行业规模快速增长

- **业绩表现**: 量化私募业绩普遍亮眼
- **驱动因素**:

  - 市场结构显著变化，扩大量化策略有效容量
  - 量化需求持续释放

### 8.2 策略创新趋势

**量化择时策略的"变形式"**
相对于纯多头的量化股多来说，量选择时策略最大的特点是仓位会灵活变动——模型看好市场时增加仓位，反之降低仓位。

**仓位调节方式：**
1. **股指期货端择时**
   - 核心思路："股票端仓位不变，股指期货端做多空调节"
   - 例如：80%资金投资量化股多策略，剩余20%用于股指期货多空调节

2. **量化+主动选股策略**
   - 核心框架："低估低波进，高估高波出"
   - 主要因子：估值因子、财务质量因子、波动率因子
   - 优势：结合量化客观性与主动管理灵活性

### 8.3 竞争核心转变
灵昀投资观点：
- **稳健与精细成为行业竞争核心**
- 量化策略需要持续优化和迭代
- 风险管理和模型稳定性愈发重要

### 8.4 市场环境影响
