# 统计学基础 - 模块3：假设检验与统计推断

## 1. 假设检验基础

### 1.1 基本概念

**假设检验的逻辑：**
- 通过样本数据推断总体参数
- 基于小概率原理进行决策
- 控制两类错误的概率

**核心要素：**

| 概念 | 定义 | 表示 |
|------|------|------|
| 原假设 | 待检验的假设 | H₀ |
| 备择假设 | 与原假设对立 | H₁ 或 Hₐ |
| 显著性水平 | 拒绝真原假设的概率 | α |
| 检验统计量 | 用于决策的统计量 | T, Z, χ², F |
| p值 | 拒绝H₀的最小显著性水平 | p-value |
| 临界值 | 决策边界 | c |

**两类错误：**

|  | H₀为真 | H₀为假 |
|--|--------|--------|
| 接受H₀ | 正确决策 | 第二类错误(β) |
| 拒绝H₀ | 第一类错误(α) | 正确决策 |

**检验功效：**
```
Power = 1 - β = P(拒绝H₀ | H₀为假)
```

### 1.2 检验步骤

**标准流程：**

1. **提出假设**
   ```
   H₀: θ = θ₀
   H₁: θ ≠ θ₀ (双侧检验)
   H₁: θ > θ₀ 或 θ < θ₀ (单侧检验)
   ```

2. **选择显著性水平**
   ```
   通常 α = 0.05 或 0.01
   ```

3. **选择检验统计量**
   ```
   Z检验：大样本或σ已知
   t检验：小样本且σ未知
   χ²检验：方差或拟合优度
   F检验：方差比或ANOVA
   ```

4. **确定拒绝域**
   ```
   双侧：|Z| > z_{α/2}
   单侧：Z > z_α 或 Z < -z_α
   ```

5. **计算检验统计量**
   ```
   从样本数据计算
   ```

6. **做出决策**
   ```
   若统计量在拒绝域，则拒绝H₀
   否则，不能拒绝H₀
   ```

7. **得出结论**
   ```
   结合实际问题解释
   ```

## 2. 参数检验

### 2.1 均值检验

**单样本均值检验：**

**Z检验（σ已知）：**
```
H₀: μ = μ₀
Z = (X̄ - μ₀) / (σ/√n) ~ N(0,1)
```

**t检验（σ未知）：**
```
H₀: μ = μ₀
t = (X̄ - μ₀) / (s/√n) ~ t_{n-1}
```

**Python实现：**
```python
from scipy import stats
import numpy as np

# 单样本t检验
sample = np.array([5.2, 5.8, 6.1, 5.5, 5.9])
mu0 = 5.5  # 假设的总体均值

t_stat, p_value = stats.ttest_1samp(sample, mu0)
print(f't统计量: {t_stat:.4f}')
print(f'p值: {p_value:.4f}')

if p_value < 0.05:
    print('拒绝原假设')
else:
    print('不能拒绝原假设')
```

**两样本均值检验：**

**独立样本t检验：**
```
H₀: μ₁ = μ₂
t = (X̄₁ - X̄₂) / √(s²ₚ(1/n₁ + 1/n₂))
```

其中：
```
s²ₚ = ((n₁-1)s²₁ + (n₂-1)s²₂) / (n₁+n₂-2)
```

**配对样本t检验：**
```
H₀: μ_d = 0
t = D̄ / (s_d/√n)
```

**Python实现：**
```python
# 独立样本t检验
group1 = np.array([25, 30, 28, 35, 32])
group2 = np.array([20, 22, 18, 25, 21])

t_stat, p_value = stats.ttest_ind(group1, group2)
print(f'独立样本t检验: t={t_stat:.4f}, p={p_value:.4f}')

# 配对样本t检验
before = np.array([120, 130, 125, 140, 135])
after = np.array([115, 125, 120, 135, 130])

t_stat, p_value = stats.ttest_rel(before, after)
print(f'配对样本t检验: t={t_stat:.4f}, p={p_value:.4f}')
```

### 2.2 方差检验

**单样本方差检验：**
```
H₀: σ² = σ²₀
χ² = (n-1)s² / σ²₀ ~ χ²_{n-1}
```

**两样本方差检验：**
```
H₀: σ²₁ = σ²₂
F = s²₁ / s²₂ ~ F_{n₁-1, n₂-1}
```

**Python实现：**
```python
# 两样本方差检验
sample1 = np.array([12, 15, 18, 14, 16, 13, 17])
sample2 = np.array([10, 20, 8, 22, 6, 24, 12])

F_stat = np.var(sample1, ddof=1) / np.var(sample2, ddof=1)
p_value = 2 * (1 - stats.f.cdf(F_stat, len(sample1)-1, len(sample2)-1))

print(f'F统计量: {F_stat:.4f}')
print(f'p值: {p_value:.4f}')
```

### 2.3 比例检验

**单样本比例检验：**
```
H₀: p = p₀
Z = (p̂ - p₀) / √(p₀(1-p₀)/n) ~ N(0,1)
```

**两样本比例检验：**
```
H₀: p₁ = p₂
Z = (p̂₁ - p̂₂) / √(p̂(1-p̂)(1/n₁ + 1/n₂))
```

其中：
```
p̂ = (x₁ + x₂) / (n₁ + n₂)
```

**Python实现：**
```python
from statsmodels.stats.proportion import proportions_ztest

# 单样本比例检验
count = 45  # 成功次数
nobs = 100  # 总次数
value = 0.5  # 假设的比例

z_stat, p_value = proportions_ztest(count, nobs, value)
print(f'Z统计量: {z_stat:.4f}, p值: {p_value:.4f}')

# 两样本比例检验
count = np.array([45, 55])
nobs = np.array([100, 120])

z_stat, p_value = proportions_ztest(count, nobs)
print(f'Z统计量: {z_stat:.4f}, p值: {p_value:.4f}')
```

## 3. 非参数检验

### 3.1 符号检验

**适用场景：**
- 中位数检验
- 配对样本比较
- 数据不满足正态性

**检验方法：**
```
S = 正差值的个数
在H₀下，S ~ B(n, 0.5)
```

**Python实现：**
```python
from scipy.stats import binom_test

data = np.array([12, 15, 18, 14, 16, 13, 17, 19])
median_test = 15

# 符号检验
positive = np.sum(data > median_test)
n = len(data)

p_value = stats.binom_test(positive, n, 0.5, alternative='two-sided')
print(f'符号检验 p值: {p_value:.4f}')
```

### 3.2 Wilcoxon符号秩检验

**适用场景：**
- 配对样本比较
- 比符号检验更有效
- 考虑差值大小

**检验统计量：**
```
W⁺ = 正差值的秩和
W⁻ = 负差值的秩和
W = min(W⁺, W⁻)
```

**Python实现：**
```python
before = np.array([120, 130, 125, 140, 135])
after = np.array([115, 125, 120, 135, 130])

# Wilcoxon符号秩检验
stat, p_value = stats.wilcoxon(before, after)
print(f'Wilcoxon符号秩检验: W={stat:.4f}, p={p_value:.4f}')
```

### 3.3 Mann-Whitney U检验

**适用场景：**
- 两独立样本比较
- 不满足正态性假设
- 比较分布位置

**检验统计量：**
```
U₁ = n₁n₂ + n₁(n₁+1)/2 - R₁
U₂ = n₁n₂ + n₂(n₂+1)/2 - R₂
U = min(U₁, U₂)
```

**Python实现：**
```python
group1 = np.array([25, 30, 28, 35, 32])
group2 = np.array([20, 22, 18, 25, 21])

# Mann-Whitney U检验
stat, p_value = stats.mannwhitneyu(group1, group2, alternative='two-sided')
print(f'Mann-Whitney U检验: U={stat:.4f}, p={p_value:.4f}')
```

### 3.4 Kruskal-Wallis检验

**适用场景：**
- 多独立样本比较
- 非参数ANOVA

**检验统计量：**
```
H = (12/N(N+1)) Σ(Rᵢ²/nᵢ) - 3(N+1)
H ~ χ²_{k-1}
```

**Python实现：**
```python
group1 = np.array([25, 30, 28])
group2 = np.array([20, 22, 18])
group3 = np.array([35, 32, 38])

# Kruskal-Wallis检验
stat, p_value = stats.kruskal(group1, group2, group3)
print(f'Kruskal-Wallis检验: H={stat:.4f}, p={p_value:.4f}')
```

## 4. 方差分析

### 4.1 单因素方差分析

**模型：**
```
Yᵢⱼ = μ + τᵢ + εᵢⱼ
```

其中：
- μ：总体均值
- τᵢ：第i组效应
- εᵢⱼ：随机误差

**假设：**
```
H₀: τ₁ = τ₂ = ... = τₖ = 0
H₁: 至少一个τᵢ ≠ 0
```

**F检验：**
```
F = MSB / MSW ~ F_{k-1, N-k}
```

其中：
```
MSB = SS_B / (k-1)  # 组间均方
MSW = SS_W / (N-k)  # 组内均方
```

**Python实现：**
```python
from scipy.stats import f_oneway

group1 = np.array([25, 30, 28, 35, 32])
group2 = np.array([20, 22, 18, 25, 21])
group3 = np.array([35, 32, 38, 40, 36])

# 单因素ANOVA
F_stat, p_value = f_oneway(group1, group2, group3)
print(f'F统计量: {F_stat:.4f}')
print(f'p值: {p_value:.4f}')

if p_value < 0.05:
    print('组间存在显著差异')
else:
    print('组间无显著差异')
```

### 4.2 多重比较

**Tukey HSD检验：**
```python
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# 准备数据
data = np.concatenate([group1, group2, group3])
groups = ['A']*len(group1) + ['B']*len(group2) + ['C']*len(group3)

# Tukey HSD
tukey = pairwise_tukeyhsd(data, groups, alpha=0.05)
print(tukey)
```

**Bonferroni校正：**
```python
from scipy.stats import ttest_ind

# 多重比较
groups_data = [group1, group2, group3]
comparisons = [(0,1), (0,2), (1,2)]

for i, j in comparisons:
    t_stat, p_value = ttest_ind(groups_data[i], groups_data[j])
    adjusted_p = p_value * len(comparisons)  # Bonferroni校正
    print(f'组{i+1} vs 组{j+1}: t={t_stat:.4f}, 调整p={min(adjusted_p, 1):.4f}')
```

### 4.3 双因素方差分析

**模型：**
```
Yᵢⱼₖ = μ + αᵢ + βⱼ + (αβ)ᵢⱼ + εᵢⱼₖ
```

其中：
- αᵢ：因素A效应
- βⱼ：因素B效应
- (αβ)ᵢⱼ：交互效应

**Python实现：**
```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.formula.api import ols

# 创建数据
data = pd.DataFrame({
    'value': [25, 30, 28, 35, 20, 22, 18, 25, 35, 32, 38, 40],
    'factor_A': ['A1']*4 + ['A1']*4 + ['A2']*4,
    'factor_B': ['B1', 'B1', 'B2', 'B2']*3
})

# 双因素ANOVA
model = ols('value ~ C(factor_A) + C(factor_B) + C(factor_A):C(factor_B)', data=data).fit()
anova_table = sm.stats.anova_lm(model, typ=2)
print(anova_table)
```

## 5. 拟合优度检验

### 5.1 χ²拟合优度检验

**检验总体分布：**
```
H₀: 总体服从某分布
χ² = Σ(Oᵢ - Eᵢ)² / Eᵢ ~ χ²_{k-m-1}
```

其中：
- Oᵢ：观察频数
- Eᵢ：期望频数
- k：类别数
- m：估计参数个数

**Python实现：**
```python
from scipy.stats import chisquare

# 观察频数
observed = np.array([30, 25, 20, 15, 10])

# 期望频数（假设均匀分布）
expected = np.array([20, 20, 20, 20, 20])

# χ²检验
chi2_stat, p_value = chisquare(observed, expected)
print(f'χ²统计量: {chi2_stat:.4f}')
print(f'p值: {p_value:.4f}')
```

### 5.2 列联表检验

**独立性检验：**
```
H₀: 两变量独立
χ² = Σ(Oᵢⱼ - Eᵢⱼ)² / Eᵢⱼ ~ χ²_{(r-1)(c-1)}
```

**Python实现：**
```python
from scipy.stats import chi2_contingency

# 列联表
observed = np.array([[50, 30], [20, 40]])

# 卡方检验
chi2_stat, p_value, dof, expected = chi2_contingency(observed)
print(f'χ²统计量: {chi2_stat:.4f}')
print(f'p值: {p_value:.4f}')
print(f'自由度: {dof}')
```

### 5.3 正态性检验

**Shapiro-Wilk检验：**
```python
from scipy.stats import shapiro

data = np.array([12, 15, 18, 14, 16, 13, 17, 19, 11, 20])

stat, p_value = shapiro(data)
print(f'Shapiro-Wilk检验: W={stat:.4f}, p={p_value:.4f}')

if p_value > 0.05:
    print('数据服从正态分布')
else:
    print('数据不服从正态分布')
```

**Kolmogorov-Smirnov检验：**
```python
from scipy.stats import kstest

# K-S检验
stat, p_value = kstest(data, 'norm', args=(data.mean(), data.std()))
print(f'K-S检验: D={stat:.4f}, p={p_value:.4f}')
```

**Jarque-Bera检验：**
```python
from scipy.stats import jarque_bera

stat, p_value = jarque_bera(data)
print(f'Jarque-Bera检验: JB={stat:.4f}, p={p_value:.4f}')
```

## 6. 金融应用

### 6.1 收益率检验

**检验收益率是否为0：**
```python
import pandas as pd
import numpy as np
from scipy import stats

# 计算日收益率
returns = pd.Series([0.01, -0.02, 0.03, -0.01, 0.02, 0.01, -0.01, 0.02, -0.01, 0.01])

# t检验
t_stat, p_value = stats.ttest_1samp(returns, 0)
print(f'收益率均值: {returns.mean():.4f}')
print(f't统计量: {t_stat:.4f}')
print(f'p值: {p_value:.4f}')

# 置信区间
confidence_level = 0.95
degrees_freedom = len(returns) - 1
sample_mean = returns.mean()
sample_standard_error = returns.std() / np.sqrt(len(returns))
confidence_interval = stats.t.interval(confidence_level, degrees_freedom, 
                                        sample_mean, sample_standard_error)
print(f'95%置信区间: {confidence_interval}')
```

## 7. 检验功效与样本量

### 7.1 检验功效分析

**功效计算：**
```python
from scipy.stats import norm
import numpy as np

def power_analysis(mu0, mu1, sigma, n, alpha=0.05):
    """
    计算检验功效
    
    参数:
    mu0: 原假设均值
    mu1: 备择假设均值
    sigma: 标准差
    n: 样本量
    alpha: 显著性水平
    
    返回:
    功效
    """
    # 临界值
    z_alpha = norm.ppf(1 - alpha/2)
    
    # 计算功效
    effect_size = abs(mu1 - mu0) / sigma
    z_beta = z_alpha - effect_size * np.sqrt(n)
    power = 1 - norm.cdf(z_beta)
    
    return power

# 示例
power = power_analysis(mu0=100, mu1=105, sigma=15, n=25, alpha=0.05)
print(f'检验功效: {power:.4f}')
```

### 7.2 样本量计算

**计算所需样本量：**
```python
def sample_size_calculation(effect_size, alpha=0.05, power=0.8):
    """
    计算所需样本量
    
    参数:
    effect_size: 效应量 (Cohen's d)
    alpha: 显著性水平
    power: 检验功效
    
    返回:
    每组样本量
    """
    from scipy.stats import norm
    
    z_alpha = norm.ppf(1 - alpha/2)
    z_beta = norm.ppf(power)
    
    n = 2 * ((z_alpha + z_beta) / effect_size) ** 2
    
    return int(np.ceil(n))

# 示例：检测中等效应
n = sample_size_calculation(effect_size=0.5, alpha=0.05, power=0.8)
print(f'每组所需样本量: {n}')
```

**使用statsmodels：**
```python
from statsmodels.stats.power import TTestIndPower

# 计算功效
power = TTestIndPower().power(effect_size=0.5, nobs1=50, alpha=0.05)
print(f'功效: {power:.4f}')

# 计算样本量
n = TTestIndPower().solve_power(effect_size=0.5, power=0.8, alpha=0.05)
print(f'所需样本量: {n:.0f}')

# 绘制功效曲线
effect_sizes = np.array([0.2, 0.5, 0.8])
sample_sizes = np.array(range(10, 200, 10))
TTestIndPower().plot_power(dep_var='nobs', nobs=sample_sizes, 
                            effect_size=effect_sizes)
```

## 8. 总结

假设检验与统计推断要点：

1. **假设检验逻辑**：提出假设 → 选择统计量 → 确定拒绝域 → 做出决策
2. **参数检验**：Z检验、t检验、F检验、χ²检验
3. **非参数检验**：符号检验、Wilcoxon、Mann-Whitney、Kruskal-Wallis
4. **方差分析**：单因素ANOVA、双因素ANOVA、多重比较
5. **拟合优度**：χ²检验、独立性检验、正态性检验
6. **金融应用**：收益率检验、事件研究、因子检验、策略回测
