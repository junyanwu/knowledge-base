## 一、深度学习在量化中的应用

### 1.1 LSTM（长短期记忆网络）

#### 1.1.1 基本原理

LSTM是一种特殊的循环神经网络(RNN)，专门设计用于解决长期依赖问题。其核心创新是引入了三个门控机制：

- **遗忘门(Forget Gate)**：决定丢弃哪些历史信息
- **输入门(Input Gate)**：决定更新哪些新信息
- **输出门(Output Gate)**：决定输出哪些信息

数学表达：
```
f_t = σ(W_f · [h_{t-1}, x_t] + b_f)  # 遗忘门
i_t = σ(W_i · [h_{t-1}, x_t] + b_i)  # 输入门
C̃_t = tanh(W_C · [h_{t-1}, x_t] + b_C)  # 候选细胞状态
C_t = f_t * C_{t-1} + i_t * C̃_t  # 细胞状态更新
o_t = σ(W_o · [h_{t-1}, x_t] + b_o)  # 输出门
h_t = o_t * tanh(C_t)  # 隐藏状态输出
```

#### 1.1.2 量化应用场景

**股价预测**
```python
import torch
import torch.nn as nn

class StockLSTM(nn.Module):
    def __init__(self, input_size, hidden_size, num_layers, output_size):
        super(StockLSTM, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, 
                           batch_first=True, dropout=0.2)
        self.fc = nn.Linear(hidden_size, output_size)
        
    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])  # 取最后一个时间步
        return out

# 模型配置
model = StockLSTM(
    input_size=20,    # 特征维度（开高低收成交量等）

    num_layers=2,     # LSTM层数
    output_size=1     # 预测目标（如收益率）
)
```

### 1.2 Transformer架构

#### 1.2.1 核心机制

Transformer通过自注意力机制捕获序列中的长距离依赖：

**多头自注意力(Multi-Head Attention)**
```
Attention(Q, K, V) = softmax(QK^T / √d_k)V

MultiHead(Q, K, V) = Concat(head_1, ..., head_h)W^O
where head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)
```

**位置编码(Positional Encoding)**
由于Transformer没有循环结构，需要显式加入位置信息：
```
PE_{(pos, 2i)} = sin(pos / 10000^{2i/d_model})
PE_{(pos, 2i+1)} = cos(pos / 10000^{2i/d_model})
```

#### 1.2.2 量化应用

**Time-Series Transformer**
```python
import torch
import torch.nn as nn
import math

class PositionalEncoding(nn.Module):
    def __init__(self, d_model, max_len=5000):
        super().__init__()
        pe = torch.zeros(max_len, d_model)
        position = torch.arange(0, max_len, dtype=torch.float).unsqueeze(1)
        div_term = torch.exp(torch.arange(0, d_model, 2).float() * 
                             (-math.log(10000.0) / d_model))
        pe[:, 0::2] = torch.sin(position * div_term)
        pe[:, 1::2] = torch.cos(position * div_term)
        self.register_buffer('pe', pe)
        
    def forward(self, x):
        return x + self.pe[:x.size(1), :]

class QuantTransformer(nn.Module):
    def __init__(self, input_size, d_model, nhead, num_layers, output_size):
        super().__init__()
        self.input_proj = nn.Linear(input_size, d_model)
        self.pos_encoder = PositionalEncoding(d_model)
        encoder_layer = nn.TransformerEncoderLayer(
            d_model=d_model, 
            nhead=nhead,
            dim_feedforward=d_model*4,
            dropout=0.1
        )
        self.transformer = nn.TransformerEncoder(encoder_layer, num_layers)
        self.fc = nn.Linear(d_model, output_size)
        
    def forward(self, x):
        # x: (batch, seq_len, features)
        x = self.input_proj(x)
        x = self.pos_encoder(x)
        x = x.permute(1, 0, 2)  # (seq_len, batch, d_model)
        out = self.transformer(x)
        out = out[-1, :, :]  # 取最后一个时间步
        return self.fc(out)
```

**应用场景**
- 多资产价格联合预测
- 跨市场信息传播建模
- 新闻情绪分析（结合NLP）

## 二、强化学习交易策略

### 2.1 强化学习基础

#### 2.1.1 核心概念

强化学习通过智能体(Agent)与环境(Environment)交互学习最优策略：

- **状态(State) s**：市场状态，如价格、持仓、资金等
- **动作(Action) a**：买入、卖出、持有
- **奖励(Reward) r**：收益、夏普比率等
- **策略(Policy) π**：状态到动作的映射
- **价值函数(Value Function) V/Q**：评估状态或状态-动作对的价值

#### 2.1.2 MDP建模

量化交易可以建模为马尔可夫决策过程(MDP)：

```python
class TradingEnv:
    def __init__(self, data, initial_balance=100000):
        self.data = data
        self.initial_balance = initial_balance
        self.reset()
        
    def reset(self):
        self.balance = self.initial_balance
        self.position = 0
        self.current_step = 0
        return self._get_state()
    
    def _get_state(self):
        # 状态包括：当前价格信息、技术指标、持仓信息
        price_features = self.data[self.current_step]
        position_info = [self.position, self.balance]
        return np.concatenate([price_features, position_info])
    
    def step(self, action):
        # action: 0=持有, 1=买入, 2=卖出
        current_price = self.data[self.current_step, 0]
        
        if action == 1 and self.balance > 0:  # 买入
            shares = self.balance // current_price
            self.position += shares
            self.balance -= shares * current_price
            
        elif action == 2 and self.position > 0:  # 卖出
            self.balance += self.position * current_price
            self.position = 0
        
        self.current_step += 1
        done = self.current_step >= len(self.data) - 1
        
        # 计算奖励
        portfolio_value = self.balance + self.position * current_price
        reward = (portfolio_value - self.initial_balance) / self.initial_balance
        
        return self._get_state(), reward, done
```

### 2.2 主要算法

#### 2.2.1 DQN (Deep Q-Network)

**核心思想**：使用神经网络近似Q函数

```python
class DQN(nn.Module):
    def __init__(self, state_size, action_size):
        super().__init__()
        self.fc1 = nn.Linear(state_size, 128)
        self.fc2 = nn.Linear(128, 64)
        self.fc3 = nn.Linear(64, action_size)
        
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        return self.fc3(x)

class DQNAgent:
    def __init__(self, state_size, action_size):
        self.model = DQN(state_size, action_size)
        self.target_model = DQN(state_size, action_size)
        self.memory = []  # 经验回放
        self.gamma = 0.99  # 折扣因子
        self.epsilon = 1.0  # 探索率
        
    def act(self, state):
        if np.random.random() < self.epsilon:
            return np.random.randint(3)  # 随机探索
        with torch.no_grad():
            q_values = self.model(torch.FloatTensor(state))
            return q_values.argmax().item()
    
    def train(self, batch_size=32):
        if len(self.memory) < batch_size:
            return
        
        batch = random.sample(self.memory, batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)
        
        states = torch.FloatTensor(np.array(states))
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(np.array(next_states))
        dones = torch.FloatTensor(dones)
        
        # 计算当前Q值
        current_q = self.model(states).gather(1, actions.unsqueeze(1))
        
        # 计算目标Q值
        with torch.no_grad():
            next_q = self.target_model(next_states).max(1)[0]
            target_q = rewards + self.gamma * next_q * (1 - dones)
        
        # 更新网络
        loss = nn.MSELoss()(current_q.squeeze(), target_q)
        optimizer = torch.optim.Adam(self.model.parameters(), lr=0.001)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
```

#### 2.2.2 PPO (Proximal Policy Optimization)

**优势**：稳定的策略更新，适合连续动作空间

```python
class PPOAgent:
    def __init__(self, state_size, action_size):
        self.actor = ActorNetwork(state_size, action_size)
        self.critic = CriticNetwork(state_size)
        self.gamma = 0.99
        self.lambda_ = 0.95  # GAE参数
        self.eps_clip = 0.2  # PPO裁剪参数
        
    def compute_gae(self, rewards, values, dones):
        """计算广义优势估计"""
        advantages = []
        gae = 0
        for t in reversed(range(len(rewards))):
            if t == len(rewards) - 1:
                next_value = 0
            else:
                next_value = values[t + 1]
            delta = rewards[t] + self.gamma * next_value * (1 - dones[t]) - values[t]
            gae = delta + self.gamma * self.lambda_ * (1 - dones[t]) * gae
            advantages.insert(0, gae)
        return advantages
    
    def update(self, states, actions, old_probs, advantages, returns):
        for _ in range(10):  # 多轮更新
            # 计算新的策略概率
            new_probs = self.actor(states).gather(1, actions.unsqueeze(1))
            
            # PPO目标函数
            ratio = new_probs / old_probs
            surr1 = ratio * advantages
            surr2 = torch.clamp(ratio, 1 - self.eps_clip, 
                               1 + self.eps_clip) * advantages
            actor_loss = -torch.min(surr1, surr2).mean()
            
            # 价值函数损失
            critic_loss = nn.MSELoss()(self.critic(states), returns)
            
            # 总损失
            loss = actor_loss + 0.5 * critic_loss
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
```

#### 2.2.3 DDPG (Deep Deterministic Policy Gradient)

**适用场景**：连续动作空间（如精确的交易比例）

```python
class DDPGAgent:
    def __init__(self, state_size, action_size):
        self.actor = ActorNetwork(state_size, action_size)
        self.critic = CriticNetwork(state_size + action_size, 1)
        self.target_actor = ActorNetwork(state_size, action_size)
        self.target_critic = CriticNetwork(state_size + action_size, 1)
        
    def act(self, state, noise_scale=0.1):
        state = torch.FloatTensor(state)
        action = self.actor(state).detach().numpy()
        noise = np.random.normal(0, noise_scale, len(action))
        return np.clip(action + noise, -1, 1)  # 添加探索噪声
    
    def soft_update(self, target, source, tau=0.005):
        """软更新目标网络"""
        for target_param, param in zip(target.parameters(), source.parameters()):
            target_param.data.copy_(tau * param.data + (1 - tau) * target_param.data)
```

```python
def calculate_momentum_factors(prices, volumes):
    """动量因子计算"""
    factors = {}
    
    # 价格动量
    factors['momentum_1m'] = prices.pct_change(20)  # 1个月动量
    factors['momentum_3m'] = prices.pct_change(60)  # 3个月动量
    factors['momentum_6m'] = prices.pct_change(120)  # 6个月动量
    
    # 动量加速度
    factors['momentum_accel'] = (
        prices.pct_change(10) - prices.pct_change(20).shift(10)
    )
    
    # 成交量加权动量
    factors['volume_weighted_momentum'] = (
        (prices.diff() * volumes).rolling(20).sum() / 
        volumes.rolling(20).sum()
    )
    
    return pd.DataFrame(factors)
```

#### 3.1.2 波动率因子

```python
def calculate_volatility_factors(prices, returns):
    """波动率因子计算"""
    factors = {}
    
    # 历史波动率
    factors['volatility_20d'] = returns.rolling(20).std() * np.sqrt(252)
    factors['volatility_60d'] = returns.rolling(60).std() * np.sqrt(252)
    
    # Parkinson波动率（基于高低价）
    factors['parkinson_vol'] = np.sqrt(
        1/(4*20*np.log(2)) * 
        (np.log(high/low)**2).rolling(20).sum()
    ) * np.sqrt(252)
    
    # 波动率偏斜
    factors['vol_skew'] = (
        returns[returns < 0].rolling(20).std() / 
        returns[returns > 0].rolling(20).std()
    )
    
    # 波动率聚集
    factors['vol_clustering'] = (
        (returns**2).rolling(20).corr((returns**2).shift(1))
    )
    
    return pd.DataFrame(factors)
```

#### 3.2.2 自动化因子挖掘

```python
from genetic_programming import GeneticProgramming

class AutoFactorMining:
    """遗传规划自动因子挖掘"""
    
    def __init__(self, population_size=100, generations=50):
        self.population_size = population_size
        self.generations = generations
        self.function_set = [
            'add', 'sub', 'mul', 'div',  # 基本运算
            'rank', 'zscore', 'demean',  # 标准化
            'ts_mean', 'ts_std', 'ts_max', 'ts_min',  # 时序统计
            'ts_rank', 'ts_delta', 'ts_corr'  # 时序运算
        ]
        
    def evolve(self, features, labels):
        """遗传规划进化"""
        gp = GeneticProgramming(
            function_set=self.function_set,
            population_size=self.population_size,
            generations=self.generations,
            metric=self._evaluate_factor
        )
        
        best_factor = gp.run(features, labels)
        return best_factor
    
    def _evaluate_factor(self, factor_values, labels):
        """评估因子质量"""
        # IC值
        ic = factor_values.corr(labels)
        
        # Rank IC
        rank_ic = factor_values.rank().corr(labels.rank())
        
        # 因子单调性
        monotonicity = self._check_monotonicity(factor_values, labels)
        
        # 综合评分
        score = 0.4 * abs(ic) + 0.4 * abs(rank_ic) + 0.2 * monotonicity
        return score
```

```python
            stability_loss = torch.mean(torch.abs(factors[:-1] - factors[1:]))
            
            total_loss = pred_loss + 0.1 * orth_loss + 0.05 * stability_loss
            
            optimizer.zero_grad()
            total_loss.backward()
            optimizer.step()
```

```python
        self.patience = patience
        self.min_delta = min_delta
```

```python
class BacktestEngine:
    """完整的回测引擎"""
    
    def __init__(self, initial_capital=1000000, commission_rate=0.0003):
        self.initial_capital = initial_capital
        self.commission_rate = commission_rate
        self.slippage_model = LinearSlippage()
        
    def run(self, signals, prices, volumes):
        """
        运行回测
        
        signals: DataFrame, 交易信号 (1=买入, -1=卖出, 0=持有)
        prices: DataFrame, 价格数据
        volumes: DataFrame, 成交量数据
        """
        portfolio_value = [self.initial_capital]
        positions = 0
        cash = self.initial_capital
        
        for i in range(1, len(signals)):
            date = signals.index[i]
            signal = signals.iloc[i]
            price = prices.iloc[i]
            volume = volumes.iloc[i]
            
            # 计算滑点
            slippage = self.slippage_model.calculate(
                signal, volume, price
            )
            
            # 执行交易
            if signal != 0:
                trade_value = abs(signal) * price * (1 + slippage)
                commission = trade_value * self.commission_rate
                
                if signal > 0:  # 买入
                    shares = (cash - commission) / (price * (1 + slippage))
                    positions += shares
                    cash -= shares * price * (1 + slippage) + commission
                else:  # 卖出
                    shares = min(abs(signal), positions)
                    cash += shares * price * (1 - slippage) - commission
                    positions -= shares
            
            # 计算组合价值
            total_value = cash + positions * price
            portfolio_value.append(total_value)
        
        return pd.Series(portfolio_value, index=signals.index)

class LinearSlippage:
    """线性滑点模型"""
    
    def __init__(self, impact_coefficient=0.1):
        self.impact_coefficient = impact_coefficient
    
    def calculate(self, signal, volume, price):
        trade_ratio = abs(signal) / (volume + 1e-6)
        slippage = self.impact_coefficient * trade_ratio ** 0.5
        return slippage
```

```python
import torch
import torch.nn as nn
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler

class LSTMAttentionModel(nn.Module):
    """带注意力机制的LSTM模型"""
    
    def __init__(self, input_size, hidden_size, num_layers, output_size):
        super().__init__()
        self.hidden_size = hidden_size
        
        # LSTM层
        self.lstm = nn.LSTM(
            input_size, hidden_size, num_layers,
            batch_first=True, dropout=0.2, bidirectional=True
        )
        
        # 注意力层
        self.attention = nn.Sequential(
            nn.Linear(hidden_size * 2, hidden_size),
            nn.Tanh(),
            nn.Linear(hidden_size, 1)
        )
        
        # 输出层
        self.fc = nn.Sequential(
            nn.Linear(hidden_size * 2, hidden_size),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(hidden_size, output_size)
        )
    
    def forward(self, x):
        # LSTM编码
        lstm_out, _ = self.lstm(x)  # (batch, seq_len, hidden*2)
        
        # 注意力权重
        attn_weights = torch.softmax(self.attention(lstm_out), dim=1)
        
        # 加权求和
        context = torch.sum(attn_weights * lstm_out, dim=1)
        
        # 输出
        output = self.fc(context)
        return output

class StockPredictionStrategy:
    """股价预测策略"""
    
    def __init__(self, model_path, lookback=60, prediction_horizon=5):

        features['obv'] = self._calculate_obv(data)

        features['volatility'] = features['returns'].rolling(20).std()

        features = self.prepare_features(data)

        features_scaled = self.scaler.fit_transform(features)
        
        # 构建序列
        sequences = self._create_sequences(features_scaled)
        
        # 预测
        self.model.eval()
        with torch.no_grad():
            predictions = self.model(torch.FloatTensor(sequences))
        
        # 转换为信号
        signals = pd.Series(predictions.numpy().flatten(), 
                           index=features.index[self.lookback:])
        
        # 标准化信号
        signals = (signals - signals.rolling(252).mean()) / signals.rolling(252).std()
        
        # 生成交易信号
        positions = np.where(signals > 1, 1, np.where(signals < -1, -1, 0))
        
        return pd.Series(positions, index=signals.index)

        sequences = []
        for i in range(len(data) - self.lookback):

    def _calculate_rsi(self, prices, period=14):
        delta = prices.diff()
        gain = (delta.where(delta > 0, 0)).rolling(period).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(period).mean()
        rs = gain / loss
        return 100 - (100 / (1 + rs))

        ema_fast = prices.ewm(span=fast).mean()
        ema_slow = prices.ewm(span=slow).mean()
        macd = ema_fast - ema_slow
        return macd

        ma = prices.rolling(period).mean()
        std = prices.rolling(period).std()
        upper = ma + std_dev * std
        lower = ma - std_dev * std
        return upper, lower
    
    def _calculate_obv(self, data):
        obv = (np.sign(data['close'].diff()) * data['volume']).cumsum()
        return obv
```

```python
        # 状态空间
        self.observation_space = spaces.Box(
            low=-np.inf, high=np.inf, shape=(10,), dtype=np.float32
        )
        
        self.reset()
    
    def reset(self):
        self.current_step = 0
        self.inventory = self.initial_inventory
        self.cash = 0
        self.trades = []
        
        return self._get_state()
    
    def _get_state(self):
        """构建状态向量"""
        ob = self.order_book.iloc[self.current_step]
        
        state = np.array([
            self.inventory / self.max_inventory,  # 当前库存
            ob['bid_price'] - ob['ask_price'],     # 买卖价差
            ob['bid_volume'],                       # 买一量
            ob['ask_volume'],                       # 卖一量
            ob['mid_price_change'],                 # 中间价变化
            ob['volatility'],                       # 波动率
            ob['order_imbalance'],                  # 订单不平衡
            ob['trade_intensity'],                  # 交易强度
            ob['spread_ma'],                        # 价差均值
            ob['volume_ma']                         # 成交量均值
        ], dtype=np.float32)
        
        return state
    
    def step(self, action):
        """执行动作并返回结果"""
        ob = self.order_book.iloc[self.current_step]
        mid_price = (ob['bid_price'] + ob['ask_price']) / 2
        
        # 解析动作：买卖价差
        bid_spread, ask_spread = action
        
        # 设置报价
        my_bid = mid_price - bid_spread
        my_ask = mid_price + ask_spread
        
        # 模拟成交
        buy_filled, sell_filled = self._simulate_fills(
            my_bid, my_ask, ob
        )
        
        # 更新库存和现金
        self.inventory += buy_filled - sell_filled
        self.cash -= buy_filled * my_bid
        self.cash += sell_filled * my_ask
        
        # 计算奖励
        reward = self._calculate_reward(mid_price)
        
        # 更新步数
        self.current_step += 1
        done = self.current_step >= len(self.order_book) - 1
        
        return self._get_state(), reward, done, {}
    
    def _simulate_fills(self, my_bid, my_ask, order_book):
        """模拟订单成交"""
        buy_filled = 0
        sell_filled = 0
        
        # 如果买价高于卖一价，则成交
        if my_bid >= order_book['ask_price']:
            volume = min(order_book['ask_volume'], 10)
            buy_filled = volume
        
        # 如果卖价低于买一价，则成交
        if my_ask <= order_book['bid_price']:
            volume = min(order_book['bid_volume'], 10)
            sell_filled = volume
        
        return buy_filled, sell_filled
    
    def _calculate_reward(self, mid_price):
        """计算奖励函数"""
        # PnL
        pnl = self.cash + self.inventory * mid_price
        
        # 库存风险惩罚
        inventory_penalty = self.risk_aversion * self.inventory ** 2
        
        # 总奖励
        reward = pnl - inventory_penalty
        
        return reward
```

```python
class MultiFactorStrategy:
    """多因子选股策略"""
    
    def __init__(self, factor_config, rebalance_frequency='monthly'):
        self.factors = {}
        self.weights = {}
        self.rebalance_frequency = rebalance_frequency
        self.factor_config = factor_config
        
    def add_factor(self, name, factor_data, weight=1.0):
        """添加因子"""
        self.factors[name] = factor_data
        self.weights[name] = weight
        
    def combine_factors(self, method='zscore_weighted'):
        """因子合成"""
        factor_df = pd.DataFrame(self.factors)
        
        if method == 'zscore_weighted':
            # Z-score标准化后加权
            factor_std = factor_df.apply(lambda x: (x - x.mean()) / x.std())
            weighted = factor_std.multiply(pd.Series(self.weights))
            combined = weighted.sum(axis=1)
            
        elif method == 'rank_weighted':
            # 排序后加权
            factor_rank = factor_df.rank()
            weighted = factor_rank.multiply(pd.Series(self.weights))
            combined = weighted.sum(axis=1)
            
        elif method == 'ic_weighted':
            # IC加权
            ic_weights = self._calculate_ic_weights()
            weighted = factor_df.multiply(ic_weights)
            combined = weighted.sum(axis=1)
        
        return combined
    
    def _calculate_ic_weights(self):
        """基于IC的因子权重"""
        ic_values = {}
        for name, factor in self.factors.items():
            # 假设有未来收益数据
            ic = factor.corr(self.future_returns)
            ic_values[name] = ic
        
        # 归一化
        total_ic = sum(abs(v) for v in ic_values.values())
        return {k: abs(v) / total_ic for k, v in ic_values.items()}
    
    def select_stocks(self, combined_factor, n_stocks=50):
        """选股"""
        # 按因子值排序
        ranked = combined_factor.groupby(level=0).rank(ascending=True)
        
        # 选择前N只股票
        selected = {}
        for date in ranked.index.get_level_values(0).unique():
            date_rank = ranked.loc[date]
            selected[date] = date_rank.nlargest(n_stocks).index.tolist()
        
        return selected
    
    def construct_portfolio(self, selected_stocks, prices, method='equal_weight'):
        """构建组合"""
        portfolio = {}
        
        for date, stocks in selected_stocks.items():
            if method == 'equal_weight':
                weights = {stock: 1/len(stocks) for stock in stocks}
            elif method == 'factor_weighted':
                # 根据因子值加权
                factor_values = self.combined_factor.loc[date, stocks]
                weights = (factor_values / factor_values.sum()).to_dict()
            elif method == 'risk_parity':
                # 风险平价
                weights = self._risk_parity_weight(prices, stocks, date)
            
            portfolio[date] = weights
        
        return portfolio
    
    def _risk_parity_weight(self, prices, stocks, date, lookback=60):
        """风险平价权重"""
        from scipy.optimize import minimize
        
        # 计算协方差矩阵
        returns = prices[stocks].pct_change().iloc[-lookback:]
        cov_matrix = returns.cov()
        
        # 优化目标：最小化风险贡献差异
        def risk_contribution(weights):
            portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
            marginal_contrib = cov_matrix @ weights
            risk_contrib = weights * marginal_contrib / portfolio_vol
            return np.sum((risk_contrib - portfolio_vol/len(stocks))**2)
        
        # 约束条件
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
        ]
        bounds = [(0, 0.1) for _ in stocks]  # 单只股票权重上限10%
        
        # 优化
        result = minimize(
            risk_contribution,
            x0=np.ones(len(stocks)) / len(stocks),
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return dict(zip(stocks, result.x))
```

```python
class FederatedLearningQuant:
    """联邦学习量化框架"""
    
    def __init__(self, num_clients, model_class):
        self.clients = [Client(model_class) for _ in range(num_clients)]

    def train(self, num_rounds=100):

        for round in range(num_rounds):
            # 1. 分发全局模型
            for client in self.clients:
                client.receive_model(self.global_model)
            
            # 2. 本地训练
            local_updates = []
            for client in self.clients:
                update = client.train_locally()
                local_updates.append(update)
            
            # 3. 聚合更新
            self.global_model = self._aggregate(local_updates)
            
            # 4. 评估
            if round % 10 == 0:
                self._evaluate()

        # 使用FedAvg算法
        aggregated = {}
        for param_name in local_updates[0].keys():
            aggregated[param_name] = torch.mean(
                torch.stack([u[param_name] for u in local_updates]), dim=0
            )
        return aggregated
```

```python
# 基于机器学习的动态因子权重模型
class DynamicFactorModel:
    """
    使用机器学习模型动态调整因子权重
    """
    def __init__(self, n_factors):
        self.n_factors = n_factors
        self.model = self._build_model()
    
    def _build_model(self):
        """
        构建因子权重预测模型
        可以使用MLP、LSTM、Transformer等架构
        """
        from sklearn.ensemble import GradientBoostingRegressor
        return GradientBoostingRegressor(
            n_estimators=100,
            max_depth=5,
            learning_rate=0.05
        )
    
    def fit(self, factor_returns, future_returns):
        """
        训练模型
        factor_returns: (n_samples, n_factors) 因子收益矩阵
        future_returns: (n_samples,) 未来收益
        """
        self.model.fit(factor_returns, future_returns)
    
    def predict_weights(self, current_factors):
        """
        预测当前因子权重
        current_factors: (n_factors,) 当前因子值
        """
        # 预测每个因子对未来收益的贡献
        factor_importance = self.model.feature_importances_
        
        # 归一化权重
        weights = factor_importance / factor_importance.sum()
        return weights
```
