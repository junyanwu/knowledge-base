### 1.6 执行算法（Execution Algorithms）

**目标**：最小化市场冲击，优化执行成本

**强化学习应用**：
- **状态**：订单簿深度、成交历史、剩余订单量
- **动作**：下单量、下单时机、下单方式
- **奖励**：执行成本、市场冲击

```python
class ExecutionEnv(gym.Env):
    def __init__(self, total_shares, time_horizon):
        self.total_shares = total_shares
        self.remaining = total_shares
        self.time_horizon = time_horizon
    
    def step(self, action):
        # action: 下单量
        executed = min(action, self.remaining)
        market_impact = self.calculate_impact(executed)
        reward = -market_impact  # 负奖励=成本
        
        self.remaining -= executed
        done = (self.remaining == 0)
        
        return self.get_state(), reward, done, {}
```

```python
def price_features(df):
    """价格相关特征"""
    features = pd.DataFrame()
    
    # 收益率系列
    features['return_1d'] = df['close'].pct_change(1)
    features['return_5d'] = df['close'].pct_change(5)
    features['return_20d'] = df['close'].pct_change(20)
    
    # 波动率
    features['volatility_5d'] = features['return_1d'].rolling(5).std()
    features['volatility_20d'] = features['return_1d'].rolling(20).std()

                                 (df['high'].rolling(20).max() - df['low'].rolling(20).min())
    
    # 动量
    features['momentum_5d'] = df['close'] / df['close'].shift(5) - 1
    features['momentum_20d'] = df['close'] / df['close'].shift(20) - 1
    
    return features
```

```python
def technical_indicators(df):
    """技术指标特征"""
    features = pd.DataFrame()
    
    # 均线系统
    for period in [5, 10, 20, 60]:
        features[f'ma{period}'] = df['close'].rolling(period).mean() / df['close'] - 1
    
    # 均线交叉
    features['ma5_ma20_cross'] = (features['ma5'] > features['ma20']).astype(int)
    
    # RSI
    for period in [6, 14, 24]:
        delta = df['close'].diff()
        gain = delta.where(delta > 0, 0).rolling(period).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(period).mean()
        features[f'rsi_{period}'] = 100 - (100 / (1 + gain / loss))
    
    # MACD
    ema12 = df['close'].ewm(span=12, adjust=False).mean()
    ema26 = df['close'].ewm(span=26, adjust=False).mean()
    features['macd'] = ema12 - ema26
    features['macd_signal'] = features['macd'].ewm(span=9, adjust=False).mean()
    features['macd_hist'] = features['macd'] - features['macd_signal']
    
    # 布林带
    features['boll_upper'] = df['close'].rolling(20).mean() + 2 * df['close'].rolling(20).std()
    features['boll_lower'] = df['close'].rolling(20).mean() - 2 * df['close'].rolling(20).std()
    features['boll_width'] = (features['boll_upper'] - features['boll_lower']) / df['close']
    features['boll_pos'] = (df['close'] - features['boll_lower']) / (features['boll_upper'] - features['boll_lower'])
    
    # KDJ
    low_min = df['low'].rolling(9).min()
    high_max = df['high'].rolling(9).max()
    features['kdj_k'] = (df['close'] - low_min) / (high_max - low_min) * 100
    features['kdj_d'] = features['kdj_k'].rolling(3).mean()
    features['kdj_j'] = 3 * features['kdj_k'] - 2 * features['kdj_d']
    
    return features
```

```python
def handle_missing_values(features):
    """处理缺失值"""
    # 策略1：向前填充（时间序列）
    features = features.fillna(method='ffill')
    
    # 策略2：行业均值填充
    for col in features.columns:
        features[col] = features.groupby('industry')[col].transform(
            lambda x: x.fillna(x.mean())
        )
    
    # 策略3：极端值处理
    features = features.clip(lower=features.quantile(0.01), 
                            upper=features.quantile(0.99), 
                            axis=1)
    
    return features
```

```python
def feature_selection(X, y, method='importance', top_k=50):
    """特征选择"""
    if method == 'importance':
        # 随机森林特征重要性
        rf = RandomForestRegressor(n_estimators=100, random_state=42, n_jobs=-1)
        rf.fit(X, y)
        importance = pd.Series(rf.feature_importances_, index=X.columns)
        selected = importance.nlargest(top_k).index.tolist()
    
    elif method == 'correlation':
        # 相关性筛选
        corr_with_target = X.corrwith(y).abs()
        selected = corr_with_target.nlargest(top_k).index.tolist()
    
    elif method == 'mutual_info':
        # 互信息
        from sklearn.feature_selection import mutual_info_regression
        mi = mutual_info_regression(X, y)
        mi_series = pd.Series(mi, index=X.columns)
        selected = mi_series.nlargest(top_k).index.tolist()
    
    return X[selected], selected
```

```python
class LLMBasedFactorMining:
```

## 深度学习

### LSTM（长短期记忆网络）

#### 1. 核心原理
- **记忆单元**：通过门控机制选择性地保留或遗忘信息
- **三个门**：输入门、遗忘门、输出门
- **优势**：解决梯度消失问题，适合长序列建模

#### 2. 股价预测应用
- **单变量预测**：仅使用历史价格序列
- **多变量预测**：结合技术指标、财务数据
- **序列长度**：通常选择20-60个交易日

#### 3. 实战案例（2026年研究）
- **上证指数预测**：基于Transformer-LSTM混合模型
- **个股预测**：京粮控股(000505)、东北制药(000597)
- **改进方向**：
  - 双向LSTM(BiLSTM)：同时捕捉前后依赖
  - Attention-LSTM：加入注意力机制
  - CEEMDAN-WPD-LSTM：结合信号降噪技术

#### 4. 代码实现要点
```python
# 数据预处理
- 归一化：MinMaxScaler或StandardScaler
- 序列构建：滑动窗口切分
- 训练集/测试集划分：时间序列不能随机划分

# 模型架构
- 典型层数：2-3层LSTM
- 隐藏单元：64-256
- Dropout：0.2-0.5防止过拟合
```

### Transformer

#### 1. 核心创新
- **自注意力机制**：捕捉序列内部长距离依赖
- **多头注意力**：并行处理多个注意力头
- **位置编码**：保留时序信息

#### 2. 股价预测优势
- **并行计算**：比LSTM训练更快
- **长序列建模**：处理更长的历史数据（100+天）
- **可解释性**：注意力权重可视化

#### 3. Transformer变体应用（2026年趋势）
- **Temporal Fusion Transformer(TFT)**：多变量时间序列预测
- **Informer**：长序列预测，O(nlogn)复杂度
- **Autoformer**：分解时序趋势和季节性
- **PatchTST**：将时间序列分patch处理
- **🔥 TradeFM**：生成式Transformer，直接学习交易流微观结构

#### 4. LSTM+Transformer混合模型
- **2026年热点**：结合两者优势
- **架构设计**：
  - LSTM层：提取局部时序特征
  - Transformer层：捕捉全局依赖
  - 实例归一化(Instance Normalization)：提升泛化能力
- **性能提升**：相比单一模型准确率提高5-10%

#### 5. TCN-Transformer-LSTM组合
- **TCN(Temporal Convolutional Network)**：因果卷积提取特征
- **融合方式**：TCN→Transformer→LSTM串行
- **应用场景**：高频交易、日内预测
- **SHAP分析**：模型可解释性增强

### 8.1 自动做市商动态均衡 (AMM Dynamic Equilibrium)

**论文来源**：arXiv:2603.08603 (2026年3月)

#### 核心创新
首个CFMM（Constant Function Market Maker）动态均衡框架，形式化套利者与流动性提供者的策略互动。

#### 技术贡献

**1. CFMM买卖冲击不对称性**
```python
class CFMMAsymmetry:
    """CFMM内在买卖冲击不对称性"""
    
    def __init__(self, reserve_x, reserve_y, fee_rate):
        self.x = reserve_x  # X代币储备
        self.y = reserve_y  # Y代币储备
        self.fee = fee_rate
    
    def buy_impact(self, delta_x):
        """买入冲击（X → Y）"""
        # 恒定乘积: (x + dx)(y - dy) = xy
        # 非对称几何结构导致买入与卖出成本不同
        effective_x = delta_x * (1 - self.fee)
        delta_y = self.y * effective_x / (self.x + effective_x)
        return delta_y / delta_x  # 实际价格
    
    def sell_impact(self, delta_y):
        """卖出冲击（Y → X）"""
        effective_y = delta_y * (1 - self.fee)
        delta_x = self.x * effective_y / (self.y + effective_y)
        return delta_x / delta_y  # 实际价格
```

### 8.2 可解释市场状态感知投资 (Wasserstein HMM Portfolio)

**论文来源**：arXiv:2603.04441 (2026年3月)

#### 核心创新

#### 技术架构

**1. Wasserstein HMM市场状态识别**
```python
class WassersteinHMM:
    """Wasserstein距离HMM市场状态识别"""
    
    def __init__(self, n_regimes=3):
        self.n_regimes = n_regimes
        self.models = []  # 多个高斯HMM候选
    
    def fit_rolling(self, returns, window=252):
        """滚动窗口训练"""
        for i in range(window, len(returns)):
            train_data = returns[i-window:i]
            # 模型阶数选择（BIC/AIC）
            best_model = self._select_model_order(train_data)
            self.models.append(best_model)
    
    def track_identity(self, new_model, prev_components):
        """模板身份追踪"""
        # 使用2-Wasserstein距离匹配成分
        from scipy.optimize import linear_sum_assignment
        
        cost_matrix = np.zeros((self.n_regimes, self.n_regimes))
        for i, comp_i in enumerate(new_model.components_):
            for j, comp_j in enumerate(prev_components):
                cost_matrix[i, j] = self._wasserstein_distance(comp_i, comp_j)
        
        row_ind, col_ind = linear_sum_assignment(cost_matrix)
        return col_ind  # 身份映射
```

**2. 交易成本感知优化**
```python
def mean_variance_with_costs(expected_returns, cov_matrix, 
                             current_weights, transaction_cost=0.001):
    """交易成本感知均值-方差优化"""
    n = len(expected_returns)
    
    def objective(w):
        portfolio_return = np.dot(w, expected_returns)
        portfolio_vol = np.sqrt(np.dot(w.T, np.dot(cov_matrix, w)))
        
        # 交易成本惩罚
        trade_size = np.abs(w - current_weights)
        cost_penalty = transaction_cost * np.sum(trade_size)
        
        # 最大化夏普比（减去交易成本）
        return -portfolio_return / portfolio_vol + cost_penalty
    
    # 约束：权重和为1
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n)]  # 多头约束
    
    from scipy.optimize import minimize
    result = minimize(objective, current_weights, 
                     method='SLSQP', bounds=bounds, constraints=constraints)
    return result.x
```

```python
import torch.nn as nn
import torch.nn.functional as F

class BayesianRiskScorer(nn.Module):
    """贝叶斯神经网络风险评分器"""
    
    def __init__(self, input_dim, hidden_dim=64, n_samples=10):
        super().__init__()
        self.n_samples = n_samples
        
        # 权重分布参数
        self.W1_mu = nn.Parameter(torch.randn(input_dim, hidden_dim) * 0.1)
        self.W1_sigma = nn.Parameter(torch.ones(input_dim, hidden_dim) * 0.1)
        self.W2_mu = nn.Parameter(torch.randn(hidden_dim, 1) * 0.1)
        self.W2_sigma = nn.Parameter(torch.ones(hidden_dim, 1) * 0.1)
    
    def forward(self, x):
        # 采样权重
        W1 = self.W1_mu + torch.exp(self.W1_sigma) * torch.randn_like(self.W1_sigma)
        W2 = self.W2_mu + torch.exp(self.W2_sigma) * torch.randn_like(self.W2_sigma)
        
        # 前向传播
        h = F.relu(torch.matmul(x, W1))
        out = torch.sigmoid(torch.matmul(h, W2))
        return out
    
    def predict_with_uncertainty(self, x):
        """预测并返回不确定性"""
        predictions = []
        for _ in range(self.n_samples):
            pred = self.forward(x)
            predictions.append(pred)
        
        predictions = torch.stack(predictions)
        mean = predictions.mean(dim=0)
        std = predictions.std(dim=0)  # 认知不确定性
        return mean, std
```

```python
class RuleLibrary:
    """透明决策规则库"""
    
    def __init__(self):
        self.rules = [
            # 规则1: 极端结果筛选
            lambda x: 1 if x['payoff_max'] > 2 * x['payoff_min'] else 0,
            
            # 规则2: 显著性
            lambda x: 1 if x['salience'] > 0.7 else 0,
            
            # 规则3: 注意力
            lambda x: 1 if x['attention_weight'] > 0.5 else 0,
            
            # 规则4: 风险厌恶
            lambda x: 0 if x['variance'] > x['expected_value'] else 1,
            
            # 规则5: 损失厌恶
            lambda x: 0 if x['potential_loss'] > x['loss_threshold'] else 1,
        ]
        self.n_rules = len(self.rules)
    
    def evaluate_all(self, x):
        """评估所有规则"""
        return np.array([rule(x) for rule in self.rules])
```

```python
import torch.nn as nn

class GatingNetwork(nn.Module):
    """可微门控网络"""
    
    def __init__(self, input_dim, n_rules):
        super().__init__()
        self.fc1 = nn.Linear(input_dim, 32)
        self.fc2 = nn.Linear(32, n_rules)
        self.softmax = nn.Softmax(dim=-1)
    
    def forward(self, x):
        h = F.relu(self.fc1(x))
        gate_weights = self.softmax(self.fc2(h))
        return gate_weights  # 环境依赖的规则倾向


class RandomRuleModel(nn.Module):
    """随机规则模型"""
    
    def __init__(self, input_dim, n_rules):
        super().__init__()
        self.rules = RuleLibrary()
        self.gate = GatingNetwork(input_dim, n_rules)
    
    def forward(self, x):
        # 评估所有规则
        rule_outputs = self.rules.evaluate_all(x)
        
        # 学习规则权重
        gate_weights = self.gate(x)
        
        # 加权决策
        final_decision = (gate_weights * rule_outputs).sum()
        return final_decision, gate_weights  # 可解释输出
```

```python
        self.target_inventory = inventory_target
        self.risk_aversion = risk_aversion
    
    def optimize_acceptance_rule(self, trade_request, current_state):
        """
        优化接受/拒绝规则
        
        状态: (库存, 拒绝评分)
        决策: (报价, 滑点容忍度)
        """
        inventory, rejection_score = current_state
        
        # HJB方程Hamiltonian
        def hamiltonian(spread, tolerance):
            # 预期收益 = 价差收益 - 延迟风险
            expected_profit = spread * trade_request['size']
            
            # 拒绝概率
            reject_prob = self._estimate_rejection(
                spread, tolerance, rejection_score
            )
            
            # 接受后期望库存变动
            inventory_cost = self.risk_aversion * (inventory ** 2)
            
            # Hamiltonian
            H = (1 - reject_prob) * (expected_profit - inventory_cost)
            return H
        
        # 优化
        from scipy.optimize import minimize_scalar
        result = minimize_scalar(
            lambda s: -hamiltonian(s, self._optimal_tolerance(s)),
            bounds=(0, 0.01),
            method='bounded'
        )
        return result.x, self._optimal_tolerance(result.x)
```

```python
import numpy as np
from scipy.special import hyp1f1  # 超几何函数

class StationaryFBM:
    """Stationary分数布朗运动核"""
    
    def __init__(self, hurst=0.7):
        self.H = hurst  # Hurst参数
    
    def spectral_density(self, omega):
        """
        Stationary fBM核的封闭形式谱密度
        
        使用超几何函数表示
        """
        # 参数
        gamma_H = np.exp(np.pi * 1j * self.H)
        c = np.sin(np.pi * self.H) * gamma_H
        
        # 超几何函数表示
        density = c * np.abs(omega) ** (1 - 2 * self.H) * \
                  hyp1f1(0.5, 1 - self.H, -1j * omega)
        return density
```

**2. 随机傅里叶特征近似**
```python
class RandomFourierFeatures:
    """随机傅里叶特征核近似"""
    
    def __init__(self, kernel_spectral_density, n_features=100):
        self.n_features = n_features
        self.spectral_density = kernel_spectral_density
    
    def sample_frequencies(self):
        """使用Hamiltonian Monte Carlo采样频率"""
        from scipy.stats import hmc
        
        # 定义目标分布（核谱密度）
        def log_prob(omega):
            return np.log(self.spectral_density(omega) + 1e-10)
        
        # HMC采样
        samples = hmc.sample(log_prob, n_samples=self.n_features)
        return samples
    
    def approximate_kernel(self, t, s):
        """近似核函数"""
        # K(t,s) ≈ Σ cos(ω_i * (t-s)) / n
        omega = self.sample_frequencies()
        kernel_approx = np.mean([np.cos(w * (t - s)) for w in omega])
        return kernel_approx
```

**3. Volterra过程模拟**
```python
def simulate_volterra(T, n_steps, kernel_approximator):
    """模拟Volterra过程"""
    dt = T / n_steps
    t = np.linspace(0, T, n_steps)
    
    # 使用RFF近似核函数
    increments = np.random.normal(0, np.sqrt(dt), n_steps)
    
    # Volterra积分
    X = np.zeros(n_steps)
    for i in range(1, n_steps):
        for j in range(i):
            X[i] += kernel_approximator(t[i], t[j]) * increments[j]
    
    return X
```

```python
def long_only_min_variance_1factor(betas, idio_vols, factor_var):
    """
    单因子模型长仓最小方差组合解析解
    
    参数:
        betas: 股票因子暴露 (n,)
        idio_vols: 股票特质波动率 (n,)
        factor_var: 因子方差
    
    返回:
        weights: 最小方差组合权重
    """
    n = len(betas)
    
    # 协方差矩阵分解
    # Σ = ββ'σ_f² + D (D为特质方差对角矩阵)
    
    # 最小方差组合权重（允许卖空）
    D_inv = np.diag(1 / (idio_vols ** 2))
    Sigma_inv = D_inv - (D_inv @ np.outer(betas, betas) @ D_inv * factor_var) / \
                (1 + factor_var * betas @ D_inv @ betas)
    
    weights_unconstrained = Sigma_inv.sum(axis=1) / Sigma_inv.sum()
    
    # 长仓约束处理（迭代投影）
```

##### 案例2：基于熵的市场状态识别

**研究背景**：
arXiv:2603.12040提出的熵框架为市场状态识别提供了新视角。

**应用实现**：
```python
import numpy as np
from scipy import stats

class Entropy_Based_Market_Regime:
    """基于熵的市场状态识别"""
    
    def __init__(self, window=20, entropy_threshold=2.0, volatility_threshold=0.02):
        self.window = window
        self.entropy_threshold = entropy_threshold
        self.vol_threshold = volatility_threshold
    
    def compute_shannon_entropy(self, returns, bins=20):
        """计算香农熵"""
        hist, _ = np.histogram(returns, bins=bins, density=True)
        hist = hist[hist > 0]  # 移除零值
        entropy = -np.sum(hist * np.log(hist))
        return entropy
    
    def compute_cumulative_entropy(self, returns):
        """计算累积熵（滑动窗口）"""
        n = len(returns)
        cum_entropy = np.zeros(n - self.window)
        
        for i in range(self.window, n):
            window_returns = returns[i-self.window:i]
            cum_entropy[i-self.window] = self.compute_shannon_entropy(window_returns)
        
        return cum_entropy
    
    def detect_regime(self, returns):
        """
        检测市场状态
        
        状态分类：
        1. 正常状态：低熵、低波动
        2. 政策驱动状态：高熵、低波动（信息密集但幅度小）
        3. 波动状态：低熵、高波动（大幅变动但方向一致）
        4. 危机状态：高熵、高波动（信息密集且大幅变动）
        """
        entropy = self.compute_cumulative_entropy(returns)
        volatility = pd.Series(returns).rolling(self.window).std().values[self.window:]
        
        # Z-score标准化
        entropy_z = (entropy - entropy.mean()) / entropy.std()
        volatility_z = (volatility - volatility.mean()) / volatility.std()
        
        regimes = np.zeros(len(entropy_z))
        
        # 分类逻辑
        regimes[(entropy_z < 0) & (volatility_z < 0)] = 1   # 正常
        regimes[(entropy_z > 0) & (volatility_z < 0)] = 2   # 政策驱动
        regimes[(entropy_z < 0) & (volatility_z > 0)] = 3   # 波动
        regimes[(entropy_z > 0) & (volatility_z > 0)] = 4   # 危机
        
        return regimes, entropy, volatility
```

```python
import pandas as pd

class Supply_Chain_Risk_Model:
    """供应链风险量化模型"""
    
    def __init__(self, input_output_table):
        """
        参数:
            input_output_table: OECD跨国家投入产出表
        """
        self.io_table = input_output_table
        self.countries = input_output_table.index
        self.sectors = input_output_table.columns
    
    def compute_vulnerability(self, target_country):
        """
        计算目标国家的供应链脆弱性
        
        方法：
        1. 单部门限制：某一国家特定部门供应中断
        2. 全面限制：某一国家所有部门供应中断
        3. 影响评估：对目标国家经济活动的影响
        """
        vulnerability_scores = {}
        
        for source_country in self.countries:
            if source_country == target_country:
                continue
            
            # 单部门脆弱性
            sector_vulnerability = {}
            for sector in self.sectors:
                impact = self._compute_sector_impact(
                    source_country, sector, target_country
                )
                sector_vulnerability[sector] = impact
            
            # 全面脆弱性
            total_impact = self._compute_total_impact(
                source_country, target_country
            )
            
            vulnerability_scores[source_country] = {
                'sector_vulnerability': sector_vulnerability,
                'total_vulnerability': total_impact
            }
        
        return vulnerability_scores
    
    def _compute_sector_impact(self, source_country, sector, target_country):
        """计算部门影响"""
        # 简化模型：基于投入产出表的直接影响
        supply_weight = self.io_table.loc[source_country, sector]
        target_demand = self.io_table.loc[target_country, :].sum()
        
        impact = supply_weight / target_demand
        return impact
    
    def _compute_total_impact(self, source_country, target_country):
        """计算全面影响"""
        total_supply = self.io_table.loc[source_country, :].sum()
        target_demand = self.io_table.loc[target_country, :].sum()
        
        impact = total_supply / target_demand
        return impact
    
    def rank_vulnerability(self, target_country):
        """
        排名脆弱性
        
        返回：
        按脆弱性排名的国家列表
        """
        vulnerability = self.compute_vulnerability(target_country)
        
        # 按全面脆弱性排序
        ranked = sorted(
            vulnerability.items(),
            key=lambda x: x[1]['total_vulnerability'],
            reverse=True
        )
        
        return ranked
```

```python
class Enhanced_Out_of_Sample_Validation:
    """增强样本外验证框架"""
    
    def __init__(self, n_splits=5, train_ratio=0.6, val_ratio=0.2):
        self.n_splits = n_splits
        self.train_ratio = train_ratio
        self.val_ratio = val_ratio
    
    def walk_forward_validation(self, data, model, metrics):
        """
        Walk-Forward验证
        
        时间序列专用验证方法：
        1. 训练集 → 验证集 → 测试集
        2. 滚动窗口向前移动
        3. 避免未来函数
        """
        n = len(data)
        train_size = int(n * self.train_ratio)
        val_size = int(n * self.val_ratio)
        test_size = n - train_size - val_size
        
        results = []
        
        for i in range(self.n_splits):
            # 窗口划分
            train_end = train_size + i * test_size
            val_end = train_end + val_size
            test_end = val_end + test_size
            
            if test_end > n:
                break
            
            train_data = data[:train_end]
            val_data = data[train_end:val_end]
            test_data = data[val_end:test_end]
            
            # 模型训练
            model.fit(train_data, val_data)
            
            # 样本外测试
            predictions = model.predict(test_data)
            
            # 评估指标
            fold_results = {
                'fold': i,
                'train_period': (train_data.index[0], train_data.index[-1]),
                'val_period': (val_data.index[0], val_data.index[-1]),
                'test_period': (test_data.index[0], test_data.index[-1])
            }
            
            for metric_name, metric_func in metrics.items():
                fold_results[metric_name] = metric_func(
                    predictions, test_data['returns']
                )
            
            results.append(fold_results)
        
        return pd.DataFrame(results)
    
    def purged_cross_validation(self, data, model, embargo_pct=0.01):
        """
        清洗交叉验证
        
        解决数据泄露问题：
        1. 在训练集和验证集之间设置禁运期
        2. 移除禁运期内的样本
        """
        n = len(data)
        embargo_size = int(n * embargo_pct)
        
        results = []
        fold_size = n // self.n_splits
        
        for i in range(self.n_splits):
            # 验证集
            val_start = i * fold_size
            val_end = (i + 1) * fold_size
            
            # 禁运期
            embargo_start = val_end
            embargo_end = min(val_end + embargo_size, n)
            
            # 训练集（排除验证集和禁运期）
            train_idx = np.concatenate([
                np.arange(0, val_start),
                np.arange(embargo_end, n)
            ])
            
            train_data = data.iloc[train_idx]
            val_data = data.iloc[val_start:val_end]
            
            # 训练和评估
            model.fit(train_data)
            predictions = model.predict(val_data)
            
            results.append({
                'fold': i,
                'train_size': len(train_data),
                'val_size': len(val_data),
                'embargo_size': embargo_size
            })
        
        return results
```

```python
class Overfitting_Detector:
    """过拟合检测器"""
    
    def __init__(self, threshold=0.2):
        """
        参数:
            threshold: 样本内外差异阈值
        """
        self.threshold = threshold
    
    def detect_overfitting(self, train_score, test_score, metric='sharpe'):
        """
        检测过拟合
        
        方法：
        1. 样本内外差异
        2. 参数敏感性
        3. 复杂度惩罚
        """
        # 样本内外差异
        gap = abs(train_score - test_score) / abs(train_score)
        
        if gap > self.threshold:
            return {
                'status': 'OVERFITTING',
                'train_score': train_score,
                'test_score': test_score,
                'gap': gap,
                'recommendation': '简化模型或增加正则化'
            }
        else:
            return {
                'status': 'OK',
                'train_score': train_score,
                'test_score': test_score,
                'gap': gap,
                'recommendation': '模型表现正常'
            }
    
    def parameter_sensitivity(self, model, data, param_name, param_range):
        """
        参数敏感性分析
        
        高敏感性提示过拟合风险
        """
        scores = []
        
        for param_value in param_range:
            model.set_params(**{param_name: param_value})
            model.fit(data)
            score = model.evaluate(data)
            scores.append(score)
        
        # 计算变异系数
        cv = np.std(scores) / np.mean(scores)
        
        if cv > 0.3:
            return {
                'status': 'HIGH_SENSITIVITY',
                'coefficient_of_variation': cv,
                'recommendation': f'{param_name}参数敏感度过高，需要优化'
            }
        else:
            return {
                'status': 'OK',
                'coefficient_of_variation': cv,
                'recommendation': '参数稳定'
            }
```

```python
class Risk_Controller:
    """风险控制器"""
    
    def __init__(self, max_position=0.1, max_sector=0.3, 
                 max_drawdown=0.15, var_limit=0.05):
        self.max_position = max_position
        self.max_sector = max_sector
        self.max_drawdown = max_drawdown
        self.var_limit = var_limit
    
    def position_limit_check(self, weights, sector_mapping):
        """仓位限制检查"""
        alerts = []
        
        # 单一资产限制
        overweight_assets = weights[weights > self.max_position]
        if len(overweight_assets) > 0:
            alerts.append(f'单一资产超限: {overweight_assets.index.tolist()}')
        
        # 行业限制
        sector_weights = weights.groupby(sector_mapping).sum()
        overweight_sectors = sector_weights[sector_weights > self.max_sector]
        if len(overweight_sectors) > 0:
            alerts.append(f'行业超限: {overweight_sectors.index.tolist()}')
        
        return alerts
    
    def drawdown_control(self, portfolio_value):
        """回撤控制"""
        peak = portfolio_value.expanding().max()
        drawdown = (portfolio_value - peak) / peak
        
        if drawdown.min() < -self.max_drawdown:
            return {
                'status': 'ALERT',
                'current_drawdown': drawdown.iloc[-1],
                'max_drawdown': drawdown.min(),
                'action': '减仓或止损'
            }
        else:
            return {
                'status': 'OK',
                'current_drawdown': drawdown.iloc[-1],
                'max_drawdown': drawdown.min(),
                'action': '维持当前仓位'
            }
    
    def var_monitoring(self, returns, confidence=0.95):
        """VaR监控"""
        # 历史模拟法
        var = np.percentile(returns, (1 - confidence) * 100)
        
        # 条件VaR (Expected Shortfall)
        es = returns[returns < var].mean()
        
        if abs(var) > self.var_limit:
            return {
                'status': 'ALERT',
                'VaR': var,
                'ES': es,
                'action': '降低风险敞口'
            }
        else:
            return {
                'status': 'OK',
                'VaR': var,
                'ES': es,
                'action': '风险可控'
            }
    
    def real_time_monitoring(self, portfolio, market_data):
        """实时监控"""
        # 综合风险检查
        position_alerts = self.position_limit_check(
            portfolio['weights'], portfolio['sector_mapping']
        )
        
        drawdown_status = self.drawdown_control(portfolio['value'])
        
        var_status = self.var_monitoring(portfolio['returns'])
        
        # 汇总报告
        report = {
            'timestamp': datetime.now(),
            'position_alerts': position_alerts,
            'drawdown_status': drawdown_status,
            'var_status': var_status,
            'overall_status': 'ALERT' if (
                position_alerts or 
                drawdown_status['status'] == 'ALERT' or 
                var_status['status'] == 'ALERT'
            ) else 'OK'
        }
        
        return report
```

```python
class A_Share_Specific_Adapter:
    """A股市场特定适配器"""
    
    def __init__(self):
        self.price_limit = 0.1  # 普通股票涨跌停限制
        self.price_limit_st = 0.05  # ST股票涨跌停限制
        self.price_limit_kcb = 0.2  # 科创板涨跌停限制
    
    def handle_price_limit(self, data):
        """处理涨跌停"""
        # 识别涨跌停
        data['is_limit_up'] = (
            data['close'] >= data['pre_close'] * (1 + self.price_limit - 0.001)
        )
        data['is_limit_down'] = (
            data['close'] <= data['pre_close'] * (1 - self.price_limit + 0.001)
        )
        
        # 调整收益率计算
        data['adjusted_return'] = np.where(
            data['is_limit_up'] | data['is_limit_down'],
            np.nan,  # 涨跌停日收益率标记为缺失
            data['close'] / data['pre_close'] - 1
        )
        
        return data
    
    def handle_t1_constraint(self, signals):
        """处理T+1约束"""
        # 买入信号当日不能卖出
        # 卖出信号可以空仓
        positions = np.zeros(len(signals))
        
        for i in range(1, len(signals)):
            if signals.iloc[i] == 1:  # 买入信号
                positions[i] = 1
            elif signals.iloc[i] == -1:  # 卖出信号
                positions[i] = 0
            else:
                positions[i] = positions[i-1]
        
        return positions
    
    def filter_stocks(self, data):
        """股票筛选"""
        # 排除ST、*ST股票
        # 排除停牌股票
        # 排除上市不足60天的股票
        
        filtered = data[
            (~data['name'].str.contains('ST|*ST')) &
            (data['suspended'] == False) &
            (data['list_days'] >= 60)
        ]
        
        return filtered
    
    def adjust_for_turnover(self, weights, max_turnover=3.0):
        """换手率约束"""
        # A股换手率过高导致交易成本显著
        # 限制换手率
        
        daily_turnover = np.abs(weights.diff()).sum()
        annual_turnover = daily_turnover * 252
        
        if annual_turnover > max_turnover:
            # 调整权重
            scale_factor = max_turnover / annual_turnover
            adjusted_weights = weights * scale_factor
            return adjusted_weights
        else:
            return weights
```

```python
class EntropicMarketSignal:
    """基于熵的市场信号"""
    
    def __init__(self, window=20):
        self.window = window
    
    def compute_entropy(self, returns):
        """计算收益率熵"""
        # 离散化
        bins = np.linspace(returns.min(), returns.max(), 20)
        hist, _ = np.histogram(returns, bins=bins, density=True)
        
        # 熵
        entropy = -np.sum(hist * np.log(hist + 1e-10))
        return entropy
    
    def cumulative_entropy(self, returns):
        """滑动窗口累积熵"""
        cum_entropy = []
        for i in range(self.window, len(returns)):
            window_returns = returns[i-self.window:i]
            entropy = self.compute_entropy(window_returns)
            cum_entropy.append(entropy)
        
        return np.array(cum_entropy)
    
    def detect_regime_change(self, returns, threshold=2.0):
        """检测市场状态变化"""
        cum_entropy = self.cumulative_entropy(returns)
        
        # Z-score标准化
        z_score = (cum_entropy - cum_entropy.mean()) / cum_entropy.std()
        
        # 超过阈值标记为状态变化
        regime_changes = np.where(np.abs(z_score) > threshold)[0]
        
        return regime_changes
```

```python
import torch
import torch.nn as nn

class NodeTransformer(nn.Module):
    """股票关系网络Transformer"""
    
    def __init__(self, num_stocks, feature_dim, num_heads=8):
        super().__init__()
        self.stock_embedding = nn.Embedding(num_stocks, feature_dim)
        self.transformer = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(
                d_model=feature_dim,
                nhead=num_heads,
                batch_first=True
            ),
            num_layers=6
        )
        self.predictor = nn.Linear(feature_dim, 1)
    
    def forward(self, x, stock_ids):
        # 股票嵌入
        stock_emb = self.stock_embedding(stock_ids)
        x = x + stock_emb
        
        # Transformer编码
        x = self.transformer(x)
        
        # 预测未来收益
        pred = self.predictor(x)
        return pred
```

### 6.3 组合优化新方法

#### Gibbs后验组合选择
```python
def gibbs_posterior_portfolio(returns, target_return, n_samples=1000):
    """
    Gibbs后验组合优化
    结合贝叶斯推断与组合选择
    """
    n_assets = returns.shape[1]
    
    # 初始化
    weights = np.ones(n_assets) / n_assets
    
    for i in range(n_samples):
        # 采样资产权重
        for j in range(n_assets):
            # 条件后验采样
            other_weights = np.delete(weights, j)
            other_weights_sum = other_weights.sum()
            
            # 满足约束：权重和为1
            w_j_min = max(0, 1 - other_weights_sum)
            w_j_max = min(1, 1 - other_weights_sum + 1)
            
            weights[j] = np.random.uniform(w_j_min, w_j_max)
        
        # 评估目标达成度
        portfolio_return = (weights @ returns.mean())
        if abs(portfolio_return - target_return) < 0.01:
            yield weights.copy()
```

```python
class EnsembleModel:
    """集成模型"""
    
    def __init__(self, models):
        self.models = models
    
    def fit(self, X, y):
        for model in self.models:
            model.fit(X, y)
    
    def predict(self, X):
        predictions = []
        for model in self.models:
            pred = model.predict(X)
            predictions.append(pred)
        
        # 平均集成
        return np.mean(predictions, axis=0)
    
    def predict_proba(self, X):
        probas = []
        for model in self.models:
            proba = model.predict_proba(X)
            probas.append(proba)
        
        return np.mean(probas, axis=0)
```

```python
class ModelMonitor:
    """模型性能监控"""
    
    def __init__(self, threshold_ic=0.02, threshold_sharpe=1.0):
        self.threshold_ic = threshold_ic
        self.threshold_sharpe = threshold_sharpe
        self.history = []
    
    def check_model_drift(self, predictions, actual_returns):
        """检查模型漂移"""
        # 计算IC
        ic = np.corrcoef(predictions, actual_returns)[0, 1]
        
        # 计算累积收益
        cumulative_return = (predictions * actual_returns).cumsum()
        
        # 检查是否低于阈值
        if ic < self.threshold_ic:
            self.send_alert(f'IC下降至{ic:.4f}，低于阈值{self.threshold_ic}')
        
        # 记录历史
        self.history.append({
            'date': datetime.now(),
            'ic': ic,
            'cumulative_return': cumulative_return[-1]
        })
        
        return ic
    
    def send_alert(self, message):
        """发送告警"""
        print(f'[ALERT] {datetime.now()}: {message}')
```

```python
class RiskController:
    """风控模块"""
    
    def __init__(self, max_position=0.1, max_sector=0.3, max_drawdown=0.15):
        self.max_position = max_position
        self.max_sector = max_sector
        self.max_drawdown = max_drawdown
    
    def check_position_limit(self, weights, sector_mapping):
        """检查仓位限制"""
        # 单一资产限制
        if (weights > self.max_position).any():
            return False, '单一资产超限'
        
        # 行业限制
        sector_weights = {}
        for asset, weight in weights.items():
            sector = sector_mapping[asset]
            sector_weights[sector] = sector_weights.get(sector, 0) + weight
        
        for sector, weight in sector_weights.items():
            if weight > self.max_sector:
                return False, f'行业{sector}超限'
        
        return True, '通过'
    
    def check_drawdown(self, portfolio_value):
        """检查回撤限制"""
        peak = portfolio_value.expanding().max()
        drawdown = (portfolio_value - peak) / peak
        
        if drawdown.min() < -self.max_drawdown:
            return False, '回撤超限'
        
        return True, '通过'
```
