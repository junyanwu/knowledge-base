## 特征工程

### 数据清洗

#### 缺失值处理
- **删除法**：适用于缺失比例很小的数据（<5%）
- **填充法**：
  - 均值/中位数填充：适用于连续变量
  - 前向填充（forward fill）：适用于时间序列数据
  - 插值法：线性插值、样条插值等
- **预测填充**：使用其他特征预测缺失值

#### 异常值处理
- **统计方法**：
  - Z-score 方法（|z| > 3 视为异常）
  - IQR 方法（超出 Q1-1.5*IQR 到 Q3+1.5*IQR 范围）
- **机器学习方法**：
  - Isolation Forest
  - Local Outlier Factor (LOF)
  - One-Class SVM

#### 数据标准化
- **Z-score 标准化**：`x' = (x - μ) / σ`
- **Min-Max 归一化**：`x' = (x - min) / (max - min)`
- **Robust Scaler**：使用中位数和IQR，对异常值鲁棒

## 模型选择

### 树模型

#### XGBoost（最新版本 3.2.0，2026年2月发布）

**核心优势**：
- 高效的并行计算
- 内置正则化防止过拟合
- 支持缺失值处理
- 支持分类特征（2025年增强）

**参数调优要点**：
```python
# 控制模型复杂度
max_depth = 6           # 树的最大深度
min_child_weight = 1    # 叶子节点最小权重
gamma = 0.1            # 分裂最小增益

# 添加随机性
subsample = 0.8        # 样本采样比例
colsample_bytree = 0.8 # 特征采样比例

# 学习率控制
eta = 0.1              # 学习率
```

**处理不平衡数据**：
- 使用 `scale_pos_weight` 平衡正负样本权重
- 使用 AUC 作为评估指标
- 对于概率预测，设置 `max_delta_step` 帮助收敛

#### LightGBM（最新版本 4.6.0）

**核心优势**：
- 更快的训练速度
- 更低的内存使用
- 支持大规模数据
- Leaf-wise 生长策略

**加速训练技巧**：
```python
# 减少树的复杂度
max_depth = -1         # 不限制深度
num_leaves = 31        # 叶子节点数量
min_gain_to_split = 0  # 分裂最小增益

# 减少分裂计算量
max_bin = 255          # 特征分箱数
feature_fraction = 0.8 # 特征采样

# 早停机制
early_stopping_round = 50
```

**内存优化**：
- 使用 `save_binary` 保存数据集
- 控制并行线程数避免内存溢出
- 使用 GPU 加速

### 神经网络

#### LSTM/GRU 时序模型
```python
# 典型架构
model = Sequential([
    LSTM(128, return_sequences=True, input_shape=(timesteps, features)),
    Dropout(0.2),
```

#### Transformer 时序模型
- 自注意力机制捕捉长距离依赖
- 位置编码保留时序信息
- 适合多变量时间序列预测

#### Temporal Fusion Transformer (TFT)
- 可解释的注意力权重
- 支持静态协变量和时变特征
- 内置变量选择机制

### 集成方法

#### 模型融合策略
1. **平均融合**：简单平均、加权平均
2. **Stacking**：使用元模型融合基础模型
3. **Blending**：使用验证集进行融合

#### 实践建议
- 基模型应具有差异性（不同算法、不同特征集）
- 使用交叉验证评估融合效果
- 注意融合可能带来的延迟增加

### 交叉验证

#### K-Fold 交叉验证
```python
from sklearn.model_selection import cross_val_score, KFold

kf = KFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(model, X, y, cv=kf, scoring='neg_mean_squared_error')
```

#### 时序交叉验证（Walk-Forward）
```python
from sklearn.model_selection import TimeSeriesSplit

tscv = TimeSeriesSplit(n_splits=5)
for train_idx, val_idx in tscv.split(X):
    X_train, X_val = X[train_idx], X[val_idx]
    y_train, y_val = y[train_idx], y[val_idx]
```

#### Purged K-Fold（金融专用）
- 在训练集和验证集之间设置隔离期
- 避免数据泄露
- 适合有重叠标签的场景

### 样本外测试

#### 关键原则
1. **严格的样本外测试**：使用未被任何训练过程接触的数据
2. **多次测试**：在不同时间段进行测试
3. **基准对比**：与简单基准策略对比

#### 常见陷阱
- **前视偏差**：使用了未来信息
- **数据泄露**：训练数据与测试数据有重叠
- **过拟合超参数**：超参数调优时的过拟合

### 2026 最新研究进展

根据 scikit-learn 文档和最新研究：

1. **嵌套交叉验证**：用于超参数调优的无偏估计
2. **Pipelines 组合**：确保预处理步骤不泄露信息
3. **学习曲线分析**：诊断模型是否需要更多数据或更复杂模型

### 强化学习

#### Deep Deterministic Policy Gradient (DDPG)

**核心原理**：
- 同时学习 Q 函数和策略
- 使用经验回放缓冲区
- 目标网络通过 Polyak 平均更新

**量化交易应用**：
```python
# DDPG 在交易中的应用
class TradingEnv(gym.Env):
    def __init__(self, data):
        self.data = data
        self.action_space = spaces.Box(
            low=-1, high=1, shape=(1,)  # 买卖信号
        )
        self.observation_space = spaces.Box(
            low=-np.inf, high=np.inf, shape=(features,)
        )
```

**关键技巧**：
- **经验回放**：存储历史交易数据用于训练
- **目标网络**：稳定训练过程
- **探索噪声**：添加高斯噪声促进探索
