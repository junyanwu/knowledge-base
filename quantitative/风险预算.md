# 风险预算

## 1. 风险预算概念

### 1.1 定义

**风险预算：** 将总风险分配到各个资产或策略，使每个资产的风险贡献达到预定目标。

### 1.2 核心公式

**风险贡献：**
```
RCᵢ = wᵢ × Cov(rᵢ, rₚ) / σₚ
```

其中：
- wᵢ：资产权重
- Cov(rᵢ, rₚ)：资产与组合的协方差
- σₚ：组合波动率

**风险贡献比例：**
```
MRCᵢ = RCᵢ / σₚ = wᵢ × βᵢ
```

## 2. 风险平价

### 2.1 原理

**风险平价：** 每个资产对组合风险的贡献相等。

```python
def risk_parity_portfolio(cov_matrix):
    """
    风险平价组合
    
    参数:
    cov_matrix: 协方差矩阵
    
    返回:
    权重
    """
    n_assets = cov_matrix.shape[0]
    
    def risk_budget_objective(weights):
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 边际风险贡献
        marginal_risk = cov_matrix @ weights / portfolio_vol
        
        # 风险贡献
        risk_contrib = weights * marginal_risk
        
        # 目标：风险贡献相等
        target_risk = portfolio_vol / n_assets
        
        return np.sum((risk_contrib - target_risk) ** 2)
    
    from scipy.optimize import minimize
    
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    result = minimize(
        risk_budget_objective,
        x0=[1/n_assets] * n_assets,
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

### 3.2 因子风险预算

```python
def factor_risk_budget(factor_exposure, factor_cov, factor_budget):
    """
    因子风险预算
    
    参数:
    factor_exposure: 因子暴露 (n_assets, n_factors)
    factor_cov: 因子协方差 (n_factors, n_factors)
    factor_budget: 因子风险预算
    
    返回:
    权重
    """
    n_assets = factor_exposure.shape[0]
    
    def objective(weights):
        # 因子风险贡献
        portfolio_exposure = weights @ factor_exposure
        
        # 因子风险
        factor_var = portfolio_exposure @ factor_cov @ portfolio_exposure
        
        # 因子边际风险贡献
        marginal_factor_risk = factor_cov @ portfolio_exposure
        
        # 因子风险贡献
        factor_risk_contrib = portfolio_exposure * marginal_factor_risk
        
        # 因子风险贡献比例
        factor_risk_pct = factor_risk_contrib / factor_var
        
        return np.sum((factor_risk_pct - factor_budget) ** 2)
    
    from scipy.optimize import minimize
    
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_assets)]
    
    result = minimize(
        objective,
        x0=[1/n_assets] * n_assets,
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

## 4. 风险预算调整

### 4.1 动态风险预算

```python
def dynamic_risk_budget(returns, window=60, target_budget=None):
    """
    动态风险预算
    
    参数:
    returns: 收益率序列
    window: 滚动窗口
    target_budget: 目标风险预算
    
    返回:
    动态权重
    """
    if target_budget is None:
        target_budget = np.ones(returns.shape[1]) / returns.shape[1]
    
    weights_series = []
    
    for i in range(window, len(returns)):
        # 滚动协方差
        window_returns = returns.iloc[i-window:i]
        cov_matrix = window_returns.cov().values
        
        # 计算权重
        weights = risk_budget_portfolio(cov_matrix, target_budget)
        weights_series.append(weights)
    
    return pd.DataFrame(weights_series, index=returns.index[window:])
```

### 4.2 风险预算再平衡

```python
def risk_budget_rebalance(current_weights, target_weights, cov_matrix, 
                         rebalance_threshold=0.05):
    """
    风险预算再平衡
    
    参数:
    current_weights: 当前权重
    target_weights: 目标权重
    cov_matrix: 协方差矩阵
    rebalance_threshold: 再平衡阈值
    
    返回:
    是否需要再平衡
    """
    # 当前风险贡献
    current_vol = np.sqrt(current_weights @ cov_matrix @ current_weights)
    current_marginal_risk = cov_matrix @ current_weights / current_vol
    current_risk_contrib = current_weights * current_marginal_risk
    current_risk_pct = current_risk_contrib / current_vol
    
    # 目标风险贡献
    target_vol = np.sqrt(target_weights @ cov_matrix @ target_weights)
    target_marginal_risk = cov_matrix @ target_weights / target_vol
    target_risk_contrib = target_weights * target_marginal_risk
    target_risk_pct = target_risk_contrib / target_vol
    
    # 风险贡献偏离
    risk_deviation = np.abs(current_risk_pct - target_risk_pct)
    
    return {
        'need_rebalance': np.any(risk_deviation > rebalance_threshold),
        'risk_deviation': risk_deviation,
        'current_risk_pct': current_risk_pct,
        'target_risk_pct': target_risk_pct
    }
```

## 5. 实战应用

### 5.1 风险预算策略

```python
class RiskBudgetStrategy:
    """风险预算策略"""
    
    def __init__(self, risk_budget=None):
        self.risk_budget = risk_budget
    
    def fit(self, returns):
        """训练"""
        if self.risk_budget is None:
            self.risk_budget = np.ones(returns.shape[1]) / returns.shape[1]
        
        self.cov_matrix = returns.cov().values
        self.weights = risk_budget_portfolio(self.cov_matrix, self.risk_budget)
        
        return self
    
    def get_weights(self):
        """获取权重"""
        return self.weights
    
    def get_risk_contrib(self):
        """获取风险贡献"""
        portfolio_vol = np.sqrt(self.weights @ self.cov_matrix @ self.weights)
        marginal_risk = self.cov_matrix @ self.weights / portfolio_vol
        risk_contrib = self.weights * marginal_risk
        risk_contrib_pct = risk_contrib / portfolio_vol
        
        return risk_contrib_pct
    
    def backtest(self, returns, rebalance_freq=20):
        """回测"""
        portfolio_returns = []
        
        for i in range(rebalance_freq, len(returns), rebalance_freq):
            window_returns = returns.iloc[i-rebalance_freq:i]
            self.fit(window_returns)
            
            # 计算下一期收益
            next_returns = returns.iloc[i:i+rebalance_freq]
            port_ret = (next_returns * self.weights).sum(axis=1)
            portfolio_returns.append(port_ret)
        
        return pd.concat(portfolio_returns)
```

### 5.2 风险监控

```python
def risk_budget_monitor(weights, cov_matrix, risk_budget, threshold=0.05):
    """
    风险预算监控
    
    参数:
    weights: 当前权重
    cov_matrix: 协方差矩阵
    risk_budget: 目标风险预算
    threshold: 预警阈值
    
    返回:
    监控报告
    """
    # 实际风险贡献
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    marginal_risk = cov_matrix @ weights / portfolio_vol
    risk_contrib = weights * marginal_risk
    risk_contrib_pct = risk_contrib / portfolio_vol
    
    # 偏离
    deviation = risk_contrib_pct - risk_budget
    
    # 预警
    alerts = []
    for i, dev in enumerate(deviation):
        if abs(dev) > threshold:
            alerts.append({
                'asset': i,
                'deviation': dev,
                'current_risk': risk_contrib_pct[i],
                'target_risk': risk_budget[i],
                'alert': 'OVER' if dev > 0 else 'UNDER'
            })
    
    return {
        'portfolio_vol': portfolio_vol,
        'risk_contrib_pct': risk_contrib_pct,
        'deviation': deviation,
        'alerts': alerts
    }
```

## 6. 总结

风险预算要点：

1. **风险平价**：风险贡献相等
2. **风险预算**：自定义风险分配
3. **因子风险预算**：基于因子的风险分配
4. **动态调整**：滚动窗口、再平衡
5. **风险监控**：偏离预警

**注意事项：**
- 协方差矩阵估计
- 风险预算合理性
- 交易成本考虑
- 定期再平衡
