## 2. 策略收益相关性分析

### 2.1 相关性度量方法

#### 2.1.1 Pearson相关系数

**定义**：衡量两个变量之间的线性相关程度

**公式**：
```
ρ_xy = Cov(X, Y) / (σ_X × σ_Y)
     = Σ[(x_i - x̄)(y_i - ȳ)] / √[Σ(x_i - x̄)² × Σ(y_i - ȳ)²]
```

**性质**：
- 取值范围：[-1, 1]
- 对称性：ρ_xy = ρ_yx
- 对线性变换敏感：ρ(aX+b, Y) = sign(a) × ρ(X, Y)

**适用场景**：
- 线性关系
- 正态分布
- 无异常值

**局限性**：
- 仅捕捉线性相关
- 对异常值敏感
- 无法反映尾部相关性

**Python实现**：

```python
import numpy as np
import pandas as pd
from scipy import stats
import matplotlib.pyplot as plt
import seaborn as sns

class StrategyCorrelationAnalysis:
    """策略相关性分析框架"""
    
    def __init__(self, strategy_returns: pd.DataFrame):
        """
        初始化
        
        Args:
            strategy_returns: 策略收益率DataFrame，每列一个策略
        """
        self.returns = strategy_returns
        self.n_strategies = strategy_returns.shape[1]
        self.strategy_names = strategy_returns.columns.tolist()
    
    def pearson_correlation(self) -> pd.DataFrame:
        """
        Pearson相关系数矩阵
        
        Returns:
            相关系数矩阵
        """
        corr_matrix = self.returns.corr(method='pearson')
        return corr_matrix
    
    def correlation_pvalues(self) -> pd.DataFrame:
        """
        相关系数的p值矩阵
        
        Returns:
            p值矩阵
        """
        n = len(self.returns)
        pvalues = pd.DataFrame(
            index=self.strategy_names,
            columns=self.strategy_names
        )
        
        for i, strat1 in enumerate(self.strategy_names):
            for j, strat2 in enumerate(self.strategy_names):
                if i == j:
                    pvalues.loc[strat1, strat2] = 0.0
                else:
                    r, p = stats.pearsonr(
                        self.returns[strat1],
                        self.returns[strat2]
                    )
                    pvalues.loc[strat1, strat2] = p
        
        return pvalues.astype(float)
    
    def plot_correlation_heatmap(self, 
                                 corr_matrix: pd.DataFrame,
                                 title: str = "策略相关性矩阵",
                                 save_path: str = None):
        """
        绘制相关性热力图
        
        Args:
            corr_matrix: 相关系数矩阵
            title: 图表标题
            save_path: 保存路径
        """
        plt.figure(figsize=(12, 10))
        
        # 热力图
        mask = np.triu(np.ones_like(corr_matrix, dtype=bool), k=1)
        sns.heatmap(
            corr_matrix,
            mask=mask,
            annot=True,
            fmt='.2f',
            cmap='RdYlGn_r',
            center=0,
            square=True,
            linewidths=1,
            cbar_kws={'label': '相关系数'}
        )
        
        plt.title(title, fontsize=16, fontweight='bold')
        plt.tight_layout()
        
        if save_path:
            plt.savefig(save_path, dpi=300, bbox_inches='tight')
            print(f"图表已保存至: {save_path}")
        
        plt.close()
    
    def correlation_significance_test(self, 
                                     alpha: float = 0.05) -> pd.DataFrame:
        """
        相关性显著性检验
        
        Args:
            alpha: 显著性水平
            
        Returns:
            显著性检验结果
        """
        corr_matrix = self.pearson_correlation()
        pvalues = self.correlation_pvalues()
        
        significance = pd.DataFrame(
            index=self.strategy_names,
            columns=self.strategy_names
        )
        
        for i in self.strategy_names:
            for j in self.strategy_names:
                if i == j:
                    significance.loc[i, j] = '-'
                else:
                    p = pvalues.loc[i, j]
                    if p < 0.001:
                        significance.loc[i, j] = '***'
                    elif p < 0.01:
                        significance.loc[i, j] = '**'
                    elif p < 0.05:
                        significance.loc[i, j] = '*'
                    else:
                        significance.loc[i, j] = ''
        
        # 合并相关系数和显著性
        result = corr_matrix.round(2).astype(str) + significance
        
        return result
```

#### 2.1.2 Spearman相关系数

**定义**：衡量两个变量之间的单调关系（秩相关）

**公式**：
```
ρ_s = 1 - 6Σd_i² / (n(n²-1))
```

其中 d_i 是秩次的差值

**优势**：
- 不依赖线性假设
- 对异常值稳健
- 捕捉单调非线性关系

**适用场景**：
- 非正态分布
- 存在异常值
- 单调非线性关系

```python
    def spearman_correlation(self) -> pd.DataFrame:
        """
        Spearman秩相关系数矩阵

            Spearman相关系数矩阵
        """
        corr_matrix = self.returns.corr(method='spearman')
        return corr_matrix
```

#### 2.1.3 Kendall相关系数

**定义**：基于一致对和不一致对的秩相关系数

**公式**：
```
τ = (C - D) / (n(n-1)/2)
```

其中：
- C：一致对数量
- D：不一致对数量

**优势**：
- 更稳健的秩相关度量
- 适合小样本
- 有明确的统计检验

```python
    def kendall_correlation(self) -> pd.DataFrame:
        """
        Kendall秩相关系数矩阵
        
        Returns:
            Kendall相关系数矩阵
        """
        corr_matrix = self.returns.corr(method='kendall')
        return corr_matrix
    
    def compare_correlation_methods(self) -> pd.DataFrame:
        """
        比较三种相关系数方法
        
        Returns:
            相关系数对比表
        """
        pearson = self.pearson_correlation()
        spearman = self.spearman_correlation()
        kendall = self.kendall_correlation()
        
        # 提取上三角元素（不含对角线）
        n = len(self.strategy_names)
        triu_idx = np.triu_indices(n, k=1)
        
        comparison = []
        for i, j in zip(triu_idx[0], triu_idx[1]):
            strat1 = self.strategy_names[i]
            strat2 = self.strategy_names[j]
            
            comparison.append({
                'Strategy Pair': f"{strat1} vs {strat2}",
                'Pearson': pearson.iloc[i, j],
                'Spearman': spearman.iloc[i, j],
                'Kendall': kendall.iloc[i, j],
                'Pearson-Spearman Diff': pearson.iloc[i, j] - spearman.iloc[i, j]
            })
        
        return pd.DataFrame(comparison)
```

### 2.2 相关性分析与解读

```python
    def analyze_correlation_structure(self) -> Dict:
        """
        分析策略相关性结构
        
        Returns:
            相关性结构分析
        """
        corr = self.pearson_correlation()
        
        # 计算平均相关性
        avg_corr = (corr.values.sum() - self.n_strategies) / (
            self.n_strategies * (self.n_strategies - 1)
        )
        
        # 找出高相关策略对
        high_corr_pairs = []
        for i in range(self.n_strategies):
            for j in range(i+1, self.n_strategies):
                if corr.iloc[i, j] > 0.7:
                    high_corr_pairs.append({
                        'Pair': f"{self.strategy_names[i]} - {self.strategy_names[j]}",
                        'Correlation': corr.iloc[i, j]
                    })
        
        # 找出低相关策略对
        low_corr_pairs = []
        for i in range(self.n_strategies):
            for j in range(i+1, self.n_strategies):
                if corr.iloc[i, j] < 0.3:
                    low_corr_pairs.append({
                        'Pair': f"{self.strategy_names[i]} - {self.strategy_names[j]}",
                        'Correlation': corr.iloc[i, j]
                    })
        
        # 计算分散化指数
        det_corr = np.linalg.det(corr.values)
        diversification_index = 1 - (avg_corr + 1) / 2  # 简化版本
        
        return {
            'average_correlation': avg_corr,
            'high_correlation_pairs': high_corr_pairs,
            'low_correlation_pairs': low_corr_pairs,
            'diversification_index': diversification_index,
            'determinant': det_corr,
            'interpretation': self._interpret_correlation(avg_corr)
        }
    
    def _interpret_correlation(self, avg_corr: float) -> str:
        """解读平均相关性"""
        if avg_corr < 0.2:
            return "极低相关性，分散化效果极佳"
        elif avg_corr < 0.4:
            return "低相关性，分散化效果良好"
        elif avg_corr < 0.6:
            return "中等相关性，分散化效果一般"
        elif avg_corr < 0.8:
            return "高相关性，分散化效果较差"
        else:
            return "极高相关性，分散化效果极差"
```

```python
    def rolling_correlation(self, 
                           window: int = 60,
                           method: str = 'pearson') -> pd.DataFrame:
        """
        滚动相关性
        
        Args:
            window: 滚动窗口（天数）
            method: 相关系数方法
            
        Returns:
            滚动相关性序列（策略对的滚动相关性）
        """
        rolling_corr = {}
        
        for i, strat1 in enumerate(self.strategy_names):
            for j, strat2 in enumerate(self.strategy_names):
                if i < j:
                    pair_name = f"{strat1}_{strat2}"
                    rolling_corr[pair_name] = self.returns[strat1].rolling(
                        window
                    ).corr(self.returns[strat2])
        
        return pd.DataFrame(rolling_corr)
    
    def plot_rolling_correlation(self, 
                                 strategy_pair: tuple,
                                 window: int = 60,
                                 save_path: str = None):
        """
        绘制滚动相关性
        
        Args:
            strategy_pair: 策略对元组
            window: 滚动窗口
            save_path: 保存路径
        """
        strat1, strat2 = strategy_pair
        
        rolling_corr = self.returns[strat1].rolling(window).corr(
            self.returns[strat2]
        )
        
        plt.figure(figsize=(14, 6))
        
        # 滚动相关性
        plt.plot(rolling_corr, label=f'{window}日滚动相关', alpha=0.8)
        
        # 平均线
        plt.axhline(y=rolling_corr.mean(), color='r', linestyle='--', 
                   label=f'平均: {rolling_corr.mean():.2f}')
        
        # 标准差带
        plt.fill_between(
            rolling_corr.index,
            rolling_corr.mean() - rolling_corr.std(),
            rolling_corr.mean() + rolling_corr.std(),
            alpha=0.2,
            color='gray',
            label='±1标准差'
        )
        
        plt.title(f'{strat1} vs {strat2} 滚动相关性', fontsize=14)
        plt.xlabel('日期')
        plt.ylabel('相关系数')
        plt.legend()
        plt.grid(True, alpha=0.3)
        plt.tight_layout()
        
        if save_path:
            plt.savefig(save_path, dpi=300, bbox_inches='tight')
        
        plt.close()
    
    def rolling_correlation_stability(self, 
                                     window: int = 60) -> pd.DataFrame:
        """
        滚动相关性稳定性分析
        
        Returns:
            相关性稳定性统计
        """
        rolling_corr = self.rolling_correlation(window)
        
        stability_stats = []
        
        for pair in rolling_corr.columns:
            corr_series = rolling_corr[pair].dropna()
            
            stability_stats.append({
                'Pair': pair,
                'Mean': corr_series.mean(),
                'Std': corr_series.std(),
                'Min': corr_series.min(),
                'Max': corr_series.max(),
                'Range': corr_series.max() - corr_series.min(),
                'CV': corr_series.std() / abs(corr_series.mean()) if corr_series.mean() != 0 else np.inf,
                'Stability': 'Stable' if corr_series.std() < 0.15 else 'Unstable'
            })
        
        return pd.DataFrame(stability_stats)
```

### 3.2 指数加权移动平均相关性（EWMA）

```python
    def ewma_correlation(self, 
                        span: int = 60) -> pd.DataFrame:
        """
        指数加权移动平均相关性
        
        Args:
            span: EWMA参数（类似半衰期）
            
        Returns:
            EWMA相关系数矩阵
        """
        # EWMA协方差
        ewma_cov = self.returns.ewm(span=span).cov()
        
        # 提取最新协方差矩阵
        latest_date = ewma_cov.index.get_level_values(0)[-1]
        cov_matrix = ewma_cov.loc[latest_date]
        
        # 转换为相关系数矩阵
        std_devs = np.sqrt(np.diag(cov_matrix))
        corr_matrix = cov_matrix / np.outer(std_devs, std_devs)
        
        return corr_matrix
```

```python
    def conditional_correlation(self, 
                               market_regime: pd.Series) -> Dict:
        """
        条件相关性（基于市场状态）
        
        Args:
            market_regime: 市场状态序列（'bull', 'bear', 'sideways'）
            
        Returns:
            不同市场状态下的相关性
        """
        regimes = market_regime.unique()
        conditional_corr = {}
        
        for regime in regimes:
            regime_returns = self.returns[market_regime == regime]
            corr_matrix = regime_returns.corr()
            conditional_corr[regime] = corr_matrix
        
        # 分析相关性变化
        correlation_shift = {}
        base_corr = self.pearson_correlation()
        
        for regime in regimes:
            diff = conditional_corr[regime] - base_corr
            correlation_shift[regime] = {
                'mean_shift': diff.values.mean(),
                'max_increase': diff.values.max(),
                'max_decrease': diff.values.min(),
                'corr_matrix': conditional_corr[regime]
            }
        
        return {
            'conditional_correlations': conditional_corr,
            'correlation_shift': correlation_shift
        }
    
    def plot_regime_correlation(self, 
                               market_regime: pd.Series,
                               save_path: str = None):
```

## 4. 策略风险相关性分析

### 4.1 尾部相关性

**Copula模型估计尾部相关性**：

```python
from scipy.stats import norm, t as student_t
from scipy.optimize import minimize

class TailDependencyAnalysis:
    """尾部相关性分析"""
    
    def __init__(self, returns: pd.DataFrame):
        """
        初始化
        
        Args:
            returns: 策略收益率DataFrame
        """
        self.returns = returns
        self.strategy_names = returns.columns.tolist()
    
    def empirical_tail_dependency(self, 
                                 quantile: float = 0.05) -> Dict:
        """
        经验尾部相关性
        
        Args:
            quantile: 分位数（左尾）
            
        Returns:
            尾部相关系数
        """
        n = len(self.returns)
        tail_dependency = {}
        
        for i, strat1 in enumerate(self.strategy_names):
            for j, strat2 in enumerate(self.strategy_names):
                if i < j:
                    # 左尾相关性
                    q1 = self.returns[strat1].quantile(quantile)
                    q2 = self.returns[strat2].quantile(quantile)
                    
                    tail_events_1 = self.returns[strat1] <= q1
                    tail_events_2 = self.returns[strat2] <= q2
                    joint_tail_events = tail_events_1 & tail_events_2
                    
                    # 尾部相关系数
                    lower_tail = joint_tail_events.sum() / (tail_events_1.sum() * tail_events_2.sum()) * n
                    
                    # 右尾相关性
                    q1_high = self.returns[strat1].quantile(1 - quantile)
                    q2_high = self.returns[strat2].quantile(1 - quantile)
                    
                    tail_events_1_high = self.returns[strat1] >= q1_high
                    tail_events_2_high = self.returns[strat2] >= q2_high
                    joint_tail_events_high = tail_events_1_high & tail_events_2_high
                    
                    upper_tail = joint_tail_events_high.sum() / (tail_events_1_high.sum() * tail_events_2_high.sum()) * n
                    
                    tail_dependency[f"{strat1}_{strat2}"] = {
                        'lower_tail': lower_tail,
                        'upper_tail': upper_tail
                    }
        
        return tail_dependency
    
    def gaussian_copula_correlation(self) -> pd.DataFrame:
        """
        高斯Copula相关性
        
        Returns:
            高斯Copula相关矩阵
        """
        # 转换为均匀分布
        u_returns = self.returns.rank() / len(self.returns)
        
        # 转换为标准正态
        z_returns = norm.ppf(u_returns)
        
        # 计算相关矩阵
        copula_corr = pd.DataFrame(
            z_returns,
            columns=self.strategy_names
        ).corr()
        
        return copula_corr
    
    def student_t_copula_fit(self, 
                            df_guess: float = 5.0) -> Dict:
        """
        学生t-Copula拟合
        
        Args:
            df_guess: 自由度初始猜测
            
        Returns:
            拟合参数
        """
        # 转换为标准正态
        u_returns = self.returns.rank() / len(self.returns)
        z_returns = norm.ppf(u_returns)
        
        def neg_log_likelihood(params):
            df = params[0]
            n = len(self.returns)
            ll = 0
            
            for i in range(n):
                z = z_returns.iloc[i].values
                # t分布对数似然
                ll += student_t.logpdf(z, df=df).sum()
            
            return -ll
        
        # 优化
        result = minimize(
            neg_log_likelihood,
            x0=[df_guess],
            bounds=[(2.0, 30.0)],
            method='L-BFGS-B'
        )
        
        return {
            'degrees_of_freedom': result.x[0],
            'log_likelihood': -result.fun
        }
```

### 4.2 极端事件相关性

```python
    def extreme_event_correlation(self, 
                                 threshold: float = -0.03) -> pd.DataFrame:
        """
        极端事件下的相关性
        
        Args:
            threshold: 极端事件阈值（日收益率）
            
        Returns:
            极端事件相关矩阵
        """
        # 找出至少有一个策略发生极端事件的日期
        extreme_dates = self.returns[
            (self.returns <= threshold).any(axis=1) |
            (self.returns >= -threshold).any(axis=1)
        ].index
        
        # 计算极端事件下的相关性
        extreme_returns = self.returns.loc[extreme_dates]
        extreme_corr = extreme_returns.corr()
        
        # 对比正常时期相关性
        normal_dates = self.returns.index.difference(extreme_dates)
        normal_returns = self.returns.loc[normal_dates]
        normal_corr = normal_returns.corr()
        
        # 相关性变化
        corr_change = extreme_corr - normal_corr
        
        return {
            'extreme_correlation': extreme_corr,
            'normal_correlation': normal_corr,
            'correlation_change': corr_change,
            'extreme_days': len(extreme_dates),
            'normal_days': len(normal_dates)
        }
    
    def correlation_breakdown_analysis(self, 
                                      window: int = 252) -> pd.DataFrame:
        """
        相关性崩解分析
        
        Args:
            window: 分析窗口
            
        Returns:
            相关性崩解统计
        """
        results = []
        
        for i in range(window, len(self.returns)):
            # 计算当前窗口相关性
            current_window = self.returns.iloc[i-window:i]
            current_corr = current_window.corr()
            
            # 计算前一个窗口相关性
            previous_window = self.returns.iloc[i-window-1:i-1]
            previous_corr = previous_window.corr()
            
            # 相关性变化
            corr_change = np.abs(current_corr.values - previous_corr.values).mean()
            
            # 平均相关性
            avg_corr = current_corr.values.mean()
            
            # 市场波动率
            market_vol = self.returns.iloc[i-window:i].std().mean()
            
            results.append({
                'date': self.returns.index[i],
                'avg_correlation': avg_corr,
                'correlation_change': corr_change,
                'market_volatility': market_vol
            })
        
        return pd.DataFrame(results).set_index('date')
```

```python
            ((self.returns > 0).any(axis=1) & (self.returns < 0).any(axis=1))
        ]
        mixed_corr = mixed.corr() if len(mixed) > 30 else None
        
        return {
            'upside_correlation': upside_corr,
            'downside_correlation': downside_corr,
```

```python
                self.returns[other_category_strategies].mean(axis=1)
            )
```
