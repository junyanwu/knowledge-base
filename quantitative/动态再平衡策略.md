## 2. 再平衡策略设计

### 2.1 定期再平衡

**定义**：按固定时间间隔进行再平衡

**频率选择**：
- 月度再平衡：每月月末
- 季度再平衡：每季度末
- 半年度再平衡：每半年末
- 年度再平衡：每年末

**优势**：
- 简单易执行
- 规律性强
- 便于资金规划

**局限**：
- 可能错过再平衡机会
- 不必要的交易
- 成本可能较高

**Python实现**：

```python
import numpy as np
import pandas as pd
from typing import Dict, List, Tuple
import matplotlib.pyplot as plt

class PeriodicRebalancer:
    """定期再平衡策略"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 target_weights: np.ndarray,
                 transaction_cost: float = 0.001):
        """
        初始化
        
        Args:
            returns: 资产收益率DataFrame
            target_weights: 目标权重
            transaction_cost: 交易成本
        """
        self.returns = returns
        self.target_weights = target_weights
        self.n_assets = len(target_weights)
        self.asset_names = returns.columns.tolist()
        self.transaction_cost = transaction_cost
    
    def rebalance(self, 
                 frequency: str = 'Q',
                 initial_capital: float = 100000) -> Dict:
        """
        定期再平衡
        
        Args:
            frequency: 再平衡频率 ('M', 'Q', '6M', 'Y')
            initial_capital: 初始资金
            
        Returns:
            回测结果
        """
        # 再平衡日期
        if frequency == 'M':
            rebalance_dates = self.returns.resample('M').last().index
        elif frequency == 'Q':
            rebalance_dates = self.returns.resample('Q').last().index
        elif frequency == '6M':
            rebalance_dates = self.returns.resample('2Q').last().index
        elif frequency == 'Y':
            rebalance_dates = self.returns.resample('Y').last().index
        else:
            raise ValueError(f"Unknown frequency: {frequency}")
        
        # 初始化
        capital = initial_capital
        positions = capital * self.target_weights.copy()
        cash = 0
        
        portfolio_values = [capital]
        rebalance_records = []
        
        for date, daily_return in self.returns.iterrows():
            # 更新持仓价值
            positions = positions * (1 + daily_return.values)
            total_value = positions.sum() + cash
            
            # 再平衡
            if date in rebalance_dates:
                target_positions = total_value * self.target_weights
                trades = target_positions - positions
                trade_value = np.abs(trades).sum()
                cost = trade_value * self.transaction_cost
                
                positions = target_positions
                cash = total_value - positions.sum() - cost
                total_value = positions.sum() + cash
                
                rebalance_records.append({
                    'date': date,
                    'portfolio_value': total_value,
                    'trade_value': trade_value,
                    'cost': cost,
                    'positions': positions.copy()
                })
            
            portfolio_values.append(total_value)
        
        # 构建结果
        portfolio_series = pd.Series(
            portfolio_values,
            index=[self.returns.index[0] - pd.Timedelta(days=1)] + self.returns.index.tolist()
        )
        
        returns = portfolio_series.pct_change().dropna()
        
        # 绩效计算
        annual_return = returns.mean() * 252
        annual_vol = returns.std() * np.sqrt(252)
        sharpe = annual_return / annual_vol if annual_vol > 0 else 0
        
        cummax = portfolio_series.cummax()
        drawdown = (portfolio_series - cummax) / cummax
        max_drawdown = drawdown.min()
        
        total_cost = sum([r['cost'] for r in rebalance_records])
        total_trades = len(rebalance_records)
        
        return {
            'portfolio_values': portfolio_series,
            'returns': returns,
            'annual_return': annual_return,
            'annual_volatility': annual_vol,
            'sharpe_ratio': sharpe,
            'max_drawdown': max_drawdown,
            'total_cost': total_cost,
            'total_trades': total_trades,
            'average_cost': total_cost / total_trades if total_trades > 0 else 0,
            'rebalance_records': rebalance_records
        }
    
    def compare_frequencies(self, 
                           frequencies: List[str] = ['M', 'Q', '6M', 'Y'],
                           initial_capital: float = 100000) -> pd.DataFrame:
        """
        对比不同再平衡频率
        
        Args:
            frequencies: 频率列表
            initial_capital: 初始资金
            
        Returns:
            对比结果
        """
        results = []
        
        for freq in frequencies:
            result = self.rebalance(freq, initial_capital)
            
            results.append({
                'Frequency': freq,
                'Annual Return': result['annual_return'],
                'Annual Volatility': result['annual_volatility'],
                'Sharpe Ratio': result['sharpe_ratio'],
                'Max Drawdown': result['max_drawdown'],
                'Total Cost': result['total_cost'],
                'Total Trades': result['total_trades'],
                'Cost per Trade': result['average_cost']
            })
        
        return pd.DataFrame(results)
```

### 2.2 阈值再平衡

**定义**：当权重偏离目标超过阈值时触发再平衡

**触发条件**：
- 单一资产偏离阈值：|w_i - w_i^*| > threshold
- 总体偏离阈值：Σ|w_i - w_i^*| > threshold
- 相对偏离阈值：|w_i - w_i^*| / w_i^* > threshold

**优势**：
- 响应及时
- 成本效率高
- 避免不必要交易

**局限**：
- 执行复杂
- 难以预测
- 可能在极端行情时失效

```python
class ThresholdRebalancer:
    """阈值再平衡策略"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 target_weights: np.ndarray,
                 transaction_cost: float = 0.001):
        """
        初始化
        
        Args:
            returns: 资产收益率DataFrame
            target_weights: 目标权重
            transaction_cost: 交易成本
        """
        self.returns = returns
        self.target_weights = target_weights
        self.n_assets = len(target_weights)
        self.asset_names = returns.columns.tolist()
        self.transaction_cost = transaction_cost
    
    def rebalance(self, 
                 threshold: float = 0.05,
                 method: str = 'absolute',
                 initial_capital: float = 100000) -> Dict:
        """
        阈值再平衡
        
        Args:
            threshold: 触发阈值
            method: 偏离度量方法
            initial_capital: 初始资金
            
        Returns:
            回测结果
        """
        capital = initial_capital
        positions = capital * self.target_weights.copy()
        cash = 0
        
        portfolio_values = [capital]
        rebalance_records = []
        
        for date, daily_return in self.returns.iterrows():
            # 更新持仓价值
            positions = positions * (1 + daily_return.values)
            total_value = positions.sum() + cash
            
            # 计算当前权重
            current_weights = positions / total_value
            
            # 检查是否触发再平衡
            trigger_rebalance = False
            
            if method == 'absolute':
                # 单一资产绝对偏离
                max_deviation = np.max(np.abs(current_weights - self.target_weights))
                trigger_rebalance = max_deviation > threshold
            
            elif method == 'total':
                # 总体偏离
                total_deviation = np.sum(np.abs(current_weights - self.target_weights))
                trigger_rebalance = total_deviation > threshold
            
            elif method == 'relative':
                # 相对偏离
                relative_deviation = np.abs(current_weights - self.target_weights) / self.target_weights
                max_relative_deviation = np.max(relative_deviation)
                trigger_rebalance = max_relative_deviation > threshold
            
            # 再平衡
            if trigger_rebalance:
                target_positions = total_value * self.target_weights
                trades = target_positions - positions
                trade_value = np.abs(trades).sum()
                cost = trade_value * self.transaction_cost
                
                positions = target_positions
                cash = total_value - positions.sum() - cost
                total_value = positions.sum() + cash
                
                rebalance_records.append({
                    'date': date,
                    'portfolio_value': total_value,
                    'current_weights': current_weights.copy(),
                    'target_weights': self.target_weights.copy(),
                    'trade_value': trade_value,
                    'cost': cost
                })
            
            portfolio_values.append(total_value)
        
        # 构建结果
        portfolio_series = pd.Series(
            portfolio_values,
            index=[self.returns.index[0] - pd.Timedelta(days=1)] + self.returns.index.tolist()
        )
        
        returns = portfolio_series.pct_change().dropna()
        
        # 绩效计算
        annual_return = returns.mean() * 252
        annual_vol = returns.std() * np.sqrt(252)
        sharpe = annual_return / annual_vol if annual_vol > 0 else 0
        
        cummax = portfolio_series.cummax()
        drawdown = (portfolio_series - cummax) / cummax
        max_drawdown = drawdown.min()
        
        total_cost = sum([r['cost'] for r in rebalance_records])
        total_trades = len(rebalance_records)
        
        return {
            'portfolio_values': portfolio_series,
            'returns': returns,
            'annual_return': annual_return,
            'annual_volatility': annual_vol,
            'sharpe_ratio': sharpe,
            'max_drawdown': max_drawdown,
            'total_cost': total_cost,
            'total_trades': total_trades,
            'average_cost': total_cost / total_trades if total_trades > 0 else 0,
            'rebalance_records': rebalance_records
        }
    
    def compare_thresholds(self, 
                          thresholds: List[float] = [0.01, 0.03, 0.05, 0.10],
                          method: str = 'absolute',
                          initial_capital: float = 100000) -> pd.DataFrame:
        """
        对比不同阈值
        
        Args:
            thresholds: 阈值列表
            method: 偏离度量方法
            initial_capital: 初始资金
            
        Returns:
            对比结果
        """
        results = []
        
        for threshold in thresholds:
            result = self.rebalance(threshold, method, initial_capital)
            
            results.append({
                'Threshold': threshold,
                'Annual Return': result['annual_return'],
                'Annual Volatility': result['annual_volatility'],
                'Sharpe Ratio': result['sharpe_ratio'],
                'Max Drawdown': result['max_drawdown'],
                'Total Cost': result['total_cost'],
                'Total Trades': result['total_trades'],
                'Average Trade Interval': len(self.returns) / result['total_trades'] if result['total_trades'] > 0 else np.inf
            })
        
        return pd.DataFrame(results)
```

### 2.3 混合再平衡策略

**定义**：结合定期和阈值触发

**策略逻辑**：
- 定期检查（如月末）
- 检查时若偏离超过阈值，则再平衡
- 否则跳过

**优势**：
- 兼顾规律性和成本效率
- 避免频繁检查
- 灵活性强

```python
class HybridRebalancer:
    """混合再平衡策略"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 target_weights: np.ndarray,
                 transaction_cost: float = 0.001):
        """
        初始化
        
        Args:
            returns: 资产收益率DataFrame
            target_weights: 目标权重
            transaction_cost: 交易成本
        """
        self.returns = returns
        self.target_weights = target_weights
        self.n_assets = len(target_weights)
        self.asset_names = returns.columns.tolist()
        self.transaction_cost = transaction_cost
    
    def rebalance(self, 
                 check_frequency: str = 'M',
                 threshold: float = 0.05,
                 initial_capital: float = 100000) -> Dict:
        """
        混合再平衡
        
        Args:
            check_frequency: 检查频率
            threshold: 触发阈值
            initial_capital: 初始资金
            
        Returns:
            回测结果
        """
        # 检查日期
        if check_frequency == 'M':
            check_dates = self.returns.resample('M').last().index
        elif check_frequency == 'Q':
            check_dates = self.returns.resample('Q').last().index
        elif check_frequency == 'W':
            check_dates = self.returns.resample('W').last().index
        else:
            raise ValueError(f"Unknown frequency: {check_frequency}")
        
        capital = initial_capital
        positions = capital * self.target_weights.copy()
        cash = 0
        
        portfolio_values = [capital]
        rebalance_records = []
        check_records = []
        
        for date, daily_return in self.returns.iterrows():
            # 更新持仓价值
            positions = positions * (1 + daily_return.values)
            total_value = positions.sum() + cash
            
            # 检查是否到检查日期
            if date in check_dates:
                # 计算当前权重
                current_weights = positions / total_value
                
                # 计算偏离
                deviation = np.max(np.abs(current_weights - self.target_weights))
                
                # 记录检查
                check_records.append({
                    'date': date,
                    'deviation': deviation,
                    'trigger_rebalance': deviation > threshold
                })
                
                # 判断是否触发再平衡
                if deviation > threshold:
                    target_positions = total_value * self.target_weights
                    trades = target_positions - positions
                    trade_value = np.abs(trades).sum()
                    cost = trade_value * self.transaction_cost
                    
                    positions = target_positions
                    cash = total_value - positions.sum() - cost
                    total_value = positions.sum() + cash
                    
                    rebalance_records.append({
                        'date': date,
                        'portfolio_value': total_value,
                        'deviation': deviation,
                        'trade_value': trade_value,
                        'cost': cost
                    })
            
            portfolio_values.append(total_value)
        
        # 构建结果
        portfolio_series = pd.Series(
            portfolio_values,
            index=[self.returns.index[0] - pd.Timedelta(days=1)] + self.returns.index.tolist()
        )
        
        returns = portfolio_series.pct_change().dropna()
        
        # 绩效计算
        annual_return = returns.mean() * 252
        annual_vol = returns.std() * np.sqrt(252)
        sharpe = annual_return / annual_vol if annual_vol > 0 else 0
        
        cummax = portfolio_series.cummax()
        drawdown = (portfolio_series - cummax) / cummax
        max_drawdown = drawdown.min()
        
        total_cost = sum([r['cost'] for r in rebalance_records])
        total_trades = len(rebalance_records)
        total_checks = len(check_records)
        
        return {
            'portfolio_values': portfolio_series,
            'returns': returns,
            'annual_return': annual_return,
            'annual_volatility': annual_vol,
            'sharpe_ratio': sharpe,
            'max_drawdown': max_drawdown,
            'total_cost': total_cost,
            'total_trades': total_trades,
            'total_checks': total_checks,
            'rebalance_ratio': total_trades / total_checks if total_checks > 0 else 0,
            'rebalance_records': rebalance_records,
            'check_records': check_records
        }
```

## 3. 再平衡成本分析

### 3.1 交易成本

**成本构成**：
- 显性成本：佣金、印花税、过户费
- 隐性成本：买卖价差、冲击成本、机会成本

```python
class TransactionCostAnalysis:
    """交易成本分析"""
    
    def __init__(self, 
                 commission_rate: float = 0.0003,
                 stamp_duty: float = 0.001,
                 bid_ask_spread: float = 0.0005,
                 market_impact_coeff: float = 0.1):
        """
        初始化
        
        Args:
            commission_rate: 佣金率
            stamp_duty: 印花税率
            bid_ask_spread: 买卖价差
            market_impact_coeff: 市场冲击系数
        """
        self.commission_rate = commission_rate
        self.stamp_duty = stamp_duty
        self.bid_ask_spread = bid_ask_spread
        self.market_impact_coeff = market_impact_coeff
    
    def calculate_total_cost(self, 
                            trade_value: float,
                            avg_daily_volume: float = 1000000) -> Dict:
        """
        计算总交易成本
        
        Args:
            trade_value: 交易金额
            avg_daily_volume: 平均日成交量
            
        Returns:
            成本明细
        """
        # 显性成本
        commission = trade_value * self.commission_rate
        stamp_duty = trade_value * self.stamp_duty
        explicit_cost = commission + stamp_duty
        
        # 隐性成本
        bid_ask_cost = trade_value * self.bid_ask_spread
        
        # 市场冲击成本（简化模型）
        participation_rate = trade_value / avg_daily_volume
        market_impact = trade_value * self.market_impact_coeff * participation_rate
        
        implicit_cost = bid_ask_cost + market_impact
        
        total_cost = explicit_cost + implicit_cost
        cost_rate = total_cost / trade_value
        
        return {
            'trade_value': trade_value,
            'commission': commission,
            'stamp_duty': stamp_duty,
            'explicit_cost': explicit_cost,
            'bid_ask_cost': bid_ask_cost,
            'market_impact': market_impact,
            'implicit_cost': implicit_cost,
            'total_cost': total_cost,
            'cost_rate': cost_rate
        }
```

class OptimalRebalanceFrequency:
    """最优再平衡频率分析"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 target_weights: np.ndarray):
        """
        初始化
        
        Args:
            returns: 收益率数据
            target_weights: 目标权重
        """
        self.returns = returns
        self.target_weights = target_weights
        self.n_assets = len(target_weights)
    
    def analyze_optimal_frequency(self, 
                                 frequencies: List[str] = ['M', 'Q', '6M', 'Y'],
                                 transaction_cost: float = 0.001,
                                 initial_capital: float = 100000) -> pd.DataFrame:
        """
        分析最优再平衡频率
        
        Args:
            frequencies: 频率列表
            transaction_cost: 交易成本
            initial_capital: 初始资金
            
        Returns:
            频率分析结果
        """
        results = []
        
        for freq in frequencies:
            rebalancer = PeriodicRebalancer(
                self.returns, self.target_weights, transaction_cost
            )
            
            result = rebalancer.rebalance(freq, initial_capital)
            
            # 跟踪误差
            target_returns = (self.returns * self.target_weights).sum(axis=1)
            actual_returns = result['returns']
            tracking_error = (actual_returns - target_returns).std() * np.sqrt(252)
            
            results.append({
                'Frequency': freq,
                'Annual Return': result['annual_return'],
                'Annual Volatility': result['annual_volatility'],
                'Sharpe Ratio': result['sharpe_ratio'],
                'Max Drawdown': result['max_drawdown'],
                'Total Cost': result['total_cost'],
                'Cost Rate': result['total_cost'] / initial_capital,
                'Total Trades': result['total_trades'],
                'Tracking Error': tracking_error,
                'Net Return': result['annual_return'] - result['total_cost'] / initial_capital
            })
        
        return pd.DataFrame(results)

## 4. 实战案例

### 4.1 ETF组合再平衡

```python
class ETFPortfolioRebalancer:
    """ETF组合再平衡实战"""
    
    def __init__(self, 
                 etf_returns: pd.DataFrame,
                 target_weights: Dict[str, float],
                 transaction_cost: float = 0.001):
        """
        初始化
        
        Args:
            etf_returns: ETF收益率数据
            target_weights: 目标权重字典
            transaction_cost: 交易成本
        """
        self.returns = etf_returns
        self.target_weights = np.array([target_weights[col] for col in etf_returns.columns])
        self.asset_names = etf_returns.columns.tolist()
        self.transaction_cost = transaction_cost
    
    def full_backtest(self, 
                     initial_capital: float = 100000) -> pd.DataFrame:
        """
        完整回测对比
        
        Args:
            initial_capital: 初始资金
            
        Returns:
            对比结果
        """
        # 1. 定期再平衡
        periodic = PeriodicRebalancer(
            self.returns, self.target_weights, self.transaction_cost
        )
        
        periodic_results = []
        for freq in ['M', 'Q', '6M', 'Y']:
            result = periodic.rebalance(freq, initial_capital)
            periodic_results.append({
                'Strategy': f'Periodic_{freq}',
                'Type': 'Periodic',
                'Annual Return': result['annual_return'],
                'Sharpe Ratio': result['sharpe_ratio'],
                'Max Drawdown': result['max_drawdown'],
                'Total Cost': result['total_cost'],
                'Total Trades': result['total_trades']
            })
        
        # 2. 阈值再平衡
        threshold = ThresholdRebalancer(
            self.returns, self.target_weights, self.transaction_cost
        )
        
        threshold_results = []
        for th in [0.01, 0.03, 0.05, 0.10]:
            result = threshold.rebalance(th, 'absolute', initial_capital)
            threshold_results.append({
                'Strategy': f'Threshold_{th}',
                'Type': 'Threshold',
                'Annual Return': result['annual_return'],
                'Sharpe Ratio': result['sharpe_ratio'],
                'Max Drawdown': result['max_drawdown'],
                'Total Cost': result['total_cost'],
                'Total Trades': result['total_trades']
            })
        
        # 3. 混合再平衡
        hybrid = HybridRebalancer(
            self.returns, self.target_weights, self.transaction_cost
        )
        
        hybrid_results = []
        for freq in ['M', 'Q']:
            for th in [0.03, 0.05]:
                result = hybrid.rebalance(freq, th, initial_capital)
                hybrid_results.append({
                    'Strategy': f'Hybrid_{freq}_{th}',
                    'Type': 'Hybrid',
                    'Annual Return': result['annual_return'],
                    'Sharpe Ratio': result['sharpe_ratio'],
                    'Max Drawdown': result['max_drawdown'],
                    'Total Cost': result['total_cost'],
                    'Total Trades': result['total_trades']
                })
        
        # 合并结果
        all_results = periodic_results + threshold_results + hybrid_results
        
        return pd.DataFrame(all_results)
```

## 5. 核心结论

### 5.1 再平衡策略选择

| 策略类型 | 适用场景 | 优势 | 劣势 |
|---------|---------|------|------|
| 定期再平衡 | 小型组合、初学者 | 简单、规律性强 | 可能错过机会、成本较高 |
| 阈值再平衡 | 大型组合、成本敏感 | 成本效率高、响应及时 | 执行复杂、难以预测 |
| 混合再平衡 | 中型组合、平衡需求 | 兼顾成本与效率 | 参数需调优 |

### 5.2 最优频率建议

**基于成本收益分析**：
- 小型组合（<100万）：季度或半年
- 中型组合（100-1000万）：月度检查+阈值触发
- 大型组合（>1000万）：阈值再平衡+分批交易

### 5.3 实践要点

1. **成本控制**：控制换手率，减少不必要的交易
2. **税务规划**：考虑持有期，避免短期资本利得税
3. **风险监控**：持续监控偏离度和风险水平
4. **灵活调整**：根据市场环境调整再平衡策略
