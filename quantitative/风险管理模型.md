# 风险管理模型

## 一、风险的概念与分类

### 1.1 风险定义
风险是指未来结果的不确定性，特别是指负面结果发生的可能性。在金融领域，风险通常指投资收益的不确定性。

### 1.2 风险分类

| 风险类型 | 定义 | 来源 |
|---------|------|------|
| 市场风险 | 价格波动风险 | 股价、利率、汇率、商品价格 |
| 信用风险 | 违约风险 | 债券、贷款、衍生品 |
| 流动性风险 | 变现困难风险 | 市场深度、交易量 |
| 操作风险 | 人为失误风险 | 系统、流程、人员 |
| 法律风险 | 法律纠纷风险 | 合同、法规变化 |
| 模型风险 | 模型失效风险 | 模型假设、参数估计 |

## 二、风险度量方法

### 2.1 波动率 (Volatility)

#### 历史波动率
```
σ = √(Σ(Ri - R̄)² / (n-1))

其中:
Ri = 第i期收益率
R̄ = 平均收益率
n = 观察期数
```

#### 隐含波动率
从期权价格反推的波动率，反映市场对未来波动的预期。

```python
import numpy as np

def calculate_volatility(returns, annualize=True):
    """
    计算波动率
    """
    vol = np.std(returns, ddof=1)
    if annualize:
        vol = vol * np.sqrt(252)  # 年化
    return vol
```

### 2.2 VaR (Value at Risk)

#### 定义
在给定置信水平和持有期内，投资组合可能遭受的最大损失。

#### 计算方法

**历史模拟法**
```python
def var_historical(returns, confidence=0.95):
    """
    历史模拟法计算VaR
    """
    return -np.percentile(returns, (1 - confidence) * 100)
```

**参数法（正态分布假设）**
```python
from scipy import stats

def var_parametric(returns, confidence=0.95):
    """
    参数法计算VaR
    """
    mu = returns.mean()
    sigma = returns.std()
    z = stats.norm.ppf(1 - confidence)
    return -(mu + z * sigma)
```

**蒙特卡洛模拟**
```python
def var_monte_carlo(returns, confidence=0.95, simulations=10000):
    """
    蒙特卡洛模拟计算VaR
    """
    mu = returns.mean()
    sigma = returns.std()
    
    simulated = np.random.normal(mu, sigma, simulations)
    return -np.percentile(simulated, (1 - confidence) * 100)
```

### 2.3 CVaR (Conditional VaR)

也称为Expected Shortfall (ES)，是VaR尾部损失的平均值。

```python
def cvar(returns, confidence=0.95):
    """
    计算CVaR
    """
    var = -np.percentile(returns, (1 - confidence) * 100)
    return -returns[returns < -var].mean()
```

### 2.4 最大回撤 (Maximum Drawdown)

```python
def max_drawdown(portfolio_value):
    """
    计算最大回撤
    """
    peak = portfolio_value.expanding(min_periods=1).max()
    drawdown = (portfolio_value - peak) / peak
    return drawdown.min()
```

## 三、风险模型

### 3.1 CAPM模型

```
E(Ri) = Rf + βi × (E(Rm) - Rf)

其中:
E(Ri) = 资产i的期望收益
Rf = 无风险利率
βi = 资产i的贝塔系数
E(Rm) = 市场组合期望收益
```

```python
import statsmodels.api as sm

def calculate_beta(asset_returns, market_returns):
    """
    计算Beta
    """
    X = sm.add_constant(market_returns)
    model = sm.OLS(asset_returns, X).fit()
    return model.params[1]
```

### 3.2 风险因子模型

#### Fama-French三因子模型
```
E(Ri) = Rf + βmkt × MKT + βsmb × SMB + βhml × HML

其中:
MKT = 市场因子
SMB = 规模因子 (Small Minus Big)
HML = 价值因子 (High Minus Low)
```

#### Carhart四因子模型
```
E(Ri) = Rf + βmkt × MKT + βsmb × SMB + βhml × HML + βmom × MOM

其中:
MOM = 动量因子
```

### 3.3 风险平价模型

风险平价策略使各资产的风险贡献相等。

```python
def risk_parity_weights(cov_matrix):
    """
    风险平价权重计算
    """
    n = len(cov_matrix)
    
    def risk_contribution(w, cov):
        portfolio_vol = np.sqrt(w @ cov @ w)
        marginal_contrib = cov @ w
        risk_contrib = w * marginal_contrib / portfolio_vol
        return risk_contrib
    
    def objective(w):
        rc = risk_contribution(w, cov_matrix)
        target_risk = 1 / n
        return np.sum((rc - target_risk) ** 2)
    
    from scipy.optimize import minimize
    
    init_weights = np.ones(n) / n
    bounds = [(0, 1) for _ in range(n)]
    constraints = {'type': 'eq', 'fun': lambda w: np.sum(w) - 1}
    
    result = minimize(objective, init_weights, method='SLSQP',
                      bounds=bounds, constraints=constraints)
    
    return result.x
```

#### ATR止损
```python
def atr_stop_loss(high, low, close, period=14, multiplier=2):
    """
    ATR动态止损
    """
    tr = np.maximum(high - low, 
                    np.maximum(abs(high - close.shift(1)),
                               abs(low - close.shift(1))))
    atr = tr.rolling(period).mean()
    return close - multiplier * atr
```

#### 移动止损
```python
def trailing_stop(close, atr, multiplier=3):
    """
    移动止损（跟踪止损）
    """
    stop = close.copy()
    for i in range(1, len(close)):
        stop[i] = max(stop[i-1], close[i] - multiplier * atr[i])
    return stop
```

#### 投资组合优化
```python
from scipy.optimize import minimize

def portfolio_optimization(returns, target_return=None):
    """
    马科维茨投资组合优化
    """
    n = len(returns.columns)
    mu = returns.mean()
    cov = returns.cov()
    
    def portfolio_volatility(w):
        return np.sqrt(w @ cov @ w)
    
    def portfolio_return(w):
        return w @ mu
    
    constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
    if target_return:
        constraints.append({'type': 'eq', 
                           'fun': lambda w: portfolio_return(w) - target_return})
    
    bounds = [(0, 1) for _ in range(n)]
    init_weights = np.ones(n) / n
    
    result = minimize(portfolio_volatility, init_weights,
                      method='SLSQP', bounds=bounds,
                      constraints=constraints)
    
    return result.x
```

### 5.3 蒙特卡洛压力测试

```python
def monte_carlo_stress_test(portfolio, n_simulations=10000, 
                           confidence_level=0.95):
    """
    蒙特卡洛压力测试
    """
    # 生成随机冲击
    shocks = np.random.normal(0, 0.02, (n_simulations, len(portfolio)))
    
    # 应用冲击
    portfolio_values = portfolio * (1 + shocks)
    
    # 计算损失分布
    losses = (portfolio.sum() - portfolio_values.sum(axis=1)) / portfolio.sum()
    
    # 计算风险指标
    var = np.percentile(losses, (1 - confidence_level) * 100)
    cvar = losses[losses > var].mean()
    
    return {
        'VaR': var,
        'CVaR': cvar,
        'max_loss': losses.max(),
        'loss_distribution': losses
    }
```

  - RLOP在大多数切片中降低shortfall频率

**实践启示**：
- 摩擦感知RL框架支持自主衍生品风险管理

```python
class OverfittingPrevention:
    """过拟合防范"""
    
    def __init__(self):
        self.train_score_history = []
        self.test_score_history = []
    
    def detect_overfitting(self, train_score, test_score, threshold=0.1):
        """检测过拟合"""
        self.train_score_history.append(train_score)
        self.test_score_history.append(test_score)
        
        # 训练集与测试集差距过大
        if train_score - test_score > threshold:
            return True, "过拟合警告：训练集得分远高于测试集"
        
        # 测试集得分下降
        if len(self.test_score_history) > 5:
            recent = self.test_score_history[-5:]
            if all(recent[i] > recent[i+1] for i in range(len(recent)-1)):
                return True, "过拟合警告：测试集得分持续下降"
        
        return False, "正常"
```

```python
class DynamicTrackingError:
    """动态跟踪误差组合管理"""
    
    def __init__(self, risk_aversion, base_tracking_limit):
        self.risk_aversion = risk_aversion
        self.base_limit = base_tracking_limit
    
    def adjust_tracking_limit(self, market_volatility, crisis_mode):
        """动态调整跟踪误差限制"""
        if crisis_mode:
            # 危机期约束成本最高，前瞻收益最丰富
            return self.base_limit * 1.5
        else:
            # 正常期保持标准限制
            return self.base_limit * (1 + 0.5 * market_volatility / 0.2)
    
    def calculate_position(self, target_weights, current_weights, tracking_limit):
        """计算仓位调整"""
        tracking_error = np.std(target_weights - current_weights)
        if tracking_error > tracking_limit:
            # 缩减调整幅度
            scale = tracking_limit / tracking_error
            return current_weights + scale * (target_weights - current_weights)
        return target_weights
```

```python
class RealTimeRiskMonitor:
    """实时风控监控"""
    
    def __init__(self, limits):
        self.limits = limits
        self.alerts = []
    
    def check_position_limit(self, positions):
        """检查仓位限制"""
        violations = []
        
        # 单一资产限制
        for asset, position in positions.items():
            if position['weight'] > self.limits.max_position:
                violations.append({
                    'type': 'position_limit',
                    'asset': asset,
                    'current': position['weight'],
                    'limit': self.limits.max_position
                })
        
        # 行业限制
        sector_weights = self._aggregate_by_sector(positions)
        for sector, weight in sector_weights.items():
            if weight > self.limits.max_sector:
                violations.append({
                    'type': 'sector_limit',
                    'sector': sector,
                    'current': weight,
                    'limit': self.limits.max_sector
                })
        
        return violations
    
    def check_var_limit(self, portfolio_value, var):
        """检查VaR限制"""
        if var > self.limits.max_var:
            return {
                'type': 'var_limit',
                'current': var,
                'limit': self.limits.max_var,
                'timestamp': datetime.now()
            }
        return None
    
    def check_drawdown_limit(self, portfolio_value):
        """检查回撤限制"""
        peak = portfolio_value.expanding().max()
        drawdown = (portfolio_value - peak) / peak
        
        if drawdown.min() < -self.limits.max_drawdown:
            return {
                'type': 'drawdown_limit',
                'current': drawdown.min(),
                'limit': -self.limits.max_drawdown,
                'timestamp': datetime.now()
            }
        return None
```

```python
class StressTestingSystem:
    """压力测试系统"""
    
    def __init__(self, portfolio):
        self.portfolio = portfolio
    
    def historical_scenarios(self):
        """历史情景压力测试"""
        scenarios = {
            '2008金融危机': {
                'date': '2008-09-15',
                'stocks': -0.40,
                'bonds': 0.05,
                'commodities': -0.30
            },
            '2020新冠疫情': {
                'date': '2020-03-09',
                'stocks': -0.35,
                'bonds': 0.10,
                'commodities': -0.25
            },
            '2026中东冲突': {
                'date': '2026-03-09',
                'stocks': -0.15,
                'bonds': -0.05,
                'commodities': 0.20,
                'oil': 0.30
            }
        }
        
        results = {}
        for name, shocks in scenarios.items():
            impact = self._apply_shocks(shocks)
            results[name] = {
                'portfolio_loss': impact['total_loss'],
                'worst_asset': impact['worst_asset'],
                'recovery_days': impact['recovery_estimate']
            }
        
        return results
    
    def hypothetical_scenarios(self):
        """假设情景压力测试"""
        scenarios = {
            '股市暴跌': {'stocks': -0.30},
            '利率上升': {'bonds': -0.10, 'stocks': -0.15},
            '通胀高企': {'bonds': -0.05, 'commodities': 0.20},
            '地缘冲突': {'stocks': -0.20, 'oil': 0.40, 'gold': 0.15},
            '流动性危机': {'stocks': -0.25, 'bonds': -0.10}
        }
        
        return self._run_scenarios(scenarios)
    
    def monte_carlo_stress(self, n_simulations=10000):
        """蒙特卡洛压力测试"""
        # 生成随机冲击
        shocks = np.random.normal(0, 0.02, (n_simulations, len(self.portfolio)))
        
        # 应用冲击
        portfolio_values = self.portfolio['values'] * (1 + shocks)
        
        # 计算损失分布
        losses = (self.portfolio['values'].sum() - portfolio_values.sum(axis=1)) / self.portfolio['values'].sum()
        
        # 计算风险指标
        var_95 = np.percentile(losses, 95)
        cvar_95 = losses[losses > var_95].mean()
        
        return {
            'VaR_95': var_95,
            'CVaR_95': cvar_95,
            'max_loss': losses.max(),
            'loss_distribution': losses
        }
```

## 十一、风险管理案例分析

### 11.1 2008金融危机案例

**风险事件**：
- 次贷危机爆发
- 雷曼兄弟破产
- 市场流动性枯竭

**教训**：
1. **相关性失效**：传统风险模型假设的相关性在危机中失效
2. **流动性风险**：无法及时变现资产
3. **尾部风险低估**：VaR低估了极端损失

**改进措施**：
- 引入CVaR度量尾部风险
- 增加流动性风险监控
- 压力测试成为必备

**风险事件**：
- 全球股市熔断
- 原油期货负价格
- 黄金价格暴涨

**教训**：
1. **黑天鹅事件**：传统模型无法预测
2. **跨市场传染**：风险快速传播
3. **政策干预**：央行救市改变市场预期

**改进措施**：
- 增加极端情景压力测试
- 动态调整风险限额
