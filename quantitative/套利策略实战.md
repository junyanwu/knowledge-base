# 套利策略实战

## 1. 统计套利

### 1.1 配对交易原理

**协整关系：**
两只股票价格存在长期均衡关系，短期偏离会回归。

**策略：**
```python
def pairs_trading(y1, y2, window=60, entry_z=2, exit_z=0.5):
    """
    配对交易策略
    
    参数:
    y1, y2: 两只股票价格
    window: 回归窗口
    entry_z: 开仓阈值
    exit_z: 平仓阈值
    
    返回:
    交易信号
    """
    # 协整回归
    spread = y1 - np.polyfit(y2, y1, 1)[0] * y2
    
    # 标准化
    spread_mean = pd.Series(spread).rolling(window).mean()
    spread_std = pd.Series(spread).rolling(window).std()
    z_score = (spread - spread_mean) / spread_std
    
    # 交易信号
    signal = pd.Series(0, index=y1.index)
    
    # 开仓
    signal[z_score > entry_z] = -1  # 做空y1，做多y2
    signal[z_score < -entry_z] = 1  # 做多y1，做空y2
    
    # 平仓
    signal[abs(z_score) < exit_z] = 0
    
    return signal, z_score
```

### 1.2 多因子套利

**因子对冲：**
```python
def factor_arbitrage(returns, factor_exposure, target_exposure):
    """
    因子套利
    
    参数:
    returns: 股票收益率
    factor_exposure: 因子暴露
    target_exposure: 目标因子暴露
    
    返回:
    对冲组合权重
    """
    from scipy.optimize import minimize
    
    n_assets = returns.shape[1]
    
    def objective(weights):
        # 最小化因子暴露差异
        exposure_diff = weights @ factor_exposure - target_exposure
        return np.sum(exposure_diff ** 2)
    
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},  # 权重和为1
        {'type': 'eq', 'fun': lambda w: w @ returns.mean()}  # 目标收益
    ]
    
    bounds = [(-1, 1) for _ in range(n_assets)]  # 允许做空
    
    result = minimize(objective, 
                     x0=[1/n_assets] * n_assets,
                     bounds=bounds,
                     constraints=constraints)
    
    return result.x
```

## 2. ETF套利

### 2.1 ETF折溢价套利

**原理：**
ETF市场价格与净值存在偏差时套利。

**策略：**
```python
def etf_arbitrage(etf_price, nav, threshold=0.5, cost=0.3):
    """
    ETF折溢价套利
    
    参数:
    etf_price: ETF市场价格
    nav: ETF净值
    threshold: 折溢价阈值（%）
    cost: 交易成本（%）
    
    返回:
    套利信号
    """
    # 折溢价率
    premium = (etf_price - nav) / nav * 100
    
    # 套利信号
    signal = pd.Series(0, index=etf_price.index)
    
    # 溢价套利：买入一篮子股票，赎回ETF份额
    signal[premium > threshold + cost] = -1
    
    # 折价套利：买入ETF份额，赎回一篮子股票
    signal[premium < -threshold - cost] = 1
    
    return signal, premium
```

### 2.2 跨市场套利

**AH股套利：**
```python
def ah_arbitrage(a_price, h_price, fx_rate, threshold=2):
    """
    AH股套利
    
    参数:
    a_price: A股价格
    h_price: H股价格（港币）
    fx_rate: 汇率
    threshold: 价差阈值（%）
    
    返回:
    套利信号
    """
    # 转换为同币种
    h_price_cny = h_price * fx_rate
    
    # 价差
    spread = (a_price - h_price_cny) / h_price_cny * 100
    
    # 套利信号
    signal = pd.Series(0, index=a_price.index)
    
    # A股高估：做空A股，做多H股
    signal[spread > threshold] = -1
    
    # A股低估：做多A股，做空H股
    signal[spread < -threshold] = 1
    
    return signal, spread
```

## 3. 期现套利

### 3.1 股指期货套利

**原理：**
期货价格与现货价格存在基差，利用基差收敛套利。

**策略：**
```python
def futures_spot_arbitrage(futures_price, spot_price, rf, dividend_yield, time_to_maturity, threshold=0.5):
    """
    期现套利
    
    参数:
    futures_price: 期货价格
    spot_price: 现货价格
    rf: 无风险利率
    dividend_yield: 股息率
    time_to_maturity: 到期时间（年）
    threshold: 套利阈值（%）
    
    返回:
    套利信号
    """
    # 理论期货价格
    theoretical = spot_price * np.exp((rf - dividend_yield) * time_to_maturity)
    
    # 基差
    basis = (futures_price - theoretical) / spot_price * 100
    
    # 套利信号
    signal = pd.Series(0, index=futures_price.index)
    
    # 期货高估：卖期货，买现货
    signal[basis > threshold] = -1
    
    # 期货低估：买期货，卖现货（需要融券）
    signal[basis < -threshold] = 1
    
    return signal, basis
```
