# 多因子模型理论 - 模块2：因子构建与检验

## 1. 因子构建方法

### 1.1 因子定义与选择

**因子分类：**

| 类型 | 代表因子 | 数据来源 |
|------|---------|---------|
| 价值因子 | B/P, E/P, D/P | 财务数据 |
| 成长因子 | 营收增长率, EPS增长率 | 财务数据 |
| 质量因子 | ROE, ROA, 毛利率 | 财务数据 |
| 动量因子 | 过去收益, 相对强度 | 行情数据 |
| 波动因子 | 波动率, 特质波动 | 行情数据 |
| 规模因子 | 市值, 流通市值 | 行情数据 |
| 流动性因子 | 换手率, Amihud非流动性 | 行情数据 |
| 情绪因子 | 分析师预期, 舆情 | 另类数据 |

**因子选择原则：**
1. **经济学逻辑**：因子应有合理的经济学解释
2. **显著性**：因子收益在统计上显著
3. **持续性**：因子效果在不同时期稳定
4. **低相关性**：因子间相关性不宜过高
5. **可投资性**：因子可实际交易

### 1.2 因子数据处理

**数据清洗：**

1. **缺失值处理**
   ```python
   # 删除缺失值
   df = df.dropna()
   
   # 填充（均值/中位数/前值）
   df = df.fillna(df.mean())
   
   # 插值
   df = df.interpolate()
   ```

2. **异常值处理**
   ```python
   # 3σ原则
   df = df[(df - df.mean()).abs() <= 3 * df.std()]
   
   # 分位数法
   lower = df.quantile(0.01)
   upper = df.quantile(0.99)
   df = df.clip(lower, upper)
   
   # MAD方法
   median = df.median()
   mad = (df - median).abs().median()
   df = df[(df - median).abs() <= 3 * mad]
   ```

3. **数据对齐**
   ```python
   # 财务数据与行情数据对齐
   df = pd.merge(price_data, financial_data, 
                  left_index=True, right_index=True, how='left')
   
   # 前向填充财务数据
   df = df.fillna(method='ffill')
   ```

### 1.4 因子正交化

**目的：** 消除因子间相关性，避免多重共线性

**方法：**

1. **施密特正交化**
   ```python
   def orthogonalize(factors):
       orthogonal = []
       for i, factor in enumerate(factors):
           ortho_factor = factor.copy()
           for prev_factor in orthogonal:
               ortho_factor -= np.dot(factor, prev_factor) / np.dot(prev_factor, prev_factor) * prev_factor
           orthogonal.append(ortho_factor)
       return orthogonal
   ```

2. **回归正交化**
   ```python
   # 因子A对因子B回归
   model = sm.OLS(factor_A, sm.add_constant(factor_B)).fit()
   factor_A_ortho = model.resid
   ```

3. **主成分正交化**
   ```python
   from sklearn.decomposition import PCA
   
   pca = PCA(n_components=len(factors))
   factors_pca = pca.fit_transform(factors)
   ```

## 2. 因子检验框架

### 2.1 IC值分析

**IC定义（信息系数）：**
```
IC = corr(factor_t, return_{t+1})
```

**IC计算方法：**

1. **普通IC**
   ```python
   from scipy.stats import spearmanr, pearsonr
   
   # Pearson IC
   ic_pearson = pearsonr(factor, forward_return)[0]
   
   # Spearman IC（更常用）
   ic_spearman = spearmanr(factor, forward_return)[0]
   ```

2. **Rank IC**
   ```python
   factor_rank = factor.rank()
   return_rank = forward_return.rank()
   ic_rank = factor_rank.corr(return_rank)
   ```

**IC统计量：**

| 指标 | 公式 | 判断标准 |
|------|------|---------|
| IC均值 | IC = mean(ICₜ) | |IC| > 0.02 较好 |
| IC标准差 | σ = std(ICₜ) | σ < 0.1 较稳定 |
| ICIR | ICIR = IC / σ | ICIR > 0.5 较好 |
| IC胜率 | IC > 0的比例 | > 50% 较好 |

**IC时序分析：**

```python
    # 年化收益
    annual_return = returns.mean() * 252
    
    # 年化波动
    annual_vol = returns.std() * np.sqrt(252)
    
    # 夏普比率
    sharpe = annual_return / annual_vol
    
    # 最大回撤
    cum = (1 + returns).cumprod()
    drawdown = (cum - cum.cummax()) / cum.cummax()
    max_dd = drawdown.min()
    
    # 胜率
    win_rate = (returns > 0).mean()
    
    # 盈亏比
    profit_loss = returns[returns > 0].mean() / abs(returns[returns < 0].mean())
    
    return {
        '年化收益': f'{annual_return:.2%}',
        '年化波动': f'{annual_vol:.2%}',
        '夏普比率': f'{sharpe:.2f}',
        '最大回撤': f'{max_dd:.2%}',
```

### 3.1 因子合成

**等权合成：**

**IC加权：**
```python

ic1 = factor1.groupby('date').apply(lambda x: spearmanr(x['factor'], x['return'])[0]).mean()
ic2 = factor2.groupby('date').apply(lambda x: spearmanr(x['factor'], x['return'])[0]).mean()
ic3 = factor3.groupby('date').apply(lambda x: spearmanr(x['factor'], x['return'])[0]).mean()

# IC加权
weights = [abs(ic1), abs(ic2), abs(ic3)]
weights = np.array(weights) / sum(weights)

combined_factor = factor1 * weights[0] + factor2 * weights[1] + factor3 * weights[2]
```

**ICIR加权：**
```python
def calc_icir(factor, returns):
    ic_series = factor.groupby('date').apply(
        lambda x: spearmanr(x['factor'], x['return'])[0]
    )
    return ic_series.mean() / ic_series.std()

icir1 = calc_icir(factor1, returns)
icir2 = calc_icir(factor2, returns)
icir3 = calc_icir(factor3, returns)

weights = [abs(icir1), abs(icir2), abs(icir3)]
weights = np.array(weights) / sum(weights)

combined_factor = factor1 * weights[0] + factor2 * weights[1] + factor3 * weights[2]
```

**最大化ICIR：**
```python
from scipy.optimize import minimize

def objective(weights, factors, returns):
    combined = sum(w * f for w, f in zip(weights, factors))
    ic_series = combined.groupby('date').apply(
        lambda x: spearmanr(x, returns.loc[x.index])[0]
    )
    return -ic_series.mean() / ic_series.std()  # 最大化ICIR

# 约束条件
constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
bounds = [(0, 1) for _ in range(len(factors))]

# 优化
result = minimize(objective, x0=[1/len(factors)]*len(factors),
                  args=(factors, returns),
                  bounds=bounds,
                  constraints=constraints)

optimal_weights = result.x
```

### 3.2 权重分配

**风险平价：**
```python
def risk_parity_weights(factor_returns):
    # 计算协方差矩阵
    cov = factor_returns.cov()
    
    # 风险平价优化
    def risk_budget(weights):
        portfolio_vol = np.sqrt(weights @ cov @ weights)
        marginal_risk = cov @ weights / portfolio_vol
        risk_contrib = weights * marginal_risk
        target_risk = portfolio_vol / len(weights)
        return sum((risk_contrib - target_risk)**2)
    
    constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
    bounds = [(0, 1) for _ in range(len(factor_returns.columns))]
    
    result = minimize(risk_budget, x0=[1/len(factor_returns.columns)]*len(factor_returns.columns),
                      bounds=bounds, constraints=constraints)
    
    return result.x
```

**均值方差优化：**
```python
def mean_variance_weights(factor_returns, target_return=None):
    # 期望收益
    mu = factor_returns.mean()
    
    # 协方差矩阵
    cov = factor_returns.cov()
    
    # 目标函数：最小化风险
    def portfolio_vol(weights):
        return np.sqrt(weights @ cov @ weights)
    
    # 约束
    constraints = [{'type': 'eq', 'fun': lambda w: sum(w) - 1}]
    if target_return:
        constraints.append({'type': 'eq', 'fun': lambda w: w @ mu - target_return})
    
    bounds = [(0, 1) for _ in range(len(mu))]
    
    result = minimize(portfolio_vol, x0=[1/len(mu)]*len(mu),
                      bounds=bounds, constraints=constraints)
    
    return result.x
```

### 4.1 A股因子构建

```python
import pandas as pd
import numpy as np

# 价值因子
def calc_value_factor(df):
    df['BP'] = df['book_value'] / df['market_cap']  # 账面市值比
    df['EP'] = df['earnings'] / df['market_cap']    # 盈利市值比
    df['DP'] = df['dividend'] / df['market_cap']    # 股息率
    df['CFP'] = df['cash_flow'] / df['market_cap']  # 现金流市值比
    return df

# 成长因子
def calc_growth_factor(df):
    df['revenue_growth'] = df['revenue'].pct_change(4)  # 营收增长率
    df['eps_growth'] = df['eps'].pct_change(4)          # EPS增长率
    df['roe_growth'] = df['roe'].diff(4)               # ROE变化
    return df

# 质量因子
def calc_quality_factor(df):
    df['ROE'] = df['net_income'] / df['equity']        # ROE
    df['ROA'] = df['net_income'] / df['assets']        # ROA
    df['gross_margin'] = df['gross_profit'] / df['revenue']  # 毛利率
    df['operating_margin'] = df['operating_profit'] / df['revenue']  # 营业利润率
    return df

# 动量因子
def calc_momentum_factor(df):
    df['momentum_12m'] = df['close'].pct_change(252)   # 12个月动量
    df['momentum_6m'] = df['close'].pct_change(126)    # 6个月动量
    df['momentum_1m'] = df['close'].pct_change(21)     # 1个月动量
    df['reversal_1m'] = -df['momentum_1m']             # 1个月反转
    return df

# 波动因子
def calc_volatility_factor(df):
    df['volatility_20d'] = df['return'].rolling(20).std()  # 20日波动率
    df['volatility_60d'] = df['return'].rolling(60).std()  # 60日波动率
    return df

# 流动性因子
def calc_liquidity_factor(df):
    df['turnover'] = df['volume'] / df['shares_outstanding']  # 换手率
    df['amihud'] = abs(df['return']) / (df['volume'] * df['close'])  # Amihud非流动性
    return df
```

### 4.3 风险因子模型

**Fama-French五因子：**

```python
def calc_ff5_factors(df):
    # 市场因子
    df['MKT'] = df['market_return'] - df['risk_free_rate']
    
    # 规模因子（SMB）
    df['size_group'] = df.groupby('date')['market_cap'].transform(
        lambda x: pd.qcut(x, 2, labels=['S', 'B'])
    )
    
    # 价值因子（HML）
    df['bp_group'] = df.groupby('date')['BP'].transform(
        lambda x: pd.qcut(x, 3, labels=['L', 'M', 'H'])
    )
    
    # 盈利因子（RMW）
    df['profit_group'] = df.groupby('date')['operating_profit'].transform(
        lambda x: pd.qcut(x, 3, labels=['W', 'M', 'R'])
    )
    
    # 投资因子（CMA）
    df['invest_group'] = df.groupby('date')['asset_growth'].transform(
        lambda x: pd.qcut(x, 3, labels=['A', 'M', 'C'])
    )
    
    # 计算因子收益
    # SMB = 小市值收益 - 大市值收益
    SMB = df[df['size_group'] == 'S']['return'].mean() - df[df['size_group'] == 'B']['return'].mean()
    
    # HML = 高BP收益 - 低BP收益
    HML = df[df['bp_group'] == 'H']['return'].mean() - df[df['bp_group'] == 'L']['return'].mean()
    
    # RMW = 高盈利收益 - 低盈利收益
    RMW = df[df['profit_group'] == 'R']['return'].mean() - df[df['profit_group'] == 'W']['return'].mean()
    
    # CMA = 低投资收益 - 高投资收益
    CMA = df[df['invest_group'] == 'C']['return'].mean() - df[df['invest_group'] == 'A']['return'].mean()
    
    return {'MKT': MKT, 'SMB': SMB, 'HML': HML, 'RMW': RMW, 'CMA': CMA}
```

### 4.4 因子投资组合

**因子组合构建：**

```python
def build_factor_portfolio(df, factor, n_groups=10, weighting='equal'):
    """
    构建因子组合
    
    参数:
    df: 包含因子和收益的数据
    factor: 因子名称
    n_groups: 分组数
    weighting: 加权方式 ('equal' 或 'cap')
    
    返回:
    组合收益
    """
    # 按因子分组
    df['group'] = df.groupby('date')[factor].transform(
        lambda x: pd.qcut(x, n_groups, labels=False, duplicates='drop')
    )
    
    # 计算组合收益
    if weighting == 'equal':
        portfolio_return = df.groupby(['date', 'group'])['forward_return'].mean().unstack()
    else:  # cap-weighted
        portfolio_return = df.groupby(['date', 'group']).apply(
            lambda x: (x['forward_return'] * x['market_cap']).sum() / x['market_cap'].sum()
        ).unstack()
    
    # 多空收益
    portfolio_return['LS'] = portfolio_return[n_groups-1] - portfolio_return[0]
    
    return portfolio_return
```

## 5. 总结

因子构建与检验要点：

1. **因子选择**：经济学逻辑 + 显著性 + 持续性
2. **数据处理**：缺失值 + 异常值 + 标准化 + 中性化
3. **因子检验**：IC分析 + 分组回测 + 多空收益 + 衰减分析
4. **因子优化**：因子合成 + 权重分配 + 动态选择
5. **实战应用**：A股因子库 + 行业模型 + 风险模型

**检验标准：**
- |IC均值| > 0.02
- ICIR > 0.5
- 多空年化收益 > 5%
- 夏普比率 > 1.0
- 最大回撤 < 10%
- 分组单调

**注意事项：**
- 避免未来函数
- 考虑交易成本
- 样本外验证
- 因子拥挤度
