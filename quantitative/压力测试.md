# 压力测试

## 1. 压力测试概述

### 1.1 定义与目的

**压力测试：** 评估极端市场情景下投资组合可能遭受的损失。

**目的：**
- 识别风险敞口
- 评估尾部风险
- 制定应急预案
- 满足监管要求

### 1.2 类型

| 类型 | 方法 | 特点 |
|------|------|------|
| 情景分析 | 历史情景、假设情景 | 直观，易理解 |
| 敏感度分析 | 参数变化 | 简单，快速 |
| 反向压力测试 | 从结果推情景 | 全面，创新 |

## 2. 历史情景分析

### 2.1 历史危机事件

**主要危机：**
- 2008年金融危机
- 2010年欧债危机
- 2015年A股股灾
- 2020年疫情冲击
- 2022年俄乌冲突

**情景构建：**
```python
def historical_scenario_analysis(returns, crisis_periods):
    """
    历史情景分析
    
    参数:
    returns: 收益率序列
    crisis_periods: 危机时期列表 [(start1, end1), ...]
    
    返回:
    情景损失
    """
    results = {}
    
    for name, (start, end) in crisis_periods.items():
        crisis_returns = returns.loc[start:end]
        
        # 损失统计
        total_loss = (1 + crisis_returns).prod() - 1
        max_drawdown = (crisis_returns.cumsum().cummax() - crisis_returns.cumsum()).max()
        volatility = crisis_returns.std() * np.sqrt(252)
        
        results[name] = {
            'total_loss': total_loss,
            'max_drawdown': max_drawdown,
            'volatility': volatility,
            'duration': len(crisis_returns)
        }
    
    return results

# 示例
crisis_periods = {
    '2008金融危机': ('2008-09-01', '2008-11-30'),
    '2020疫情冲击': ('2020-02-20', '2020-03-23'),
    '2022俄乌冲突': ('2022-02-24', '2022-03-08')
}

scenario_results = historical_scenario_analysis(returns, crisis_periods)
```

### 3.2 因子情景

**因子冲击：**
```python
def factor_shock_analysis(portfolio, factor_exposure, factor_shocks):
    """
    因子冲击分析
    
    参数:
    portfolio: 组合权重
    factor_exposure: 因子暴露
    factor_shocks: 因子冲击
    
    返回:
    情景损失
    """
    # 组合因子暴露
    portfolio_exposure = portfolio @ factor_exposure
    
    # 情景损失
    losses = {}
    for scenario, shocks in factor_shocks.items():
        loss = portfolio_exposure @ shocks
        losses[scenario] = loss
    
    return losses

# 示例
factor_shocks = {
    '市场暴跌': {
        'market': -0.30,
        'size': -0.10,
        'value': 0.05,
        'momentum': -0.15
    },
    '价值回归': {
        'market': -0.05,
        'size': 0.10,
        'value': 0.20,
        'momentum': -0.25
    }
}
```

## 4. 敏感度分析

### 4.1 单因子敏感度

**参数扫描：**
```python
def sensitivity_analysis(price_function, param_range, param_name):
    """
    单因子敏感度分析
    
    参数:
    price_function: 定价函数
    param_range: 参数范围
    param_name: 参数名称
    
    返回:
    敏感度曲线
    """
    results = []
    
    for param_value in param_range:
        price = price_function(**{param_name: param_value})
        results.append({'param': param_value, 'price': price})
    
    return pd.DataFrame(results)

# 示例：利率敏感度
def bond_price(rate, face_value=100, coupon=0.05, maturity=10):
    """债券定价"""
    cash_flows = [coupon * face_value] * (maturity - 1)
    cash_flows.append(face_value + coupon * face_value)
    
    price = sum(cf / (1 + rate) ** i for i, cf in enumerate(cash_flows, 1))
    return price

rate_range = np.linspace(0.01, 0.10, 100)
sensitivity = sensitivity_analysis(bond_price, rate_range, 'rate')
```

### 4.2 多因子敏感度

**网格分析：**
```python
def grid_sensitivity(price_function, param_ranges):
    """
    多因子敏感度网格分析
    
    参数:
    price_function: 定价函数
    param_ranges: 参数范围字典
    
    返回:
    敏感度矩阵
    """
    from itertools import product
    
    # 参数组合
    keys = param_ranges.keys()
    values = param_ranges.values()
    combinations = list(product(*values))
    
    results = []
    for combo in combinations:
        params = dict(zip(keys, combo))
        price = price_function(**params)
        results.append({**params, 'price': price})
    
    return pd.DataFrame(results)
```

## 5. 反向压力测试

### 5.1 定义

**反向压力测试：** 从给定的损失水平出发，推导可能导致该损失的情景。

### 5.2 实现

```python
def reverse_stress_test(portfolio_value, target_loss, returns, n_scenarios=10000):
    """
    反向压力测试
    
    参数:
    portfolio_value: 组合价值
    target_loss: 目标损失
    returns: 历史收益率
    n_scenarios: 模拟情景数
    
    返回:
    可能情景
    """
    # 模拟情景
    mu = returns.mean()
    cov = returns.cov()
    
    scenarios = []
    for _ in range(n_scenarios):
        # 随机生成收益率
        simulated_returns = np.random.multivariate_normal(mu, cov)
        
        # 计算损失
        loss = -portfolio_value * simulated_returns.sum()
        
        if loss >= target_loss:
            scenarios.append({
                'returns': simulated_returns,
                'loss': loss
            })
    
    # 分析情景特征
    if scenarios:
        scenario_returns = np.array([s['returns'] for s in scenarios])
        avg_returns = scenario_returns.mean(axis=0)
        
        return {
            'n_severe_scenarios': len(scenarios),
            'avg_returns': avg_returns,
            'scenarios': scenarios
        }
    
    return None
```

## 6. 风险报告

### 6.1 压力测试报告

```python
def generate_stress_test_report(portfolio, scenarios):
    """
    生成压力测试报告
    
    参数:
    portfolio: 组合
    scenarios: 情景列表
    
    返回:
    报告
    """
    report = {
        'date': pd.Timestamp.now(),
        'portfolio_value': portfolio.value,
        'scenarios': []
    }
    
    for name, shock in scenarios.items():
        loss = portfolio.apply_shock(shock)
        
        report['scenarios'].append({
            'name': name,
            'shock': shock,
            'loss': loss,
            'loss_pct': loss / portfolio.value
        })
    
    # 汇总
    report['worst_case'] = max(report['scenarios'], key=lambda x: x['loss'])
    report['average_loss'] = np.mean([s['loss'] for s in report['scenarios']])
    
    return report
```

## 7. 总结

压力测试要点：

1. **历史情景**：参考历史危机事件
2. **假设情景**：设计极端但可能的情景
3. **敏感度分析**：评估参数变化影响
4. **反向压力测试**：从损失推情景
5. **风险报告**：定期报告压力测试结果

**注意事项：**
- 情景要有合理性
- 覆盖多种风险类型
- 定期更新情景
- 结合VaR使用
