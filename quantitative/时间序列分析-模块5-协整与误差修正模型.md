# 时间序列分析 - 模块5：协整与误差修正模型

## 1. 单位根检验

### 1.1 单位根问题

**单位根过程：**
```
yₜ = yₜ₋₁ + εₜ
```

**特征：**
- 方差无限增长
- 冲击永久持续
- 不平稳

### 1.2 ADF检验

**Augmented Dickey-Fuller检验：**
```python
from statsmodels.tsa.stattools import adfuller

def adf_test(series, maxlag=None):
    """
    ADF单位根检验
    
    参数:
    series: 时间序列
    maxlag: 最大滞后阶数
    
    返回:
    检验结果
    """
    result = adfuller(series, maxlag=maxlag, autolag='AIC')
    
    print(f'ADF统计量: {result[0]:.4f}')
    print(f'p值: {result[1]:.4f}')
    print(f'临界值:')
    for key, value in result[4].items():
        print(f'  {key}: {value:.4f}')
    
    if result[1] < 0.05:
        print('结论: 拒绝原假设，序列平稳')
    else:
        print('结论: 不能拒绝原假设，序列可能存在单位根')
    
    return result
```

### 1.3 PP检验

**Phillips-Perron检验：**
```python
from statsmodels.tsa.stattools import pp

def pp_test(series):
    """
    PP单位根检验
    """
    result = pp(series, lags='auto')
    
    print(f'PP统计量: {result[0]:.4f}')
    print(f'p值: {result[1]:.4f}')
    
    if result[1] < 0.05:
        print('结论: 拒绝原假设，序列平稳')
    else:
        print('结论: 不能拒绝原假设，序列可能存在单位根')
    
    return result
```

## 2. 协整理论

### 2.1 协整定义

**协整关系：**
如果 y₁ₜ 和 y₂ₜ 都是 I(1)（一阶单整），但存在线性组合 zₜ = y₁ₜ - βy₂ₜ 是 I(0)（平稳），则称 y₁ₜ 和 y₂ₜ 协整。

**经济意义：**
- 长期均衡关系
- 短期偏离会回归
- 可用于预测

### 2.2 Engle-Granger两步法

**步骤：**

1. **协整回归：**
   ```
   y₁ₜ = α + βy₂ₜ + εₜ
   ```

2. **残差平稳性检验：**
   ```
   êₜ = y₁ₜ - α̂ - β̂y₂ₜ
   对êₜ进行ADF检验
   ```

**Python实现：**
```python
import numpy as np
import pandas as pd
from statsmodels.tsa.stattools import adfuller, coint
from statsmodels.regression.linear_model import OLS
import statsmodels.api as sm

def engle_granger(y1, y2):
    """
    Engle-Granger协整检验
    
    参数:
    y1: 时间序列1
    y2: 时间序列2
    
    返回:
    协整关系、检验结果
    """
    # 第一步：协整回归
    X = sm.add_constant(y2)
    model = OLS(y1, X).fit()
    resid = model.resid
    
    # 第二步：残差ADF检验
    adf_result = adfuller(resid, maxlag=1)
    
    print('协整回归结果:')
    print(f'  系数: {model.params[1]:.4f}')
    print(f'  截距: {model.params[0]:.4f}')
    print(f'  R²: {model.rsquared:.4f}')
    
    print('\n残差平稳性检验:')
    print(f'  ADF统计量: {adf_result[0]:.4f}')
    print(f'  p值: {adf_result[1]:.4f}')
    
    if adf_result[1] < 0.05:
        print('  结论: 存在协整关系')
        cointegrated = True
    else:
        print('  结论: 不存在协整关系')
        cointegrated = False
    
    return {
        'coefficient': model.params[1],
        'intercept': model.params[0],
        'residuals': resid,
        'adf_stat': adf_result[0],
        'p_value': adf_result[1],
        'cointegrated': cointegrated
    }
```

### 2.3 Johansen检验

**向量自回归（VAR）：**
```
Yₜ = A₁Yₜ₋₁ + ... + AₚYₜ₋ₚ + εₜ
```

**误差修正形式（VECM）：**
```
ΔYₜ = ΠYₜ₋₁ + Γ₁ΔYₜ₋₁ + ... + Γₚ₋₁ΔYₜ₋ₚ₊₁ + εₜ
```

其中：
```
Π = αβ'
```
- α：调整系数
- β：协整向量

**Python实现：**
```python
from statsmodels.tsa.vector_ar.vecm import coint_johansen

def johansen_test(data, det_order=0, k_ar_diff=1):
    """
    Johansen协整检验
    
    参数:
    data: 多变量时间序列 (n_obs, n_vars)
    det_order: 确定性趋势项
        -1: 无确定性趋势
        0: 有截距
        1: 有截距和趋势
    k_ar_diff: VAR滞后阶数
    
    返回:
    检验结果
    """
    result = coint_johansen(data, det_order, k_ar_diff)
    
    print('迹统计量检验:')
    print('秩 | 迹统计量 | 临界值(5%) | 结论')
    print('-' * 50)
    for i in range(len(result.lr1)):
        print(f'{i}  | {result.lr1[i]:8.4f} | {result.cvt[i, 1]:10.4f} | ', end='')
        if result.lr1[i] > result.cvt[i, 1]:
            print('拒绝')
        else:
            print('接受')
    
    print('\n最大特征值检验:')
    print('秩 | 特征值 | 临界值(5%) | 结论')
    print('-' * 50)
    for i in range(len(result.lr2)):
        print(f'{i}  | {result.lr2[i]:8.4f} | {result.cvm[i, 1]:10.4f} | ', end='')
        if result.lr2[i] > result.cvm[i, 1]:
            print('拒绝')
        else:
            print('接受')
    
    print('\n协整向量:')
    print(result.evec)
    
    return result
```

## 3. 误差修正模型

### 3.1 ECM理论

**长期均衡关系：**
```
y₁ₜ = α + βy₂ₜ + εₜ
```

**短期调整：**
```
Δy₁ₜ = γΔy₂ₜ + λ(y₁ₜ₋₁ - α - βy₂ₜ₋₁) + uₜ
```

其中：
- λ：误差修正系数（负值）
- γ：短期调整系数

**经济含义：**
- 长期关系：协整关系
- 短期偏离：误差修正机制
- 调整速度：|λ|越大，调整越快

### 3.2 Python实现

```python
from statsmodels.tsa.api import VECM
import statsmodels.api as sm

def vecm_model(data, coint_rank=1, k_ar_diff=1):
    """
    向量误差修正模型
    
    参数:
    data: 多变量时间序列 (n_obs, n_vars)
    coint_rank: 协整秩
    k_ar_diff: 差分滞后阶数
    
    返回:
    模型结果
    """
    # VECM模型
    model = VECM(data, k_ar_diff=k_ar_diff, coint_rank=coint_rank, deterministic='ci')
    result = model.fit()
    
    print('协整向量:')
    print(result.beta)
    
    print('\n调整系数:')
    print(result.alpha)
    
    print('\n短期系数:')
    print(result.gamma)
    
    return result
```

### 3.3 单方程ECM

```python
def ecm_single_equation(y1, y2, lags=1):
    """
    单方程误差修正模型
    
    参数:
    y1: 被解释变量
    y2: 解释变量
    lags: 滞后阶数
    
    返回:
    ECM模型结果
    """
    # 差分
    dy1 = np.diff(y1)
    dy2 = np.diff(y2)
    
    # 协整回归
    X = sm.add_constant(y2)
    coint_model = OLS(y1, X).fit()
    resid = coint_model.resid
    
    # ECM模型
    ecm_y = dy1[lags:]
    ecm_X = np.column_stack([
        dy2[lags:],  # Δy₂ₜ
        resid[:-lags-1]  # 误差修正项
    ])
    
    for i in range(1, lags+1):
        ecm_X = np.column_stack([ecm_X, dy1[lags-i:-i]])
    
    ecm_X = sm.add_constant(ecm_X)
    ecm_model = OLS(ecm_y, ecm_X).fit()
    
    print('误差修正模型结果:')
    print(ecm_model.summary())
    
    print(f'\n误差修正系数: {ecm_model.params[2]:.4f}')
    if ecm_model.params[2] < 0:
        print('符合理论预期（负值）')
    else:
        print('不符合理论预期')
    
    return ecm_model
```

## 4. 金融应用

### 4.1 配对交易

**原理：**
两只协整股票的价格有长期均衡关系，短期偏离时可以套利。

**策略：**
```python
def pairs_trading(y1, y2, window=60, entry_threshold=2, exit_threshold=0.5):
    """
    配对交易策略
    
    参数:
    y1, y2: 两只股票价格
    window: 滚动窗口
    entry_threshold: 开仓阈值
    exit_threshold: 平仓阈值
    
    返回:
    交易信号、收益
    """
    # 协整关系
    spread = y1 - np.polyfit(y2, y1, 1)[0] * y2
    
    # 滚动标准化
    spread_mean = pd.Series(spread).rolling(window).mean()
    spread_std = pd.Series(spread).rolling(window).std()
    z_score = (spread - spread_mean) / spread_std
    
    # 交易信号
    signals = pd.Series(0, index=range(len(spread)))
    
    # 开仓
    signals[z_score > entry_threshold] = -1  # 做空y1，做多y2
    signals[z_score < -entry_threshold] = 1   # 做多y1，做空y2
    
    # 平仓
    signals[(z_score > -exit_threshold) & (z_score < exit_threshold)] = 0
    
    # 计算收益
    returns = signals.shift(1) * (np.diff(y1) - np.polyfit(y2, y1, 1)[0] * np.diff(y2))
    
    return signals, returns
```

### 4.3 汇率协整

**购买力平价检验：**
```python
def ppp_test(exchange_rate, price_domestic, price_foreign):
    """
    购买力平价协整检验
    
    参数:
    exchange_rate: 汇率
    price_domestic: 国内价格
    price_foreign: 国外价格
    
    返回:
    协整检验结果
    """
    # 实际汇率
    real_rate = np.log(exchange_rate) + np.log(price_foreign) - np.log(price_domestic)
    
    # ADF检验
    adf_result = adfuller(real_rate)
    
    if adf_result[1] < 0.05:
        print('实际汇率平稳，支持PPP')
        ppp_holds = True
    else:
        print('实际汇率非平稳，不支持PPP')
        ppp_holds = False
    
    return {
        'real_rate': real_rate,
        'adf_stat': adf_result[0],
        'p_value': adf_result[1],
        'ppp_holds': ppp_holds
    }
```

## 5. 总结

协整与误差修正模型要点：

1. **单位根检验**：ADF、PP、KPSS

- 先检验单整阶数
- 协整关系需经济意义
- 样本量要足够
- 结构变化影响协整
