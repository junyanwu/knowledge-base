# 多因子轮动模型

## 一、多因子模型基础

### 1.1 因子分类体系

**Fama-French五因子模型**：
$$E(R_i) - R_f = \beta_{MKT} \cdot MKT + \beta_{SMB} \cdot SMB + \beta_{HML} \cdot HML + \beta_{RMW} \cdot RMW + \beta_{CMA} \cdot CMA$$

| 因子 | 名称 | 含义 | A股适用性 |
|------|------|------|----------|
| MKT | 市场因子 | 股票市场超额收益 | 适用 |
| SMB | 规模因子 | 小市值股票超额收益 | 适用，但近年减弱 |
| HML | 价值因子 | 高B/M股票超额收益 | 适用，周期性强 |
| RMW | 盈利因子 | 高盈利股票超额收益 | 适用 |
| CMA | 投资因子 | 低投资股票超额收益 | 适用性一般 |

**A股特色因子**：
| 因子类型 | 具体因子 | A股特征 |
|---------|---------|---------|
| **技术因子** | 动量、反转、换手率 | 反转因子更显著 |
| **情绪因子** | 北向资金、融资余额 | 散户情绪影响大 |
| **政策因子** | 产业政策指数 | 政策市特征明显 |
| **事件因子** | 并购重组、股权激励 | 事件驱动机会多 |

### 1.2 行业轮动因子体系

**核心因子矩阵**：

```
           价值因子  动量因子  质量因子  情绪因子  政策因子
行业A         ✓         ✓         ✗         ✓         ✓
行业B         ✓         ✗         ✓         ✓         ✗
行业C         ✗         ✓         ✓         ✗         ✓
...
```

**因子权重动态调整**：
$$w_i = \frac{IC_i^{\lambda}}{\sum_{j=1}^{n} IC_j^{\lambda}}$$

其中 $IC_i$ 为因子i的信息系数，$\lambda$ 为衰减系数

## 二、价值+动量+质量组合

### 2.1 价值因子

#### 2.1.1 价值因子定义

**核心指标**：
$$B/P = \frac{账面价值}{市值}$$

$$E/P = \frac{净利润}{市值} = \frac{1}{PE}$$

$$CF/P = \frac{经营现金流}{市值}$$

**价值因子组合**：
$$Value\_Score = w_1 \cdot B/P_{zscore} + w_2 \cdot E/P_{zscore} + w_3 \cdot CF/P_{zscore}$$

#### 2.1.2 A股价值因子特征

| 特征 | 表现 | 原因 |
|------|------|------|
| **周期性强** | 周期股价值因子表现突出 | 周期股盈利波动大 |
| **均值回归** | 高估值板块终将回归 | A股估值泡沫最终破裂 |
| **行业分化** | 不同行业估值中枢差异大 | 需行业内比较 |
| **失效期** | 成长股牛市时跑输 | 2013-2015, 2019-2020 |

#### 2.1.3 改进价值因子

**相对价值因子**：
$$Relative\_Value = \frac{V_i}{\bar{V}_{industry}}$$

**价值+成长因子**：
$$GARP = \frac{盈利增速}{PE} = \frac{g}{PE}$$

**PEG因子**：
$$PEG = \frac{PE}{g}$$
