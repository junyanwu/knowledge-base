# 市场环境识别方法

## 概述

市场环境识别是量化投资的核心问题。正确识别当前市场状态（牛市/熊市/震荡市）对策略选择和参数调整至关重要。

---

## 1. ADX指标趋势强度判断

### 原理
ADX（Average Directional Index）由Wilder开发，用于衡量趋势强度，不区分方向。

### 计算公式
```
+DM = High_today - High_yesterday (if positive, else 0)
-DM = Low_yesterday - Low_today (if positive, else 0)

+DI = 100 * EMA(+DM, n) / ATR(n)
-DI = 100 * EMA(-DM, n) / ATR(n)

DX = 100 * |+DI - -DI| / (+DI + -DI)
ADX = EMA(DX, n)
```

### 市场环境识别规则
| ADX值 | 趋势强度 | 市场状态 |
|-------|---------|---------|
| < 20 | 无趋势 | 震荡市 |
| 20-25 | 趋势形成 | 趋势萌芽 |
| 25-50 | 强趋势 | 明显趋势 |
| 50-75 | 很强趋势 | 强趋势 |
| > 75 | 极强趋势 | 可能反转 |

### 优点
- 简单直观，易于实现
- 参数少，不易过拟合
- 适合趋势跟踪策略

### 缺点
- 滞后性明显（通常14-28周期）
- 无法识别趋势方向
- 震荡市中信号频繁

### 实践应用
```python
def identify_regime_adx(df, period=14):
    df['adx'] = ta.ADX(df['high'], df['low'], df['close'], timeperiod=period)
    df['plus_di'] = ta.PLUS_DI(df['high'], df['low'], df['close'], timeperiod=period)
    df['minus_di'] = ta.MINUS_DI(df['high'], df['low'], df['close'], timeperiod=period)
    
    # 市场环境分类
    df['regime'] = 'sideways'  # 震荡市
    df.loc[df['adx'] > 25, 'regime'] = 'trending'  # 趋势市
    
    # 趋势方向
    df['trend_direction'] = 'neutral'
    df.loc[(df['adx'] > 25) & (df['plus_di'] > df['minus_di']), 'trend_direction'] = 'bull'
    df.loc[(df['adx'] > 25) & (df['plus_di'] < df['minus_di']), 'trend_direction'] = 'bear'
    
    return df
```

## 2. 均线系统趋势方向判断

### 原理
利用多条均线的排列关系判断趋势方向和强度。

### 常用组合
1. **经典三均线系统**
   - 短期：MA5/MA10
   - 中期：MA20/MA30
   - 长期：MA60/MA120

2. **多空头排列**
   - 多头排列：MA5 > MA10 > MA20 > MA60（牛市）
   - 空头排列：MA5 < MA10 < MA20 < MA60（熊市）
   - 交叉排列：震荡市

### 市场环境识别规则
```python
def identify_regime_ma(df):
    df['ma5'] = df['close'].rolling(5).mean()
    df['ma10'] = df['close'].rolling(10).mean()
    df['ma20'] = df['close'].rolling(20).mean()
    df['ma60'] = df['close'].rolling(60).mean()
    
    # 多头排列
    bull = (df['ma5'] > df['ma10']) & (df['ma10'] > df['ma20']) & (df['ma20'] > df['ma60'])
    # 空头排列
    bear = (df['ma5'] < df['ma10']) & (df['ma10'] < df['ma20']) & (df['ma20'] < df['ma60'])
    
    df['regime'] = 'sideways'
    df.loc[bull, 'regime'] = 'bull'
    df.loc[bear, 'regime'] = 'bear'
    
    return df
```

### 优点
- 直观易懂
- 适合中长期趋势识别
- 结合支撑阻力位效果更好

### 缺点
- 滞后性强
- 参数选择主观
- 震荡市中频繁假信号

### 改进方法
- 使用EMA代替SMA（降低滞后性）
- 结合价格位置判断（价格在均线上方/下方）
- 加入斜率判断（均线角度）

## 3. 波动率状态分类

### 原理
波动率反映市场的不确定性，不同波动率状态对应不同的交易策略。

### 常用指标
1. **历史波动率（HV）**
   ```
   HV = sqrt(252) * std(log(return), n)
   ```

2. **ATR（Average True Range）**
   ```
   TR = max(H-L, |H-C_prev|, |L-C_prev|)
   ATR = EMA(TR, n)
   ```

3. **VIX指数**（期权隐含波动率）

### 波动率状态分类
```python
def classify_volatility_regime(df, window=20):
    # 计算历史波动率
    df['returns'] = df['close'].pct_change()
    df['volatility'] = df['returns'].rolling(window).std() * np.sqrt(252)
    
    # 波动率分位数
    df['vol_percentile'] = df['volatility'].rolling(252).rank(pct=True)
    
    # 分类
    df['vol_regime'] = 'normal'
    df.loc[df['vol_percentile'] < 0.25, 'vol_regime'] = 'low'  # 低波动
    df.loc[df['vol_percentile'] > 0.75, 'vol_regime'] = 'high'  # 高波动
    
    return df
```

### 应用场景
| 波动率状态 | 适用策略 | 风险控制 |
|-----------|---------|---------|
| 低波动 | 趋势跟踪、突破策略 | 放宽止损 |
| 正常波动 | 标准策略 | 标准止损 |
| 高波动 | 均值回归、波动率策略 | 收紧止损 |

### 优点
- 直接反映市场风险
- 对极端行情敏感
- 适合动态调整仓位

### 缺点
- 波动率聚类效应
- 难以预测波动率转折点
- 需要较长历史数据

## 4. Regime-Switching模型（HMM）

### 原理
Hidden Markov Model（隐马尔可夫模型）假设市场存在K个隐状态，通过观测序列推断当前状态。

### 模型结构
```
隐状态：S = {s1, s2, ..., sK}  （如：牛市、熊市、震荡市）
观测变量：O = {o1, o2, ..., oT} （收益率、波动率等）

参数：
- 初始概率：π = P(s1)
- 转移概率：A = P(st|st-1)
- 发射概率：B = P(ot|st)
```

### Python实现
```python
from hmmlearn import hmm
import numpy as np

def train_hmm_regime(returns, n_states=3):
    # 准备数据
    X = returns.values.reshape(-1, 1)
    
    # 训练HMM
    model = hmm.GaussianHMM(
        n_components=n_states,
        covariance_type='full',
        n_iter=1000
    )
    model.fit(X)
    
    # 预测隐状态
    hidden_states = model.predict(X)
    
    # 解读状态（基于均值和方差）
    means = model.means_.flatten()
    variances = np.sqrt(model.covars_.flatten())
    
    # 根据统计特征命名状态
    state_names = {}
    sorted_by_mean = np.argsort(means)
    
    # 均值最低 = 熊市
    state_names[sorted_by_mean[0]] = 'bear'
    # 均值中等 = 震荡
    state_names[sorted_by_mean[1]] = 'sideways'
    # 均值最高 = 牛市
    state_names[sorted_by_mean[2]] = 'bull'
    
    return model, hidden_states, state_names
```

### HMM优势
- 理论框架完整
- 自动学习状态转移概率
- 可以预测未来状态概率

### HMM劣势
- 需要大量数据训练
- 状态数量选择主观
- 假设状态转移是马尔可夫的
- 对初始化敏感

### 改进方法
1. **特征扩展**：使用收益率+波动率+成交量等多维特征
2. **时变转移概率**：允许转移概率随时间变化
3. **贝叶斯HMM**：使用MCMC进行参数估计
4. **深度学习结合**：使用LSTM-HMM混合模型

## 5. 市场情绪指标

### 常用情绪指标

#### 5.1 VIX恐慌指数
- VIX > 30：市场恐慌
- VIX 20-30：波动偏高
- VIX 15-20：正常波动
- VIX < 15：市场平静

#### 5.2 Put/Call Ratio
- PCR > 1.0：过度看空（反向指标，可能反弹）
- PCR 0.7-1.0：轻度看空
- PCR < 0.7：过度看多（反向指标，可能回调）

#### 5.3 涨跌家数比
```python
def calculate_breadth_ratio(df_market):
    advancing = (df_market['close'] > df_market['open']).sum()
    declining = (df_market['close'] < df_market['open']).sum()
    breadth_ratio = advancing / (advancing + declining)
    
    # 极端值判断
    if breadth_ratio > 0.75:
        sentiment = 'extreme_greed'
    elif breadth_ratio > 0.6:
        sentiment = 'greed'
    elif breadth_ratio < 0.25:
        sentiment = 'extreme_fear'
    elif breadth_ratio < 0.4:
        sentiment = 'fear'
    else:
        sentiment = 'neutral'
    
    return sentiment, breadth_ratio
```

#### 5.4 成交量分析
```python
def analyze_volume_trend(df):
    df['volume_ma'] = df['volume'].rolling(20).mean()
    df['volume_ratio'] = df['volume'] / df['volume_ma']
    
    # 放量上涨
    high_vol_up = (df['close'] > df['open']) & (df['volume_ratio'] > 1.5)
    # 缩量下跌
    low_vol_down = (df['close'] < df['open']) & (df['volume_ratio'] < 0.7)
    
    return high_vol_up, low_vol_down
```

### 综合情绪指数
```python
def calculate_sentiment_index(vix=None, put_call_ratio=None, 
                              breadth_ratio=None, news_sentiment=None):
    """
    构建综合情绪指数（0-100）
    """
    score = 50  # 基准
    
    if vix is not None:
        vix_score = max(0, min(100, 100 - vix * 2))
        score = 0.3 * vix_score + 0.7 * score
    
    if put_call_ratio is not None:
        # PCR反向指标
        pcr_score = max(0, min(100, 100 - put_call_ratio * 80))
        score = 0.25 * pcr_score + 0.75 * score
    
    if breadth_ratio is not None:
        breadth_score = breadth_ratio * 100
        score = 0.25 * breadth_score + 0.75 * score
    
    return score
```

### 情绪指标优缺点
**优点**：
- 反映市场心理
- 可作为反向指标
- 有助于识别极端行情

**缺点**：
- 主观性强
- 数据获取困难（部分指标）
- 滞后性

```python
def comprehensive_regime_identification(df):
    """
    综合市场环境识别
    """
    # 1. ADX判断趋势强度
    df = identify_regime_adx(df)
    
    # 2. 均线系统判断趋势方向
    df = identify_regime_ma(df)
    
    # 3. 波动率分类
    df = classify_volatility_regime(df)
    
    # 4. 综合判断（投票法）
    df['final_regime'] = 'sideways'
    
    # 如果多个指标一致，则确认
    bull_votes = 0
    bear_votes = 0
    
    if df['adx'].iloc[-1] > 25:  # 有趋势
        if df['trend_direction'].iloc[-1] == 'bull':
            bull_votes += 1
        elif df['trend_direction'].iloc[-1] == 'bear':
            bear_votes += 1
    
    if df['regime'].iloc[-1] == 'bull':
        bull_votes += 1
    elif df['regime'].iloc[-1] == 'bear':
        bear_votes += 1
    
    if bull_votes >= 2:
        df['final_regime'].iloc[-1] = 'bull'
    elif bear_votes >= 2:
        df['final_regime'].iloc[-1] = 'bear'
    
    return df
```
