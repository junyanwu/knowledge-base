# 多因子模型理论 - 模块3：因子组合构建

## 1. 因子组合构建框架

### 1.1 组合构建流程

**完整流程：**
```
数据准备 → 因子计算 → 因子处理 → 因子检验 → 组合构建 → 回测优化
```

**核心步骤：**

| 步骤 | 内容 | 关键点 |
|------|------|--------|
| 数据准备 | 行情、财务、另类数据 | 数据清洗、对齐 |
| 因子计算 | 计算122个因子 | 标准化公式 |
| 因子处理 | 标准化、中性化、正交化 | 去极值、填充 |
| 因子检验 | IC、分组、回归 | 有效性验证 |
| 组合构建 | 选股、加权、风控 | 优化目标 |
| 回测优化 | 绩效分析、参数调优 | 样本外验证 |

### 1.2 组合类型

**因子组合分类：**

1. **单因子组合**
   - 单一因子选股
   - 多空组合
   - 因子暴露组合

2. **多因子组合**
   - 因子加权
   - 风险模型
   - 优化组合

3. **因子择时组合**
   - 动态因子权重
   - 市场状态切换
   - 因子动量

### 2.1 简单分组法

**构建步骤：**

1. **因子计算**
   ```python
   # 计算因子
   df['factor'] = df['book_value'] / df['market_cap']  # BP因子
   ```

2. **因子处理**
   ```python
   # 去极值
   def winsorize(data, upper=0.99, lower=0.01):
       upper_limit = data.quantile(upper)
       lower_limit = data.quantile(lower)
       return data.clip(lower_limit, upper_limit)
   
   df['factor'] = winsorize(df['factor'])
   
   # 标准化
   df['factor_std'] = (df['factor'] - df['factor'].mean()) / df['factor'].std()
   
   # 行业中性化
   def industry_neutralize(data, industry_col, factor_col):
       return data.groupby(industry_col)[factor_col].transform(
           lambda x: x - x.mean()
       )
   
   df['factor_neutral'] = industry_neutralize(df, 'industry', 'factor_std')
   ```

3. **分组选股**
   ```python
   # 按因子分组
   df['group'] = df.groupby('date')['factor_neutral'].transform(
       lambda x: pd.qcut(x, 10, labels=False, duplicates='drop')
   )
   
   # 选择多头组合（第9组）
   long_portfolio = df[df['group'] == 9].copy()
   
   # 选择空头组合（第0组）
   short_portfolio = df[df['group'] == 0].copy()
   ```

4. **等权组合收益**
   ```python
   # 计算组合收益
   ```

### 2.3 行业中性组合

**行业内选股：**
```python
def industry_neutral_portfolio(data, factor_col, n_groups=5):
    """
    行业中性组合
    
    参数:
    data: 数据框
    factor_col: 因子列名
    n_groups: 每个行业内的分组数
    
    返回:
    组合收益
    """
    # 行业内分组
    data['industry_group'] = data.groupby(['date', 'industry'])[factor_col].transform(
        lambda x: pd.qcut(x, n_groups, labels=False, duplicates='drop')
    )
    
    # 各行业多头组合
    long_portfolio = data[data['industry_group'] == n_groups - 1].copy()
    
    # 各行业空头组合
    short_portfolio = data[data['industry_group'] == 0].copy()
    
    # 行业内等权
    long_return = long_portfolio.groupby(['date', 'industry'])['forward_return'].mean()
    short_return = short_portfolio.groupby(['date', 'industry'])['forward_return'].mean()
    
    # 各行业等权平均
    long_return_final = long_return.groupby('date').mean()
    short_return_final = short_return.groupby('date').mean()
    
    ls_return = long_return_final - short_return_final
    
    return ls_return
```

## 3. 多因子组合构建

### 3.1 因子合成方法

**1. 等权合成**
```python
def equal_weight_combination(factors):
    """
    等权合成
    
    参数:
    factors: 因子列表 [(factor1_name, factor1_series), ...]
    
    返回:
    合成因子
    """
    # 标准化各因子
    standardized = []
    for name, factor in factors:
        factor_std = (factor - factor.mean()) / factor.std()
        standardized.append(factor_std)
    
    # 等权平均
    combined = sum(standardized) / len(standardized)
    
    return combined
```

**2. IC加权**
```python
def ic_weight_combination(factors, returns, dates):
    """
    IC加权合成
    
    参数:
    factors: 因子列表
    returns: 收益率序列
    dates: 日期序列
    
    返回:
    合成因子
    """
    from scipy.stats import spearmanr
    
    # 计算各因子IC
    ics = []
    for name, factor in factors:
        ic_series = factor.groupby(dates).apply(
            lambda x: spearmanr(x, returns.loc[x.index])[0]
        )
        ics.append(ic_series.mean())
    
    # IC加权
    weights = [abs(ic) for ic in ics]
    weights = np.array(weights) / sum(weights)
    
    # 合成因子
    standardized = [(f - f.mean()) / f.std() for _, f in factors]
    combined = sum(w * s for w, s in zip(weights, standardized))
    
    return combined, weights
```

**3. ICIR加权**
```python
def icir_weight_combination(factors, returns, dates):
    """
    ICIR加权合成
    
    参数:
    factors: 因子列表
    returns: 收益率序列
    dates: 日期序列
    
    返回:
    合成因子
    """
    from scipy.stats import spearmanr
    
    # 计算各因子ICIR
    icirs = []
    for name, factor in factors:
        ic_series = factor.groupby(dates).apply(
            lambda x: spearmanr(x, returns.loc[x.index])[0]
        )
        icir = ic_series.mean() / ic_series.std()
        icirs.append(icir)
    
    # ICIR加权
    weights = [abs(icir) for icir in icirs]
    weights = np.array(weights) / sum(weights)
    
    # 合成因子
    standardized = [(f - f.mean()) / f.std() for _, f in factors]
    combined = sum(w * s for w, s in zip(weights, standardized))
    
    return combined, weights
```

**4. 最大化ICIR**
```python
from scipy.optimize import minimize

def maximize_icir_combination(factors, returns, dates):
    """
    最大化ICIR合成
    
    参数:
    factors: 因子列表
    returns: 收益率序列
    dates: 日期序列
    
    返回:
    合成因子, 最优权重
    """
    from scipy.stats import spearmanr
    
    n_factors = len(factors)
    
    # 计算IC序列
    ic_matrix = []
    for name, factor in factors:
        ic_series = factor.groupby(dates).apply(
            lambda x: spearmanr(x, returns.loc[x.index])[0]
        )
        ic_matrix.append(ic_series.values)
    
    ic_matrix = np.array(ic_matrix).T  # (n_dates, n_factors)
    
    # 目标函数：最大化ICIR
    def neg_icir(weights):
        combined_ic = ic_matrix @ weights
        return -combined_ic.mean() / combined_ic.std()
    
    # 约束条件
    constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
    bounds = [(0, 1) for _ in range(n_factors)]
    
    # 优化
    result = minimize(
        neg_icir,
        x0=[1/n_factors] * n_factors,
        bounds=bounds,
        constraints=constraints
    )
    
    optimal_weights = result.x
    
    # 合成因子
    standardized = [(f - f.mean()) / f.std() for _, f in factors]
    combined = sum(w * s for w, s in zip(optimal_weights, standardized))
    
    return combined, optimal_weights
```

### 3.2 均值方差优化

**经典马科维茨优化：**
```python
from scipy.optimize import minimize

def mean_variance_optimization(expected_returns, cov_matrix, 
                                target_return=None, risk_aversion=1):
    """
    均值方差优化
    
    参数:
    expected_returns: 期望收益 (n,)
    cov_matrix: 协方差矩阵 (n, n)
    target_return: 目标收益
    risk_aversion: 风险厌恶系数
    
    返回:
    最优权重
    """
    n = len(expected_returns)
    
    # 目标函数：最大化效用
    def neg_utility(weights):
        portfolio_return = weights @ expected_returns
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        return -(portfolio_return - 0.5 * risk_aversion * portfolio_vol**2)
    
    # 约束条件
    constraints = [{'type': 'eq', 'fun': lambda w: sum(w) - 1}]
    if target_return:
        constraints.append({
            'type': 'eq',
            'fun': lambda w: w @ expected_returns - target_return
        })
    
    # 边界条件
    bounds = [(0, 1) for _ in range(n)]
    
    # 优化
    result = minimize(
        neg_utility,
        x0=[1/n] * n,
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

**风险平价：**
```python
def risk_parity_optimization(cov_matrix):
    """
    风险平价优化
    
    参数:
    cov_matrix: 协方差矩阵
    
    返回:
    风险平价权重
    """
    n = cov_matrix.shape[0]
    
    def risk_budget_objective(weights):
        # 组合波动率
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        
        # 边际风险贡献
        marginal_risk = cov_matrix @ weights / portfolio_vol
        
        # 风险贡献
        risk_contrib = weights * marginal_risk
        
        # 目标：各资产风险贡献相等
        target_risk = portfolio_vol / n
        
        return sum((risk_contrib - target_risk)**2)
    
    # 约束条件
    constraints = {'type': 'eq', 'fun': lambda w: sum(w) - 1}
    bounds = [(0, 1) for _ in range(n)]
    
    # 优化
    result = minimize(
        risk_budget_objective,
        x0=[1/n] * n,
        bounds=bounds,
        constraints=constraints
    )
    
    return result.x
```

### 3.3 风险模型优化

**因子风险模型：**
```python
def factor_risk_model_optimization(alpha, factor_exposure, factor_cov, 
                                    idio_vol, constraints=None):
    """
    因子风险模型优化
    
    参数:
    alpha: 股票alpha (n,)
    factor_exposure: 因子暴露 (n, k)
    factor_cov: 因子协方差矩阵 (k, k)
    idio_vol: 特质波动率 (n,)
    constraints: 额外约束
    
    返回:
    最优权重
    """
    n = len(alpha)
    
    # 协方差矩阵
    idio_var = np.diag(idio_vol**2)
    cov_matrix = factor_exposure @ factor_cov @ factor_exposure.T + idio_var
    
    # 目标函数：最大化风险调整后收益
    def neg_sharpe(weights):
        portfolio_return = weights @ alpha
        portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
        return -portfolio_return / portfolio_vol
    
    # 基础约束
    cons = [{'type': 'eq', 'fun': lambda w: sum(w) - 1}]
    if constraints:
        cons.extend(constraints)
    
    # 边界
    bounds = [(0, 1) for _ in range(n)]
    
    # 优化
    result = minimize(
        neg_sharpe,
        x0=[1/n] * n,
        bounds=bounds,
        constraints=cons
    )
    
    return result.x
```

```python
def portfolio_constraints(n_stocks, stock_ids=None):

    组合约束

    constraints = []

    max_weight = 0.05  # 5%
    # 已在bounds中设置
    
    # 3. 行业权重上限
```

```python
def risk_budget_constraint(weights, cov_matrix, max_vol=0.2):

    风险预算约束
```

**因子暴露约束：**
```python
def factor_exposure_constraint(weights, factor_exposure, factor_limits):
    """
    因子暴露约束
    
    参数:
    weights: 组合权重
    factor_exposure: 因子暴露矩阵 (n, k)
    factor_limits: 因子暴露限制 {factor_name: (min, max)}
    
    返回:
    约束列表
    """
    constraints = []
    
    for i, (factor_name, (min_exp, max_exp)) in enumerate(factor_limits.items()):
        # 因子暴露
        exposure = lambda w, idx=i: w @ factor_exposure[:, idx]
        
        # 下限
        constraints.append({
            'type': 'ineq',
            'fun': lambda w, idx=i, min_e=min_exp: exposure(w, idx) - min_e
        })
        
        # 上限
        constraints.append({
            'type': 'ineq',
            'fun': lambda w, idx=i, max_e=max_exp: max_e - exposure(w, idx)
        })
    
    return constraints
```

### 4.3 交易约束

**换手率约束：**
```python
def turnover_constraint(new_weights, old_weights, max_turnover=0.3):
    """
    换手率约束
    
    参数:
    new_weights: 新权重
    old_weights: 旧权重
    max_turnover: 最大换手率
    
    返回:
    约束条件
    """
    turnover = np.sum(np.abs(new_weights - old_weights)) / 2
    return max_turnover - turnover
```

**交易成本：**
```python
def transaction_cost_objective(weights, old_weights, buy_cost=0.001, sell_cost=0.001):
    """
    交易成本
    
    参数:
    weights: 新权重
    old_weights: 旧权重
    buy_cost: 买入成本
    sell_cost: 卖出成本
    
    返回:
    交易成本
    """
    trades = weights - old_weights
    buy_amount = np.maximum(trades, 0).sum()
    sell_amount = np.maximum(-trades, 0).sum()
    
    return buy_amount * buy_cost + sell_amount * sell_cost
```

## 5. 组合优化实战

### 5.1 完整优化流程

```python
import numpy as np
import pandas as pd
from scipy.optimize import minimize

class PortfolioOptimizer:
    """组合优化器"""
    
    def __init__(self, data, factors, returns):
        """
        初始化
        
        参数:
        data: 股票数据（包含市值、行业等）
        factors: 因子数据 (n_stocks, n_factors)
        returns: 收益率数据 (n_stocks,)
        """
        self.data = data
        self.factors = factors
        self.returns = returns
        self.n_stocks = len(data)
        
    def calc_expected_returns(self):
        """计算期望收益"""
        # 使用因子预测收益
        # alpha = factor_exposure @ factor_returns
        # 简化处理：使用因子值作为alpha
        alpha = self.factors.mean(axis=1)
        return alpha
    
    def calc_cov_matrix(self):
        """计算协方差矩阵"""
        # 样本协方差矩阵
        return self.returns.cov()
    
    def optimize(self, objective='sharpe', constraints=None, bounds=None):
        """
        组合优化
        
        参数:
        objective: 目标函数 ('sharpe', 'min_vol', 'max_return')
        constraints: 约束条件
        bounds: 权重边界
        
        返回:
        最优权重
        """
        alpha = self.calc_expected_returns()
        cov = self.calc_cov_matrix()
        
        # 目标函数
        if objective == 'sharpe':
            def obj_func(w):
                ret = w @ alpha
                vol = np.sqrt(w @ cov @ w)
                return -ret / vol
        
        elif objective == 'min_vol':
            def obj_func(w):
                return np.sqrt(w @ cov @ w)
        
        elif objective == 'max_return':
            def obj_func(w):
                return -w @ alpha
        
        # 默认约束
        if constraints is None:
            constraints = [{'type': 'eq', 'fun': lambda w: sum(w) - 1}]
        
        # 默认边界
        if bounds is None:
            bounds = [(0, 0.05) for _ in range(self.n_stocks)]
        
        # 优化
        result = minimize(
            obj_func,
            x0=[1/self.n_stocks] * self.n_stocks,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )
        
        return result.x
    
    def backtest(self, weights, periods=12):
        """
        回测
        
        参数:
        weights: 初始权重
        periods: 回测期数
        
        返回:
        回测结果
        """
        portfolio_returns = []
        portfolio_weights = [weights]
        
        for i in range(periods):
            # 计算当期收益
            period_return = weights @ self.returns.iloc[i]
            portfolio_returns.append(period_return)
            
            # 更新权重（使用前一期数据优化）
            if i < periods - 1:
                # 简化：保持权重不变
                weights = weights
                portfolio_weights.append(weights)
        
        # 计算绩效
        portfolio_returns = pd.Series(portfolio_returns)
        cum_return = (1 + portfolio_returns).cumprod()
        
        sharpe = portfolio_returns.mean() / portfolio_returns.std() * np.sqrt(252)
        max_dd = (cum_return / cum_return.cummax() - 1).min()
        
        return {
            'returns': portfolio_returns,
            'cum_return': cum_return,
            'sharpe': sharpe,
            'max_drawdown': max_dd,
            'weights': portfolio_weights
        }
```

```python
def process_factors(factors):

def select_stocks(combined_factor, n_stocks=50):
    """选股"""
    # 选择因子值最高的n只股票
    selected = combined_factor.nlargest(n_stocks)

# 构建组合
def build_portfolio(selected_stocks, data, method='equal_weight'):
    """构建组合"""
    portfolio = pd.DataFrame(index=selected_stocks)
    
    if method == 'equal_weight':
        portfolio['weight'] = 1 / len(selected_stocks)
    
    elif method == 'market_cap_weight':
        market_cap = data.loc[selected_stocks, 'market_cap']
        portfolio['weight'] = market_cap / market_cap.sum()
    
    elif method == 'factor_weight':
        factor_value = combined_factor[selected_stocks]
        portfolio['weight'] = factor_value / factor_value.sum()
    
    return portfolio

# 回测
def backtest_portfolio(portfolio, returns, periods):
    """回测"""
    portfolio_returns = []
    
    for i in range(periods):
        period_return = (portfolio['weight'] * returns.iloc[i]).sum()
        portfolio_returns.append(period_return)
    
    return pd.Series(portfolio_returns)
```

```python
def calc_return_metrics(returns):
    """
    计算收益指标
    
    参数:
    returns: 收益率序列
    
    返回:
    收益指标字典
    """
    n = len(returns)
    
    # 总收益
    total_return = (1 + returns).prod() - 1
    
    # 年化收益
    annual_return = (1 + returns).prod() ** (252 / n) - 1
    
    # 月度收益
    monthly_return = returns.resample('M').apply(lambda x: (1 + x).prod() - 1)
    
    # 正收益月份比例
    positive_months = (monthly_return > 0).sum() / len(monthly_return)
    
    return {
        'total_return': total_return,
        'annual_return': annual_return,
        'positive_months': positive_months
    }
```

**风险指标：**
```python
def calc_risk_metrics(returns):
    """
    计算风险指标
    
    参数:
    returns: 收益率序列
    
    返回:
    风险指标字典
    """
    # 波动率
    volatility = returns.std() * np.sqrt(252)
    
    # 下行波动率
    downside_returns = returns[returns < 0]
    downside_vol = downside_returns.std() * np.sqrt(252)
    
    # 最大回撤
    cum_return = (1 + returns).cumprod()
    drawdown = cum_return / cum_return.cummax() - 1
    max_drawdown = drawdown.min()
    
    # VaR
    var_95 = returns.quantile(0.05)
    var_99 = returns.quantile(0.01)
    
    # CVaR
    cvar_95 = returns[returns <= var_95].mean()
    cvar_99 = returns[returns <= var_99].mean()
    
    return {
        'volatility': volatility,
        'downside_volatility': downside_vol,
        'max_drawdown': max_drawdown,
        'var_95': var_95,
        'var_99': var_99,
        'cvar_95': cvar_95,
        'cvar_99': cvar_99
    }
```

**风险调整收益：**
```python
def calc_risk_adjusted_metrics(returns, benchmark_returns=None, risk_free_rate=0.03):
    """
    计算风险调整收益指标
    
    参数:
    returns: 策略收益率
    benchmark_returns: 基准收益率
    risk_free_rate: 无风险利率
    
    返回:
    风险调整收益指标字典
    """
    n = len(returns)
    
    # 夏普比率
    excess_returns = returns - risk_free_rate / 252
    sharpe = excess_returns.mean() / returns.std() * np.sqrt(252)
    
    # 索提诺比率
    downside_returns = returns[returns < 0]
    sortino = excess_returns.mean() / downside_returns.std() * np.sqrt(252)
    
    # 卡玛比率
    annual_return = (1 + returns).prod() ** (252 / n) - 1
    cum_return = (1 + returns).cumprod()
    max_dd = (cum_return / cum_return.cummax() - 1).min()
    calmar = annual_return / abs(max_dd)
    
    if benchmark_returns is not None:
        # 信息比率
        active_return = returns - benchmark_returns
        information_ratio = active_return.mean() / active_return.std() * np.sqrt(252)
        
        # Alpha和Beta
        cov = np.cov(returns, benchmark_returns)[0, 1]
        var = np.var(benchmark_returns)
        beta = cov / var
        alpha = annual_return - beta * ((1 + benchmark_returns).prod() ** (252 / n) - 1)
        
        # 特雷诺比率
        treynor = (annual_return - risk_free_rate) / beta
        
        return {
            'sharpe': sharpe,
            'sortino': sortino,
            'calmar': calmar,
            'information_ratio': information_ratio,
            'alpha': alpha,
            'beta': beta,
            'treynor': treynor
        }
    
    return {
        'sharpe': sharpe,
        'sortino': sortino,
        'calmar': calmar
    }
```

### 6.2 归因分析

**Brinson归因：**
```python
def brinson_attribution(portfolio_returns, benchmark_returns, 
                        portfolio_weights, benchmark_weights,
                        asset_returns):
    """
    Brinson归因分析
    
    参数:
    portfolio_returns: 组合收益率
    benchmark_returns: 基准收益率
    portfolio_weights: 组合权重
    benchmark_weights: 基准权重
    asset_returns: 各资产收益率
    
    返回:
    归因结果
    """
    # 总收益差异
    total_difference = portfolio_returns - benchmark_returns
    
    # 配置效应（配置收益 - 基准收益）
    allocation_effect = (portfolio_weights - benchmark_weights) @ asset_returns
    
    # 选择效应
    selection_effect = benchmark_weights @ (portfolio_returns - benchmark_returns)
    
    # 交互效应
    interaction_effect = (portfolio_weights - benchmark_weights) @ (portfolio_returns - benchmark_returns)
    
    return {
        'total_difference': total_difference,
        'allocation_effect': allocation_effect,
        'selection_effect': selection_effect,
        'interaction_effect': interaction_effect
    }
```

**因子归因：**
```python
def factor_attribution(portfolio_returns, factor_returns, factor_exposure):
    """
    因子归因分析
    
    参数:
    portfolio_returns: 组合收益率
    factor_returns: 因子收益率 (n_periods, n_factors)
    factor_exposure: 因子暴露 (n_factors,)
    
    返回:
    因子归因结果
    """
    # 因子贡献
    factor_contribution = factor_returns * factor_exposure
    
    # 总因子收益
    total_factor_return = factor_contribution.sum(axis=1)
    
    # 特质收益
    idiosyncratic_return = portfolio_returns - total_factor_return
    
    return {
        'factor_contribution': factor_contribution,
        'total_factor_return': total_factor_return,
        'idiosyncratic_return': idiosyncratic_return
    }
```
