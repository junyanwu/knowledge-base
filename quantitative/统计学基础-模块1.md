## 一、概率论基础

### 1.1 概率的定义

#### 古典概率

**定义**：在等可能的情况下，事件A发生的概率。

**公式**：
```
P(A) = 有利情况数 / 所有可能情况数 = n(A) / n(Ω)
```

**案例**：
```
掷骰子，出现偶数的概率
P(偶数) = 3/6 = 1/2
```

#### 统计概率

**定义**：在大量重复试验中，事件发生的频率趋于稳定的值。

**大数定律**：
```
lim(n→∞) P(|fₙ(A) - P(A)| < ε) = 1
```

**案例**：
```
抛硬币10000次，正面出现频率 ≈ 0.5
```

#### 主观概率

**定义**：基于个人信念或经验对事件发生可能性的判断。

**贝叶斯观点**：概率是信念程度的度量。

### 1.2 概率的基本性质

#### 概率公理（Kolmogorov公理）

**公理1（非负性）**：
```
P(A) ≥ 0
```

**公理2（规范性）**：
```
P(Ω) = 1
```

**公理3（可列可加性）**：
```
若 A₁, A₂, ... 互不相容，则
P(A₁ ∪ A₂ ∪ ...) = P(A₁) + P(A₂) + ...
```

#### 基本性质

**性质1**：
```
P(∅) = 0
```

**性质2（有限可加性）**：
```
若 A₁, A₂, ..., Aₙ 互不相容，则
P(A₁ ∪ A₂ ∪ ... ∪ Aₙ) = P(A₁) + P(A₂) + ... + P(Aₙ)
```

**性质3**：
```
P(Ā) = 1 - P(A)
```

**性质4**：
```
若 A ⊂ B，则 P(A) ≤ P(B)
```

### 1.3 条件概率与独立性

#### 条件概率

**定义**：在事件B发生的条件下，事件A发生的概率。

**公式**：
```
P(A|B) = P(A∩B) / P(B),  P(B) > 0
```

**案例**：
```
某班男生60%，戴眼镜40%，男生中戴眼镜30%

P(戴眼镜|男生) = 30%
P(男生|戴眼镜) = P(男生∩戴眼镜) / P(戴眼镜) = 0.6×0.3 / 0.4 = 45%
```

#### 乘法公式

**两事件**：
```
P(A∩B) = P(A)P(B|A) = P(B)P(A|B)
```

**多事件**：
```
P(A₁∩A₂∩...∩Aₙ) = P(A₁)P(A₂|A₁)P(A₃|A₁A₂)...P(Aₙ|A₁A₂...Aₙ₋₁)
```

**案例**：
```
袋中有5个白球，3个红球，不放回抽取2个球都是白球的概率

P(白白) = P(第一个白)×P(第二个白|第一个白)
       = (5/8)×(4/7) = 20/56 = 5/14
```

#### 全概率公式

**公式**：
```
设 B₁, B₂, ..., Bₙ 是完备事件组，则对任意事件A：

P(A) = Σ P(Bᵢ)P(A|Bᵢ)
```

**案例**：
```
某工厂三条生产线生产同一产品：
- 生产线1：产量50%，次品率2%
- 生产线2：产量30%，次品率3%
- 生产线3：产量20%，次品率4%

随机抽一件产品，求次品率：

P(次品) = 0.5×0.02 + 0.3×0.03 + 0.2×0.04
        = 0.01 + 0.009 + 0.008
        = 0.027 = 2.7%
```

#### 贝叶斯公式

**公式**：
```
P(Bᵢ|A) = P(Bᵢ)P(A|Bᵢ) / Σ P(Bⱼ)P(A|Bⱼ)
```

**案例**：
```
接上例，已知抽到次品，求来自生产线1的概率

P(生产线1|次品) = P(生产线1)P(次品|生产线1) / P(次品)
                = 0.5×0.02 / 0.027
                = 0.01 / 0.027
                ≈ 37%
```

#### 事件的独立性

**定义**：如果 P(A|B) = P(A)，则称事件A与B独立。

**等价条件**：
```
P(A∩B) = P(A)P(B)
```

**多个事件独立**：
```
A₁, A₂, ..., Aₙ 相互独立 ⟺ 
对任意子集 I ⊂ {1,2,...,n}：
P(∩_{i∈I} Aᵢ) = ∏_{i∈I} P(Aᵢ)
```

## 二、随机变量与分布

### 2.1 随机变量

#### 定义

**随机变量**：定义在样本空间Ω上的实值函数。

**类型**：
- **离散型**：取值为有限或可数无限个
- **连续型**：取值为某区间内的任意值

### 2.2 离散型随机变量

#### 概率质量函数（PMF）

**定义**：
```
P(X = xᵢ) = pᵢ

满足：
1. pᵢ ≥ 0
2. Σ pᵢ = 1
```

#### 常见离散分布

##### 1) 伯努利分布（Bernoulli）

**概率质量函数**：
```
P(X = k) = p^k (1-p)^{1-k},  k = 0, 1
```

**期望与方差**：
```
E(X) = p
Var(X) = p(1-p)
```

**案例**：
```
抛硬币，正面朝上的概率p=0.5
X~Bernoulli(0.5)
E(X) = 0.5
Var(X) = 0.25
```

##### 2) 二项分布（Binomial）

**概率质量函数**：
```
P(X = k) = C(n,k) p^k (1-p)^{n-k},  k = 0, 1, ..., n
```

**期望与方差**：
```
E(X) = np
Var(X) = np(1-p)
```

**案例**：
```
抛硬币10次，求正面出现6次的概率

X~Binomial(10, 0.5)
P(X=6) = C(10,6) × 0.5^6 × 0.5^4 = 210 × 0.5^10 ≈ 0.205
```

##### 3) 泊松分布（Poisson）

**概率质量函数**：
```
P(X = k) = (λ^k e^{-λ}) / k!,  k = 0, 1, 2, ...
```

**期望与方差**：
```
E(X) = λ
Var(X) = λ
```

**案例**：
```
某银行平均每小时接到5个投诉，求某小时接到3个投诉的概率

X~Poisson(5)
P(X=3) = (5^3 e^{-5}) / 3! = 125e^{-5}/6 ≈ 0.140
```

##### 4) 几何分布（Geometric）

**概率质量函数**：
```
P(X = k) = p(1-p)^{k-1},  k = 1, 2, 3, ...
```

**期望与方差**：
```
E(X) = 1/p
Var(X) = (1-p)/p^2
```

**案例**：
```
射击命中率0.3，求第5次才命中的概率

X~Geometric(0.3)
P(X=5) = 0.3 × 0.7^4 ≈ 0.072
```

### 2.3 连续型随机变量

#### 概率密度函数（PDF）

**定义**：
```
f(x) 满足：
1. f(x) ≥ 0
2. ∫_{-∞}^{+∞} f(x)dx = 1

P(a ≤ X ≤ b) = ∫_a^b f(x)dx
```

#### 累积分布函数（CDF）

**定义**：
```
F(x) = P(X ≤ x) = ∫_{-∞}^x f(t)dt
```

**性质**：
```
1. F(x) 单调不减
2. lim_{x→-∞} F(x) = 0
3. lim_{x→+∞} F(x) = 1
4. f(x) = F'(x)
```

#### 常见连续分布

##### 1) 均匀分布（Uniform）

**概率密度函数**：
```
f(x) = 1/(b-a),  a ≤ x ≤ b
f(x) = 0,  其他
```

**期望与方差**：
```
E(X) = (a+b)/2
Var(X) = (b-a)^2/12
```

**案例**：
```
某公交车间隔10分钟发车，乘客等待时间X~U(0,10)
E(X) = 5分钟
Var(X) = 100/12 ≈ 8.33分钟²
```

##### 2) 指数分布（Exponential）

**概率密度函数**：
```
f(x) = λe^{-λx},  x ≥ 0
f(x) = 0,  x < 0
```

**期望与方差**：
```
E(X) = 1/λ
Var(X) = 1/λ^2
```

**无记忆性**：
```
P(X > s+t | X > s) = P(X > t)
```

**案例**：
```
某设备平均寿命500小时，求设备运行1000小时以上的概率

X~Exponential(1/500)
P(X > 1000) = e^{-1000/500} = e^{-2} ≈ 0.135
```

##### 3) 正态分布（Normal/Gaussian）

**概率密度函数**：
```
f(x) = (1/√(2πσ^2)) exp(-(x-μ)^2/(2σ^2))
```

**记号**：
```
X ~ N(μ, σ^2)
```

**期望与方差**：
```
E(X) = μ
Var(X) = σ^2
```

**标准正态分布**：
```
Z ~ N(0, 1)
f(z) = (1/√(2π)) exp(-z^2/2)
```

**标准化**：
```
若 X ~ N(μ, σ^2)，则
Z = (X - μ)/σ ~ N(0, 1)
```

**经验法则（68-95-99.7法则）**：
```
P(μ-σ < X < μ+σ) ≈ 68.27%
P(μ-2σ < X < μ+2σ) ≈ 95.45%
P(μ-3σ < X < μ+3σ) ≈ 99.73%
```

**案例**：
```
某股票日收益率服从正态分布，均值0.1%，标准差2%
求日收益率大于5%的概率

P(X > 5%) = P(Z > (5-0.1)/2) = P(Z > 2.45) ≈ 0.71%
```

##### 4) t分布（Student's t）

**概率密度函数**：
```
f(x) = Γ((n+1)/2) / (Γ(n/2)√(nπ)) × (1 + x^2/n)^{-(n+1)/2}
```

**自由度**：n

**性质**：
```
1. 对称分布，E(X) = 0 (n > 1)
2. Var(X) = n/(n-2) (n > 2)
3. 当 n → ∞，t分布 → 标准正态分布
```

**应用**：小样本均值检验

##### 5) χ²分布（Chi-square）

**概率密度函数**：
```
f(x) = x^{(n/2)-1} e^{-x/2} / (2^{n/2} Γ(n/2)),  x > 0
```

**自由度**：n

**性质**：
```
1. E(X) = n
2. Var(X) = 2n
3. 若 X₁, X₂, ..., Xₙ 独立同分布于 N(0,1)，则
   X = X₁² + X₂² + ... + Xₙ² ~ χ²(n)
```

**应用**：方差检验、拟合优度检验

##### 6) F分布

**概率密度函数**：
```
f(x) = ...（复杂公式）
```

**性质**：
```
若 U ~ χ²(n₁), V ~ χ²(n₂) 独立，则
F = (U/n₁) / (V/n₂) ~ F(n₁, n₂)
```

**应用**：方差分析、回归检验

## 三、随机变量的数字特征

### 3.1 期望（均值）

#### 定义

**离散型**：
```
E(X) = Σ xᵢP(X = xᵢ)
```

**连续型**：
```
E(X) = ∫_{-∞}^{+∞} x f(x) dx
```

#### 性质

**性质1（线性性）**：
```
E(aX + b) = aE(X) + b
```

**性质2**：
```
E(X₁ + X₂) = E(X₁) + E(X₂)
```

**性质3**：
```
若X₁, X₂独立，则 E(X₁X₂) = E(X₁)E(X₂)
```

**性质4（条件期望）**：
```
E(X) = E[E(X|Y)]
```

### 3.2 方差

#### 定义

```
Var(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2
```

#### 性质

**性质1**：
```
Var(aX + b) = a²Var(X)
```

**性质2**：
```
Var(X) ≥ 0
```

**性质3**：
```
若X₁, X₂独立，则
Var(X₁ + X₂) = Var(X₁) + Var(X₂)
```

**性质4（切比雪夫不等式）**：
```
P(|X - μ| ≥ kσ) ≤ 1/k²
```

### 3.3 协方差与相关系数

#### 协方差

**定义**：
```
Cov(X, Y) = E[(X - E(X))(Y - E(Y))]
          = E(XY) - E(X)E(Y)
```

**性质**：
```
1. Cov(X, X) = Var(X)
2. Cov(X, Y) = Cov(Y, X)
3. Cov(aX, bY) = ab Cov(X, Y)
4. Cov(X₁ + X₂, Y) = Cov(X₁, Y) + Cov(X₂, Y)
```

#### 相关系数

**定义**：
```
ρ(X, Y) = Cov(X, Y) / (σ_X × σ_Y)
```

**性质**：
```
1. |ρ| ≤ 1
2. ρ = ±1 ⟺ Y = aX + b
3. ρ = 0 表示X与Y不相关（但不一定独立）
```

### 3.4 矩

**k阶原点矩**：
```
E(X^k)
```

**k阶中心矩**：
```
E[(X - E(X))^k]
```

**偏度（Skewness）**：
```
Skewness = E[(X - μ)^3] / σ³
```

- Skewness > 0：右偏（正偏）
- Skewness < 0：左偏（负偏）
- Skewness = 0：对称

**峰度（Kurtosis）**：
```
Kurtosis = E[(X - μ)^4] / σ⁴ - 3
```

- Kurtosis > 0：尖峰分布
- Kurtosis < 0：扁平分布
- Kurtosis = 0：正态分布

## 四、统计推断

### 4.1 抽样与抽样分布

#### 总体与样本

**总体**：研究对象的全体。

**样本**：从总体中抽取的部分个体。

**样本性质**：
- 随机性：每个个体被抽中的概率相同
- 独立性：每次抽取相互独立

#### 统计量

**定义**：样本的函数，不含未知参数。

**常见统计量**：

**样本均值**：
```
X̄ = (1/n) Σ Xᵢ
```

**样本方差**：
```
S² = (1/(n-1)) Σ (Xᵢ - X̄)²
```

**样本标准差**：
```
S = √S²
```

#### 抽样分布

##### 1) 样本均值的分布

**定理**：设 X₁, X₂, ..., Xₙ 独立同分布，E(Xᵢ) = μ，Var(Xᵢ) = σ²，则

```
E(X̄) = μ
Var(X̄) = σ²/n
```

**中心极限定理（CLT）**：
```
当 n → ∞ 时，
(X̄ - μ)/(σ/√n) → N(0, 1)
```

**正态总体**：
```
若 Xᵢ ~ N(μ, σ²)，则
X̄ ~ N(μ, σ²/n)
```

##### 2) 样本方差的分布

**定理**：设 Xᵢ ~ N(μ, σ²)，则

```
(n-1)S²/σ² ~ χ²(n-1)
```

**独立性**：
```
X̄ 与 S² 相互独立
```

##### 3) t统计量

**定理**：设 Xᵢ ~ N(μ, σ²)，则

```
T = (X̄ - μ)/(S/√n) ~ t(n-1)
```

### 4.2 点估计

#### 估计方法

##### 1) 矩估计法

**思想**：用样本矩估计总体矩。

**步骤**：
1. 计算总体矩 E(X^k)
2. 计算样本矩 (1/n) Σ Xᵢ^k
3. 令总体矩 = 样本矩，解方程

**案例**：
```
设 X ~ N(μ, σ²)，求 μ 和 σ² 的矩估计

E(X) = μ  →  X̄ = μ̂
E(X²) = μ² + σ²  →  (1/n)ΣXᵢ² = μ̂² + σ̂²

解得：
μ̂ = X̄
σ̂² = (1/n)Σ(Xᵢ - X̄)²
```

##### 2) 极大似然估计法（MLE）

**思想**：选择使样本出现概率最大的参数值。

**似然函数**：
```
L(θ) = L(θ; x₁, x₂, ..., xₙ) = ∏ f(xᵢ; θ)
```

**对数似然函数**：
```
l(θ) = ln L(θ) = Σ ln f(xᵢ; θ)
```

**求解**：
```
dl(θ)/dθ = 0
```

**案例**：
```
设 X₁, X₂, ..., Xₙ ~ N(μ, σ²)，求 μ 和 σ² 的MLE

l(μ, σ²) = -n/2 ln(2π) - n/2 ln(σ²) - (1/2σ²)Σ(xᵢ-μ)²

∂l/∂μ = (1/σ²)Σ(xᵢ-μ) = 0  →  μ̂ = X̄
∂l/∂σ² = -n/(2σ²) + (1/2σ⁴)Σ(xᵢ-μ)² = 0  →  σ̂² = (1/n)Σ(Xᵢ-X̄)²
```

#### 估计量的评价标准

##### 1) 无偏性

**定义**：
```
E(θ̂) = θ
```

**案例**：
```
样本方差 S² = (1/(n-1))Σ(Xᵢ-X̄)² 是 σ² 的无偏估计
样本二阶中心矩 Sₙ² = (1/n)Σ(Xᵢ-X̄)² 是 σ² 的有偏估计
```

##### 2) 有效性

**定义**：在无偏估计中，方差最小者最有效。

**案例**：
```
θ̂₁ 和 θ̂₂ 都是 θ 的无偏估计
若 Var(θ̂₁) < Var(θ̂₂)，则 θ̂₁ 比 θ̂₂ 更有效
```

##### 3) 一致性

**定义**：
```
对任意 ε > 0，lim_{n→∞} P(|θ̂ - θ| < ε) = 1
```

**大数定律保证**：样本均值是总体均值的一致估计。

### 4.3 区间估计

#### 置信区间

**定义**：对于参数 θ，若统计量 θ̂_L 和 θ̂_U 满足

```
P(θ̂_L ≤ θ ≤ θ̂_U) = 1 - α
```

则称 [θ̂_L, θ̂_U] 为 θ 的置信水平为 1-α 的置信区间。

#### 正态总体均值的置信区间

**σ已知**：
```
[X̄ - z_{α/2} σ/√n, X̄ + z_{α/2} σ/√n]
```

**σ未知**：
```
[X̄ - t_{α/2}(n-1) S/√n, X̄ + t_{α/2}(n-1) S/√n]
```

**案例**：
```
某股票日收益率样本：
n = 30, X̄ = 0.1%, S = 2%

95%置信区间：
[0.1% - t_{0.025}(29) × 2%/√30, 0.1% + t_{0.025}(29) × 2%/√30]
[0.1% - 2.045 × 0.365%, 0.1% + 2.045 × 0.365%]
[-0.65%, 0.85%]
```

#### 正态总体方差的置信区间

```
[(n-1)S²/χ²_{α/2}(n-1), (n-1)S²/χ²_{1-α/2}(n-1)]
```

### 4.4 假设检验

#### 基本概念

**原假设（H₀）**：待检验的假设。

**备择假设（H₁）**：与原假设对立的假设。

**显著性水平（α）**：犯第一类错误的概率上限。

**检验统计量**：用于判断的统计量。

**拒绝域**：使检验统计量落入的区域。

**两类错误**：
- 第一类错误（弃真）：H₀为真但拒绝H₀，P = α
- 第二类错误（取伪）：H₀为假但接受H₀，P = β

#### 检验步骤

1. 提出假设 H₀ 和 H₁
2. 选择检验统计量
3. 确定显著性水平 α
4. 计算检验统计量的值
5. 确定拒绝域
6. 做出判断

#### 正态总体均值检验

**σ已知（Z检验）**：
```
检验统计量：Z = (X̄ - μ₀)/(σ/√n)

拒绝域：
|Z| > z_{α/2}（双边检验）
Z > z_α（右边检验）
Z < -z_α（左边检验）
```

**σ未知（t检验）**：
```
检验统计量：t = (X̄ - μ₀)/(S/√n)

拒绝域：
|t| > t_{α/2}(n-1)（双边检验）
t > t_α(n-1)（右边检验）
t < -t_α(n-1)（左边检验）
```

**案例**：
```
某基金声称平均收益率为8%，随机抽取25个月度数据：
X̄ = 7.2%, S = 3%, α = 0.05

H₀: μ = 8% vs H₁: μ ≠ 8%

t = (7.2% - 8%)/(3%/√25) = -1.333

t_{0.025}(24) = 2.064

|t| < t_{0.025}(24)，不能拒绝H₀

结论：没有足够证据拒绝基金声称
```

#### 正态总体方差检验（χ²检验）

```
检验统计量：χ² = (n-1)S²/σ₀²

拒绝域：
χ² > χ²_{α/2}(n-1) 或 χ² < χ²_{1-α/2}(n-1)（双边检验）
χ² > χ²_α(n-1)（右边检验）
χ² < χ²_{1-α}(n-1)（左边检验）
```

#### p值

**定义**：在H₀为真的条件下，检验统计量取值比观测值更极端的概率。

**判断准则**：
```
若 p < α，则拒绝H₀
若 p ≥ α，则不能拒绝H₀
```

### 5.1 金融数据分析案例

**案例：股票收益率分析**

```python
import numpy as np
import pandas as pd
from scipy import stats

# 假设股票收益率数据
returns = np.array([0.02, -0.01, 0.03, 0.01, -0.02, 0.04, -0.01, 0.02, 0.01, -0.03])

# 描述性统计
mean_return = np.mean(returns)
std_return = np.std(returns, ddof=1)
skewness = stats.skew(returns)
kurtosis = stats.kurtosis(returns)

print(f"均值: {mean_return:.4f}")
print(f"标准差: {std_return:.4f}")
print(f"偏度: {skewness:.4f}")
print(f"峰度: {kurtosis:.4f}")

# 正态性检验（Jarque-Bera检验）
jb_stat, jb_pvalue = stats.jarque_bera(returns)
print(f"JB统计量: {jb_stat:.4f}")
print(f"p值: {jb_pvalue:.4f}")

# 均值检验（t检验）
t_stat, t_pvalue = stats.ttest_1samp(returns, 0)
print(f"t统计量: {t_stat:.4f}")
print(f"p值: {t_pvalue:.4f}")
```
