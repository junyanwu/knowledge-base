## 二、趋势跟踪策略

### 2.1 核心原理

**趋势跟踪假设**：
- 市场存在持续性趋势
- 趋势一旦形成会持续一段时间
- 可以通过技术指标识别趋势

**关键要素**：
- 趋势识别：均线、突破、动量
- 入场时机：趋势确认、突破信号
- 出场机制：趋势反转、止损止盈

### 2.2 经典策略

#### 均线交叉策略
```python
def ma_cross_strategy(prices, short_window=5, long_window=20):
    """均线交叉策略"""
    signals = pd.DataFrame(index=prices.index)
    signals['short_ma'] = prices.rolling(short_window).mean()
    signals['long_ma'] = prices.rolling(long_window).mean()
    signals['signal'] = 0
    
    # 金叉买入
    signals.loc[signals['short_ma'] > signals['long_ma'], 'signal'] = 1
    # 死叉卖出
    signals.loc[signals['short_ma'] < signals['long_ma'], 'signal'] = -1
    
    return signals
```

#### 突破策略
- **唐奇安通道**：价格突破N日最高价买入，跌破N日最低价卖出
- **海龟交易法则**：20日突破入场，10日突破离场
- **波动率突破**：价格突破波动率通道

#### 动态止损机制

```python
def dynamic_stop_loss(position, price, volatility):
    """动态止损"""
    # 基于波动率的止损幅度
    stop_distance = 2 * volatility
    
    if position > 0:  # 多头
        stop_price = price * (1 - stop_distance)
    else:  # 空头
        stop_price = price * (1 + stop_distance)
    
    return stop_price
```

## 三、均值回归策略

### 3.1 核心原理

**均值回归假设**：
- 价格围绕价值波动
- 极端偏离后会回归均值
- 可以通过统计方法识别偏离

**关键要素**：
- 均值定义：移动平均、价值中枢
- 偏离度量：标准差、百分位、Z-Score
- 回归时机：极端偏离、反转信号

### 3.2 经典策略

#### 布林带策略
```python
def bollinger_strategy(prices, window=20, num_std=2):
    """布林带均值回归策略"""
    signals = pd.DataFrame(index=prices.index)
    
    # 计算布林带
    signals['middle'] = prices.rolling(window).mean()
    signals['std'] = prices.rolling(window).std()
    signals['upper'] = signals['middle'] + num_std * signals['std']
    signals['lower'] = signals['middle'] - num_std * signals['std']
    
    # 交易信号
    signals['signal'] = 0
    # 触及下轨买入
    signals.loc[prices < signals['lower'], 'signal'] = 1
    # 触及上轨卖出
    signals.loc[prices > signals['upper'], 'signal'] = -1
    
    return signals
```

#### 配对交易
```python
def pairs_trading(asset1, asset2, window=30, threshold=2):
    """配对交易策略"""
    # 计算价差
    spread = asset1 - asset2
    mean = spread.rolling(window).mean()
    std = spread.rolling(window).std()
    z_score = (spread - mean) / std
    
    signals = pd.DataFrame(index=spread.index)
    signals['spread'] = spread
    signals['z_score'] = z_score
    
    # 交易信号
    signals['signal'] = 0
    signals.loc[z_score > threshold, 'signal'] = -1  # 做空价差
    signals.loc[z_score < -threshold, 'signal'] = 1  # 做多价差
    
    return signals
```

### 3.3 风险管理

#### 极端行情下的风险
- **均值回归策略可能持续失效**：极端行情下价格偏离可能持续扩大
- **建议**：暂停或大幅减仓，等待波动率回归正常

#### 止损机制
```python
def mean_reversion_stop_loss(entry_price, current_price, max_deviation=3):
    """均值回归止损"""
    deviation = abs(current_price - entry_price) / entry_price
    
    if deviation > max_deviation:
        return True  # 触发止损
    return False
```

## 四、波动率择时

### 4.1 波动率周期

**波动率聚类**：
- 波动率具有聚集性
- 高波动期后会跟随高波动期
- 低波动期后会跟随低波动期

**波动率均值回归**：
- 波动率长期会回归历史均值
- 极端高波动不可持续
- 极端低波动后通常爆发

### 4.2 VIX择时策略

```python
def vix_timing_strategy(vix, threshold_high=30, threshold_low=15):
    """VIX择时策略"""
    signals = pd.DataFrame(index=vix.index)
    signals['vix'] = vix
    signals['signal'] = 0
    
    # 高波动期：降低仓位
    signals.loc[vix > threshold_high, 'signal'] = -1
    # 低波动期：增加仓位
    signals.loc[vix < threshold_low, 'signal'] = 1
    
    return signals
```

### 4.3 动态波动率管理（DVM）

```python
class DynamicVolatilityManagement:
    """动态波动率管理"""
    
    def __init__(self, base_volatility=0.15):
        self.base_vol = base_volatility
    
    def adjust_position(self, current_volatility, base_position=0.6):
        """根据波动率调整仓位"""
        # 仓位调整系数 = 基准波动率 / 当前波动率
        position_multiplier = self.base_vol / current_volatility
        
        # 极端行情系数：单日波动>10%时，额外降仓30%
        if current_volatility > 0.10:
            position_multiplier *= 0.7
        
        adjusted_position = base_position * position_multiplier
        return min(adjusted_position, 1.0)  # 不超过100%
    
    def adjust_stop_loss(self, current_volatility):
        """根据波动率调整止损幅度"""
        # 止损调整系数 = sqrt(基准波动率 / 当前波动率)
        stop_multiplier = np.sqrt(self.base_vol / current_volatility)
        
        base_stop = 0.08  # 基准止损8%
        return base_stop * stop_multiplier
```

### 4.4 隐马尔可夫模型（HMM）择时

#### HMM择时原理
- 假设市场存在"隐藏"状态（牛市、熊市、震荡）
- 通过收益率序列推断当前状态
- 不同状态采用不同策略

```python
from hmmlearn.hmm import GaussianHMM

def train_hmm_model(returns, n_states=2):
    """训练HMM模型识别市场状态"""
    model = GaussianHMM(
        n_components=n_states,
        covariance_type="full",
        n_iter=1000
    )
    model.fit(returns)
    return model

# 应用：低波动状态允许交易，高波动状态禁止开仓
```
