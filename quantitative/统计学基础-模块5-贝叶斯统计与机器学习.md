## 1. 贝叶斯统计基础

### 1.1 贝叶斯定理

**贝叶斯公式：**
```
P(θ|D) = P(D|θ)P(θ) / P(D)
```

其中：
- P(θ|D)：后验概率（Posterior）
- P(D|θ)：似然函数（Likelihood）
- P(θ)：先验概率（Prior）
- P(D)：边际似然（Marginal Likelihood）

**贝叶斯推断流程：**
```
先验信息 + 数据 → 后验分布
```

### 1.2 先验分布

**共轭先验：**

| 似然 | 参数 | 共轭先验 | 后验分布 |
|------|------|---------|---------|
| 二项分布 | p | Beta分布 | Beta分布 |
| 泊松分布 | λ | Gamma分布 | Gamma分布 |
| 正态分布(σ²已知) | μ | 正态分布 | 正态分布 |
| 正态分布(μ已知) | σ² | 逆Gamma分布 | 逆Gamma分布 |

**示例：正态分布均值推断**
```python
import numpy as np
from scipy.stats import norm

def bayesian_normal_mean(data, prior_mean, prior_var, data_var):
    """
    正态分布均值的贝叶斯推断
    
    参数:
    data: 观测数据
    prior_mean: 先验均值
    prior_var: 先验方差
    data_var: 数据方差（已知）
    
    返回:
    后验均值、后验方差
    """
    n = len(data)
    data_mean = np.mean(data)
    
    # 后验方差
    posterior_var = 1 / (1/prior_var + n/data_var)
    
    # 后验均值
    posterior_mean = posterior_var * (prior_mean/prior_var + n*data_mean/data_var)
    
    return posterior_mean, posterior_var

# 示例
data = np.random.normal(5, 2, 100)  # 真实均值5
posterior_mean, posterior_var = bayesian_normal_mean(
    data, 
    prior_mean=0, 
    prior_var=10,
    data_var=4
)

print(f'后验均值: {posterior_mean:.2f}')
print(f'后验方差: {posterior_var:.4f}')
```

### 1.3 MCMC方法

**Metropolis-Hastings算法：**
```python
def metropolis_hastings(data, log_likelihood, log_prior, n_samples=10000):
    """
    Metropolis-Hastings采样
    
    参数:
    data: 观测数据
    log_likelihood: 对数似然函数
    log_prior: 对数先验函数
    n_samples: 采样数量
    
    返回:
    后验样本
    """
    samples = []
    current = np.random.randn()  # 初始值
    
    for i in range(n_samples):
        # 提议分布
        proposal = current + np.random.randn() * 0.5
        
        # 计算接受概率
        log_ratio = (log_likelihood(proposal, data) + log_prior(proposal) - 
                     log_likelihood(current, data) - log_prior(current))
        
        # 接受或拒绝
        if np.log(np.random.rand()) < log_ratio:
            current = proposal
        
        samples.append(current)
    
    return np.array(samples)

# 示例：估计正态分布均值
def log_likelihood(theta, data):
    return np.sum(norm.logpdf(data, loc=theta, scale=1))

def log_prior(theta):
    return norm.logpdf(theta, loc=0, scale=10)  # 先验N(0, 10)

data = np.random.normal(5, 1, 100)
samples = metropolis_hastings(data, log_likelihood, log_prior)

# 后验统计
print(f'后验均值: {samples[1000:].mean():.2f}')  # 舍弃burn-in
print(f'后验标准差: {samples[1000:].std():.2f}')
```

## 2. 贝叶斯线性回归

### 2.1 模型设定

**贝叶斯线性回归：**
```
y = Xβ + ε, ε ~ N(0, σ²I)

β ~ N(μ₀, Σ₀)  # 参数先验
σ² ~ Inv-χ²(ν₀, σ₀²)  # 方差先验
```

**后验分布：**
```
β|y,X,σ² ~ N(μₙ, Σₙ)

μₙ = Σₙ(Σ₀⁻¹μ₀ + X'y/σ²)
Σₙ = (Σ₀⁻¹ + X'X/σ²)⁻¹
```

### 2.2 Python实现

```python

with pm.Model() as model:
    # 先验
    beta = pm.Normal('beta', mu=0, sd=10, shape=2)
    sigma = pm.HalfNormal('sigma', sd=1)
    
    # 似然
    mu = X @ beta
    likelihood = pm.Normal('y', mu=mu, sd=sigma, observed=y)
    
    # 采样
    trace = pm.sample(3000, tune=1000, cores=2)
```

## 3. 贝叶斯模型选择

### 3.1 贝叶斯因子

**定义：**
```
BF = P(D|M₁) / P(D|M₂)
```

**解释：**
- BF > 1：模型1更优
- BF < 1：模型2更优
- BF > 10：强证据支持模型1
- BF > 100：决定性证据

**Python实现：**
```python
def bayes_factor(trace1, trace2):
    """
    计算贝叶斯因子
    
    参数:
    trace1: 模型1的MCMC样本
    trace2: 模型2的MCMC样本
    
    返回:
    贝叶斯因子
    """
    # 边际似然估计（调和平均）
    log_marginal1 = -np.log(np.mean(1 / np.exp(trace1['log_likelihood'])))
    log_marginal2 = -np.log(np.mean(1 / np.exp(trace2['log_likelihood'])))
    
    log_bf = log_marginal1 - log_marginal2
    
    return np.exp(log_bf)
```

### 3.2 信息准则

**DIC（Deviance Information Criterion）：**
```python
def dic(trace, model):
    """
    计算DIC
    
    参数:
    trace: MCMC样本
    model: PyMC3模型
    
    返回:
    DIC值
    """
    # 后验均值处的偏差
    mean_params = {var: trace[var].mean() for var in trace.varnames}
    D_mean = model.logp(mean_params)
    
    # 平均偏差
    D_bar = np.mean([model.logp(point) for point in trace.points()])
    
    # 有效参数数量
    p_D = D_bar - D_mean
    
    # DIC
    dic = D_bar + p_D
    
    return dic
```

**WAIC（Widely Applicable Information Criterion）：**
```python
def waic(trace, model, y):
    """
    计算WAIC
    
    参数:
    trace: MCMC样本
    model: PyMC3模型
    y: 观测数据
    
    返回:
    WAIC值
    """
    # 点对点对数似然
    log_lik = np.array([[model.logp(point, y_i) for y_i in y] 
                        for point in trace.points()])
    
    # lppd
    lppd = np.sum(np.log(np.mean(np.exp(log_lik), axis=0)))
    
    # p_waic
    p_waic = np.sum(np.var(log_lik, axis=0))
    
    # WAIC
    waic = -2 * (lppd - p_waic)
    
    return waic
```

## 4. 贝叶斯时间序列

### 4.1 贝叶斯AR模型

```python
import pymc3 as pm

def bayesian_ar(data, p=1):
    """
    贝叶斯AR(p)模型
    
    参数:
    data: 时间序列数据
    p: AR阶数
    
    返回:
    模型、后验样本
    """
    with pm.Model() as model:
        # 滞后矩阵
        X = np.array([data[i:-p+i-1] for i in range(p)]).T
        y = data[p+1:]
        
        # 先验
        phi = pm.Normal('phi', mu=0, sd=10, shape=p)
        sigma = pm.HalfNormal('sigma', sd=1)
        
        # 似然
        mu = X @ phi
        likelihood = pm.Normal('y', mu=mu, sd=sigma, observed=y)
        
        # 采样
        trace = pm.sample(2000, tune=1000)
    
    return model, trace

# 示例
data = np.random.randn(100)
model, trace = bayesian_ar(data, p=2)

print('AR系数后验:')
print(trace['phi'].mean(axis=0))
```

### 4.2 贝叶斯状态空间模型

```python
def bayesian_state_space(y):
    """
    贝叶斯状态空间模型（局部水平模型）
    
    参数:
    y: 观测序列
    
    返回:
    模型、后验样本
    """
    with pm.Model() as model:
        n = len(y)
        
        # 先验
        sigma_obs = pm.HalfNormal('sigma_obs', sd=1)
        sigma_state = pm.HalfNormal('sigma_state', sd=1)
        
        # 状态随机游走
        level = pm.GaussianRandomWalk('level', sd=sigma_state, shape=n)
        
        # 观测
        likelihood = pm.Normal('y', mu=level, sd=sigma_obs, observed=y)
        
        # 采样
        trace = pm.sample(2000, tune=1000)
    
    return model, trace
```

```python
def bayesian_var(returns, alpha=0.05, n_samples=10000):
    """
    贝叶斯VaR估计
    
    参数:
    returns: 收益率序列
    alpha: 置信水平
    n_samples: 采样数量
    
    返回:
    VaR分布
    """
    with pm.Model() as model:
        # 均值先验
        mu = pm.Normal('mu', mu=0, sd=0.1)
        
        # 标准差先验
        sigma = pm.HalfNormal('sigma', sd=0.1)
        
        # 似然（t分布）
        nu = pm.Exponential('nu', 1/10)
        likelihood = pm.StudentT('returns', nu=nu, mu=mu, sd=sigma, observed=returns)
        
        # 采样
        trace = pm.sample(n_samples, tune=1000)
    
    # 计算VaR
    var_samples = []
    for i in range(n_samples):
        mu_sample = trace['mu'][i]
        sigma_sample = trace['sigma'][i]
        nu_sample = trace['nu'][i]
        
        # t分布分位数
        var = mu_sample + sigma_sample * pm.tdist.ppf(alpha, nu_sample)
        var_samples.append(var)
    
    return np.array(var_samples)
```

## 6. 总结

贝叶斯统计要点：

1. **贝叶斯定理**：后验 = 似然 × 先验 / 边际似然
2. **先验选择**：共轭先验、无信息先验
3. **MCMC方法**：Metropolis-Hastings、Gibbs采样
4. **贝叶斯线性回归**：参数不确定性量化
5. **模型选择**：贝叶斯因子、DIC、WAIC
6. **金融应用**：投资组合优化、VaR估计

**优势：**
- 参数不确定性量化
- 小样本有效
- 自然处理模型不确定性
- 先验信息融入
