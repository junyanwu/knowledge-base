## 2. 风险预算理论

### 2.1 风险贡献计算

**风险贡献定义**：

对于组合权重 w 和协方差矩阵 Σ，组合波动率为：
```
σ_p = √(w'Σw)
```

**边际风险贡献（MRC）**：
```
MRC_i = ∂σ_p / ∂w_i = (Σw)_i / σ_p
```

**风险贡献（RC）**：
```
RC_i = w_i × MRC_i = w_i × (Σw)_i / σ_p
```

**风险贡献比例（RC%）**：
```
RC%_i = RC_i / σ_p = w_i × (Σw)_i / σ_p²
```

**性质**：
- Σ RC_i = σ_p
- Σ RC%_i = 1

```python
import numpy as np
import pandas as pd
from scipy.optimize import minimize
from typing import Dict, List, Tuple

class RiskBudgetCalculator:
    """风险预算计算器"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 weights: np.ndarray = None):
        """
        初始化
        
        Args:
            returns: 资产收益率DataFrame
            weights: 组合权重
        """
        self.returns = returns
        self.n_assets = returns.shape[1]
        self.asset_names = returns.columns.tolist()
        
        # 默认等权重
        if weights is None:
            self.weights = np.ones(self.n_assets) / self.n_assets
        else:
            self.weights = weights
        
        # 计算协方差矩阵
        self.cov_matrix = returns.cov().values * 252
    
    def calculate_portfolio_volatility(self, 
                                      weights: np.ndarray = None) -> float:
        """
        计算组合波动率
        
        Args:
            weights: 权重（None则使用self.weights）
            
        Returns:
            组合波动率
        """
        if weights is None:
            weights = self.weights
        
        return np.sqrt(np.dot(weights.T, np.dot(self.cov_matrix, weights)))
    
    def calculate_marginal_risk_contribution(self, 
                                            weights: np.ndarray = None) -> np.ndarray:
        """
        计算边际风险贡献
        
        Args:
            weights: 权重
            
        Returns:
            边际风险贡献向量
        """
        if weights is None:
            weights = self.weights
        
        portfolio_vol = self.calculate_portfolio_volatility(weights)
        marginal_risk = np.dot(self.cov_matrix, weights) / portfolio_vol
        
        return marginal_risk
    
    def calculate_risk_contribution(self, 
                                   weights: np.ndarray = None) -> np.ndarray:
        """
        计算风险贡献
        
        Args:
            weights: 权重
            
        Returns:
            风险贡献向量
        """
        if weights is None:
            weights = self.weights
        
        portfolio_vol = self.calculate_portfolio_volatility(weights)
        marginal_risk = self.calculate_marginal_risk_contribution(weights)
        risk_contrib = weights * marginal_risk
        
        return risk_contrib
    
    def calculate_risk_contribution_pct(self, 
                                       weights: np.ndarray = None) -> np.ndarray:
        """
        计算风险贡献比例
        
        Args:
            weights: 权重
            
        Returns:
            风险贡献比例向量
        """
        if weights is None:
            weights = self.weights
        
        portfolio_vol = self.calculate_portfolio_volatility(weights)
        risk_contrib = self.calculate_risk_contribution(weights)
        risk_contrib_pct = risk_contrib / portfolio_vol
        
        return risk_contrib_pct
    
    def risk_decomposition_report(self, 
                                 weights: np.ndarray = None) -> pd.DataFrame:
        """
        风险分解报告
        
        Args:
            weights: 权重
            
        Returns:
            风险分解报告
        """
        if weights is None:
            weights = self.weights
        
        portfolio_vol = self.calculate_portfolio_volatility(weights)
        marginal_risk = self.calculate_marginal_risk_contribution(weights)
        risk_contrib = self.calculate_risk_contribution(weights)
        risk_contrib_pct = self.calculate_risk_contribution_pct(weights)
        
        # 资产波动率
        asset_vols = np.sqrt(np.diag(self.cov_matrix))
        
        report = pd.DataFrame({
            'Asset': self.asset_names,
            'Weight': weights,
            'Volatility': asset_vols,
            'MRC': marginal_risk,
            'RC': risk_contrib,
            'RC%': risk_contrib_pct
        })
        
        return report
```

### 2.2 边际风险贡献

**定义**：边际风险贡献表示权重微小变化对组合风险的影响

**应用**：
- 风险归因：识别主要风险来源
- 风险调整：调整权重以降低风险
- 风险预算：设定风险预算目标

```python
    def analyze_marginal_risk(self, 
                             weights: np.ndarray = None) -> Dict:
        """
        分析边际风险贡献
        
        Args:
            weights: 权重
            
        Returns:
            边际风险分析
        """
        if weights is None:
            weights = self.weights
        
        marginal_risk = self.calculate_marginal_risk_contribution(weights)
        portfolio_vol = self.calculate_portfolio_volatility(weights)
        
        # 识别高边际风险资产
        high_mrc_assets = [
            self.asset_names[i] for i in np.argsort(marginal_risk)[-3:]
        ]
        
        # 识别低边际风险资产
        low_mrc_assets = [
            self.asset_names[i] for i in np.argsort(marginal_risk)[:3]
        ]
        
        # 风险敏感度
        sensitivity = marginal_risk * weights / portfolio_vol
        
        return {
            'marginal_risk': dict(zip(self.asset_names, marginal_risk)),
            'high_mrc_assets': high_mrc_assets,
            'low_mrc_assets': low_mrc_assets,
            'sensitivity': dict(zip(self.asset_names, sensitivity)),
            'mrc_contribution': dict(zip(self.asset_names, marginal_risk * weights))
        }
    
    def risk_reduction_simulation(self, 
                                  weights: np.ndarray = None,
                                  reduction_target: float = 0.1) -> Dict:
        """
        风险降低模拟
        
        Args:
            weights: 权重
            reduction_target: 风险降低目标（%）
            
        Returns:
            风险降低方案
        """
        if weights is None:
            weights = self.weights
        
        current_vol = self.calculate_portfolio_volatility(weights)
        target_vol = current_vol * (1 - reduction_target)
        
        # 方法1：降低高边际风险资产权重
        marginal_risk = self.calculate_marginal_risk_contribution(weights)
        high_mrc_idx = np.argmax(marginal_risk)
        
        adjusted_weights = weights.copy()
        reduction = 0.0
        step = 0.01
        
        while self.calculate_portfolio_volatility(adjusted_weights) > target_vol:
            adjusted_weights[high_mrc_idx] -= step
            adjusted_weights = adjusted_weights / adjusted_weights.sum()
            reduction += step
            
            if adjusted_weights[high_mrc_idx] <= 0.01:
                break
        
        # 方法2：风险平价
        rp_weights = self.risk_parity_weights()
        rp_vol = self.calculate_portfolio_volatility(rp_weights)
        
        return {
            'current_volatility': current_vol,
            'target_volatility': target_vol,
            'method1': {
                'weights': adjusted_weights,
                'volatility': self.calculate_portfolio_volatility(adjusted_weights),
                'reduction': reduction
            },
            'method2': {
                'weights': rp_weights,
                'volatility': rp_vol
            }
        }
```

### 2.3 风险预算分配方法

**风险平价**：每个资产风险贡献相等

**逆波动率**：权重与波动率成反比

**自定义风险预算**：根据投资观点设定风险预算

```python
    def risk_parity_weights(self) -> np.ndarray:
        """
        风险平价权重（等风险贡献）
        
        Returns:
            风险平价权重
        """
        def risk_parity_objective(weights):
            portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(self.cov_matrix, weights)))
            marginal_risk = np.dot(self.cov_matrix, weights) / portfolio_vol
            risk_contrib_pct = weights * marginal_risk / portfolio_vol
            target = np.ones(self.n_assets) / self.n_assets
            return np.sum((risk_contrib_pct - target) ** 2)
        
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        bounds = tuple((0.001, 1) for _ in range(self.n_assets))
        init_weights = np.ones(self.n_assets) / self.n_assets
        
        result = minimize(
            risk_parity_objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        return result.x
    
    def inverse_volatility_weights(self) -> np.ndarray:
        """
        逆波动率权重
        
        Returns:
            逆波动率权重
        """
        asset_vols = np.sqrt(np.diag(self.cov_matrix))
        weights = 1 / asset_vols
        return weights / np.sum(weights)
    
    def risk_budget_weights(self, 
                           risk_budget: np.ndarray = None) -> np.ndarray:
        """
        风险预算权重
        
        Args:
            risk_budget: 风险预算（None则等风险贡献）
            
        Returns:
            风险预算权重
        """
        if risk_budget is None:
            risk_budget = np.ones(self.n_assets) / self.n_assets
        
        risk_budget = risk_budget / np.sum(risk_budget)
        
        def risk_budget_objective(weights):
            portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(self.cov_matrix, weights)))
            marginal_risk = np.dot(self.cov_matrix, weights) / portfolio_vol
            risk_contrib_pct = weights * marginal_risk / portfolio_vol
            return np.sum((risk_contrib_pct - risk_budget) ** 2)
        
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        bounds = tuple((0.001, 1) for _ in range(self.n_assets))
        init_weights = np.ones(self.n_assets) / self.n_assets
        
        result = minimize(
            risk_budget_objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        return result.x
    
    def compare_risk_methods(self) -> pd.DataFrame:
        """
        对比不同风险预算方法
        
        Returns:
            对比结果
        """
        # 等权重
        equal_weights = np.ones(self.n_assets) / self.n_assets
        equal_rc = self.calculate_risk_contribution_pct(equal_weights)
        
        # 逆波动率
        iv_weights = self.inverse_volatility_weights()
        iv_rc = self.calculate_risk_contribution_pct(iv_weights)
        
        # 风险平价
        rp_weights = self.risk_parity_weights()
        rp_rc = self.calculate_risk_contribution_pct(rp_weights)
        
        results = []
        for i, asset in enumerate(self.asset_names):
            results.append({
                'Asset': asset,
                'Equal_Weight': equal_weights[i],
                'Equal_RC%': equal_rc[i],
                'IV_Weight': iv_weights[i],
                'IV_RC%': iv_rc[i],
                'RP_Weight': rp_weights[i],
                'RP_RC%': rp_rc[i]
            })
        
        return pd.DataFrame(results)
```

```python
class AssetLevelRiskBudget:
    """资产层级风险预算"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 asset_classes: Dict[str, str]):
        """
        初始化
        
        Args:
            returns: 资产收益率
            asset_classes: 资产类别映射 {资产: 类别}
        """
        self.returns = returns
        self.asset_classes = asset_classes
        self.asset_names = returns.columns.tolist()
        self.class_names = list(set(asset_classes.values()))
    
    def class_level_risk_budget(self, 
                               class_budget: Dict[str, float] = None) -> Dict:
        """
        资产类别层级风险预算
        
        Args:
            class_budget: 类别风险预算
            
        Returns:
            类别风险预算分配
        """
        if class_budget is None:
            # 等风险预算
            class_budget = {c: 1.0 / len(self.class_names) for c in self.class_names}
        
        # 计算类别收益
        class_returns = {}
        for cls in self.class_names:
            assets = [a for a in self.asset_names if self.asset_classes[a] == cls]
            class_returns[cls] = self.returns[assets].mean(axis=1)
        
        class_returns_df = pd.DataFrame(class_returns)
        class_cov = class_returns_df.cov().values * 252
        
        # 类别权重
        class_budget_arr = np.array([class_budget[c] for c in self.class_names])
        
        def class_budget_objective(weights):
            portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(class_cov, weights)))
            marginal_risk = np.dot(class_cov, weights) / portfolio_vol
            risk_contrib_pct = weights * marginal_risk / portfolio_vol
            return np.sum((risk_contrib_pct - class_budget_arr) ** 2)
        
        n_classes = len(self.class_names)
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        bounds = tuple((0.001, 1) for _ in range(n_classes))
        init_weights = np.ones(n_classes) / n_classes
        
        result = minimize(
            class_budget_objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        class_weights = dict(zip(self.class_names, result.x))
        
        # 类别内资产权重
        asset_weights = {}
        for cls in self.class_names:
            assets = [a for a in self.asset_names if self.asset_classes[a] == cls]
            n_assets = len(assets)
            
            # 类别内等权重
            for asset in assets:
                asset_weights[asset] = class_weights[cls] / n_assets
        
        return {
            'class_weights': class_weights,
            'asset_weights': asset_weights,
            'class_budget': class_budget
        }
```

```python
class StrategyLevelRiskBudget:
    """策略层级风险预算"""
    
    def __init__(self, 
                 strategy_returns: pd.DataFrame,
                 strategy_categories: Dict[str, str]):
        """
        初始化
        
        Args:
            strategy_returns: 策略收益率
            strategy_categories: 策略分类 {策略: 类别}
        """
        self.returns = strategy_returns
        self.categories = strategy_categories
        self.strategy_names = strategy_returns.columns.tolist()
        self.category_names = list(set(strategy_categories.values()))
    
    def strategy_risk_budget(self, 
                            category_budget: Dict[str, float] = None) -> Dict:
        """
        策略风险预算
        
        Args:
            category_budget: 类别风险预算
            
        Returns:
            策略权重分配
        """
        if category_budget is None:
            category_budget = {c: 1.0 / len(self.category_names) for c in self.category_names}
        
        # 类别内策略数量
        strategy_per_category = {}
        for cat in self.category_names:
            strategies = [s for s in self.strategy_names if self.categories[s] == cat]
            strategy_per_category[cat] = strategies
        
        # 类别权重（基于风险预算）
        class_risk_budget = StrategyLevelRiskBudget(
            self.returns.groupby(self.categories, axis=1).mean(),
            {c: c for c in self.category_names}
        )
        
        # 类别权重
        category_budget_arr = np.array([category_budget[c] for c in self.category_names])
        category_cov = self.returns.groupby(self.categories, axis=1).mean().cov().values * 252
        
        def cat_budget_objective(weights):
            portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(category_cov, weights)))
            marginal_risk = np.dot(category_cov, weights) / portfolio_vol
            risk_contrib_pct = weights * marginal_risk / portfolio_vol
            return np.sum((risk_contrib_pct - category_budget_arr) ** 2)
        
        n_cat = len(self.category_names)
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        bounds = tuple((0.001, 1) for _ in range(n_cat))
        init_weights = np.ones(n_cat) / n_cat
        
        result = minimize(
            cat_budget_objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        category_weights = dict(zip(self.category_names, result.x))
        
        # 策略权重
        strategy_weights = {}
        for cat in self.category_names:
            strategies = strategy_per_category[cat]
            n_strategies = len(strategies)
            
            # 类别内等风险贡献
            for strategy in strategies:
                strategy_weights[strategy] = category_weights[cat] / n_strategies
        
        return {
            'category_weights': category_weights,
            'strategy_weights': strategy_weights,
            'category_budget': category_budget
        }
```

```python
class FactorLevelRiskBudget:
    """因子层级风险预算"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 factor_exposures: np.ndarray,
                 factor_names: List[str]):
        """
        初始化
        
        Args:
            returns: 资产收益率
            factor_exposures: 因子暴露矩阵 (n_assets, n_factors)
            factor_names: 因子名称
        """
        self.returns = returns
        self.factor_exposures = factor_exposures
        self.factor_names = factor_names
        self.n_assets = returns.shape[1]
        self.n_factors = len(factor_names)
    
    def factor_risk_budget(self, 
                          factor_budget: np.ndarray = None) -> Dict:
        """
        因子风险预算
        
        Args:
            factor_budget: 因子风险预算
            
        Returns:
            因子风险预算分配
        """
        if factor_budget is None:
            factor_budget = np.ones(self.n_factors) / self.n_factors
        
        factor_budget = factor_budget / np.sum(factor_budget)
        
        # 估计因子协方差矩阵
        factor_returns = self.returns.values @ np.linalg.pinv(self.factor_exposures.T)
        factor_cov = np.cov(factor_returns.T) * 252
        
        def factor_budget_objective(weights):
            # 因子暴露
            portfolio_factor_exposure = weights @ self.factor_exposures
            
            # 因子风险贡献
            portfolio_factor_var = portfolio_factor_exposure @ factor_cov @ portfolio_factor_exposure
            marginal_factor_risk = factor_cov @ portfolio_factor_exposure / np.sqrt(portfolio_factor_var)
            factor_risk_contrib = portfolio_factor_exposure * marginal_factor_risk / np.sqrt(portfolio_factor_var)
            
            return np.sum((factor_risk_contrib - factor_budget) ** 2)
        
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        bounds = tuple((0.001, 1) for _ in range(self.n_assets))
        init_weights = np.ones(self.n_assets) / self.n_assets
        
        result = minimize(
            factor_budget_objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        asset_weights = result.x
        
        # 计算因子风险贡献
        portfolio_factor_exposure = asset_weights @ self.factor_exposures
        portfolio_factor_var = portfolio_factor_exposure @ factor_cov @ portfolio_factor_exposure
        marginal_factor_risk = factor_cov @ portfolio_factor_exposure / np.sqrt(portfolio_factor_var)
        factor_risk_contrib = portfolio_factor_exposure * marginal_factor_risk / np.sqrt(portfolio_factor_var)
        
        return {
            'asset_weights': asset_weights,
            'factor_exposures': portfolio_factor_exposure,
            'factor_risk_contrib': dict(zip(self.factor_names, factor_risk_contrib)),
            'factor_budget': factor_budget
        }
```

```python
class RiskBudgetOptimizer:
    """风险预算优化器"""
    
    def __init__(self, 
                 returns: pd.DataFrame):
        """
        初始化
        
        Args:
            returns: 收益率数据
        """
        self.returns = returns
        self.n_assets = returns.shape[1]
        self.asset_names = returns.columns.tolist()
        self.mean_returns = returns.mean() * 252
        self.cov_matrix = returns.cov() * 252
    
    def optimize_with_risk_budget(self, 
                                  risk_budget: np.ndarray,
                                  target_return: float = None,
                                  max_weight: float = 1.0) -> Dict:
        """
        风险预算约束优化
        
        Args:
            risk_budget: 风险预算
            target_return: 目标收益（可选）
            max_weight: 最大权重
            
        Returns:
            优化结果
        """
        risk_budget = risk_budget / np.sum(risk_budget)
        
        def objective(weights):
            # 组合波动率
            portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(self.cov_matrix, weights)))
            
            # 风险贡献
            marginal_risk = np.dot(self.cov_matrix, weights) / portfolio_vol
            risk_contrib_pct = weights * marginal_risk / portfolio_vol
            
            # 风险预算偏离
            budget_error = np.sum((risk_contrib_pct - risk_budget) ** 2)
            
            return budget_error
        
        # 约束条件
        constraints = [{'type': 'eq', 'fun': lambda w: np.sum(w) - 1}]
        
        # 目标收益约束
        if target_return is not None:
            constraints.append({
                'type': 'eq',
                'fun': lambda w, r=target_return: np.dot(w, self.mean_returns) - r
            })
        
        bounds = tuple((0.001, max_weight) for _ in range(self.n_assets))
        init_weights = np.ones(self.n_assets) / self.n_assets
        
        result = minimize(
            objective,
            init_weights,
            method='SLSQP',
            bounds=bounds,
            constraints=constraints,
            options={'ftol': 1e-12, 'maxiter': 1000}
        )
        
        weights = result.x
        
        # 计算风险贡献
        portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(self.cov_matrix, weights)))
        marginal_risk = np.dot(self.cov_matrix, weights) / portfolio_vol
        risk_contrib_pct = weights * marginal_risk / portfolio_vol
        
        portfolio_return = np.dot(weights, self.mean_returns)
        
        return {
            'weights': dict(zip(self.asset_names, weights)),
            'expected_return': portfolio_return,
            'volatility': portfolio_vol,
            'sharpe_ratio': portfolio_return / portfolio_vol,
            'risk_contribution': dict(zip(self.asset_names, risk_contrib_pct)),
            'risk_budget': dict(zip(self.asset_names, risk_budget)),
            'optimization_success': result.success
        }
```

### 4.2 风险预算监控

```python
class RiskBudgetMonitor:
    """风险预算监控"""
    
    def __init__(self, 
                 returns: pd.DataFrame,
                 target_risk_budget: np.ndarray):
        """
        初始化
        
        Args:
            returns: 收益率数据
            target_risk_budget: 目标风险预算
        """
        self.returns = returns
        self.target_risk_budget = target_risk_budget / np.sum(target_risk_budget)
        self.n_assets = returns.shape[1]
        self.asset_names = returns.columns.tolist()
    
    def monitor_risk_budget(self, 
                           current_weights: np.ndarray,
                           window: int = 60) -> Dict:
        """
        监控风险预算偏离
        
        Args:
            current_weights: 当前权重
            window: 滚动窗口
            
        Returns:
            监控结果
        """
        # 滚动协方差
        recent_returns = self.returns.iloc[-window:]
        cov_matrix = recent_returns.cov().values * 252
        
        # 当前风险贡献
        portfolio_vol = np.sqrt(np.dot(current_weights.T, np.dot(cov_matrix, current_weights)))
        marginal_risk = np.dot(cov_matrix, current_weights) / portfolio_vol
        current_risk_contrib = current_weights * marginal_risk / portfolio_vol
        
        # 偏离
        deviation = current_risk_contrib - self.target_risk_budget
        
        # 超标资产
        over_budget = [
            self.asset_names[i] for i in range(self.n_assets) 
            if deviation[i] > 0.01  # 超过1%
        ]
        
        # 低于预算资产
        under_budget = [
            self.asset_names[i] for i in range(self.n_assets) 
            if deviation[i] < -0.01  # 低于1%
        ]
        
        return {
            'current_risk_contrib': dict(zip(self.asset_names, current_risk_contrib)),
            'target_risk_budget': dict(zip(self.asset_names, self.target_risk_budget)),
            'deviation': dict(zip(self.asset_names, deviation)),
            'over_budget_assets': over_budget,
            'under_budget_assets': under_budget,
            'max_deviation': np.max(np.abs(deviation)),
            'mean_deviation': np.mean(np.abs(deviation))
        }
    
    def generate_rebalance_signal(self, 
                                  current_weights: np.ndarray,
                                  threshold: float = 0.05) -> Dict:
        """
        生成再平衡信号
        
        Args:
            current_weights: 当前权重
            threshold: 触发阈值
            
        Returns:
            再平衡信号
        """
        monitor_result = self.monitor_risk_budget(current_weights)
        
        max_deviation = monitor_result['max_deviation']
        
        if max_deviation > threshold:
            signal = 'REBALANCE'
            reason = f"风险预算偏离{max_deviation:.2%}超过阈值{threshold:.2%}"
        else:
            signal = 'HOLD'
            reason = f"风险预算偏离{max_deviation:.2%}在阈值内"
        
        return {
            'signal': signal,
            'reason': reason,
            'max_deviation': max_deviation,
            'over_budget': monitor_result['over_budget_assets'],
            'under_budget': monitor_result['under_budget_assets']
        }
```

## 5. 核心结论

### 5.1 风险预算分配原则

1. **明确风险目标**：设定整体风险预算和各层级风险预算
2. **风险平价基准**：风险平价是风险分散的良好起点
3. **动态调整**：根据市场环境动态调整风险预算
4. **多层级管理**：资产、策略、因子多层风险预算

### 5.2 实践建议

| 风险预算方法 | 适用场景 | 优势 | 劣势 |
|------------|---------|------|------|
| 等风险贡献 | 分散化投资 | 简单、稳健 | 忽略收益差异 |
| 逆波动率 | 低波动环境 | 降低风险 | 忽略相关性 |
| 自定义预算 | 有投资观点 | 灵活性强 | 需要准确判断 |
| 因子风险预算 | 因子投资 | 因子层面控制 | 需要因子模型 |

### 5.3 风险预算监控

- **偏离监控**：持续监控风险贡献偏离
- **阈值触发**：偏离超过阈值触发再平衡
- **动态调整**：根据市场变化调整风险预算
- **风险报告**：定期生成风险预算报告
