## 一、因子模型的理论基础

### 1.1 资产定价的基本问题

**核心问题**：如何解释资产收益率的差异？

**两种方法**：
1. **时间序列方法**：分析单个资产收益率的时序特征
2. **横截面方法**：分析不同资产收益率的截面差异

**因子模型的本质**：
```
资产超额收益 = 因子暴露 × 因子收益 + 特质收益
```

### 1.2 从CAPM到多因子模型

#### CAPM的局限

**CAPM模型**：
```
E(Rᵢ) - Rᶠ = βᵢ [E(Rₘ) - Rᶠ]
```

**实证异象（无法解释）**：
1. **规模效应**：小市值股票收益高于大市值股票
2. **价值效应**：低P/B股票收益高于高P/B股票
3. **动量效应**：过去表现好的股票继续表现好
4. **盈利能力效应**：高盈利能力股票收益更高
5. **投资效应**：低投资股票收益更高

**Roll批评（1977）**：
- 市场组合不可观测
- 使用代理组合检验CAPM存在问题

#### 从单因子到多因子

**理论基础**：
1. **Ross (1976)**：套利定价理论（APT）
2. **Fama & French (1993)**：三因子模型
3. **Carhart (1997)**：四因子模型
4. **Fama & French (2015)**：五因子模型
5. **Hou, Xue, Zhang (2020)**：q因子模型

## 二、套利定价理论（APT）

### 2.1 APT的假设

**假设1**：资产收益由K个因子生成

**因子模型**：
```
Rᵢ = E(Rᵢ) + βᵢ₁F₁ + βᵢ₂F₂ + ... + βᵢₖFₖ + εᵢ
```

其中：
- Rᵢ：资产i的收益率
- βᵢₖ：资产i对因子k的敏感度
- Fₖ：因子k的收益
- εᵢ：特质收益，E(εᵢ) = 0

**假设2**：无套利机会

### 2.2 APT的推导

**套利组合**：
```
权重向量 w = (w₁, w₂, ..., wₙ)
满足：
1. Σ wᵢ = 0（无需资金）
2. Σ wᵢβᵢₖ = 0, k = 1, ..., K（无因子风险）
3. Σ wᵢ² → 0（充分分散化）
```

**无套利条件**：
```
如果套利组合无风险，则其预期收益为0：
Σ wᵢE(Rᵢ) = 0
```

**推导结果**：
```
E(Rᵢ) - Rᶠ = λ₁βᵢ₁ + λ₂βᵢ₂ + ... + λₖβᵢₖ

其中：
λₖ = 因子k的风险溢价
```

### 2.3 APT vs CAPM

| 维度 | CAPM | APT |
|------|------|-----|
| 因子数量 | 1个（市场因子） | K个（多个因子） |
| 因子识别 | 明确（市场组合） | 不明确（需实证识别） |
| 假设 | 投资者偏好（风险厌恶） | 无套利 |
| 理论基础 | 均衡定价 | 套利定价 |
| 可检验性 | 存在争议 | 更可检验 |

### 2.4 APT的局限

**局限1**：因子数量不确定
- APT没有说明应该有多少个因子
- 因子的选择依赖实证研究

**局限2**：因子识别困难
- APT没有说明具体是哪些因子
- 不同研究者可能得到不同的因子

**局限3**：因子定价未定
- APT不提供因子风险溢价的理论值
- 需要实证估计

## 三、Fama-French三因子模型

### 3.1 模型提出

**Fama & French (1993)**：

**三因子模型**：
```
E(Rᵢ) - Rᶠ = βᵢ [E(Rₘ) - Rᶠ] + sᵢ E(SMB) + hᵢ E(HML)
```

**三个因子**：

**1) 市场因子（MKT）**
```
Rₘ - Rᶠ = 市场组合超额收益
```

**2) 规模因子（SMB）**
```
SMB = 小市值股票收益 - 大市值股票收益
SMB = Small Minus Big
```

**3) 价值因子（HML）**
```
HML = 高账面市值比股票收益 - 低账面市值比股票收益
HML = High Minus Low
```

### 3.2 因子构建

#### 2×3分类法

**步骤1**：按市值分组
```
6月市值中位数分为两组：
- Small（小市值）
- Big（大市值）
```

**步骤2**：按账面市值比分组
```
按B/M分为三组：
- Low（低B/M，成长股）：B/M < 30分位数
- Medium（中等B/M）：30分位数 ≤ B/M < 70分位数
- High（高B/M，价值股）：B/M ≥ 70分位数
```

**步骤3**：构建6个组合
```
SL, SM, SH：小市值 × 低/中/高B/M
BL, BM, BH：大市值 × 低/中/高B/M
```

**步骤4**：计算因子收益
```
SMB = (SL + SM + SH) / 3 - (BL + BM + BH) / 3

HML = (SH + BH) / 2 - (SL + BL) / 2
```

### 3.3 模型估计

**时间序列回归**：
```
Rᵢₜ - Rᶠₜ = αᵢ + βᵢ(Rₘₜ - Rᶠₜ) + sᵢ SMBₜ + hᵢ HMLₜ + εᵢₜ
```

**估计方法**：OLS（普通最小二乘法）

**案例：某基金的Fama-French三因子回归**
```python
import statsmodels.api as sm

# 数据准备
data = pd.read_csv('fund_returns.csv')
data['excess_return'] = data['fund_return'] - data['rf']
data['mkt_excess'] = data['mkt'] - data['rf']

# 回归
X = data[['mkt_excess', 'smb', 'hml']]
X = sm.add_constant(X)
y = data['excess_return']

model = sm.OLS(y, X).fit()
print(model.summary())

# 结果解读
print(f'Alpha: {model.params[0]:.4f}')
print(f'Beta (MKT): {model.params[1]:.4f}')
print(f'Beta (SMB): {model.params[2]:.4f}')
print(f'Beta (HML): {model.params[3]:.4f}')
print(f'R²: {model.rsquared:.4f}')
```

### 3.4 模型解释力

**实证结果**：
```
三因子模型可解释：
- 股票平均收益的横截面差异
- 规模效应和价值效应
- CAPM无法解释的异象
```

**R²统计**：
```
CAPM: R² ≈ 60-70%
三因子模型: R² ≈ 90-95%
```

### 3.5 Alpha的含义

**Jensen's Alpha**：
```
αᵢ = E(Rᵢ) - [Rᶠ + βᵢ(E(Rₘ) - Rᶠ) + sᵢ E(SMB) + hᵢ E(HML)]
```

**解释**：
- α > 0：基金经理创造超额收益
- α < 0：基金经理表现不佳
- α ≈ 0：收益可由因子风险解释

## 四、Carhart四因子模型

### 4.1 动量因子

**Jegadeesh & Titman (1993)**：发现动量效应

**动量因子（MOM）**：
```
MOM = 过去表现好的股票收益 - 过去表现差的股票收益
MOM = Winners Minus Losers (WML)
```

### 4.2 四因子模型

**Carhart (1997)**：

**四因子模型**：
```
E(Rᵢ) - Rᶠ = βᵢ [E(Rₘ) - Rᶠ] + sᵢ E(SMB) + hᵢ E(HML) + mᵢ E(MOM)
```

**因子构建**：
```
MOM构建方法：
1. 计算过去12个月收益（跳过最近1个月）
2. 按收益排序分为10组
3. Winners = 前10%（高收益组）
4. Losers = 后10%（低收益组）
5. MOM = Winners收益 - Losers收益
```

### 4.3 动量效应的解释

**行为金融学解释**：
1. **过度反应**：投资者对新信息反应过度
2. **反应不足**：投资者对新信息反应不足
3. **处置效应**：投资者过早卖出盈利股票

**风险解释**：
1. 动量收益是对某种风险的补偿
2. 动量因子可能捕捉了宏观经济风险

## 五、Fama-French五因子模型

### 5.1 模型提出

**Fama & French (2015)**：

**五因子模型**：
```
E(Rᵢ) - Rᶠ = βᵢ [E(Rₘ) - Rᶠ] + sᵢ E(SMB) + hᵢ E(HML) + rᵢ E(RMW) + cᵢ E(CMA)
```

**新增因子**：

**1) 盈利能力因子（RMW）**
```
RMW = 强盈利能力股票收益 - 弱盈利能力股票收益
RMW = Robust Minus Weak

盈利能力衡量：
Operating Profitability (OP) = (Revenue - COGS - SG&A - Interest) / Book Equity
```

**2) 投资因子（CMA）**
```
CMA = 保守投资股票收益 - 激进投资股票收益
CMA = Conservative Minus Aggressive

投资衡量：
Investment = ΔTotal Assets / Lagged Total Assets
```

### 5.2 因子构建（2×2×2×2×2）

**分类维度**：
1. **市值**：Small, Big
2. **账面市值比**：High, Low
3. **盈利能力**：Robust, Weak
4. **投资**：Conservative, Aggressive

**因子收益计算**：
```
SMB = 1/3 (S/L + S/H + S/R + S/W + S/C + S/A) 
    - 1/3 (B/L + B/H + B/R + B/W + B/C + B/A)

HML = 1/2 (S/H + B/H) - 1/2 (S/L + B/L)

RMW = 1/2 (S/R + B/R) - 1/2 (S/W + B/W)

CMA = 1/2 (S/C + B/C) - 1/2 (S/A + B/A)
```

### 5.3 模型解释力

**实证发现**：
```
五因子模型可解释：
- 三因子模型可解释的所有异象
- 盈利能力效应
- 投资效应
- 更多的收益横截面差异
```

**争议**：
```
Hou, Xue, Zhang (2020)：
- HML因子在五因子模型中变得不显著
- 四因子模型（q因子模型）可能更好
```

## 六、q因子模型

### 6.1 理论基础

**Hou, Xue, Zhang (2015, 2020)**：

**投资CAPM**：
```
基于投资的资本资产定价模型（Investment CAPM）
```

**核心思想**：
```
企业投资决策与股票收益相关
投资收益率 = 净现值 / 投资
```

### 6.2 四因子模型

**q因子模型**：
```
E(Rᵢ) - Rᶠ = βᵢ [E(Rₘ) - Rᶠ] + sᵢ E(ME) + iᵢ E(I/A) + rᵢ E(ROE)
```

**四个因子**：

**1) 市场因子（MKT）**
```
Rₘ - Rᶠ
```

**2) 规模因子（ME）**
```
ME = 小市值股票收益 - 大市值股票收益
```

**3) 投资因子（I/A）**
```
I/A = 低投资股票收益 - 高投资股票收益
Investment = ΔTotal Assets / Lagged Total Assets
```

**4) 盈利能力因子（ROE）**
```
ROE = 高ROE股票收益 - 低ROE股票收益
ROE = Net Income / Book Equity
```

### 6.3 q因子模型 vs Fama-French五因子模型

| 维度 | Fama-French五因子 | q因子模型 |
|------|-------------------|-----------|
| 理论基础 | 经验驱动 | 理论驱动 |
| 盈利因子 | RMW (Operating Profitability) | ROE |
| 投资因子 | CMA | I/A |
| 因子构建 | 2×3或2×2×2×2×2 | 2×3×3 |
| 解释力 | 较好 | 更好 |

## 七、因子模型的估计方法

### 7.1 时间序列回归

**方法**：
```
对每个资产i，运行时间序列回归：
Rᵢₜ - Rᶠₜ = αᵢ + βᵢ₁F₁ₜ + βᵢ₂F₂ₜ + ... + βᵢₖFₖₜ + εᵢₜ
```

**优点**：
- 简单直观
- 可得到每个资产的因子暴露

**缺点**：
- 需要足够长的时间序列
- 因子暴露假设为常数

### 7.2 横截面回归

**方法**：

**步骤1**：估计因子收益
```
每期运行横截面回归：
Rᵢₜ = λ₀ₜ + λ₁ₜβᵢ₁ + λ₂ₜβᵢ₂ + ... + λₖₜβᵢₖ + εᵢₜ

其中βᵢₖ通过时间序列回归预先估计
```

**步骤2**：计算因子风险溢价
```
λₖ = (1/T) Σ λₖₜ
```

**优点**：
- 可直接估计因子风险溢价
- 因子暴露可时变

**缺点**：
- 两步估计存在误差
- 需要先估计因子暴露

### 7.3 Fama-MacBeth回归

**Fama & MacBeth (1973)**：

**步骤1**：估计因子暴露
```
对每个资产i，运行时间序列回归得到βᵢₖ
```

**步骤2**：横截面回归
```
每期t运行横截面回归：
Rᵢₜ = λ₀ₜ + λ₁ₜβᵢ₁ + λ₂ₜβᵢ₂ + ... + λₖₜβᵢₖ + εᵢₜ
```

**步骤3**：计算平均风险溢价
```
λₖ = (1/T) Σ λₖₜ
SE(λₖ) = σ(λₖₜ) / √T
t统计量 = λₖ / SE(λₖ)
```

**优点**：
- 可估计因子风险溢价
- 标准误修正了横截面相关

**案例**：
```python
import statsmodels.api as sm
import numpy as np

# Fama-MacBeth回归
def fama_macbeth(returns, factors, betas):
    """
    returns: (T, N) 收益率矩阵
    factors: (T, K) 因子收益矩阵
    betas: (N, K) 因子暴露矩阵
    """
    T, N = returns.shape
    K = betas.shape[1]
    
    # 存储每期的因子风险溢价
    risk_premia = np.zeros((T, K+1))
    
    # 每期横截面回归
    for t in range(T):
        # 横截面回归：R_it = λ0 + λ1*β_i1 + ... + λk*β_ik + ε_it
        X_t = sm.add_constant(betas)  # (N, K+1)
        y_t = returns[t, :]  # (N,)
        
        model = sm.OLS(y_t, X_t).fit()
        risk_premia[t, :] = model.params
    
    # 计算平均风险溢价
    mean_premia = np.mean(risk_premia, axis=0)
    se_premia = np.std(risk_premia, axis=0) / np.sqrt(T)
    t_stats = mean_premia / se_premia
    
    return mean_premia, se_premia, t_stats

# 使用示例
returns = pd.read_csv('stock_returns.csv').values  # (T, N)
factors = pd.read_csv('factors.csv').values  # (T, K)
betas = pd.read_csv('betas.csv').values  # (N, K)

mean_premia, se_premia, t_stats = fama_macbeth(returns, factors, betas)

print('因子风险溢价：')
for i, premia in enumerate(mean_premia):
    print(f'因子{i}: {premia:.4f} (t={t_stats[i]:.2f})')
```

### 7.4 主成分分析（PCA）

**方法**：
```
从大量资产收益中提取主成分，作为隐含因子
```

**步骤**：
1. 计算收益率协方差矩阵
2. 特征值分解
3. 选择前K个主成分

**优点**：
- 数据驱动
- 不需要预先指定因子

**缺点**：
- 因子经济学含义不明确
- 解释困难

### 7.5 估计方法比较

| 方法 | 因子暴露 | 因子收益 | 因子风险溢价 | 优点 | 缺点 |
|------|---------|---------|-------------|------|------|
| 时间序列回归 | ✓ | ✓ | ✗ | 简单 | 需要长时序 |
| 横截面回归 | ✗ | ✓ | ✓ | 直接估计风险溢价 | 两步误差 |
| Fama-MacBeth | ✓ | ✗ | ✓ | 标准误修正 | 两步估计 |
| PCA | ✓ | ✓ | ✗ | 数据驱动 | 因子含义不明 |

## 八、因子模型的检验

### 8.1 GRS检验

**Gibbons, Ross, Shanken (1989)**：

**原假设**：
```
H₀: α₁ = α₂ = ... = αₙ = 0
（所有资产的Alpha为零）
```

**检验统计量**：
```
GRS = (T/N) × (T-N-K)/(T-K-1) × α̂' Σ̂⁻¹ α̂ / (1 + μ̂' Ω̂⁻¹ μ̂)

其中：
- T：时间序列长度
- N：资产数量
- K：因子数量
- α̂：Alpha向量
- Σ̂：残差协方差矩阵
- μ̂：因子收益均值向量
- Ω̂：因子收益协方差矩阵
```

**分布**：
```
GRS ~ F(N, T-N-K)
```

**判断准则**：
```
如果 GRS > F_{α}(N, T-N-K)，则拒绝原假设
```

### 8.2 横截面R²

**公式**：
```
R² = 1 - Σ(Rᵢ - Řᵢ)² / Σ(Rᵢ - R̄)²

其中：
- Rᵢ：资产i的平均收益
- Řᵢ：因子模型预测的收益
```

**解释**：
```
R²越高，模型解释力越强
```

### 8.3 最大夏普比率平方

**公式**：
```
SR²(factors) = μ' Σ⁻¹ μ

其中：
- μ：因子收益均值向量
- Σ：因子收益协方差矩阵
```

**解释**：
```
最大夏普比率平方衡量因子的定价能力
SR²越大，因子组合的夏普比率越高
```

## 九、因子模型的应用

### 9.1 资产定价

**应用**：计算资产的预期收益

**方法**：
```
E(Rᵢ) = Rᶠ + βᵢ₁λ₁ + βᵢ₂λ₂ + ... + βᵢₖλₖ
```

**案例**：
```
某股票的因子暴露：
- Beta (MKT) = 1.2
- Beta (SMB) = 0.3
- Beta (HML) = -0.2

因子风险溢价：
- MKT = 6%
- SMB = 3%
- HML = 4%

预期收益：
E(R) = 2% + 1.2×6% + 0.3×3% + (-0.2)×4% = 8.8%
```

### 9.2 投资组合管理

**应用**：
1. 风险归因
2. 业绩评估
3. 组合优化

**风险归因**：
```
总风险 = 因子风险 + 特质风险

因子风险分解：
方差 = β' Σ_f β + σ_ε²
```

**业绩评估**：
```
Alpha = 实际收益 - 因子模型预测收益

正Alpha：基金经理创造价值
负Alpha：基金经理跑输市场
```

### 9.3 风险管理

**应用**：
1. 因子风险预测
2. 压力测试
3. 风险预算

**因子风险预测**：
```
组合波动率 = √(β' Σ_f β + σ_ε²)

因子VaR = z_α × √(β' Σ_f β)
```

## 十、多因子模型的发展趋势

### 10.1 因子动物园

**问题**：
```
文献中识别的因子超过400个（Harvey et al., 2016）
很多因子可能是数据挖掘的结果
```

**解决方案**：
1. 提高统计显著性标准
2. 样本外检验
3. 理论驱动而非数据驱动

### 10.2 因子拥挤

**问题**：

## 十二、小结与思考题

### 12.1 核心要点

1. **从CAPM到多因子**
   - CAPM无法解释市场异象
   - 多因子模型提供更好的定价能力

2. **主要因子模型**
   - APT：理论框架
   - Fama-French三因子：规模、价值
   - Carhart四因子：增加动量
   - Fama-French五因子：增加盈利、投资
   - q因子模型：投资CAPM

3. **估计方法**
   - 时间序列回归
   - 横截面回归
   - Fama-MacBeth回归

4. **模型检验**
   - GRS检验
   - 横截面R²
   - 最大夏普比率

### 12.2 思考题

1. **理论题**：
   推导APT的无套利定价公式，并解释λₖ的经济含义。

2. **实证题**：
   使用Fama-French三因子模型分析某A股基金的业绩，判断基金经理是否创造超额收益。

3. **比较题**：
   比较Fama-French五因子模型和q因子模型的异同，讨论哪个模型更适合A股市场。

4. **应用题**：
   某投资组合的因子暴露为：β_MKT=1.1, β_SMB=0.5, β_HML=0.3。已知因子风险溢价为：λ_MKT=5%, λ_SMB=2%, λ_HML=3%，无风险利率为2%。计算该组合的预期收益。
