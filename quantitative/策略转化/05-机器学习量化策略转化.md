## 二、策略思路描述

### 2.1 LSTM股价预测策略

**核心思想**：
- 利用LSTM捕捉价格序列的长期依赖关系
- 预测未来N日的收益率或价格方向

**策略框架**：
```
数据准备 → 特征工程 → 序列构建 → 模型训练 → 预测交易
```

**详细步骤**：
1. **数据准备**：获取OHLCV数据，计算技术指标
2. **特征工程**：标准化、滑动窗口构建序列
3. **模型训练**：划分训练/验证/测试集，训练LSTM
4. **预测交易**：预测上涨则买入，预测下跌则卖出

### 2.2 Transformer多资产策略

**核心思想**：
- 使用多头注意力机制捕捉资产间关系
- 联合预测多资产收益

**策略框架**：
```
多资产数据 → 特征对齐 → Transformer编码 → 联合预测 → 组合优化
```

**优势**：
- 捕捉跨市场信息传播
- 并行计算效率高
- 长距离依赖能力强

### 2.3 强化学习交易策略

**核心思想**：
- 将交易建模为MDP（马尔可夫决策过程）
- 智能体通过与环境交互学习最优策略

**环境设计**：
- **状态**：价格信息、技术指标、持仓、资金
- **动作**：买入、卖出、持有（或连续仓位比例）
- **奖励**：收益率、夏普比率、风险调整收益

**训练流程**：
```
初始化策略 → 与环境交互 → 收集经验 → 更新策略 → 评估性能
```

### 2.4 因子挖掘机器学习策略

**核心思想**：
- 使用机器学习自动挖掘因子
- 特征重要性分析识别有效因子

**方法**：
1. **特征工程**：生成大量候选因子（100+）
2. **因子筛选**：使用随机森林/XGBoost筛选重要因子
3. **因子组合**：使用模型预测或加权组合
4. **组合优化**：基于预测收益构建投资组合

### 3.1 LSTM预测模型

```python
import torch
import torch.nn as nn
import numpy as np
from sklearn.preprocessing import StandardScaler

class StockLSTM(nn.Module):
    """LSTM股价预测模型"""
    
    def __init__(self, input_size, hidden_size=128, num_layers=2, 
                 output_size=1, dropout=0.2):
        super().__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        
        self.lstm = nn.LSTM(
            input_size, hidden_size, num_layers,
            batch_first=True, dropout=dropout
        )
        self.fc = nn.Linear(hidden_size, output_size)
        
    def forward(self, x):
        # x: (batch, seq_len, features)
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])  # 取最后一个时间步
        return out


class LSTMPredictor:
    """LSTM预测器"""
    
    def __init__(self, seq_length=20, hidden_size=128, num_layers=2):
        self.seq_length = seq_length
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.scaler = StandardScaler()
        self.model = None
        
    def create_features(self, df):
        """创建特征"""
        features = pd.DataFrame(index=df.index)
        
        # 价格特征
        features['return'] = df['close'].pct_change()
        features['log_return'] = np.log(df['close'] / df['close'].shift(1))
        
        # 技术指标
        features['ma_ratio_5_20'] = df['close'].rolling(5).mean() / df['close'].rolling(20).mean()
        features['volatility_20'] = features['return'].rolling(20).std()
        
        # RSI
        delta = df['close'].diff()
        gain = delta.where(delta > 0, 0).rolling(14).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
        features['rsi'] = 100 - 100 / (1 + gain / loss)
        
        # MACD
        ema12 = df['close'].ewm(span=12).mean()
        ema26 = df['close'].ewm(span=26).mean()
        features['macd'] = ema12 - ema26
        
        return features.dropna()
    
    def create_sequences(self, data):
        """构建序列"""
        X, y = [], []
        for i in range(len(data) - self.seq_length):
            X.append(data[i:(i + self.seq_length)])
            # 预测下一天的收益率
            y.append(data[i + self.seq_length, 0])
        return np.array(X), np.array(y)
    
    def train(self, train_data, epochs=100, batch_size=32, lr=0.001):
        """训练模型"""
        # 标准化
        scaled_data = self.scaler.fit_transform(train_data)
        
        # 构建序列
        X, y = self.create_sequences(scaled_data)
        X = torch.FloatTensor(X)
        y = torch.FloatTensor(y).unsqueeze(1)
        
        # 初始化模型
        input_size = X.shape[2]
        self.model = StockLSTM(input_size, self.hidden_size, self.num_layers)
        
        # 训练
        criterion = nn.MSELoss()
        optimizer = torch.optim.Adam(self.model.parameters(), lr=lr)
        
        for epoch in range(epochs):
            self.model.train()
            for i in range(0, len(X), batch_size):
                batch_X = X[i:i+batch_size]
                batch_y = y[i:i+batch_size]
                
                optimizer.zero_grad()
                outputs = self.model(batch_X)
                loss = criterion(outputs, batch_y)
                loss.backward()
                optimizer.step()
            
            if (epoch + 1) % 10 == 0:
                print(f'Epoch [{epoch+1}/{epochs}], Loss: {loss.item():.4f}')
    
    def predict(self, recent_data):
        """预测"""
        self.model.eval()
        scaled_data = self.scaler.transform(recent_data)
        X = torch.FloatTensor(scaled_data[-self.seq_length:]).unsqueeze(0)
        
        with torch.no_grad():
            prediction = self.model(X)
        
        return prediction.item()
    
    def generate_signal(self, recent_data, threshold=0.0):
        """生成交易信号"""
        pred_return = self.predict(recent_data)
        
        if pred_return > threshold:
            return 1   # 买入
        elif pred_return < -threshold:
            return -1  # 卖出
        else:
            return 0   # 持有
```

### 3.2 强化学习交易环境

```python
import numpy as np
import pandas as pd

class TradingEnvironment:
    """交易环境"""
    
    def __init__(self, data, initial_balance=100000, 
                 transaction_cost=0.001, max_position=1.0):
        self.data = data
        self.initial_balance = initial_balance
        self.transaction_cost = transaction_cost
        self.max_position = max_position
        
        self.reset()
    
    def reset(self):
        """重置环境"""
        self.balance = self.initial_balance
        self.position = 0  # 持仓比例
        self.current_step = 0
        self.total_value = self.initial_balance
        self.history = []
        
        return self._get_state()
    
    def _get_state(self):
        """获取当前状态"""
        # 价格特征
        price_features = self.data.iloc[self.current_step].values
        
        # 持仓信息
        position_info = np.array([
            self.position,
            self.balance / self.initial_balance,
            self.total_value / self.initial_balance
        ])
        
        return np.concatenate([price_features, position_info])
    
    def step(self, action):
        """执行动作"""
        current_price = self.data.iloc[self.current_step]['close']
        
        # 动作映射
        if isinstance(action, int):
            # 离散动作: 0=卖出, 1=持有, 2=买入
            target_position = action / 2  # 0, 0.5, 1
        else:
            # 连续动作: 直接作为目标仓位
            target_position = np.clip(action, 0, self.max_position)
        
        # 计算交易量
        trade_size = target_position - self.position
        
        # 执行交易
        if trade_size > 0:  # 买入
            cost = trade_size * self.total_value * (1 + self.transaction_cost)
            if cost <= self.balance:
                self.position += trade_size
                self.balance -= cost
        
        elif trade_size < 0:  # 卖出
            revenue = -trade_size * self.total_value * (1 - self.transaction_cost)
            self.position += trade_size
            self.balance += revenue
        
        # 更新状态
        self.current_step += 1
        new_price = self.data.iloc[self.current_step]['close']
        
        # 计算新的组合价值
        self.total_value = self.balance + self.position * self.total_value * (new_price / current_price)
        
        # 计算奖励
        prev_value = self.history[-1]['total_value'] if self.history else self.initial_balance
        reward = (self.total_value - prev_value) / prev_value
        
        # 记录历史
        self.history.append({
            'step': self.current_step,
            'position': self.position,
            'balance': self.balance,
            'total_value': self.total_value,
            'reward': reward
        })
        
        # 判断是否结束
        done = self.current_step >= len(self.data) - 1
        
        return self._get_state(), reward, done
    
    def get_portfolio_value(self):
        """获取组合价值"""
        return self.total_value
```

### 3.3 DQN交易代理

```python
import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np
from collections import deque
import random

class DQNAgent:
    """DQN交易代理"""
    
    def __init__(self, state_size, action_size, 
                 learning_rate=0.001, gamma=0.95,
                 epsilon_start=1.0, epsilon_end=0.01, epsilon_decay=0.995,
                 buffer_size=10000, batch_size=64):
        self.state_size = state_size
        self.action_size = action_size
        self.gamma = gamma
        self.epsilon = epsilon_start
        self.epsilon_min = epsilon_end
        self.epsilon_decay = epsilon_decay
        self.batch_size = batch_size
        
        # 经验回放
        self.memory = deque(maxlen=buffer_size)
        
        # 主网络和目标网络
        self.policy_net = self._build_network()
        self.target_net = self._build_network()
        self.target_net.load_state_dict(self.policy_net.state_dict())
        
        self.optimizer = optim.Adam(self.policy_net.parameters(), lr=learning_rate)
        
    def _build_network(self):
        """构建Q网络"""
        return nn.Sequential(
            nn.Linear(self.state_size, 128),
            nn.ReLU(),
            nn.Linear(128, 64),
            nn.ReLU(),
            nn.Linear(64, self.action_size)
        )
    
    def remember(self, state, action, reward, next_state, done):
        """存储经验"""
        self.memory.append((state, action, reward, next_state, done))
    
    def act(self, state, training=True):
        """选择动作"""
        if training and random.random() < self.epsilon:
            return random.randrange(self.action_size)
        
        with torch.no_grad():
            state = torch.FloatTensor(state).unsqueeze(0)
            q_values = self.policy_net(state)
            return q_values.argmax().item()
    
    def replay(self):
        """经验回放训练"""
        if len(self.memory) < self.batch_size:
            return
        
        # 采样
        batch = random.sample(self.memory, self.batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)
        
        states = torch.FloatTensor(np.array(states))
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(np.array(next_states))
        dones = torch.FloatTensor(dones)
        
        # 计算当前Q值
        current_q = self.policy_net(states).gather(1, actions.unsqueeze(1))
        
        # 计算目标Q值
        with torch.no_grad():
            next_q = self.target_net(next_states).max(1)[0]
            target_q = rewards + (1 - dones) * self.gamma * next_q
        
        # 计算损失
        loss = nn.MSELoss()(current_q.squeeze(), target_q)
        
        # 反向传播
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        # 更新epsilon
        self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)
    
    def update_target_network(self):
        """更新目标网络"""
        self.target_net.load_state_dict(self.policy_net.state_dict())


def train_dqn_agent(env, episodes=1000, target_update=10):
    """训练DQN代理"""
    state_size = env._get_state().shape[0]
    action_size = 3  # 买入、持有、卖出
    
    agent = DQNAgent(state_size, action_size)
    
    for episode in range(episodes):
        state = env.reset()
        total_reward = 0
        
        while True:
            action = agent.act(state)
            next_state, reward, done = env.step(action)
            agent.remember(state, action, reward, next_state, done)
            
            state = next_state
            total_reward += reward
            
            if done:
                break
        
        # 训练
        agent.replay()
        
        # 更新目标网络
        if episode % target_update == 0:
            agent.update_target_network()
        
        if (episode + 1) % 100 == 0:
            print(f"Episode {episode+1}, Total Reward: {total_reward:.4f}, "
                  f"Epsilon: {agent.epsilon:.4f}")
    
    return agent
```

### 3.4 随机森林因子挖掘

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.preprocessing import StandardScaler
import pandas as pd
import numpy as np

class FactorMiner:
    """因子挖掘器"""
    
    def __init__(self, n_estimators=100, max_depth=10):
        self.model = RandomForestClassifier(
            n_estimators=n_estimators,
            max_depth=max_depth,
            random_state=42,
            n_jobs=-1
        )
        self.scaler = StandardScaler()
        self.feature_importance = None
    
    def generate_candidate_factors(self, df):
        """生成候选因子"""
        factors = pd.DataFrame(index=df.index)
        
        # 动量因子
        for period in [5, 10, 20, 60]:
            factors[f'momentum_{period}'] = df['close'] / df['close'].shift(period) - 1
        
        # 波动因子
        for period in [5, 10, 20, 60]:
            factors[f'volatility_{period}'] = df['close'].pct_change().rolling(period).std()
        
        # 技术指标
        # RSI
        delta = df['close'].diff()
        gain = delta.where(delta > 0, 0).rolling(14).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
        factors['rsi'] = 100 - 100 / (1 + gain / loss)
        
        # MACD
        ema12 = df['close'].ewm(span=12).mean()
        ema26 = df['close'].ewm(span=26).mean()
        factors['macd'] = ema12 - ema26
        factors['macd_signal'] = factors['macd'].ewm(span=9).mean()
        
        # 布林带
        ma20 = df['close'].rolling(20).mean()
        std20 = df['close'].rolling(20).std()
        factors['bollinger_upper'] = ma20 + 2 * std20
        factors['bollinger_lower'] = ma20 - 2 * std20
        factors['bollinger_position'] = (df['close'] - ma20) / (2 * std20)
        
        # 成交量因子
        factors['volume_ratio'] = df['volume'] / df['volume'].rolling(20).mean()
        
        return factors.dropna()
    
    def create_labels(self, df, forward_period=5, threshold=0.02):
        """创建标签"""
        forward_return = df['close'].shift(-forward_period) / df['close'] - 1
        
        labels = pd.cut(
            forward_return,
            bins=[-np.inf, -threshold, threshold, np.inf],
            labels=[0, 1, 2]  # 下跌、震荡、上涨
        )
        
        return labels
    
    def train(self, factors, labels):
        """训练模型"""
        # 标准化
        factors_scaled = self.scaler.fit_transform(factors)
        
        # 训练
        self.model.fit(factors_scaled, labels)
        
        # 保存特征重要性
        self.feature_importance = pd.Series(
            self.model.feature_importances_,
            index=factors.columns
        ).sort_values(ascending=False)
        
        return self.model
    
    def get_top_factors(self, top_n=20):
        """获取最重要的因子"""
        return self.feature_importance.head(top_n)
    
    def predict(self, factors):
        """预测"""
        factors_scaled = self.scaler.transform(factors)
        return self.model.predict(factors_scaled)
    
    def predict_proba(self, factors):
        """预测概率"""
        factors_scaled = self.scaler.transform(factors)
        return self.model.predict_proba(factors_scaled)
```

### 6.1 机器学习特有风险

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 过拟合 | 模型在训练集表现好，测试集差 | 样本外验证、正则化 |
| 数据泄露 | 未来信息进入特征 | 严格的时间划分 |
| 模型漂移 | 市场变化导致模型失效 | 定期重训练、监控 |
| 黑盒风险 | 模型决策不可解释 | 特征重要性分析 |

```python
def train_test_split_time_series(data, test_size=0.2):
    """时序数据划分"""
    split_idx = int(len(data) * (1 - test_size))
    train = data[:split_idx]
    test = data[split_idx:]
    return train, test

# 早停防止过拟合
class EarlyStopping:
    def __init__(self, patience=10, min_delta=0):
        self.patience = patience
        self.min_delta = min_delta
        self.counter = 0
        self.best_loss = None
        self.early_stop = False
    
    def __call__(self, val_loss):
        if self.best_loss is None:
            self.best_loss = val_loss
        elif val_loss > self.best_loss - self.min_delta:
            self.counter += 1
            if self.counter >= self.patience:
                self.early_stop = True
        else:
            self.best_loss = val_loss
            self.counter = 0
```
