## 二、策略思路描述

### 2.1 双均线趋势跟踪策略

**核心逻辑**：
- 趋势形成后具有持续性
- 短期均线反应快，长期均线反应慢
- 均线交叉代表趋势转换信号

**策略规则**：
1. 当短期均线上穿长期均线时，建立多头仓位
2. 当短期均线下穿长期均线时，平仓或做空
3. 在震荡市场中频繁止损，需要趋势过滤器

**适用环境**：
- 明显趋势市场（牛市或熊市）
- ADX > 25时效果更好
- 震荡市需结合均值回归策略

### 2.2 多因子选股策略

**核心逻辑**：
- 多因子分散风险
- 因子暴露获取超额收益
- 定期调仓保持因子有效性

**策略规则**：
1. 因子计算：每月末计算所有股票的因子值
2. 因子筛选：剔除ST、停牌、流动性差的股票
3. 因子打分：Z-score标准化后加权合成
4. 选股建仓：选取综合得分最高的前N只股票
5. 调仓频率：月度调仓

**权重分配**：
- 价值因子：30%
- 质量因子：25%
- 动量因子：25%
- 低波因子：20%

### 2.3 股指期货择时策略（变形式）

**核心逻辑**：
- 80%资金投资量化股多策略
- 20%资金用于股指期货多空调节
- 根据择时信号调整期货端仓位

**策略规则**：
1. 构建股票端：多因子选股，保持稳定仓位
2. 构建期货端：根据择时信号调整
   - 看多市场：做多期货
   - 看空市场：做空期货
   - 中性判断：空仓期货端
3. 风险控制：期货端保证金不超过总资金20%

### 2.4 均值回归套利策略

**核心逻辑**：
- 价格偏离均值后会回归
- 统计套利捕捉定价错误
- 配对交易降低市场风险

**策略规则**：
1. 选择协整关系的股票对
2. 计算价差的Z-score
3. 当Z-score > 2时，做空价差
4. 当Z-score < -2时，做多价差
5. 当Z-score回归0附近时平仓

## 三、实现伪代码

### 3.1 双均线策略实现

```python
class DualMAStrategy:
    """双均线趋势跟踪策略"""
    
    def __init__(self, short_period=5, long_period=20):
        self.short_period = short_period
        self.long_period = long_period
        
    def calculate_signals(self, prices):
        """计算交易信号"""
        # 计算均线
        ma_short = prices.rolling(self.short_period).mean()
        ma_long = prices.rolling(self.long_period).mean()
        
        # 生成信号
        signals = []
        position = 0
        
        for i in range(len(prices)):
            if ma_short[i] > ma_long[i] and position == 0:
                signals.append(1)   # 买入
                position = 1
            elif ma_short[i] < ma_long[i] and position == 1:
                signals.append(-1)  # 卖出
                position = 0
            else:
                signals.append(0)   # 持有
        
        return signals
    
    def backtest(self, prices, initial_capital=100000):
        """回测"""
        signals = self.calculate_signals(prices)
        capital = initial_capital
        position = 0
        portfolio = []
        
        for i, signal in enumerate(signals):
            if signal == 1:  # 买入
                position = capital / prices[i]
                capital = 0
            elif signal == -1:  # 卖出
                capital = position * prices[i]
                position = 0
            
            portfolio.append(capital + position * prices[i])
        
        return portfolio
```

### 3.2 多因子选股策略实现

```python
class MultiFactorStrategy:
    """多因子选股策略"""
    
    def __init__(self, factor_weights=None):
        self.factor_weights = factor_weights or {
            'value': 0.30,
            'quality': 0.25,
            'momentum': 0.25,
            'low_vol': 0.20
        }
    
    def calculate_factors(self, stock_data):
        """计算各因子"""
        factors = {}
        
        # 价值因子：低P/B
        factors['value'] = -stock_data['pb']  # 负号表示低PB更好
        
        # 质量因子：高ROE
        factors['quality'] = stock_data['roe']
        
        # 动量因子：过去6月收益率
        factors['momentum'] = stock_data['return_6m']
        
        # 低波因子：低波动率
        factors['low_vol'] = -stock_data['volatility_60d']
        
        return factors
    
    def standardize(self, factor_values):
        """Z-score标准化"""
        return (factor_values - factor_values.mean()) / factor_values.std()
    
    def calculate_score(self, stock_data):
        """计算综合得分"""
        factors = self.calculate_factors(stock_data)
        
        # 标准化
        scores = {}
        for factor_name, factor_values in factors.items():
            scores[factor_name] = self.standardize(factor_values)
        
        # 加权合成
        total_score = sum(
            scores[factor] * weight 
            for factor, weight in self.factor_weights.items()
        )
        
        return total_score
    
    def select_stocks(self, stock_data, top_n=30):
        """选股"""
        scores = self.calculate_score(stock_data)
        selected = scores.nlargest(top_n).index.tolist()
        return selected
```

### 3.3 择时策略实现（RSRS斜率指标）

```python
class RSRSStrategy:
    """RSRS斜率择时策略"""
    
    def __init__(self, lookback=18, buy_threshold=1.0, sell_threshold=0.8):
        self.lookback = lookback
        self.buy_threshold = buy_threshold
        self.sell_threshold = sell_threshold
    
    def calculate_beta(self, highs, lows):
        """计算RSRS斜率"""
        from sklearn.linear_model import LinearRegression
        
        X = lows.reshape(-1, 1)
        y = highs
        
        model = LinearRegression()
        model.fit(X, y)
        
        return model.coef_[0]
    
    def generate_signals(self, high_prices, low_prices):
        """生成交易信号"""
        signals = []
        
        for i in range(self.lookback, len(high_prices)):
            # 取最近N天数据
            highs = high_prices[i-self.lookback:i]
            lows = low_prices[i-self.lookback:i]
            
            # 计算斜率
            beta = self.calculate_beta(highs, lows)
            
            # 生成信号
            if beta > self.buy_threshold:
                signals.append(1)   # 看多
            elif beta < self.sell_threshold:
                signals.append(-1)  # 看空
            else:
                signals.append(0)   # 中性
        
        return signals
```

1. **历史表现不代表未来**：策略历史回测表现优异，实盘可能偏差
2. **市场环境变化**：策略在不同市场环境下表现差异大

4. **流动性风险**：大资金调仓可能面临流动性不足
