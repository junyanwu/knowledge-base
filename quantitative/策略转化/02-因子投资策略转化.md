## 二、策略思路描述

### 2.1 多因子选股策略

**核心逻辑**：
- 多因子分散风险，单一因子可能失效
- 因子加权合成综合得分
- 定期调仓保持因子有效性

**策略框架**：
```
股票池筛选 → 因子计算 → 因子标准化 → 因子合成 → 选股 → 组合优化
```

**详细步骤**：
1. **股票池筛选**：剔除ST、停牌、流动性差的股票
2. **因子计算**：计算所有股票的因子值
3. **异常值处理**：缩尾处理，消除极端值
4. **标准化**：Z-score标准化
5. **因子正交化**：消除因子共线性
6. **因子合成**：加权计算综合得分
7. **选股**：选取得分最高的N只股票
8. **权重分配**：等权或风险预算

### 2.2 因子择时策略

**核心逻辑**：
- 因子收益具有时变特征
- 通过预测因子收益调整因子暴露
- 宏观、市场、因子信号驱动择时

**择时信号**：

| 宏观信号 | 因子偏好 |
|---------|---------|
| 经济上行 | 价值+动量 |
| 经济下行 | 质量+低波 |
| 利率上行 | 金融+价值 |
| 利率下行 | 成长+动量 |

| 市场信号 | 因子偏好 |
|---------|---------|
| 牛市初期 | 动量 |
| 牛市中期 | 成长+动量 |
| 牛市末期 | 质量+低波 |
| 熊市 | 低波+红利 |
| 震荡市 | 多因子均衡 |

### 2.3 因子轮动策略

**核心逻辑**：
- A股风格轮动周期短（3-6个月）
- 根据因子动量和拥挤度调整权重
- 动态配置提高收益

**轮动规则**：
```python
# 因子动量权重
factor_momentum = factor_return.rolling(3).mean()
factor_weight = factor_momentum / factor_momentum.sum()

# 因子拥挤度调整
crowding_score = calculate_crowding(factor_exposure)
adj_weight = factor_weight * (1 - crowding_score)
```

### 2.4 指数增强策略

**核心逻辑**：
- 在跟踪指数基础上获取超额收益
- 因子选股增强 + 风险模型控制
- 严格跟踪误差约束

**增强框架**：
```
目标函数：max E[r_portfolio - r_bench]
约束条件：
  - 跟踪误差 < 3%
  - 行业偏离 < 2%
  - 个股权重偏离 < 0.5%
  - 换手率 < 100%/年
```

### 3.1 因子计算模块

```python
import pandas as pd
import numpy as np

class FactorCalculator:
    """因子计算模块"""
    
    def __init__(self, data):
        self.data = data  # 股票数据DataFrame
    
    # ========== 价值因子 ==========
    def calc_value_factors(self):
        """计算价值因子"""
        factors = pd.DataFrame(index=self.data.index)
        
        # P/B (市净率)
        factors['pb'] = self.data['market_cap'] / self.data['net_assets']
        
        # P/E (市盈率)
        factors['pe'] = self.data['market_cap'] / self.data['net_profit']
        
        # P/CF (市现率)
        factors['pcf'] = self.data['market_cap'] / self.data['operating_cf']
        
        # EV/EBITDA
        factors['ev_ebitda'] = (self.data['market_cap'] + self.data['debt'] - 
                                self.data['cash']) / self.data['ebitda']
        
        # 股息率
        factors['div_yield'] = self.data['dividend'] / self.data['price']
        
        return factors
    
    # ========== 动量因子 ==========
    def calc_momentum_factors(self, prices):
        """计算动量因子"""
        factors = pd.DataFrame(index=prices.index)
        
        # 3月动量
        factors['mom_3m'] = prices / prices.shift(63) - 1
        
        # 6月动量
        factors['mom_6m'] = prices / prices.shift(126) - 1
        
        # 12月动量
        factors['mom_12m'] = prices / prices.shift(252) - 1
        
        # 风险调整动量
        returns = prices.pct_change()
        factors['risk_adj_mom'] = (factors['mom_6m'] / 
                                   returns.rolling(126).std() * np.sqrt(252))
        
        return factors
    
    # ========== 质量因子 ==========
    def calc_quality_factors(self):
        """计算质量因子"""
        factors = pd.DataFrame(index=self.data.index)
        
        # ROE
        factors['roe'] = self.data['net_profit'] / self.data['net_assets']
        
        # ROA
        factors['roa'] = self.data['net_profit'] / self.data['total_assets']
        
        # 毛利率
        factors['gross_margin'] = (self.data['revenue'] - self.data['cost']) / self.data['revenue']
        
        # 净利率
        factors['net_margin'] = self.data['net_profit'] / self.data['revenue']
        
        # 资产负债率
        factors['debt_ratio'] = self.data['total_liability'] / self.data['total_assets']
        
        return factors
    
    # ========== 低波因子 ==========
    def calc_lowvol_factors(self, returns):
        """计算低波因子"""
        factors = pd.DataFrame(index=returns.index)
        
        # 历史波动率
        factors['volatility'] = returns.rolling(60).std() * np.sqrt(252)
        
        # 下行波动率
        neg_returns = returns[returns < 0]
        factors['downside_vol'] = neg_returns.rolling(60).std() * np.sqrt(252)
        
        # Beta
        market_returns = self.get_market_returns()
        factors['beta'] = returns.rolling(60).cov(market_returns) / market_returns.rolling(60).var()
        
        return factors
    
    # ========== 规模因子 ==========
    def calc_size_factors(self):
        """计算规模因子"""
        factors = pd.DataFrame(index=self.data.index)
        
        # 总市值
        factors['market_cap'] = self.data['price'] * self.data['total_shares']
        
        # 流通市值
        factors['float_cap'] = self.data['price'] * self.data['float_shares']
        
        # 取对数
        factors['ln_cap'] = np.log(factors['market_cap'])
        
        return factors
```

### 3.2 因子预处理模块

```python
class FactorPreprocessor:
    """因子预处理模块"""
    
    @staticmethod
    def winsorize(factor, lower=0.025, upper=0.975):
        """缩尾处理"""
        lower_bound = factor.quantile(lower)
        upper_bound = factor.quantile(upper)
        return factor.clip(lower_bound, upper_bound)
    
    @staticmethod
    def standardize(factor):
        """Z-score标准化"""
        return (factor - factor.mean()) / factor.std()
    
    @staticmethod
    def orthogonalize(factor, benchmark_factor):
        """因子正交化（残差化）"""
        from sklearn.linear_model import LinearRegression
        
        model = LinearRegression()
        model.fit(benchmark_factor.values.reshape(-1, 1), factor.values)
        residual = factor - model.predict(benchmark_factor.values.reshape(-1, 1))
        
        return residual
    
    def process_factors(self, factors, industry_dummies=None):
        """完整预处理流程"""
        processed = {}
        
        for name, factor in factors.items():
            # 1. 缩尾
            factor = self.winsorize(factor)
            
            # 2. 标准化
            factor = self.standardize(factor)
            
            # 3. 行业中性化（可选）
            if industry_dummies is not None:
                factor = self.orthogonalize(factor, industry_dummies)
            
            processed[name] = factor
        
        return processed
```

class FactorCombiner:
    """因子合成模块"""
    
    def __init__(self, factor_weights=None):
        self.factor_weights = factor_weights or {
            'value': 0.25,
            'momentum': 0.25,
            'quality': 0.25,
            'low_vol': 0.15,
            'size': 0.10
        }
    
    def combine_factors(self, factor_scores):
        """加权合成因子得分"""
        total_score = 0
        
        for factor_name, weight in self.factor_weights.items():
            if factor_name in factor_scores:
                total_score += factor_scores[factor_name] * weight
        
        return total_score
    
    def ic_weighted_combine(self, factor_scores, factor_ics):
        """IC加权合成"""
        ic_weights = factor_ics / factor_ics.abs().sum()
        
        total_score = 0
        for factor_name, score in factor_scores.items():
            total_score += score * ic_weights[factor_name]
        
        return total_score
    
    def dynamic_weight_combine(self, factor_scores, factor_returns):
        """动态权重合成"""
        # 基于因子历史收益计算权重
        weights = factor_returns.rolling(12).mean()
        weights = weights / weights.abs().sum()
        
        total_score = 0
        for factor_name, score in factor_scores.items():
            total_score += score * weights[factor_name]
        
        return total_score


class StockSelector:
    """选股模块"""
    
    def __init__(self, top_n=30):
        self.top_n = top_n
    
    def select_stocks(self, scores, exclude_list=None):
        """选择得分最高的股票"""
        if exclude_list:
            scores = scores[~scores.index.isin(exclude_list)]
        
        selected = scores.nlargest(self.top_n).index.tolist()
        return selected
    
    def select_with_neutralization(self, scores, industry_map, max_per_industry=5):
        """行业中性选股"""
        selected = []
        industry_count = {}
        
        sorted_stocks = scores.sort_values(ascending=False)
        
        for stock in sorted_stocks.index:
            industry = industry_map.get(stock, 'Other')
            
            if industry_count.get(industry, 0) < max_per_industry:
                selected.append(stock)
                industry_count[industry] = industry_count.get(industry, 0) + 1
            
            if len(selected) >= self.top_n:
                break
        
        return selected

### 3.4 因子有效性检验模块

```python
class FactorValidator:
    """因子有效性检验"""
    
    def calc_ic(self, factor_values, forward_returns):
        """计算IC值"""
        # Spearman相关系数
        from scipy.stats import spearmanr
        ic, p_value = spearmanr(factor_values, forward_returns)
        return ic, p_value
    
    def calc_ic_series(self, factor_values, forward_returns, periods=252):
        """计算IC时间序列"""
        ic_series = []
        
        for i in range(periods, len(factor_values)):
            factor = factor_values.iloc[i-periods:i]
            returns = forward_returns.iloc[i-periods:i]
            ic, _ = self.calc_ic(factor, returns)
            ic_series.append(ic)
        
        return pd.Series(ic_series)
    
    def calc_icir(self, ic_series):
        """计算ICIR"""
        return ic_series.mean() / ic_series.std()
    
    def layer_backtest(self, factor_values, forward_returns, n_layers=5):
        """分层回测"""
        # 将股票分为N层
        labels = pd.qcut(factor_values, n_layers, labels=False)
        
        layer_returns = {}
        for layer in range(n_layers):
            stocks_in_layer = labels[labels == layer].index
            layer_returns[layer] = forward_returns.loc[stocks_in_layer].mean()
        
        # 多空收益
        long_short = layer_returns[n_layers-1] - layer_returns[0]
        
        return layer_returns, long_short
    
    def factor_report(self, factor_name, factor_values, forward_returns):
        """生成因子报告"""
        ic_series = self.calc_ic_series(factor_values, forward_returns)
        layer_returns, ls_return = self.layer_backtest(factor_values, forward_returns)
        
        report = {
            'factor_name': factor_name,
            'IC_mean': ic_series.mean(),
            'IC_std': ic_series.std(),
            'ICIR': self.calc_icir(ic_series),
            'IC_positive_ratio': (ic_series > 0).mean(),
            'long_short_return': ls_return,
            'layer_returns': layer_returns
        }
        
        return report
```
