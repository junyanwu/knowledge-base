## 二、策略思路描述

### 2.1 RSRS斜率择时策略

**核心原理**：
- 最高价和最低价存在线性关系：high = β × low + α + ε
- β（斜率）反映市场阻力与支撑的相对强度
- β > 1：高价随低价上涨更快，市场强势
- β < 1：高价随低价上涨更慢，市场弱势

**策略逻辑**：
```
市场强势 → β > 1 → 阻力弱于支撑 → 买入
市场弱势 → β < 0.8 → 阻力强于支撑 → 卖出
```

**优化方向**：
1. **标准化处理**：消除不同市场时期斜率均值差异
2. **拟合优度加权**：β × R²，提高信号可靠性
3. **动态阈值**：根据市场波动调整买卖阈值

### 2.2 量选择时策略（股指期货端）

**核心思路**：
- 80%资金投资量化股多策略（稳定收益来源）
- 20%资金用于股指期货多空调节（择时收益增强）

**仓位调节逻辑**：
```python
if market_signal == 'bullish':
    futures_position = 0.2  # 做多期货，总仓位达100%
elif market_signal == 'neutral':
    futures_position = 0    # 维持80%股票仓位
elif market_signal == 'bearish':
    futures_position = -0.2  # 做空期货，总仓位降至60%
```

**优势**：
- 股票端稳定运行，不受择时信号频繁调整
- 期货端交易成本低，调整灵活
- 风险管理清晰，两端分离

### 2.3 多因子择时模型

**因子体系**：

| 因子类别 | 具体指标 | 权重 |
|---------|---------|------|
| 估值因子 | PE分位数、PB分位数、股债收益比 | 30% |


### 2.4 趋势跟踪+均值回归组合

**核心思路**：
- 趋势市场用趋势跟踪策略
- 震荡市场用均值回归策略
- 通过市场状态识别自动切换

**市场状态识别**：
```python
def identify_market_state(prices, window=60):
    """识别市场状态"""
    
    # 计算ADX判断趋势强度
    adx = calculate_adx(prices, window)
    
    # 计算价格波动率
    volatility = prices.pct_change().rolling(window).std()
    
    # 计算价格区间
    high_low_range = (prices.rolling(window).max() - 
                       prices.rolling(window).min()) / prices.rolling(window).mean()
    
    if adx > 25 and volatility < volatility.quantile(0.7):
        return 'trending'  # 趋势市场
    elif adx < 20 and high_low_range < 0.2:
        return 'oscillating'  # 震荡市场
    else:
        return 'transitional'  # 过渡期
```

### 3.1 RSRS择时策略完整实现

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

class RSRSTimingStrategy:
    """RSRS斜率择时策略"""
    
    def __init__(self, lookback=18, buy_threshold=1.0, sell_threshold=0.8,
                 use_standardized=True):
        self.lookback = lookback
        self.buy_threshold = buy_threshold
        self.sell_threshold = sell_threshold
        self.use_standardized = use_standardized
        
        # 存储历史beta值用于标准化
        self.beta_history = []
    
    def calculate_beta(self, highs, lows):
        """计算RSRS斜率"""
        X = lows.reshape(-1, 1)
        y = highs
        
        model = LinearRegression()
        model.fit(X, y)
        
        beta = model.coef_[0]
        r_squared = model.score(X, y)
        
        return beta, r_squared
    
    def calculate_standardized_beta(self, beta):
        """标准化beta值"""
        self.beta_history.append(beta)
        
        if len(self.beta_history) < self.lookback:
            return beta
        
        # 使用历史数据标准化
        recent_betas = self.beta_history[-self.lookback:]
        mean_beta = np.mean(recent_betas)
        std_beta = np.std(recent_betas)
        
        if std_beta > 0:
            standardized_beta = (beta - mean_beta) / std_beta
        else:
            standardized_beta = 0
        
        return standardized_beta
    
    def generate_signal(self, high_prices, low_prices):
        """生成交易信号"""
        if len(high_prices) < self.lookback:
            return 0, 0, 0  # 数据不足
        
        # 取最近N天数据
        highs = high_prices[-self.lookback:]
        lows = low_prices[-self.lookback:]
        
        # 计算beta和R²
        beta, r_squared = self.calculate_beta(highs, lows)
        
        # 修正beta
        adjusted_beta = beta * r_squared
        
        # 标准化
        if self.use_standardized:
            signal_beta = self.calculate_standardized_beta(beta)
        else:
            signal_beta = adjusted_beta
        
        # 生成信号
        if signal_beta > self.buy_threshold:
            signal = 1  # 买入
        elif signal_beta < self.sell_threshold:
            signal = -1  # 卖出
        else:
            signal = 0  # 持有
        
        return signal, beta, r_squared
    
    def backtest(self, high_prices, low_prices, close_prices, 
                 initial_capital=100000, fee_rate=0.0003):
        """回测"""
        n = len(close_prices)
        
        # 结果存储
        signals = []
        positions = []
        portfolio_values = [initial_capital]
        
        capital = initial_capital
        position = 0
        
        for i in range(self.lookback, n):
            high_window = high_prices[:i]
            low_window = low_prices[:i]
            
            signal, beta, r2 = self.generate_signal(high_window, low_window)
            signals.append(signal)
            
            current_price = close_prices[i]
            
            # 执行交易
            if signal == 1 and position == 0:  # 买入
                shares = capital / current_price
                position = shares
                capital = 0
                # 扣除手续费
                capital -= shares * current_price * fee_rate
                
            elif signal == -1 and position > 0:  # 卖出
                capital = position * current_price
                # 扣除手续费
                capital -= capital * fee_rate
                position = 0
            
            # 记录仓位和净值
            positions.append(position)
            portfolio_value = capital + position * current_price
            portfolio_values.append(portfolio_value)
        
        return {
            'signals': signals,
            'positions': positions,
            'portfolio_values': portfolio_values,
            'final_return': (portfolio_values[-1] - initial_capital) / initial_capital
        }


# 使用示例
strategy = RSRSTimingStrategy(lookback=18, buy_threshold=0.7, sell_threshold=-0.7)

# 假设已有数据
result = strategy.backtest(high_prices, low_prices, close_prices)

print(f"最终收益率: {result['final_return']:.2%}")
```

### 3.2 多因子择时模型实现

```python
import pandas as pd
import numpy as np
from sklearn.ensemble import RandomForestClassifier
from sklearn.preprocessing import StandardScaler

class MultiFactorTimingModel:
    """多因子择时模型"""
    
    def __init__(self):
        self.factor_weights = {
            'valuation': 0.30,
            'sentiment': 0.25,
            'technical': 0.25,
            'macro': 0.20
        }
        self.scaler = StandardScaler()
    
    # ========== 因子计算 ==========
    
    def calc_valuation_factor(self, pe, pb, bond_yield):
        """估值因子"""
        # PE分位数
        pe_percentile = pe.rolling(252).rank(pct=True)
        # PB分位数
        pb_percentile = pb.rolling(252).rank(pct=True)
        # 股债收益比
        earnings_yield = 1 / pe
        equity_bond_spread = earnings_yield - bond_yield
        
        # 估值得分（低估值=高分）
        val_score = -(pe_percentile + pb_percentile) / 2 + equity_bond_spread
        
        return val_score
    
    def calc_sentiment_factor(self, vix, turnover, margin_balance):
        """情绪因子"""
        # VIX得分（低恐慌=高分）
        vix_score = -vix
        
        # 换手率得分（适度换手=高分）
        turnover_ma = turnover.rolling(60).mean()
        turnover_score = -abs(turnover - turnover_ma) / turnover_ma
        
        # 融资余额变化得分（增加=高分）
        margin_change = margin_balance.pct_change(20)
        margin_score = margin_change
        
        sentiment_score = vix_score + turnover_score + margin_score
        
        return sentiment_score
    
    def calc_technical_factor(self, prices):
        """技术因子"""
        # MACD
        ema12 = prices.ewm(span=12).mean()
        ema26 = prices.ewm(span=26).mean()
        macd = ema12 - ema26
        macd_signal = macd.ewm(span=9).mean()
        macd_hist = macd - macd_signal
        
        # RSI
        delta = prices.diff()
        gain = delta.where(delta > 0, 0).rolling(14).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
        rs = gain / loss
        rsi = 100 - 100 / (1 + rs)
        
        # 趋势强度
        ma20 = prices.rolling(20).mean()
        ma60 = prices.rolling(60).mean()
        trend_strength = (ma20 - ma60) / ma60
        
        # 技术得分
        tech_score = macd_hist + (rsi - 50) / 50 + trend_strength
        
        return tech_score
    
    def calc_macro_factor(self, pmi, interest_rate_change, m2_growth):
        """宏观因子"""
        # PMI得分
        pmi_score = (pmi - 50) / 10  # PMI>50为扩张
        
        # 利率变化得分（利率下降=高分）
        rate_score = -interest_rate_change
        
        # M2增长得分
        liquidity_score = m2_growth
        
        macro_score = pmi_score + rate_score + liquidity_score
        
        return macro_score
    
    # ========== 信号生成 ==========
    
    def generate_timing_signal(self, data):
        """生成择时信号"""
        # 计算各因子
        val_factor = self.calc_valuation_factor(
            data['pe'], data['pb'], data['bond_yield']
        )
        sent_factor = self.calc_sentiment_factor(
            data['vix'], data['turnover'], data['margin_balance']
        )
        tech_factor = self.calc_technical_factor(data['close'])
        macro_factor = self.calc_macro_factor(
            data['pmi'], data['interest_rate_change'], data['m2_growth']
        )
        
        # 标准化
        factors = pd.DataFrame({
            'valuation': val_factor,
            'sentiment': sent_factor,
            'technical': tech_factor,
            'macro': macro_factor
        })
        
        factors_normalized = pd.DataFrame(
            self.scaler.fit_transform(factors),
            columns=factors.columns,
            index=factors.index
        )
        
        # 加权合成
        total_score = sum(
            factors_normalized[col] * weight 
            for col, weight in self.factor_weights.items()
        )
        
        # 生成信号
        signals = []
        for score in total_score:
            if score > 0.5:
                signals.append(1)   # 看多
            elif score < -0.5:
                signals.append(-1)  # 看空
            else:
                signals.append(0)   # 中性
        
        return signals, total_score


# 使用示例
model = MultiFactorTimingModel()
signals, scores = model.generate_timing_signal(market_data)
```

### 3.3 机器学习择时实现

```python
import pandas as pd
import numpy as np
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report

class MLTimingStrategy:
    """机器学习择时策略"""
    
    def __init__(self, model=None, lookback=60, forward_period=5):
        self.model = model or RandomForestClassifier(
            n_estimators=100,
            max_depth=10,
            min_samples_split=5,
            random_state=42
        )
        self.lookback = lookback
        self.forward_period = forward_period
        self.feature_scaler = StandardScaler()
    
    def create_features(self, prices, volumes):
        """创建特征"""
        features = pd.DataFrame(index=prices.index)
        
        # 价格特征
        features['returns'] = prices.pct_change()
        features['log_returns'] = np.log(prices / prices.shift(1))
        
        # 均线特征
        features['ma_ratio_5_20'] = prices.rolling(5).mean() / prices.rolling(20).mean()
        features['ma_ratio_10_60'] = prices.rolling(10).mean() / prices.rolling(60).mean()
        
        # 动量特征
        features['momentum_5'] = prices / prices.shift(5) - 1
        features['momentum_20'] = prices / prices.shift(20) - 1
        features['momentum_60'] = prices / prices.shift(60) - 1
        
        # 波动特征
        features['volatility_5'] = features['returns'].rolling(5).std()
        features['volatility_20'] = features['returns'].rolling(20).std()
        
        # RSI
        delta = prices.diff()
        gain = delta.where(delta > 0, 0).rolling(14).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
        features['rsi'] = 100 - 100 / (1 + gain / loss)
        
        # MACD
        ema12 = prices.ewm(span=12).mean()
        ema26 = prices.ewm(span=26).mean()
        features['macd'] = ema12 - ema26
        features['macd_signal'] = features['macd'].ewm(span=9).mean()
        
        # 成交量特征
        features['volume_ratio'] = volumes / volumes.rolling(20).mean()
        features['price_volume_corr'] = prices.rolling(20).corr(volumes)
        
        # 布林带位置
        ma20 = prices.rolling(20).mean()
        std20 = prices.rolling(20).std()
        features['bollinger_position'] = (prices - ma20) / (2 * std20)
        
        return features
    
    def create_labels(self, prices, threshold=0.02):
        """创建标签"""
        # 计算未来收益
        forward_returns = prices.shift(-self.forward_period) / prices - 1
        
        # 标签：1=上涨，-1=下跌，0=震荡
        labels = pd.cut(
            forward_returns,
            bins=[-np.inf, -threshold, threshold, np.inf],
            labels=[-1, 0, 1]
        )
        
        return labels
    
    def train(self, prices, volumes):
        """训练模型"""
        # 创建特征和标签
        features = self.create_features(prices, volumes)
        labels = self.create_labels(prices)
        
        # 对齐数据
        valid_idx = features.dropna().index.intersection(labels.dropna().index)
        X = features.loc[valid_idx]
        y = labels.loc[valid_idx]
        
        # 划分训练集和测试集
        X_train, X_test, y_train, y_test = train_test_split(
            X, y, test_size=0.2, shuffle=False
        )
        
        # 标准化
        X_train_scaled = self.feature_scaler.fit_transform(X_train)
        X_test_scaled = self.feature_scaler.transform(X_test)
        
        # 训练模型
        self.model.fit(X_train_scaled, y_train)
        
        # 评估
        train_pred = self.model.predict(X_train_scaled)
        test_pred = self.model.predict(X_test_scaled)
        
        print(f"训练集准确率: {accuracy_score(y_train, train_pred):.4f}")
        print(f"测试集准确率: {accuracy_score(y_test, test_pred):.4f}")
        print(classification_report(y_test, test_pred))
        
        return self.model
    
    def predict(self, prices, volumes):
        """预测"""
        features = self.create_features(prices, volumes)
        features_scaled = self.feature_scaler.transform(features)
        
        predictions = self.model.predict(features_scaled)
        probabilities = self.model.predict_proba(features_scaled)
        
        return predictions, probabilities
    
    def get_feature_importance(self):
        """获取特征重要性"""
        return pd.Series(
            self.model.feature_importances_,
            index=self.feature_scaler.feature_names_in_
        ).sort_values(ascending=False)


# 使用示例
ml_strategy = MLTimingStrategy(lookback=60, forward_period=5)
ml_strategy.train(train_prices, train_volumes)

predictions, probabilities = ml_strategy.predict(test_prices, test_volumes)
```

## 六、风险提示与应对

### 6.1 择时策略固有风险

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 信号失效 | 市场环境变化导致信号失效 | 多信号组合、定期回测 |
| 假信号 | 频繁错误信号导致亏损 | 信号过滤、置信度阈值 |
| 滞后性 | 趋势确认滞后导致收益回吐 | 多周期确认、止损机制 |
| 过拟合 | 历史表现好，实盘差 | 样本外测试、简化模型 |

### 6.2 应对策略

```python
class RiskManagedTiming:
    """风险管理择时"""
    
    def __init__(self, base_strategy, max_drawdown=0.15, stop_loss=0.05):
        self.base_strategy = base_strategy
        self.max_drawdown = max_drawdown
        self.stop_loss = stop_loss
        self.peak_value = 0
    
    def check_risk(self, current_value):
        """风险检查"""
        # 更新峰值
        self.peak_value = max(self.peak_value, current_value)
        
        # 回撤检查
        drawdown = (self.peak_value - current_value) / self.peak_value
        if drawdown > self.max_drawdown:
            return 'max_drawdown_breach'
        
        # 止损检查
        if drawdown > self.stop_loss:
            return 'stop_loss_triggered'
        
        return 'normal'
    
    def adjust_signal(self, signal, risk_status):
        """根据风险状态调整信号"""
        if risk_status == 'max_drawdown_breach':
            return -1  # 强制减仓
        elif risk_status == 'stop_loss_triggered':
            return 0   # 暂停交易
        else:
            return signal
```
