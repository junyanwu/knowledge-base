# 统计学基础 - 模块2：回归分析

## 1. 线性回归基础

### 1.1 简单线性回归模型

**模型形式：**
```
Y = β₀ + β₁X + ε
```

其中：
- Y：因变量（被解释变量）
- X：自变量（解释变量）
- β₀：截距项
- β₁：斜率系数
- ε：误差项

**OLS估计方法：**

最小二乘法（Ordinary Least Squares）通过最小化残差平方和来估计参数：

```
min Σ(Yᵢ - β₀ - β₁Xᵢ)²
```

**估计公式：**
```
β̂₁ = Σ(Xᵢ - X̄)(Yᵢ - Ȳ) / Σ(Xᵢ - X̄)²
β̂₀ = Ȳ - β̂₁X̄
```

### 1.2 多元线性回归

**模型形式：**
```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ + ε
```

**矩阵形式：**
```
Y = Xβ + ε
```

**OLS估计：**
```
β̂ = (X'X)⁻¹X'Y
```

**假设条件（高斯-马尔可夫）：**
1. **线性性**：Y与X呈线性关系
2. **随机抽样**：样本独立同分布
3. **零条件均值**：E(ε|X) = 0
4. **同方差性**：Var(ε|X) = σ²
5. **无完全多重共线性**：X矩阵列满秩

### 1.3 回归假设检验

**t检验（单系数）：**
```
t = β̂ⱼ / SE(β̂ⱼ) ~ tₙ₋ₖ₋₁
```

**F检验（整体显著性）：**
```
F = (R²/(k-1)) / ((1-R²)/(n-k)) ~ Fₖ₋₁,ₙ₋ₖ
```

**R²决定系数：**
```
R² = 1 - SSE/SST = SSR/SST
```

**调整R²：**
```
R̄² = 1 - (1-R²)(n-1)/(n-k-1)
```

## 2. 回归诊断

### 2.1 残差分析

**残差定义：**
```
eᵢ = Yᵢ - Ŷᵢ
```

**标准化残差：**
```
rᵢ = eᵢ / SE(eᵢ)
```

**学生化残差：**
```
tᵢ = eᵢ / (s√(1-hᵢᵢ))
```

其中hᵢᵢ是帽子矩阵H = X(X'X)⁻¹X'的对角元素。

**残差图分析：**
- 残差 vs 拟合值：检验同方差性
- 残差 vs 自变量：检验线性性
- Q-Q图：检验正态性
- 残差时序图：检验独立性

### 2.2 异方差检验

**异方差的影响：**
- OLS估计仍无偏但不有效
- 标准误估计有偏
- t检验和F检验失效

**检验方法：**

**1. White检验：**
```
辅助回归：e²ᵢ = α₀ + α₁Xᵢ + α₂Xᵢ² + νᵢ
检验：H₀: α₁ = α₂ = 0
统计量：nR² ~ χ²ₖ
```

**2. Breusch-Pagan检验：**
```
辅助回归：e²ᵢ = α₀ + α₁Z₁ᵢ + ... + αₘZₘᵢ + νᵢ
检验：H₀: α₁ = ... = αₘ = 0
统计量：nR² ~ χ²ₘ
```

**处理方法：**
- 加权最小二乘（WLS）
- 稳健标准误（White's HC标准误）
- 对数变换

### 2.3 多重共线性

**诊断指标：**

**方差膨胀因子（VIF）：**
```
VIFⱼ = 1 / (1 - Rⱼ²)
```

其中Rⱼ²是Xⱼ对其他自变量回归的R²。

**判断标准：**
- VIF < 5：轻度共线性
- 5 ≤ VIF < 10：中度共线性
- VIF ≥ 10：严重共线性

**处理方法：**
- 剔除高度相关的自变量
- 主成分回归
- 岭回归（Ridge）
- LASSO回归

### 2.4 自相关问题

**Durbin-Watson检验：**
```
DW = Σ(eᵢ - eᵢ₋₁)² / Σeᵢ²
```

**判断：**
- DW ≈ 2：无自相关
- DW < 1.5：正自相关
- DW > 2.5：负自相关

**处理方法：**
- Cochrane-Orcutt迭代法
- 广义最小二乘（GLS）
- Newey-West标准误

## 3. 回归扩展方法

### 3.1 加权最小二乘（WLS）

**适用场景：** 异方差已知形式

**模型：**
```
min Σwᵢ(Yᵢ - β₀ - β₁Xᵢ)²
```

**权重选择：**
- wᵢ = 1/σᵢ²（方差倒数）
- wᵢ = 1/Xᵢ²（正比于自变量平方）

### 3.2 稳健回归

**M估计：**
```
min Σρ(eᵢ/σ)
```

其中ρ是稳健损失函数：
- Huber：ρ(e) = e²/2 if |e| ≤ k, k|e| - k²/2 otherwise
- Tukey bisquare：ρ(e) = k²/6(1 - (1 - (e/k)²)³)

**迭代加权最小二乘（IRLS）：**
```
β̂^(m+1) = (X'W^(m)X)⁻¹X'W^(m)Y
```

### 3.3 分位数回归

**模型：**
```
Qτ(Y|X) = X'β(τ)
```

**估计方法：**
```
min Σ ρτ(Yᵢ - X'ᵢβ)
```

其中ρτ(e) = e(τ - I(e < 0))

**优点：**
- 对异常值稳健
- 提供条件分布的全貌
- 不假设误差分布

### 3.4 非线性回归

**多项式回归：**
```
Y = β₀ + β₁X + β₂X² + ... + βₖXᵏ + ε
```

**Logistic回归：**
```
P(Y=1|X) = 1 / (1 + exp(-X'β))
```

**泊松回归：**
```
E(Y|X) = exp(X'β)
```

## 4. 金融应用

### 4.1 因子模型回归

**CAPM模型：**
```
Rᵢ - Rf = αᵢ + βᵢ(Rm - Rf) + εᵢ
```

**Fama-French三因子：**
```
Rᵢ - Rf = αᵢ + βᵐ(Rm - Rf) + βˢSMB + βʰHML + εᵢ
```

**Carhart四因子：**
```
Rᵢ - Rf = αᵢ + βᵐ(Rm - Rf) + βˢSMB + βʰHML + βᵐᵒMOM + εᵢ
```

**回归检验：**
- α显著为0：市场有效
- α > 0：超额收益
- β值：系统性风险暴露

### 4.2 事件研究

**步骤：**

1. **估计窗口：** [-T, -1]
   估计正常收益模型

2. **事件窗口：** [0, τ]
   计算异常收益

3. **异常收益（AR）：**
   ```
   ARᵢₜ = Rᵢₜ - E(Rᵢₜ|Xₜ)
   ```

4. **累计异常收益（CAR）：**
   ```
   CAR(τ₁, τ₂) = Σ ARᵢₜ
   ```

5. **显著性检验：**
   ```
   t = CAR / SE(CAR)
   ```

### 4.3 收益预测

**预测模型：**
```
Rₜ₊₁ = α + β₁Rₜ + β₂DPₜ + β₃EPₜ + β₄TERMₜ + εₜ₊₁
```

其中：
- DP：股息率
- EP：盈利收益率
- TERM：期限利差

**样本外预测：**
- 滚动窗口回归
- 递归估计
- 交叉验证

**预测评估：**
- RMSE：均方根误差
- MAE：平均绝对误差
- R²ₒₒₛ：样本外R²

### 4.4 风险建模

**波动率建模：**
```
σ²ₜ = α₀ + α₁ε²ₜ₋₁ + βσ²ₜ₋₁
```

**VaR估计：**
```
VaR = μ - zₐσ
```

**条件VaR（CVaR）：**
```
CVaR = E(L|L > VaR)
```

## 5. Python实现

### 5.1 线性回归

```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
from sklearn.linear_model import LinearRegression

# 使用statsmodels（含详细统计）
X = sm.add_constant(data[['X1', 'X2']])
model = sm.OLS(data['Y'], X).fit()
print(model.summary())

# 使用sklearn（预测导向）
model = LinearRegression()
model.fit(data[['X1', 'X2']], data['Y'])
predictions = model.predict(data[['X1', 'X2']])
```

### 5.2 回归诊断

```python
# 残差分析
residuals = model.resid
fitted = model.fittedvalues

# 异方差检验
from statsmodels.stats.diagnostic import het_breuschpagan, het_white
bp_test = het_breuschpagan(residuals, X)
white_test = het_white(residuals, X)

# 多重共线性
from statsmodels.stats.outliers_influence import variance_inflation_factor
vif = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]

# 自相关检验
from statsmodels.stats.stattools import durbin_watson
dw = durbin_watson(residuals)
```

### 5.3 稳健回归

```python
from statsmodels.robust.robust_linear_model import RLM

# Huber稳健回归
model = RLM(data['Y'], X, M=sm.robust.norms.HuberT())
results = model.fit()
```

### 5.4 分位数回归

```python
import statsmodels.formula.api as smf

# 中位数回归
model = smf.quantreg('Y ~ X1 + X2', data)
results = model.fit(q=0.5)

# 多分位数
quantiles = [0.1, 0.25, 0.5, 0.75, 0.9]
models = [model.fit(q=q) for q in quantiles]
```

## 6. 实践建议

### 6.1 建模流程

1. **数据探索**
   - 描述性统计
   - 相关性分析
   - 散点图矩阵

2. **模型设定**
   - 选择自变量
   - 考虑交互项
   - 检查线性性

3. **参数估计**
   - OLS估计
   - 检查系数符号

4. **模型诊断**
   - 残差分析
   - 异方差检验
   - 多重共线性

5. **模型改进**
   - 变量变换
   - 异常值处理
   - 模型选择

### 6.2 常见陷阱

1. **过度拟合**
   - 变量过多
   - 样本量不足
   - 解决：交叉验证

2. **遗漏变量偏误**
   - 重要变量缺失
   - 内生性问题
   - 解决：工具变量

3. **样本选择偏误**
   - 非随机样本
   - 幸存者偏差
   - 解决：Heckman修正

4. **伪回归**
   - 单位根过程
   - 缺乏协整
   - 解决：差分或协整
