### 1.1 订单簿

**订单簿结构：**
```
买方（Bid）           卖方（Ask）
价格    数量         价格    数量
100.05  1000         100.10  800
100.04  1500         100.11  1200
100.03  2000         100.12  1000
100.02  3000         100.13  1500
100.01  5000         100.14  2000
```

**关键指标：**
- **最优买价（Best Bid）**：100.05
- **最优卖价（Best Ask）**：100.10
- **价差（Spread）**：0.05
- **中间价（Mid Price）**：100.075
- **深度（Depth）**：买方深度12500股，卖方深度6500股

**订单簿分析：**
```python
def order_book_analysis(bids, asks):
    """
    订单簿分析
    
    参数:
    bids: 买方订单 [(price, size), ...]
    asks: 卖方订单 [(price, size), ...]
    
    返回:
    订单簿指标
    """
    # 最优价
    best_bid = bids[0][0]
    best_ask = asks[0][0]
    
    # 价差
    spread = best_ask - best_bid
    spread_bps = spread / best_bid * 10000  # 基点
    
    # 中间价
    mid = (best_bid + best_ask) / 2
    
    # 深度
    bid_depth = sum(size for _, size in bids[:5])  # 前5档深度
    ask_depth = sum(size for _, size in asks[:5])
    
    # 不平衡
    imbalance = (bid_depth - ask_depth) / (bid_depth + ask_depth)
    
    # 加权价格
    bid_vwap = sum(p * s for p, s in bids[:5]) / bid_depth
    ask_vwap = sum(p * s for p, s in asks[:5]) / ask_depth
    
    return {
        'best_bid': best_bid,
        'best_ask': best_ask,
        'spread': spread,
        'spread_bps': spread_bps,
        'mid': mid,
        'bid_depth': bid_depth,
        'ask_depth': ask_depth,
        'imbalance': imbalance,
        'bid_vwap': bid_vwap,
        'ask_vwap': ask_vwap
    }
```

### 1.2 成交数据

**逐笔成交：**
```python
def analyze_trades(trades):
    """
    成交数据分析
    
    参数:
    trades: 成交数据 [(price, size, side, time), ...]
    
    返回:
    成交统计
    """
    df = pd.DataFrame(trades, columns=['price', 'size', 'side', 'time'])
    
    # 买卖比例
    buy_volume = df[df['side'] == 'buy']['size'].sum()
    sell_volume = df[df['side'] == 'sell']['size'].sum()
    buy_ratio = buy_volume / (buy_volume + sell_volume)
    
    # 成交加权平均价
    vwap = (df['price'] * df['size']).sum() / df['size'].sum()
    
    # 成交速率
    df['time'] = pd.to_datetime(df['time'])
    df = df.set_index('time')
    trade_rate = df['size'].resample('1s').sum()  # 每秒成交量
    
    # 大单识别
    avg_size = df['size'].mean()
    large_trades = df[df['size'] > avg_size * 3]
    
    return {
        'buy_ratio': buy_ratio,
        'vwap': vwap,
        'trade_rate': trade_rate,
        'large_trades': large_trades
    }
```

## 2. 高频策略

### 2.1 做市策略

**原理：**
在买卖双边挂单，赚取价差。

**策略：**
```python
def market_making(best_bid, best_ask, inventory, target_inventory=0, 
                  max_inventory=1000, spread_target=0.02):
    """
    做市策略
    
    参数:
    best_bid: 最优买价
    best_ask: 最优卖价
    inventory: 当前持仓
    target_inventory: 目标持仓
    max_inventory: 最大持仓
    spread_target: 目标价差
    
    返回:
    报价
    """
    mid = (best_bid + best_ask) / 2
    
    # 库存调整
    inventory_skew = (inventory - target_inventory) / max_inventory
    
    # 报价偏移
    skew = inventory_skew * spread_target / 2
    
    # 报价
    bid_price = mid - spread_target / 2 + skew
    ask_price = mid + spread_target / 2 + skew
    
    # 挂单量
    bid_size = min(100, max_inventory - inventory) if inventory < max_inventory else 0
    ask_size = min(100, inventory + max_inventory) if inventory > -max_inventory else 0
    
    return {
        'bid_price': bid_price,
        'ask_price': ask_price,
        'bid_size': bid_size,
        'ask_size': ask_size
    }
```

### 2.2 统计套利高频版

**日内配对交易：**
```python
def intraday_pairs(spread, z_threshold=2, holding_period=60):
    """
    日内配对交易
    
    参数:
    spread: 价差序列
    z_threshold: 开仓阈值
    holding_period: 持仓时间（秒）
    
    返回:
    交易信号
    """
    # 标准化
    spread_mean = spread.rolling(300).mean()  # 5分钟均值
    spread_std = spread.rolling(300).std()
    z = (spread - spread_mean) / spread_std
    
    # 信号
    signal = pd.Series(0, index=spread.index)
    
    # 开仓
    signal[z > z_threshold] = -1
    signal[z < -z_threshold] = 1
    
    # 持仓时间限制
    signal = signal.replace(0, np.nan).ffill(limit=holding_period).fillna(0)
    
    return signal
```

### 2.3 订单流策略

**订单流不平衡：**
```python
def order_flow_strategy(order_flow, window=60):
    """
    订单流策略
    
    参数:
    order_flow: 订单流数据 (buy_size - sell_size)
    window: 窗口期
    
    返回:
    交易信号
    """
    # 累计订单流
    cumulative_flow = order_flow.rolling(window).sum()
    
    # 订单流不平衡
    imbalance = cumulative_flow / cumulative_flow.rolling(window).std()
    
    # 信号
    signal = pd.Series(0, index=order_flow.index)
    
    # 买单占优
    signal[imbalance > 2] = 1
    
    # 卖单占优
    signal[imbalance < -2] = -1
    
    return signal
```

## 3. 风险控制

### 3.1 延迟风险

**延迟测量：**
```python
def measure_latency(timestamps):
    """
    测量延迟
    
    参数:
    timestamps: 时间戳 [(sent_time, received_time), ...]
    
    返回:
    延迟统计
    """
    latencies = [(recv - sent).total_seconds() * 1000 
                 for sent, recv in timestamps]
    
    return {
        'mean': np.mean(latencies),
        'median': np.median(latencies),
        'p99': np.percentile(latencies, 99),
        'max': np.max(latencies)
    }
```

### 3.2 滑点控制

**滑点估计：**
```python
def estimate_slippage(order_size, avg_depth, spread):
    """
    估计滑点
    
    参数:
    order_size: 订单大小
    avg_depth: 平均深度
    spread: 价差
    
    返回:
    预期滑点
    """
    # 线性市场冲击模型
    market_impact = spread / 2 + (order_size / avg_depth) * spread
    
    return market_impact
```
