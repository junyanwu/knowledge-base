## 策略优化流程

### 1. 学习 → 提炼
```
研报/文章 → 核心观点 → 可量化因子
```

### 2. 提炼 → 策略
```
因子 → 交易规则 → 策略代码
```

### 3. 策略 → 回测
```
策略代码 → 历史数据 → 回测结果
```

### 4. 回测 → 优化
```
回测结果 → 参数调整 → 优化策略
```

### 5. 优化 → 实盘
```
优化策略 → 模拟账户 → 实盘表现
```
